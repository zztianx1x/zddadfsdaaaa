### 中共拟立法禁强制转让技术 专家：为时已晚
------------------------

<p>
 【大纪元2018年12月24日讯】
 <span class="s1">
  （大纪元记者骆亚采访报导）
 </span>
 <span class="s1">
  中共全国人大
  <span class="s3">
   12
  </span>
  月
  <span class="s3">
   23
  </span>
  日审议“外商
  <a href="http://www.epochtimes.com/gb/tag/%E6%8A%95%E8%B5%84%E6%B3%95%E8%8D%89%E6%A1%88.html">
   投资法草案
  </a>
  ”，其中包含不得利用行政手段强制转让技术等
 </span>
 <span class="s1">
  。这是中美双方
 </span>
 <span class="s3">
  90
 </span>
 <span class="s1">
  天协议框架中，中方承诺美国的重要内容之一。专家认为目前这些小的政策变化，一来外界不再相信它，二来也为时已晚。
 </span>
</p>
<p class="p3">
 <span class="s1">
  官媒称，
 </span>
 <span class="s1">
  “外商
  <a href="http://www.epochtimes.com/gb/tag/%E6%8A%95%E8%B5%84%E6%B3%95%E8%8D%89%E6%A1%88.html">
   投资法草案
  </a>
  规定”的具体内容包括，“不得违法减损外商投资企业的合法权益或者增加其义务”，“不得违法设置市场准入和退出条件、干预或影响外商企业正常生产经营”
 </span>
 等。
</p>
<p class="p3">
 <span class="s1">
  此外，草案还规定，鼓励基于“自愿原则”和商业规则开展技术合作，技术合作条件由投资各方协商确定，“不得利用行政手段强制转让技术”。并要
 </span>
 <span class="s1">
  各地政府履行对外商的“政策承诺”以及各类合同。
 </span>
</p>
<p class="p5">
 <span class="s5">
  美国
 </span>
 <span class="s1">
  南卡罗莱纳大学艾肯商学院教授
 </span>
 <span class="s5">
  谢田
 </span>
 <span class="s1">
  接受大纪元采访时表示，这显然是对中美贸易战提供自己的“武器”了。中共试图表明它已经在立法上制定一些法律来保护外国投资者的权益，防止强迫技术转让发生。
 </span>
</p>
<p class="p5">
 <span class="s1">
  但谢田认为，“中共现在做什么事情都为时已晚，这种政策上的小修小补，世界已经不相信它了，也不管是西方的投资者还是政界。没有人会相信法律中共会真正的去实施。”
 </span>
</p>
<p class="p5">
 <span style="font-family: 细明体;">
  日经新闻英文版24日的报导质疑道
 </span>
 <span style="font-family: 细明体;">
  ，从外资的观点来看，所谓的自愿性“技术合作”，恐怕就是强制技转的代名词。在国际社会的严厉挞伐下，北京当局虽不再插手，陆企却接续过去的技转做法，以合约的方式鼓动外企自愿把技术交给中国合作伙伴。
 </span>
</p>
<p class="p5">
 谢田还强调，“中共并不是没有法，它制定了很多的法律，包括很漂亮的宪法，但是宪法中所有受保护的自由和权益，老百姓都享受不到。”
</p>
<p class="p5">
 <span class="s1">
  谢田举例，自己在台湾、越南见了一些企业界人士和政界人士，“他们都提到外商在中国投资最常见的问题就是投进去的钱不容易拿出来。投资赚的钱拿不出来换不成外汇、换不成本国货币，中国就像一个陷阱一样，只能把钱在中国继续发展。但这些钱只能以人民币形式存在，一旦人民币出现贬值这些外商投资都受损。”
 </span>
</p>
<p class="p7">
 <span class="s1">
  谢田表示，刚结束的中共的
  <a href="http://www.epochtimes.com/gb/tag/%E6%94%B9%E9%9D%A9.html">
   改革
  </a>
  开放
 </span>
 <span class="s4">
  40
 </span>
 <span class="s1">
  年周年纪念会给外界强烈的感觉是，在关键的问题上、在政治体制问题上、经济结构的根本问题上，中共根本不打算让步。外企或者私营企业作为中共控制之外的独立体是无法在中国生存。而中美贸易战导致产业链的问题，外资撤走已经形成趋势，现在再提保护，一来不相信，二来为时已晚。这些满足不了美国的胃口，中共是徒劳的。
 </span>
</p>
<p class="p5">
 而中共
 <a href="http://www.epochtimes.com/gb/tag/%E6%94%B9%E9%9D%A9.html">
  改革
 </a>
 已死在海内外的一批专家、学者中早已成共识。
</p>

 责任编辑：林诗远
</p>
