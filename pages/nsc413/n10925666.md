### 吴敬琏“改革十大忠告”文章遭封杀
------------------------

<p>
 【大纪元2018年12月22日讯】（大纪元记者骆亚报导）近日著名经济学家
 <a href="http://www.epochtimes.com/gb/tag/%E5%90%B4%E6%95%AC%E7%90%8F.html">
  吴敬琏
 </a>
 大胆进言“
 <a href="http://www.epochtimes.com/gb/tag/%E6%94%B9%E9%9D%A9.html">
  改革
 </a>
 十大忠告”，提到中国改革方向就是市场化、法治化、民主化，别无出路。其言论迅速遭到
 <a href="http://www.epochtimes.com/gb/tag/%E5%B0%81%E6%9D%80.html">
  封杀
 </a>
 。
</p>
<p class="p3">
 <span class="s1">
  近期大陆一批知识分子借中共
  <a href="http://www.epochtimes.com/gb/tag/%E6%94%B9%E9%9D%A9.html">
   改革
  </a>
  四十周年纪念时机，重新将改革话题炒热。
 </span>
 <span class="s3">
  12
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  20
 </span>
 <span class="s1">
  日，“
  <a href="http://www.epochtimes.com/gb/tag/%E5%90%B4%E6%95%AC%E7%90%8F.html">
   吴敬琏
  </a>
  十大改革忠告”文章出现在大陆金融家智库官微上。吴敬琏一上来就要求“真刀真枪地进行改革，这是包括我在内的大多数人的希望。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  他强调，“建立市场化、法制化的社会是大势所趋，除此之外，中国别无出路。
 </span>
 <span class="s1">
  在此关键时刻，我们每个人都要做出努力。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  第二个忠告说，为了避免社会危机，必须当机立断，痛下决心，真实地而非口头上推进市场化，、法制化、民主化的改革，建立包容性的经济体制和政治体制，实现从威权发展模式到民主发展模式的转型。
 </span>
 <span class="s1">
  并强调，这是中国唯一可能的出路。
 </span>
</p>
<p class="p3">
 <span class="s1">
  在第三个忠告中，他认为，改革向前推进的必要前提，就是对于改革的理论和实践问题进行自由切实地讨论。
 </span>
</p>
<p class="p3">
 <span class="s1">
  第四个忠告中，他提醒，“苏联式的意识形态在我们这一代人身上还是非常强烈的，这是一个沉重的包袱。”并忧虑下一代也会因“思维方式的惯性”继续，有些人打着这个旗帜来反对改革。
 </span>
</p>
<p class="p3">
 <span class="s1">
  在其它几个忠告中，他警告，“如果放任行政权力干预市场，并且通过理论包装使其得到某种正当性，将是相当危险的。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  他批评中共当局说，“近些年来由于政府不断干预微观经济活动，使权力而不是市场成为资源配置的主导力量。失去自由竞争的市场只能是一种貌似市场的‘
 </span>
 <span class="s1">
  伪市场’
 </span>
 <span class="s1">
  。”
 </span>
</p>
<p class="p4">
 <span class="s1">
  他呼吁经济改革要与政治改革并提，并认为，“目前经济改革的落后的方面，像国有经济的改革、政府经济管理职能的改革，都无不与政治改革、政府改革滞后有关。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  但此文很快遭微博
  <a href="http://www.epochtimes.com/gb/tag/%E5%B0%81%E6%9D%80.html">
   封杀
  </a>
  ，在微博上用标题“吴敬琏改革十大忠告”查找，显示未找到相关结果。
 </span>
</p>
<p class="p3">
 <span class="s1">
  如果用吴敬琏名字查询，显示零星的相关内容，多是网民在微博上以截图形式转发的图片格式，还原了当时文章发表过的痕迹。
 </span>
</p>
<figure class="wp-caption aligncenter" id="attachment_10925686" style="width: 452px">
 <a href="http://i.epochtimes.com/assets/uploads/2018/12/d5c1caa7a5e562e5fc1121aac5fc5099.png">

 </a>
 <br/><figcaption class="wp-caption-text">
  吴敬琏的十大改革忠告在微博上遭封杀，通过搜索找不到。（网页截图）
 </figcaption><br/>
</figure><br/>
<p class="p1">
 <span class="s1">
  而如果用金融家智库官微上其它没有被封杀文章的标题去查询，则能正常显示该微博推荐的大头像配标题和网址。
 </span>
</p>
<p class="p1">
 此外，中共官方公布百名改革贡献者名单未列入吴敬琏，也遭舆论的抨击。#
</p>
