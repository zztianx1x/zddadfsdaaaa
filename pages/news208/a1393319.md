 </a>
 响应国际

 <br/>

 <br/>
 <br/>
 2. 中共强迫技转 美化工、电动车受害
 <br/>
 <br/>
 《华尔街日报》揭露，中共系统性强迫外企技术转让的4大伎俩，包括：施压美企合资对像要求放弃技术、利用法院宣布美企的专利和授权安排无效等等，让美国化工、电动车、半导体等业者都深受其害。
 <br/>
 <br/>

  马斯克
 </a>
 诈欺 特斯拉面临危机
 <br/>
 <br/>
 美国电动车大厂特斯拉执行长马斯克，周四遭美国证券交易委员会（SEC）以诈欺起诉，称其发布错误和误导性声明，没有适当将公司私有化的计画知会监管机构。特斯拉股价周四盘后暴跌超过11%。
