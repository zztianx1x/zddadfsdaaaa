------------------------

 【新唐人2018年10月02日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.欧元区的失业率，从7月的8.2%下降到8.1%，达到近10年低点，其中意大利失业率由10.2%下降到9.7%，失业率最低的是德国，维持在3.4%。欧元区8月的失业人口为1千322万人，较7月减少10.2万人。
 <br/>
 <br/>
 2.日本央行公布第3季企业短期经济观测调查，大型制造业者信心指数从6月的21下滑至19，10年来第一次出现连续3季下滑，反映出日本企业对贸易战与一连串自然灾害的疑虑。
 <br/>
 <br/>
 3.韩国8月的汽车产量较7月增加21.8%，创下5年来最大增幅，带动8月的工业生产较上月增加1.4%，较去年同期增长2.5%。韩国的汽车与芯片产业增产，提振了整体工业产出。
 <br/>
 <br/>
 新唐人记者杰森综合报导
