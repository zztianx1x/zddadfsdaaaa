### 10月4日财经速瞄
------------------------

<div class="wysiwyg">
 【新唐人2018年10月04日讯】下面是今天的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_财经速瞄.html" target="_blank">
  财经速瞄
 </a>
 <br/>
 <br/>
 周四， 美国商务部数据显示, 8月美国工厂订单终值2.3％，大于预期增幅2.1％，创下2017年9月以来的最大单月涨幅。其中交运设备订单跳涨13.1％，创2017年6月以来最高。
 <br/>
 <br/>
 周四，随着美国经济数据利好，美国10年期国债收益率持续攀升，突破3.20%关口，来到7年来高点。市场担心通货膨涨加速，导致周四美股普遍走低。道指收盘下跌201点，是6个交易日连涨之后的首度下跌，标普下跌23.90点，纳指下跌145.75点。
 <br/>
 <br/>
 周四，美国总统川普转发华尔街日报的一张世界主要股指走势图表，数据显示，美国标普指数今年以来已经上涨9.4%, 超过日经指数的6.5%。而与此同时，欧洲斯托克600指数则下跌1.4%, 英国富时指数下跌2.5%, 中国的上证指数则下跌14.7%
</div>
