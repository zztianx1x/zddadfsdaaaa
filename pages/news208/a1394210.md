### 中共间谍晶片效应扩大 中国科技股应声下跌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】外媒彭博大篇幅报导中植
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍晶片.html" target="_blank">
  间谍晶片
 </a>
 事件，包括全球知名厂商伺服器，可能都遭到渗透。除了身陷风暴中心的美超微，4日股价暴跌41%，今天中国科技股，像是联想、中兴通讯，应声下跌，外媒也指出对产业将有深远影响。
 <br/>
 <br/>
 彭博大篇幅报导，中共秘密将微晶片植入伺服器主机板，出售给包括苹果（Apple）、亚马逊（Amazon）等大约30家美国企业，引发业界哗然。彭博历时一年多的深度调查追踪，进行超过100次的访谈，宣称有凭有据。
 <br/>
 <br/>
 彭博新闻记者 Jordan Robertson：〝中共军队，企图在电脑主机板植入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_间谍晶片.html" target="_blank">
  间谍晶片
 </a>
 ，并出售给美国企业，这种伺服器在非常关键、非常广泛的公司，包括苹果和亚马逊。这种间谍晶片被用来绕过软体，装在硬体上，某种程度上你完全没有办法侦测。〞
 <br/>
 <br/>
 尽管苹果、亚马逊和美超微公司都郑重否认此事，却也突显，美国对中共长期技术窃取行为，无法容忍。香港证券分析师（Ray KW Kwok）指出，中国生产的电子产品，可能因此被视为不安全。
 <br/>
 <br/>
 淡江大学整合战略与科技中心执行长 苏紫云：〝它核心就是科技战，因为科技不只是影响到未来经济的核心，那些5G的通讯是一个指标。中国制造的这些资通讯产品都有安全漏洞，所以一旦变成中国规格，那西方的民主国家基本上就会很危险。〞
 <br/>
 <br/>
 外媒报导一出，中国科技股今天全面下跌，联想一度狂崩23%，创下十年来最大跌幅，中兴通讯跌10.4%，瑞声下跌5.5%，中芯国际下跌4.5%。
 <br/>
 <br/>
 科技新闻网站TechRepublic指出，事件可能对科技产业产生持久的三大影响：除了贸易战愈演愈烈，美国可能禁用中国制主机板；中国生产晶片一旦列为黑名单，产品价格可能走扬；此外，中国以外的硬体工作机会也因此增加。
 <br/>
 <br/>
 台经院副研究员 刘佩真：〝美国会禁止出口相关的一些产品到中国，或是禁止中国呢，相关的产品出口到美国。中美的一个贸易战会有持续升温的一个态势，也就是说，未来会朝向一个非关税的相互制裁。〞
 <br/>
 <br/>
 学者认为，全球
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_半导体.html" target="_blank">
  半导体
 </a>
 供应链势必受到影响，不过台湾
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_ICT.html" target="_blank">
  ICT
 </a>
 出口东南亚国家，出口占比达到27%，市场地位更加提升下，或许可利用时机强化全球产业布局。
 <br/>
 <br/>
 (新唐人亚太电视 胡宗翰 张东旭 沈唯同 台湾台北报导)
