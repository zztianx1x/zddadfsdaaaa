### 10月8日环球财经简讯
------------------------

 【新唐人2018年10月09日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 <br/>
 1.国际货币基金（IMF）公布世界经济展望（WEO）最新报告，与7月的估值相较，IMF下调今、明两年全球经济成长率0.2个百分点，成为3.7%，并且维持对美国今年经济成长率预测为2.9%，明年成长率则下修0.2个百分点，成为2.5%。
 <br/>
 <br/>
 2.美国连锁超市业者沃尔玛(Walmart)表示，将与美国电影制片公司米高梅(MGM)合作，为沃尔玛旗下在线媒体点播平台Vudu创造更多独家原创媒体内容，以提高收视率，吸引更多客户，并与Netflix及Hulu做出市场区隔。
 <br/>
 <br/>
 3.全球最大付费在线视频流服务商奈飞(Netflix)宣布，将并购影视制作室ABQ Studios，以为奈飞提供更多的原创电影和电视节目，这项收购案进一步模糊了视频流业者，与传统电影制片商好莱坞工作室之间的界限。
 <br/>
 <br/>
 <br/>
 新唐人记者杰森综合报导
