### 10月2日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月03日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.波音公司宣布，联合航空向波音订购9架787-9订单，扩大了787梦想飞机机队。依据目前牌价，这笔交易价值25.3亿美元。波音在全球已经交付超过700架787梦想飞机，今年的订单总数已突破100架。
 <br/>
 <br/>
 2.美国食品药物管理局表示，上周五对电子烟制造商Juul旧金山总部突击检查，要求其提供更多关于销售和营销的信息。今年四月，FDA要求Juul提供其吸引青少年消费者的营销手段信息，并检查其生产设施。FDA指，电子烟使青少年产生一种定期使用尼古丁的流行病，必须采取行动。
 <br/>
 <br/>
 3.亚马逊长期受到批评，称其员工薪酬差距过大。最近，亚马逊宣布，11月开始加薪计画，最低时薪将调到15美元。有分析师认为，随着亚马逊自动化程度的加深，将来的薪水支出反而会减少。
 <br/>
 <br/>
 4.英国首相特雷莎．梅宣布，英国脱欧后，欧盟会员国劳工将不能自由进入英国就职，具体法案将在2019年公布。同时，英国将在全球范围内吸引具有英国所需技能的移民，移民数量将受移民政策限制。
 <br/>
 <br/>
 新唐人记者米西、明玉综合报导
