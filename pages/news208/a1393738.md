### 美加墨达成贸易协议 股市应声上涨
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月02日讯】美国和加拿大经过紧张谈判，在30日午夜12点最后期限前，宣布达成新的美加墨三方贸易协议（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_USMCA.html" target="_blank">
  USMCA
 </a>
 ），取代20多年前签署的北美自贸协定（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_NAFTA.html" target="_blank">
  NAFTA
 </a>
 ）。投资人因此欢呼庆祝，欧洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 周一（10月1日）也应声上涨。
 <br/>
 <br/>
 欧洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 周一走高，带动全球金融市场以稳健的步伐跨入第4季的第一天。
 <br/>
 <br/>
 道琼工业平均指数当天上涨192.9点或0.73%报26651.21点，雪佛龙和波音跑赢大盘。
 <br/>
 <br/>
 欧洲三大股市盘初温和上扬，伦敦FTSE 100指数微涨0.02%，法国CAC 40指数小涨0.13%，德国DAX 30指数涨升0.60%。
 <br/>
 <br/>
 标准普尔500指数上涨10.61点或0.36%报2924.59点，能源、材料和工业领涨。但纳斯达克综合指数下跌9.05点或0.11%报8037.3点，因为脸书和英特尔的下跌限制了科技股的涨幅。
 <br/>
 <br/>
 廉价航空瑞恩航空控股公司（Ryanair）因下修全年度财测，并表示财报恐恶化，导致股价重挫7.5%，跌幅居冠。
 <br/>
 <br/>
 瑞恩航空表示，如果近来欧洲各地串连罢工持续重创空中交通和订票，公司财报恐将进一步恶化。
 <br/>
 <br/>
 Peel Hunt策略师威廉斯（Ian Williams）认为，美加前夜达成协议，拯救北美自由贸易协定的消息，在第4季之初可能对全球风险胃口带来提振。
 <br/>
 <br/>
 FBB Capital Partners研究主管贝利(Mike Bailey)称，贸易战是股市最大的风险因素，现在威胁稍微调低了一点。
 <br/>
 <br/>
 Tower Bridge Advisors总裁欧格(Maris Ogg)则表示，股市反应热烈，因为我们消除了不确定性。
 <br/>
 <br/>
 按照美加两国的联合声明，新版的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_NAFTA.html" target="_blank">
  NAFTA
 </a>
 将改名为美国—墨西哥—加拿大协定（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_USMCA.html" target="_blank">
  USMCA
 </a>
 ），挽回了由美加墨三国参与北美自由贸易协定的格局，将使美国对加拿大乳制品市场的准入大幅提高，同时让加拿大避免美国可能开征的汽车关税。过程中，双方都有让步。
 <br/>
 <br/>
 据悉，加拿大有超过2/3的出口商品销往美国，相当于加国GDP的20%；另一方面，加拿大也是美国最大的出口市场。
 <br/>
 <br/>
 旧版的NAFTA长达24年之久，被川普（特朗普）总统批评为〝史上最糟协议〞，让美国制造业工作流失，川普要求对协议进行重大修改。NAFTA构成的年贸易额为1.2万亿美元，市场曾担心该协议若废止将给经济造成重大破坏。美加为此重启谈判一年多。
 <br/>
 <br/>
 周一，在白宫举行的新闻会上，川普将三国间重新谈判的贸易协定称为〝对我们国家乃至整个世界的真正的历史性新闻〞。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
