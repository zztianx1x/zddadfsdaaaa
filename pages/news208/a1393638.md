### 9月30日环球财经简讯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月01日讯】下面请看一组
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_环球财经简讯.html" target="_blank">
  环球财经简讯
 </a>
 <br/>
 <br/>
 1.美国证券监管当局表示，被控证券诈欺罪名的马斯克已经同意和解，他将辞去特斯拉董事长职位，并且支付2千万美元罚款。马斯克将不能担任董事长，为期3年，将由独立董事长取代。
 <br/>
 <br/>
 2.英特尔表示，为了提振产量，2018年的资本支出将增加约10亿美元，达到150亿美元，而且随着更好的PC市场需求，引发芯片短缺，有机会达成全年的营收目标。
 <br/>
 <br/>
 3.严重亏损的韩国乐金显示器，将进行创立以来第一次裁员，而且要求员工透过自愿退休方式离职，估计将占员工总数的65%，最终裁员规模尚未确定。乐金显示器计画将战略目标转向OLED面板业务。
 <br/>
 <br/>
 4.韩国82.8%的受访者认为，韩国社会严重依赖智慧手机等电子产品。其中71.9%在工作或学习之余玩手机。最常用使用的是即时通讯，其次是看新闻、网路购物、上网浏览、社交网站。
 <br/>
 <br/>
 新唐人记者保罗、阳阳综合报导
