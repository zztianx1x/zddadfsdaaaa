 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 前总书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵紫阳.html" target="_blank">

 </a>
 政治秘书
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鲍彤.html" target="_blank">
  鲍彤
 </a>
 表示，中共从土匪军起家，到抗战胜利后，中共立刻诬告蒋介石偷桃子，挑动内战夺权。
 <br/>
 <br/>
 中共前总书记赵紫阳政治秘书鲍彤：〝（中共）杀了多少，他自己也死了多少，然后说中国人民做了历史性选择。毛泽东在天安门宣布，说是中国人民站起来，怎么站起来，就是大家乖乖地听毛泽东改造。69年来，我们大家任人摆布、任人改造、任人设计。〞
 <br/>
 <br/>
 鲍彤认为，中共窃国69年来，一直都把老百姓当试验品。
 <br/>
 <br/>
 中共前总书记赵紫阳政治秘书鲍彤：〝共产党成立才几十年，几千年土地的就全是它的了。（共产党说），我要怎么试验就怎么试验，我说你是反革命，所有抗战的，民选的人大代表通通杀光。〞
 <br/>
 <br/>
 前陕西电台编辑马晓明认为中共最大的邪恶，在于谎言治国。
 <br/>
 <br/>
 前陕西电台编辑马晓明：〝活摘器官，镇压法轮功都封锁住了。我说这个邪恶最邪的就是说假话，封锁信息。这是对公民信息权利的最公然的，最粗暴的一种剥夺。〞
 <br/>
 <br/>
 曾遭受中共酷刑迫害的北京天主教徒夏天庆辉，则呼吁民众不要信中共。
 <br/>
 <br/>
 北京天主教徒夏天庆辉：〝我想通过我所亲身经历的这种酷刑，告诉大家不要轻信共产党。共产党是一个邪恶的组织，它们所做的事情都是对人权的践踏。〞
 <br/>
 <br/>
 目前，已有3亿1千5百万勇敢的中国人声明退出中共党、团、队组织，也有集会人士认为，中共解体势在必行，呼应民众退党自救。
 <br/>
