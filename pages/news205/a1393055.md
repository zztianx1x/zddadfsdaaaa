### 逃港大陆天主教徒：梵蒂冈不了解中共邪恶
------------------------

<div class="wysiwyg">
 【新唐人2018年09月26日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梵蒂冈.html" target="_blank">
  梵蒂冈
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 早前签订主教任命临时协议。一名遭中共酷刑迫害而逃亡香港的大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天主教徒.html" target="_blank">
  天主教徒
 </a>
 ，批评梵蒂冈与魔鬼交易。
 <br/>
 <br/>
 北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天主教徒.html" target="_blank">
  天主教徒
 </a>
 夏天庆辉因调查天主教徒受迫害，两年前逃亡香港，首次公开露面，控诉遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 酷刑折磨的经历。
 <br/>
 <br/>
 北京天主教徒夏天庆辉：〝不知道妳们有没有见过老虎凳？有种铁凳子，人就绑在那里，电击我，电的我的手，疼的受不了，我就骂他了---他就把电棍，插到我这里，电我的生殖器，电得人就昏过去。〞
 <br/>
 <br/>
 对于中梵在9月22日签订主教任命临时协议。他认为教宗和魔鬼做交易。
 <br/>
 <br/>
 北京天主教徒夏天庆辉：〝因为（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梵蒂冈.html" target="_blank">
  梵蒂冈
 </a>
 ）他根本不了解中共的邪恶体制，假如梵蒂冈认可的话，对于中国地下教会，是非常严重的伤害，导致他们不承认教宗的合法性。〞
 <br/>
 <br/>
 香港荣休主教陈日君枢机认为，中梵临时协议并无交代具体内容，令人失望及很不放心，亦不会令中国的地下教会放心。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
