### 坠集资炒股骗案 数百大陆苦主来港维权
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】去年起，一个名为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麒麟.html" target="_blank">
  麒麟
 </a>
 韦尔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融平台.html" target="_blank">
  金融平台
 </a>
 蓝筹计划，涉嫌在大陆非法
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集资.html" target="_blank">
  集资
 </a>
 来港炒股，但金融平台今年八月突然关闭，投资者血本无归，涉案多达数万人，金额达数十亿。数百受害苦主，近日来港维权，希望港府调查并追讨损失。
 <br/>
 <br/>
 口号：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麒麟.html" target="_blank">
  麒麟
 </a>
 诈骗，还我血汗钱〞
 <br/>
 <br/>
 自今年八月起，约三百名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_麒麟韦尔.html" target="_blank">
  麒麟韦尔
 </a>
 金融骗案苦主，先后来港维权。本周更一连数日到香港警署报案，并在涉案公司楼下抗议，要求追讨损失。
 <br/>
 <br/>
 有大陆苦主控诉，一个名为韦尔资本的香港公司，打着麒麟集团和中共央企中军金控的名义，以沪深港通开通，香港股票回报率高为诱饵，吸引他们
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集资.html" target="_blank">
  集资
 </a>
 来港炒股，按要求将钱打入不同个人帐号，指定炒作香港两只股票。
 <br/>
 <br/>
 北京金融苦主王小姐：〝他写着的，中军金控、麒麟，韦尔，生和国际，麒麟控股集团，（福泽集团）、（麒麟集团控股），香港的股票，所以麒麟作为融资项目，我们都是他们旗下要融资的这块。所以我们相信了。我们还来到这儿（香港公司）签的，我们都有合同签的。〞
 <br/>
 <br/>
 山西金融苦主崔先生：〝我们也问过为何这个钱不能打给麒麟，他说麒麟是一个上市公司，上市公司不能对这些散户。他就用韦尔、包括生和国际、凤凰国际，他说其实是他们一家的，都是麒麟旗下的。〞
 <br/>
 <br/>
 没想到麒麟韦尔
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融平台.html" target="_blank">
  金融平台
 </a>
 今年八月突然关闭，数万人投资血本无归。
 <br/>
 <br/>
 山西金融苦主崔先生：〝我总共投了240万，一分钱没有拿回来。〞
 <br/>
 <br/>
 北京金融苦主王小姐：〝大家都被骗得特别惨，我真的可怜他们，对我来说66万是一个钱，但是我没有他们（其他苦主）惨。〞
 <br/>
 <br/>
 山西金融苦主李小姐：〝我希望媒体、社会能够关注这个事情，给我们这些投资麒麟韦尔金融平台的（苦主）一个说法。〞
 <br/>
 <br/>
 麒麟集团控股早前发表公告，否认和麒麟韦尔有任何业务交易。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙、陈仲明香港报导
</div>
