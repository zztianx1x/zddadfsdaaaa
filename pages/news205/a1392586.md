### 陈日君才批〝背叛信仰〞 教廷：已签梵中协议
------------------------

<div class="wysiwyg">
 【新唐人2018年09月22日讯】梵蒂冈周六主动宣布，和中共达成〝历史性的〞主教任命的临时协议，希望促进双边对话。不过，香港荣休枢机主教
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈日君.html" target="_blank">
  陈日君
 </a>
 ，日前质疑，这是彻底投降，炮轰
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_教廷.html" target="_blank">
  教廷
 </a>
 国务卿帕洛林〝背叛天主教信仰〞，该辞职下台。
 <br/>
 <br/>
 据媒体先前报导，即便梵中签署协议，相关文本，仍不会公开。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_教廷.html" target="_blank">
  教廷
 </a>
 强调，会维持和中华民国的邦交。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈日君.html" target="_blank">
  陈日君
 </a>
 则认为，教廷高层，不了解中共意图，却和中共〝秘密交易〞，〝正在将羊群送入虎口，这是难以置信的背叛〞，他认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_梵中协议.html" target="_blank">
  梵中协议
 </a>
 ，将重创整个天主教会的威信，并产生长远的，悲惨后果。

 <br/>
