### 专访大提琴家李垂谊 想做21世纪的马可波罗
------------------------

<div class="wysiwyg">
 【新唐人2018年09月23日讯】国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大提琴家.html" target="_blank">
  大提琴家
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李垂谊.html" target="_blank">
  李垂谊
 </a>
 （Trey Lee），近年来醉心作曲和培育年轻音乐家，架起中西方音乐和文化交流的桥梁。李垂谊日前接受本台
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_专访.html" target="_blank">
  专访
 </a>
 ，分享他对音乐，以及对中国文化的热爱，一起来看看。
 <br/>
 <br/>
 国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大提琴家.html" target="_blank">
  大提琴家
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李垂谊.html" target="_blank">
  李垂谊
 </a>
 ：〝我是21世纪的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马可波罗.html" target="_blank">
  马可波罗
 </a>
 （笑）。〞
 <br/>
 <br/>
 出生在香港、生长在美国、定居欧洲的李垂谊，梦想是做21世纪的马可波罗，架起中西方文化交流的桥梁。这源于他对中国文化的热爱。
 <br/>
 <br/>
 国际大提琴家李垂谊：〝欧洲
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_古典音乐.html" target="_blank">
  古典音乐
 </a>
 之外，我个人也对中国音乐很有兴趣，因为我改编了很多好像《红楼梦》，这几年我做了不同的《红楼梦》（版本），大提琴跟乐团、钢琴还有中乐的乐器，所以这个音乐让我回归到中国文化。〞
 <br/>
 <br/>
 《马可波罗重游威尼斯嘉年华》儿童音乐会是他中国情的又一延续，今年11月中旬将在由他创办的〝乐．谊国际音乐节〞上再度演出。
 <br/>
 <br/>
 国际著名大提琴家李垂谊：〝我选择马可波罗，是因为他和中国有重要的历史，第二他也是威尼斯人，我觉得可以吸引很多年轻小孩来看。他们又学到历史，又学到音乐，而且和东西方的关系，我们可以利用音乐来讲这个故事。〞
 <br/>
 <br/>
 李垂谊希望帮助华人音乐家走向世界，寄语年轻人一定要先了解古典音乐，同时争取上台演奏的机会。
 <br/>
 <br/>
 国际著名大提琴家李垂谊：〝需要有在音乐厅演奏，在观众面前演奏的机会，所以我们乐社，特别是音乐节，希望每年这样做，希望多一点让香港音乐家和外国音乐家一起合作。〞
 <br/>
 <br/>
 李垂谊被意大利权威杂志列为华人古典音乐界的〝金童〞。
 <br/>
 <br/>
 新唐人记者梁珍、陈仲明香港报导
</div>
