### 贸易战令港出口增长减半 港商撤出大陆
------------------------

<div class="wysiwyg">
 【新唐人2018年09月19日讯】美国对2,000亿美元中国货物加征10%关税，香港贸发局调低出口增长预测从6%到3%。商务及经济发展局局长邱腾华表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港商.html" target="_blank">
  港商
 </a>
 的确有迁出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆.html" target="_blank">
  大陆
 </a>
 ，政府也会协助开拓其他市场。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 升级，香港作为中、美贸易的重要转口港，受到波及。贸发局降低今年出口增长预测至3%，预测明年会更差，商务及经济发展局局长邱腾华对前景也不乐观。
 <br/>
 <br/>
 商务及经济发展局局长邱腾华：〝现在2000亿美元的关税，加上原先（500亿美元）的关税，已占中国经香
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港出口.html" target="_blank">
  港出口
 </a>
 至美国大约一半的货值，假设
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆.html" target="_blank">
  大陆
 </a>
 实施反制措施，亦会影响美国经香港出口至内地的贸易。〞
 <br/>
 <br/>
 邱腾华表示，为应对贸易战，不少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港商.html" target="_blank">
  港商
 </a>
 已逐步迁出大陆。
 <br/>
 <br/>
 商务及经济发展局局长邱腾华：〝中美贸易战可能要一段时间，也有很多人从采购方面或者生产线方面，确实逐步向其他地方迁移。因此在市场拓展方面，或者采购拓展我们有做多一些工作，包括去东南亚国家、东欧国家以及非洲的国家都有很多人参加。〞
 <br/>
 <br/>
 有在大陆设厂的港商预测，如果进一步加税到25%，港商几乎都要关门。
 <br/>
 <br/>
 新唐人记者梁珍、宋碧龙香港报导
</div>
