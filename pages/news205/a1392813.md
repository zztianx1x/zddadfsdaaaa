### 颱风后大澳首办灯会 香港舞火龙祈福
------------------------

<div class="wysiwyg">
 【新唐人2018年09月24日讯】超强颱风〝山竹〞袭港一周，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 市民重拾心情，欢度中秋。除了市区三场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彩灯会.html" target="_blank">
  彩灯会
 </a>
 外，风灾后的古老渔村大澳，今年民间首办彩灯会，重现古香港情怀，此外大坑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舞火龙.html" target="_blank">
  舞火龙
 </a>
 一连三日热闹登场，为市民祈福。
 <br/>
 <br/>
 山竹袭港一周，重灾区大澳居民已重建家园，迎接传统中秋佳节的到来。有民间团体自发举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彩灯会.html" target="_blank">
  彩灯会
 </a>
 ，用300盏手绘彩灯和巨型鲤鱼灯笼，点亮古老渔村，带来浓浓的节日气氛。
 <br/>
 <br/>
 大澳非茂里创办人张敏芳：〝灯笼高高挂和鲤鱼的概念，背后讲述其实每一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 人怀念老香港的感觉，回忆他们以前小时候的人情味和地道情怀，因为现今人们都遗忘了传统日子。〞
 <br/>
 <br/>
 大澳非茂里创办人Hiyuki：〝因为我们都怕水浸，都很幸运、够胆将整条鱼（巨型鲤鱼花灯）捧上到二楼，算是逃过大劫之后，我们都希望鲤鱼的尾巴，都可以将颱风拨走。〞
 <br/>
 <br/>
 市区四处张灯结彩，湾仔利东街有巨型发光月兔，配上超过800个彩色花灯，加上灯笼扎作、木偶戏曲等多种传统手工艺市集，以及LED
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舞火龙.html" target="_blank">
  舞火龙
 </a>
 ，气氛热闹。
 <br/>
 <br/>
 另外，位于铜锣湾的大坑，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中秋节.html" target="_blank">
  中秋节
 </a>
 前一天起一连三天，举行有过百年历史的传统舞火龙仪式。全长220呎的火龙，龙身插满两万多支香烛，由约300名表演者轮流舞动，祈求来年风调雨顺，国泰民安，吸引上千名市民、游客围观。
 <br/>
 <br/>
 新唐人记者梁珍、曾莲香港报导
 <br/>
 <br/>
