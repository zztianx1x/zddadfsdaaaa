### 支联会中秋前夕 促中共释放异见人士
------------------------

 【新唐人2018年09月24日讯】

 </a>

 等团体趁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中秋节.html" target="_blank">
  中秋节
 </a>
 前夕，游行到中联办，要求中共立即停止暴政，释放长期被无理拘禁的异见及维权人士，让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_良心犯.html" target="_blank">
  良心犯
 </a>
 早日回家与家人团聚。

 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中秋节.html" target="_blank">
  中秋节
 </a>
 本是一家共聚天伦的日子，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_香港.html" target="_blank">
  香港
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_支联会.html" target="_blank">
  支联会
 </a>
 、社民连、公民党等约40人由西区警署游行到中联办，要求中共停止暴政，释放长期被监禁的政治犯和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_良心犯.html" target="_blank">
  良心犯
 </a>
 ，包括秦永敏、黄琦及王全璋等人回家团聚。
 <br/>
 <br/>
 支联会主席何俊仁：〝竟然大家看到有那么多著名的不同政见人士受打压、摧残及各种不人道的酷刑，这个所谓复兴的民族对我们而言是一种最大的耻辱，如果强大的国家要建立在劳役人民的基础上，有那么多不同政见者还有很多不知名的，在各州各省各市被关押在监狱中。包括不少宗教人士。〞
 <br/>
 <br/>
 示威人士将请愿信贴在中联办门前，并将示威标语扔进中联办内后，和平离去。
 <br/>
 <br/>
 新唐人记者林秀宜在香港报导
