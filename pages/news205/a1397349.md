### 武侠小说家金庸去世 享年94岁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】金庸（原名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_查良镛.html" target="_blank">
  查良镛
 </a>
 ）于2018年10月30日在香港养和医院逝世，享年94岁。
 <br/>
 <br/>
 当日，金庸之子查传倜回应媒体时证实：〝下午走了，很安详！〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_查良镛.html" target="_blank">
  查良镛
 </a>
 以金庸为笔名的作品以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_武侠小说.html" target="_blank">
  武侠小说
 </a>
 为主，兼有政论、散文等。
 <br/>
 <br/>
 自1955年的《书剑恩仇录》开始至1972年的《鹿鼎记》封笔，金庸共创作了十五部武侠小说，包括《射雕英雄传》、《神雕侠侣》、《倚天屠龙记》、《天龙八部》、《笑傲江湖》等。其笔下的著作屡次被改编为电视剧、电影。
 <br/>
 <br/>
 武侠小说的金庸热最初是从港澳开始，接着延烧到台湾和中国大陆，继而几乎影响了整个华语圈。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
