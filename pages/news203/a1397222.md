### 美重量级参议员力挺川普   撰文呼吁强力回击中共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】美国参议院司法委员会主席查克．
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_格拉斯利.html" target="_blank">
  格拉斯利
 </a>
 (Sen. Chuck Grassley, R-IA)日前在美国媒体上发表文章，呼吁美国形成一个〝强有力的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_统一战线.html" target="_blank">
  统一战线
 </a>
 〞，以便继续保持强硬立场来应对中共在贸易等领域的种种恶劣行径。
 <br/>
 <br/>
 共和党参议员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_格拉斯利.html" target="_blank">
  格拉斯利
 </a>
 是美国政坛的一名重量级参议员，他本月28日在美国媒体《福克斯新闻》杂志上发表了一篇措辞强硬文章。
 <br/>
 <br/>
 格拉斯利在这篇文章中表示，美中关系的确是21世纪〝最重要的双边关系〞，但同时也是〝最复杂的关系〞，因为统治这个国家的是一个专制政权，而这个政权还每时每刻都在试图扩大其在全球的影响力并试图〝控制其它国家〞。
 <br/>
 <br/>
 文章认为，美国政府过去对中共专制政权的种种危害〝掉以轻心〞，但现在到了必须改变这种状况的时候了。
 <br/>
 <br/>
 文章称，〝美国必须继续展现出一个强有力的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_统一战线.html" target="_blank">
  统一战线
 </a>
 ，回击中国以触目惊心的谎言、盗窃和欺骗等手段成为世纪经济领袖。否则，美国必将（在博弈中）处于下风。〞
 <br/>
 <br/>
 格拉斯利在文章中表示，他曾投票赞成中国（中共政府）加入世界贸易组织，但中国（中共）长期以来都没有遵守其对国际社会的承诺，并没有遵守世界贸易组织的规则。
 <br/>
 <br/>
 据美国知识产权委员会估计，在过去的这些年间，美国在软件和商业机密方面的损失每年高达6千亿美元。
 <br/>
 <br/>
 事实上，格拉斯利参议员此前还曾于本月25日在美国《政治杂志》发表文章，严词谴责中共严重侵犯宗教群体的人权。他在那一篇文章中曾直言，为宗教自由而战应是美中关系的核心部分。
 <br/>
 <br/>
 次日（10月26日），美国国务卿蓬佩奥在接受美国电台采访时也表态称，美国政府今后对于中共在各个领域中的〝每一个恶行〞将不仅仅是口头谴责，而是要采取将强力的行动作为回应。
 <br/>
 <br/>
 蓬佩奥在那次谈话中指出，中共利用〝一带一路〞贿赂世界各国，损害各国人民以及美国的利益；中共长期奉行的是不公平的贸易政策，中共在南海的军事扩张、在国内暴力镇压宗教信仰人士，在国外则偷盗他国的知识产权，还在国际贸易中实行政府补贴、强迫技术转让等等，这些全部都是恶意行为，而且与美国奉行的价值观相抵触。
 <br/>
 <br/>
 当时，蓬佩奥曾明确表态：美国政府将对中共的上述恶劣行为给予强烈的反击，并会持续下去。他还强调，美国政府将会在〝每一个领域〞里都强力应对中共的〝每一个挑战〞。
 <br/>
 <br/>
 蓬佩奥披露，川普（特朗普）总统已经要求美国政府严肃处理上述问题，今后美国政府将不仅仅是对那些恶行提出批评，而是要动用美国和世界的影响力来应对这些对基本人权的挑战。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
 <br/>
