### 孟晚舟被捕细节流出：加国转机时被带走
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】 针对中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 副董事长兼CFO孟晚舟被捕一事，12月6日，华为董事、高级副总裁陈黎芳在推特披露了相关细节。
 <br/>
 <br/>
 12月6日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 董事、高级副总裁陈黎芳在推特说，近期，华为公司CFO孟晚舟女士在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加拿大.html" target="_blank">
  加拿大
 </a>
 转机时，被加拿大当局代表美国政府暂时扣留，美国正在寻求对孟晚舟女士的引渡，面临纽约东区未指明的指控。
 <br/>
 <br/>
 加拿大《环球邮报》(The Global and Mail)12月5日说，加拿大当局应美国要求，逮捕了华为公司副董事长兼首席财务官孟晚舟。她是华为创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿，因涉嫌违反美国对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伊朗.html" target="_blank">
  伊朗
 </a>
 的贸易制裁而将被引渡到美国。
 <br/>
 <br/>
 加拿大司法部发言人说，孟晚舟于12月1日在温哥华被捕，定于7日举行保释听证，因孟晚舟申请的〝禁止报导令〞有效，当局此刻无法提供更多详情。
 <br/>
 <br/>
 禁令规定，在司法程序中途，或者聆讯期间，禁止报导案情。根据法院资料，禁止报道令可根据《加拿大刑法》（Criminal Code of Canada）、《加拿大引渡法》（Extradition Act）等法律依据发出，而此案的禁令，就是根据《引渡法》第26节发出。
 <br/>
 <br/>
 禁令亦称，孟晚舟以Cathy或Sabrina为别名。随母姓。1972年出生，现年46岁。
 <br/>
 <br/>
 据报导，纽约联邦检察官至少从2017年开始，就一直在调查华为，检察官就原产于美国的产品运往伊朗和其它国家、违反美国出口和制裁法令的指控进行调查。
 <br/>
 <br/>
 孟晚舟是华为创办人任正非与第一任妻子孟军所生的长女，现任公司CFO、副董事长，担任华为系旗下共11间公司的董事。
 <br/>
 <br/>
 华为是全球最大的电讯设备制造商。多年以来，西方情报机构一直质疑华为设备的安全性，认为它受制于中共政府，存在间谍和安全隐患。
 <br/>
 <br/>
 美国此前已经采取了一系列行动，阻止或减少华为和中兴通讯公司进入美国市场。
 <br/>
 <br/>
 美国调查华为的主要原因有三点：一，华为有中共军方背景，可能被中共利用在海外从事技术间谍活动；二，华为违规向伊朗和朝鲜等被制裁国家出口电讯设备和提供技术支持；三，华为号称将〝引领5G时代〞，威胁全世界资讯安全。
 <br/>
 <br/>
 华为公司于1987年在中国深圳成立，自2012年以来一直是全球最大的电信设备制造商。华为也生产消费类电子产品，今年超过苹果成为全球第二大智能手机制造商，仅次于三星电子。
 <br/>
 <br/>
 华为公司的产品从智能手机、到路由器、交换机、基站等等，目前全世界145个国家及500个电讯运营商使用华为产品。
 <br/>
 <br/>
 华为从未上市，管理架构相当神秘，被指长期帮助中共监视中国民众。其创办人任正非曾是中共军队团职干部；刚刚卸任的董事长孙亚芳，加入华为之前，曾长期在中共国安部任职。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
