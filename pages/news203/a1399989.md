### 阻大篷车移民越境 美封西半球最大入境口岸
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】周一（11月19日），美国边境官员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关闭.html" target="_blank">
  关闭
 </a>
 了加州境内一个最繁忙的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入境口岸.html" target="_blank">
  入境口岸
 </a>
 ，并加紧安装障碍物以应对即将到来的数千名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法越境.html" target="_blank">
  非法越境
 </a>
 。
 <br/>
 <br/>
 圣伊西德罗
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_入境口岸.html" target="_blank">
  入境口岸
 </a>
 （San Ysidro Port of Entry）是连接墨西哥和加州的重要边境口岸，每天能容纳20,000行人和70,000车辆，是从蒂华纳（Tijuana）进入圣地亚哥（San Diego）的交通要塞。但周一，美国边境官员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关闭.html" target="_blank">
  关闭
 </a>
 了该口岸所有的客运和商业车道，并把两个行人入境口岸中的其中一个也关掉。
 <br/>
 <br/>
 圣伊西德罗入境口岸共有26条客运车道和22条人行道，是西半球交通容纳力最强的边境口岸。
 <br/>
 <br/>
 美国海关和边境保护局（CBP）周一声明，他们已暂时关闭该口岸的北行车道，以便安装更多基础设施稳固边界。
 <br/>
 <br/>
 为阻止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 移民擅自入境，边防人员采用一些临时性措施和障碍物加以阻拦，包括带刺铁丝网、防碰撞护栏等。
 <br/>
 <br/>
 CBP在声明中表示，这些障碍物安置好之后，就会恢复一些精选车道的北行服务。
 <br/>
 <br/>
 CBP官员承认，关闭入境口岸是为了应对即将到达的数千名中美洲移民。
 <br/>
 <br/>
 周一的行动并不影响南行的车辆。
 <br/>
 <br/>
 上周，CBP关闭了圣伊西德罗入境口岸和附近的奥塔伊梅萨（Otay Mesa）入境口岸的一些车道，以便安装防护设施。
 <br/>
 <br/>
 不过，在周一的声明中，CBP没有说明北行车道将会关闭多久。
 <br/>
 <br/>
 据CBP官员估计，由于移民官员每天能处理的庇护申请非常有限，估计很多抵达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的大篷车移民会选择非法入境。
 <br/>
 <br/>
 另外，国土安全部官员周一下午表示，大约有500名犯罪分子混在大篷车队伍中，正在圣地亚哥市过境口岸的墨西哥一侧等待入境。
 <br/>
