### 美媒曝G20习川会若失败 美对华所有商品征税
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 持续升级之际，中美首脑将在阿根廷G20峰会上会晤。美媒称，若有关会面不能缓和中美之间的贸易战，美国计划对尚未
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_征收关税.html" target="_blank">
  征收关税
 </a>
 的全部
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国货品.html" target="_blank">
  中国货品
 </a>
 征收关税，并将于2019年2年初中国黄历新年前后实施。
 <br/>
 <br/>
 《彭博》引述3名熟悉内幕的消息人士说，新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_征收关税.html" target="_blank">
  征收关税
 </a>
 清单将包括之前几轮尚未征收关税的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国货品.html" target="_blank">
  中国货品
 </a>
 。根据2017年数据，这些货品总值约2,570亿美元。
 <br/>
 <br/>
 假如美国在12月宣布有关措施，根据一贯在宣布的60天后实施的做法，美国的新一轮关税措施将在2019年2年初中国黄历新年前后实施。
 <br/>
 <br/>
 两名消息人士披露，白宫还在考虑另一个可能的方案，即
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 谈将排除贸易议题，不过完全不谈也不可能。美国官员显然是做好准备，一旦中美首脑峰会无法就贸易争端取得进展，就会实施有关举措。
 <br/>
 <br/>
 不过，他们也强调，华府尚未做最后决定。
 <br/>
 <br/>
 《华尔街日报》早前引述消息人士的话爆料，美国总统川普（特朗普）和习近平计划在20国峰会期间举行会晤。中方希望川习会晤，对中美贸易关系产生积极的影响。
 <br/>
 <br/>
 20国集团会议将于11月30日至12月1日在阿根廷布宜诺斯艾利斯举行。
 <br/>
 <br/>
 白宫高级官员库德洛（Larry Kudlow）日前也证实，川普和习近平将在阿根廷G20峰会上会晤。
 <br/>
 <br/>
 不过，美国媒体爆料称，美国对这次会晤并不抱太大的期望。
 <br/>
 <br/>
 10月21日，美国Axios新闻网站引述3名消息人士的话称，川普无意放缓对中国征收关税。一名了解川普想法的消息人士称，川普认为他的关税持续的越长，他就会拥有更多的筹码。
 <br/>
 <br/>
 一名熟悉川普谈话的消息人士称，川普同中共的贸易战还处在〝开始的开始〞。川普的团队并不对暂定于11月的G20〝习川会〞抱有太大的期待。
 <br/>
 <br/>
 《华尔街日报》引述中美官员的话说，华府已经要求中方就强制技术转移及其他中美经贸议题，交出具体的解决及让步方案，否则美方拒绝重新展开贸易谈判。
 <br/>
 <br/>
 一名白宫高级官员表示：〝如果中方想20国集团的双边会谈有意义，我们需要做一些基本工作。如果他们（中方）不给予我们任何资讯，实在很难见得到这如何变得富有成效。〞
 <br/>
 <br/>
 知情人士表示，中共担心先提出让步方案，将暴露中方的谈判立场，川普也可能公开宣称成功，令中共妥协。报导称，中方把美国提出的要求分为142项，并表明可以谈判其中122项，余下则因涉国家安全而拒绝让步，但没有进一步说明如何处理每项要求。
 <br/>
 <br/>
 美方官员形容这属〝概念〞，不足以令谈判有进展，要求中方提交详细及具体的方案。
 <br/>
 <br/>
 多名消息人士称，美财长团队对中共说，谈论购买更多的美国产品没有意义，因为关键的优先议题必须解决，如偷窃知识产权和市场准入权等结构性的问题。消息人士还称，川普并不为中共的报复措施感到担心，他准备全力出击。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿)
</div>
