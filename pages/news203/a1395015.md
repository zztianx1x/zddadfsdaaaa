### 川普喊话北京:美国人不是傻瓜 中共好日子到头了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 本周四接受美国媒体的采访时对北京发出警告说，在中美之间的这场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 中，他还有很多招数可用。他同时告诫北京当局：不要把美国人都看成是〝傻子〞，过去美国之所以在贸易方面吃了中共的大亏，只是因为是那时的〝领头人〞太差。
 <br/>
 <br/>
 据《美国之音》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 日前接受福克斯电视台访问时表示，中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 开打以来，已经给中国的经济造成了强大的冲击，现在中共治下的经济已经明显的下滑，而且当下的情况是〝如果我想要，我就能采取更多行动。〞
 <br/>
 <br/>
 川普接着又说：〝我并不想这么做，但前提是他们（中共政府）得回到谈判桌来。〞
 <br/>
 <br/>
 川普表示，北京方面其实还是愿意谈判的，但他不认为中共政府已经准备妥协了。
 <br/>
 <br/>
 川普再次批评前几任美国总统长期容忍中共明显不公平的贸易做法。他说，他现在想要告诉北京，这样的好事〝结束了〞。
 <br/>
 <br/>
 〝（中共的）好日子过得太久了。〞川普说：〝我估计他们认为美国人是傻瓜。但美国人不是傻瓜。我们过去在贸易方面仅仅是领头人太差。〞
 <br/>
 <br/>
 《美国之音》认为，川普的上述发言显示，他在和中共展开的贸易战方面〝没有任何退让的迹象〞。
 <br/>
 <br/>
 就在一周前（10月4日），美国副总统彭斯发表了一次给国际社会带来了震撼的演讲。在这次演讲中，彭斯痛陈中共政府过去针对美国所做过的种种恶行与罪状。
 <br/>
 <br/>
 彭斯这次的演讲几乎对中共进行了全方位的抨击，内容涉及贸易不公、南海军事化、侵犯宗教自由、对美国进行政治渗透、网络攻击、干预美国选举、破坏美国自由等等。
 <br/>
 <br/>
 外界注意到，彭斯在演讲时，特别注意了区分〝中共〞和〝中国〞。他在每一次指出指出中方的某一种恶劣行径时，都会点名〝中国共产党〞，而不是按照惯例笼统地称呼〝中国〞或〝中国政府〞。
 <br/>
 <br/>
 彭斯在开足火力对中共的一系列恶行一通炮轰后，直接向北京当局喊话，表示希望北京能用实际行动〝重新尊重美国〞。他并警告说，在获得公平、对等和主权尊重前，美国政府绝不会让步。
 <br/>
 <br/>
 彭斯最后明确指出，贸易上的竞争〝并不总是意味着敌意〞。他说：川普总统希望共同促进中美双方的繁荣与安全，而不是分离。北京领导人现在仍可以改变路线，回归〝改革开放〞的精神。
 <br/>
 <br/>
 彭斯表示，他相信美国人民和中国人民之间的持久友谊，也希望促进中美两国的繁荣，他说：〝中国人民理应得到更多。〞
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
</div>
