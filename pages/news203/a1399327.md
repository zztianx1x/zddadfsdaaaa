### 美媒：华府制定新计划打击中共盗窃智慧产权
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】肆意盗窃或掠夺美国企业智慧产权，是引发美中以加征关税为主线的贸易战的一个重要原因。不过，日前有美国媒体披露，川普（特朗普）政府并未把一切希望寄托在关税战上，而是已经开始制定新的计划，准备动用关税以外的其他多方面综合手段来打击中共盗窃智慧产权的行为。
 <br/>
 <br/>
 《华尔街日报》援引知情人士透露，美国政府正制定一个包括动用出口管制、诉讼和其他工具的新综合计划，以打击中共盗窃美国企业智慧产权。
 <br/>
 <br/>
 该知情者表示，新的计划需动用广泛的资源，进行跨机构协调，逐一解决每宗盗窃案件，因此只适用在特定情况。
 <br/>
 <br/>
 据指，新计划会特别注重帮助那些拥有独家技术的美国公司避免遭到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济间谍.html" target="_blank">
  经济间谍
 </a>
 盗窃其技术的危害。而当下，美国官员正在寻找更多中共盗窃关键技术之类的案例，希望透过具体的反击行动，让更多美国的企业看到成效，从而愿意与政府合作，共同打击中共的经济间谍行为。
 <br/>
 <br/>
 此前，美国政府曾于11月初指控中国福建晋华集成电路有限公司盗取美国企业美光科技（Micron）的独家技术，同时宣布禁止所有的美国公司向晋华出口和转让美国原创技术，并把这个中资企业划入了对美国国家安全构成威胁的黑名单中，还以民事诉讼阻止晋华向美国出售其盗窃的技术。
 <br/>
 <br/>
 根据指控，美国Micron公司损失87.5亿美元。美国希望引渡被指控者，他们如被判决有罪，将面临最高15年有期徒刑和500万美元左右的罚款。
 <br/>
 <br/>
 同时被起诉的，还包括涉嫌与晋华公司合谋实施了技术盗窃行动的台湾联华公司，以及三名涉嫌盗窃美光公司商业机密的台湾人。
 <br/>
 <br/>
 在这个案例中，被盗窃技术的美光公司与美国政府进行了合作。
 <br/>
 <br/>
 （记者黎明报导/责任编辑：明轩）
