### 美国漫威首个亚洲超级英雄形象 灵感源于李小龙
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_漫威.html" target="_blank">
  漫威
 </a>
 漫画公司12月3日宣布，将推出首部以华人超级英雄为主角的电影，该角色定名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Shang-Chi.html" target="_blank">
  Shang-Chi
 </a>
 〞。
 <br/>
 <br/>
 12月4日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_漫威.html" target="_blank">
  漫威
 </a>
 漫画官方透露，目前该电影正在筹备中，美籍华裔编剧戴夫．卡拉哈姆会参与〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_Shang-Chi.html" target="_blank">
  Shang-Chi
 </a>
 〞的制作；漫威正在寻找来自亚洲的或者美籍亚裔电影制片人，以确保电影能够准确诠释〝Shang-Chi〞的亚裔身份。
 <br/>
 <br/>
 据《好莱坞报导》网站12月4日的报导，漫威工作室的负责人凯文．法伊格说，他希望在未来的漫威电影中增加更多样的角色，而且角色的展示不应只在镜头前，应更专注于人物背后的故事。
 <br/>
 <br/>
 据报导，〝Shang-Chi〞角色最初的灵感来源于华裔武打电影明星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李小龙.html" target="_blank">
  李小龙
 </a>
 ，1973年主演的电影《龙争虎斗》。当时，功夫电影刚刚开始在美国流行。这次漫威塑造的〝Shang-Chi〞角色，拥有丰富的历史色彩，电影也准备从一个新的视角诠释。
 <br/>
 <br/>
 根据漫威的故事，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_动漫.html" target="_blank">
  动漫
 </a>
 人物〝〝Shang-Chi〞出生在大陆湖南，从小隐居习武，与世隔绝。由父亲与师父授授武艺。
 <br/>
 <br/>
 〝Shang-Chi〞的父亲野心勃勃，企图统治世界，把儿子〝Shang-Chi〞当作统治世界的工具，对〝Shang-Chi〞进行严酷的武术训练。
 <br/>
 <br/>
 然而，Shang-Chi〞在武术的修习过程中，有所参悟，他开始怀疑其父的做法；当他来到外面的世界，得知自己尊敬的父亲其实非常邪恶之后，便开始奋起反抗。最终摆脱其父的控制，开始独自闯荡江湖。
 <br/>
 <br/>
 与其他漫威漫画的超级英雄形象不同的是，〝Shang-Chi〞不具备超能力，但是他精通中华十八般武艺，不输拥有超能力的其他人或形象。
 <br/>
 <br/>
 〝Shang-Chi〞的特点是：精于徒手战斗，精通多种武术和格斗技巧。另外，〝Shang-Chi〞还擅长冥想，可以通过冥想看到目力不能及的景象。
 <br/>
 <br/>
 美国漫威漫画公司（Marvel Comics）创建于1939年，于1961年正式定名Marvel；2010年9月，MARVEL宣布其正式中文名称为〝漫威〞。知名的动漫形象
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蜘蛛侠.html" target="_blank">
  蜘蛛侠
 </a>
 、金刚狼等均出自漫威漫画。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：文慧）
</div>
