### 中期选举投票在即 美国两党做最后动员
------------------------

<div class="wysiwyg">
 【新唐人2018年11月05日讯】11月6日，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 在即，两党都在做最后动员。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 、奥巴马以及州长和国会候选人们，纷纷呼吁选民，周二出来投票。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ：〝这次大选的对比不能再清晰了，民主党在制造暴徒，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 在创造就业！〞
 <br/>
 <br/>
 川普总统周一下午出席俄州助选集会，这是他今天三场集会的第一场。
 <br/>
 <br/>
 美国总统川普：〝民主党引来一拨又一拨的大篷车移民，这不好吧，这些非法入境的外国人，涌入我们的国家，‘占据’你们的社区。〞
 <br/>
 <br/>
 川普呼吁选民周二为共和党人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_投票.html" target="_blank">
  投票
 </a>
 。
 <br/>
 <br/>
 同一天，前总统奥巴马出现在弗吉尼亚州，继续为民主党人助选。
 <br/>
 <br/>
 美国前总统奥巴马：〝我想我们不能坐待其变，我们要努力争取，不要（给对手）喝倒彩，去投票！〞
 <br/>
 <br/>
 而在选前最后一刻，乔治亚州共和党籍州长候选人，同时也是主掌该州选举事务的州务卿坎普，指控民主党人可能涉嫌骇客入侵选民登记系统，并要求FBI调查，不过并未提供详细犯案情况。
 <br/>
 <br/>
 乔治亚州共和党籍州长候选人坎普：〝当我发现这些可能的网络犯罪时，我立即联系了FBI，国土安全部和乔治亚州调查局〞
 <br/>
 <br/>
 他与民主党州长竞选对手艾布兰的选战也是备受关注的焦点战役。目前坎普平均民调领先艾布兰2.8%。
 <br/>
 <br/>
 在德州，30年多来，其联邦参议员席位一直由共和党执掌。本次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 ，民主党后起之秀奥罗克挑战资深共和党参议员克鲁兹，克鲁兹平均民调比前者多6%。
 <br/>
 <br/>
 共和党联邦参议员克鲁兹：〝奥罗克在法官投票上，会投票反对戈萨奇，投票反对卡瓦诺。等你明天（周二）出来投票的时候，直接投给选票上的共和党。〞
 <br/>
 <br/>
 观察人士分析，川普总统对本次共和党中期选举帮助很大。
 <br/>
 <br/>
 美联社华盛顿分社主任 Julie Pace：〝川普现在就是共和党。他（川普）已经得到证明，他对很多共和党选民有真实持久的吸引力。所以你看到很多共和党选民，倾向他的政策。〞
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
