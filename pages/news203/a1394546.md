### 蓬佩奥晤王毅言辞严厉场面尴尬 直言不见习近平
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】美国国务卿蓬佩奥10月8日访华，只与中共外长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 和中共外事办主任杨洁篪匆匆会面。外媒报导，蓬佩奥和王毅会面时言辞激烈，场面紧张。蓬佩奥当日稍早时直言不会和北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 见面。
 <br/>
 <br/>
 《美国之音》报导，蓬佩奥和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王毅.html" target="_blank">
  王毅
 </a>
 在会晤后的联合记者会上，都直言不讳地重复了双方不同的立场。
 <br/>
 <br/>
 王毅指责美国不断升级贸易战，在台湾问题上损害中方〝权利〞，并对中共的内政和外交政策〝无端指责〞。
 <br/>
 <br/>
 蓬佩奥则回应称，在王毅所列举的问题上，美方和中方存在根本分歧，美国对中共采取的行动十分关注。
 <br/>
 <br/>
 蓬佩奥还批评北京最近取消两国国防部长的安全对话，并称这个对话是进行讨论的〝重要机会〞。
 <br/>
 <br/>
 中共官媒在会晤后的通稿中，强调了王毅对美方的指责，并突出蓬佩奥的表态：〝（美国）无意围堵中国，也没有全面遏制中国的政策〞。不过，通稿中刻意忽略了蓬佩奥对王毅的反击。
 <br/>
 <br/>
 《彭博》报导，外界普遍认为蓬佩奥此行不会与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 见面，而蓬佩奥于8日稍早也证实了这点。蓬佩奥回答媒体提问时，非常肯定地说：〝不，我不会和他见面〞。
 <br/>
 <br/>
 蓬佩奥此次亚洲四国之行，已先行访问了日本、韩国和朝鲜。10月8日上午抵达北京是其访问的最后一站，显示美方对此次访问北京并未寄予太多期望。蓬佩奥在抵达当日上午和下午先后会见了王毅及杨洁篪，傍晚即搭机离开北京，行程十分紧凑。
 <br/>
 <br/>
 在此之前，美国副总统彭斯10月4日在华盛顿发表演讲，对中共恶劣行径进行全方位的严厉谴责，被认为是〝新冷战宣言〞。中共外交部一日两度回应，言辞激烈。
 <br/>
 <br/>
 外界认为，蓬佩奥此次访华，主要聚焦朝核问题，但仍会延续彭斯演讲带来的紧张气氛。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：云涛）
