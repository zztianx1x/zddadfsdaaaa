### 美炸弹邮包案嫌犯落网 正面照曝光 有犯罪前科
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】向美国民主党政要和商界大佬寄送〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹邮包.html" target="_blank">
  炸弹邮包
 </a>
 〞的一名嫌犯26日早间在迈阿密地区落网，该嫌犯的正面照随即被媒体曝光。美国司法部将在美东时间下午2时30分就此案召开记者会。迄今为止，全美在最近的三天内已经发现了至少12个被寄出的可疑邮包。
 <br/>
 <br/>
 当地时间10月26日早间，联邦警察在佛罗里达州劳德岱堡（Ft.Lauderdale）附近的种植园市（Plantation）逮捕了一名涉嫌寄送爆炸包裹的男子。

 据美国有线电视新闻网（CNN）26日报导，嫌犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢亚克.html" target="_blank">
  谢亚克
 </a>
 (Cesar Sayoc) 现年56岁，有犯罪前科。此人有个住址在佛州艾文杜拉（Aventura），不过他与纽约也有关系。
 <br/>
 <br/>
 报导引述一名执法官员表示，在侦办近日接连发生寄送可疑邮包案件的过程中，执法人员通过追踪找到的各种线索，聚焦到了佛罗里达州南部以及这个地区的一个邮件处理中心，最终顺利将嫌犯抓获。从本月24日到目前为止，全美已查获12件
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_炸弹邮包.html" target="_blank">
  炸弹邮包
 </a>
 。
 <br/>
 <br/>
 随后有多家美国媒体曝光了谢亚克的正面照。
