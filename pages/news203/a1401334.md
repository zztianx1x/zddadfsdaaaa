### 库德洛：中方回应令人失望 川习会晚宴只是个派对
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】中美双方均无实质让步迹象，下月初
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 达成和解的可能性越来越低。白宫首席经济顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 日前表示，虽然总统对达成协议持开放态度，但中方迄今对美方要求的回应令人失望。他还透露，川习会是个类似
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_派对.html" target="_blank">
  派对
 </a>
 的晚宴。
 <br/>
 <br/>
 路透社周二（11月27日）引述
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库德洛.html" target="_blank">
  库德洛
 </a>
 （Larry Kudlow）透露，川普（特朗普）总统曾告诉顾问们，他认为达成协议的可能性很大。但前提是〝必须满足某些条件〞，中共必须解决的问题包括窃取知识产权、强制技术转让、美国公司在中国的持股比例、大宗商品的高关税和非关税壁垒以及商业黑客行为等等。
 <br/>
 <br/>
 库德洛暗示，白宫认为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晚宴是令贸易战出现转机的机会，但是美方迄今对中方在贸易问题上的回应感到失望。
 <br/>
 <br/>
 他说，〝我们发现他们（中共）的做法没有多大改变。〞
 <br/>
 <br/>
 川普日前表示，美方准备把对2,000亿美元中国商品加征的关税从10%提高到25%，如果川习会不能取得进展，还很可能对另外2,670亿美元的中国产品加征关税。
 <br/>
 <br/>
 库德洛说，〝我们都知道，他（总统）说话算数。〞
 <br/>
 <br/>
 对于12月1日的川习会晚宴后双方会否发表联合声明，库德洛称〝我不太确定〞。
 <br/>
 <br/>
 而据香港《南华早报》消息，晚宴仅持续〝超过两小时〞，晚宴后没有安排记者会，也没有联合声明。
 <br/>
 <br/>
 库德洛透露，并没有双方顾问当场举行会谈的安排。他说，〝这是一场类似
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_派对.html" target="_blank">
  派对
 </a>
 的晚宴。〞
 <br/>
 <br/>
 库德洛拒绝透露美方有谁将出席晚宴。但据此前消息，美国第一夫人梅拉尼娅、国务卿蓬佩奥、贸易代表莱特希泽、国家安全顾问博尔顿和库德洛都将陪同川普出席G20峰会。
 <br/>
 <br/>
 本月中旬，中方曾向美方提供了一份让步清单，被指〝老调重谈〞，美方表示拒绝接受。曾有消息称，刘鹤将赴美谈判，为川习会铺路，但随后改成直接赶赴G20主办国阿根廷会谈。不过，最终刘鹤在G20前去了德国，没有和美方团队接触。
 <br/>
 <br/>
 虽然中方不断隔空喊话，要加强知识产权保护，继续开放市场等，但与美方的要求相去甚远。
 <br/>
 <br/>
 外界认为，缺乏双方谈判团队事前的实质接触，中美首脑仅凭〝私人感情〞叫停贸易战的可能性很低。即使勉强达成暂时缓解的临时协议，如果中方缺乏后续的实质让步行动，协议也难以持久。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
</div>
