### 中共国安特工狱中正面照曝光 或面临25年监禁
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】中共国家安全部官员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐燕军.html" target="_blank">
  徐燕军
 </a>
 (Yanjun Xu，音译)今年稍早时候在比利时被美国联邦探员成功诱捕并被引渡到美国。日前，徐因涉嫌企图窃取美国航太公司的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商业机密.html" target="_blank">
  商业机密
 </a>
 在美出庭受审，成为被押送到美国并遭起诉的首名中共特工，随后他的正面照也被当地媒体曝光。
 <br/>
 <br/>
 当地时间10月10日，中共国安间谍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐燕军.html" target="_blank">
  徐燕军
 </a>
 在美国辛辛那提联邦法庭受审，他被控企图窃取美国航太公司的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商业机密.html" target="_blank">
  商业机密
 </a>
 。
 <br/>
 <br/>
 美国司法部近年来虽然时不时也宣布起诉一些在美犯案的外国特工，但一直未能把嫌犯押进美国法庭接受审判，该部门因此经常遭到美国各界批评。今年4月1日，徐燕军在比利时被诱捕后于本月9日引渡回美国，现在开始接受美国法庭的审讯，美国司法的官员总算扬眉吐气了。
 <br/>
 <br/>
 美国司法部官员日前对当地媒体披露，徐燕军是中共国安部江苏省国安厅六局副处长，专门负责窃取航太企业的商业机密，而徐涉嫌企图窃取美国奇异电气公司 (GE)旗下的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奇异航空.html" target="_blank">
  奇异航空
 </a>
 的商业机密。
 <br/>
 <center>
  <br/>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-12/p9076171a448122809.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-12/p9076171a448122809-ss.jpg"/>
   <br/>
   <font size="-1">
    遭到美方在比利时诱捕并引渡回美受审的中共国安官员徐燕军。(图片来源：辛辛那提WCPO网站网页截图)
   </font>
  </a>
  <br/>
  <br/>
 </center>
 <br/>
 资料显示，总部设在辛辛那提的奇异航空，是全球最大的航太公司中的一家，也是航空载具科技领域的领军企业。该公司为波音和空中巴士大飞机提供引擎，还正在为商业飞机和载重军用直升机发展新一代引擎。
 <br/>
 <br/>
 徐案的起诉书称，徐燕军从2013年起就开始锁定美国内外的一些航太公司，通过邀请这些公司的专家到中国旅行或到中国的大学演讲来拉拢他们，并为这些专家支付各种费用和津贴。
 <br/>
 <br/>
 据起诉书的讲述，徐燕军2017年3月，利用南京航空航天大学一名主管与奇异航空的一个工程师通电邮，然后邀请后者到中国去进行学术交流。同年5月和6月，那位奇异航空的工程师把五份公司文件存在私人手提电脑中并带到中国去进行演讲，期间与自称来自江苏科技促进协会的徐燕军会过面。
 <br/>
 <br/>
 今年2月，徐燕军开始与这个工程师讨论，希望在这名工程师到欧洲出差时再次与他会面，徐还要求这个工程师把他的工作电脑制作档案目录传给自己。随后，徐燕军又进一步要求对方在比利时会面时，把其手提电脑里的资料录制到自己的随身碟中。
 <br/>
 <br/>
 按照美国的相关法律，参与共谋和企图从事经济间谍活动者，最高可判处15年徒刑；共谋和企图窃取商业机密者，最高可判处10年徒刑。
 <br/>
 <br/>
 联邦调查局(FBI)指出，中共情报官员徐燕军案，证明中共政府直接监督并实施了对美国的经济间谍活动。
 <br/>
 <br/>
 美国司法部的官员也说，徐案并非孤立事件，而是中国（中共政府）企图利用美国公司和专门科技发展经济的最新事例。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
