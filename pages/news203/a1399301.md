### 彭斯：北京须有〝极大改变〞 否则将面对新冷战
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】11月底中美首脑会晤在即，在关键经贸问题上北京仍无让步的迹象，已令中美和解可能性大大降低。日前美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 再次喊话加压，将南海问题也加入谈判条件，警告中方必须做出〝极大改变〞，并直言提及中美〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 〞。
 <br/>
 <br/>
 周二（11月13日），正在日本访问的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 接受《华盛顿邮报》专访时表示，中方要想避免与美国及其盟友陷入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 ，必须〝从根本上〞改变其行为。
 <br/>
 <br/>
 彭斯谈到G20峰会上的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 晤时说，川普总统愿意和北京达成协议，但前提是北京也须在保护知识产权、强迫技术转移、限制外企进入中国市场、尊重国际规则及在国际水域航行自由等问题上做出实质让步。他深信中方已经充分了解美国的立场，并指川普期待接到中方的具体方案。
 <br/>
 <br/>
 彭斯警告说，如果要想避免新冷战，这次川习会是中方的最佳时机（如果不是最后机会的话）。
 <br/>
 <br/>
 他还表示，如果北京拒绝做出显著、实质的让步，美国已经准备好在经济、外交和政治对中方施加更大压力，包括将加征关税的中国商品数额翻倍。他相信强劲的美国经济足以承受这样的升级，而中国经济则不能持久。
 <br/>
 <br/>
 彭斯此行的目的是参加第33届东盟峰会以及亚太经济合作组织（APEC）领导人峰会，并顺道访问亚洲。外媒指，彭斯这次出访的使命是向亚太的美国盟友宣布，针对中共〝一带一路〞倡议，美国将向该地区提供一个真实、有竞争性的替代选项。
 <br/>
 <br/>
 彭斯10月4日全面谴责中共的重磅演讲，曾令外界惊呼新冷战开启。但彭斯本人公开使用〝新冷战〞对北京发出警告，似乎还是第一次。此前，白宫官员一般会尽量回避直接对〝冷战〞一词做出评论。
 <br/>
 <br/>
 观察人士指，这或许显示了川普政府对中共战略的根本转变。
 <br/>
 <br/>
 此外，彭斯列出的中美和解条件中，除了贸易不公问题，还加入了〝国际水域航行自由〞，据信是指南海问题。
 <br/>
 <br/>
 据香港《南华早报》报导，11月9日的中美第二轮外交安全对话上，中方要求美国停止向南海派遣军舰和军机，美方则称必须维护国际法允许范围内的航行自由。随后，美国务院发表声明，要求中方撤除部署在南沙群岛的导弹。
 <br/>
 <br/>
 这是美国首次就南海问题向中共提出此种要求。外界分析，中共2015年就已承诺不将南海军事化，美国不过是要中共兑现承诺。
 <br/>
 <br/>
 其实，即使不加入南海问题，中美此次达成贸易和解的可能性已经很低。
 <br/>
 <br/>
 《华尔街日报》消息，中共副总理刘鹤11月9日和美国财长梅努钦通话，美方依然坚持中方必须先提交回应美方要求的具体方案，通话没有取得任何进展。《南华早报》称，刘鹤将亲自赴美。
 <br/>
 <br/>
 不过，摩根大通中国首席经济学家朱海斌认为，双方都没有做出实质让步迹象，未来数月贸易战将继续升级。
 <br/>
 <br/>
 在这种情况下，美方再将谈判条件加码，再次印证了川普有关〝美方不急于达成协议〞的说法。
 <br/>
 <br/>
 外界普遍认为，美国中期选举的结果，反而加强了美国政府对中共的强硬立场。
 <br/>
 <br/>
 近年以来，中共的〝经济侵略〞对美国经济造成致命伤害，而中共对美国及西方的政治渗透、在国际上的经济和军事扩张，令中共称霸世界的野心暴露无遗，也让西方社会感受到实实在在的威胁。遏制中共已是美国朝野和两党的共识。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
