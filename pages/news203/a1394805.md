### 疑涉及经济间谍活动 德州华人学者卷入间谍门
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞最近在美国的华裔学者圈中，突然变成了一个敏感词。近期一些参与该计划的华裔学者相继被捕判刑。美国官员近日透露，联邦调查局（FBI）可能正在调查参与该计划的德州华人学者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢克平.html" target="_blank">
  谢克平
 </a>
 ，是否涉嫌参与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济间谍.html" target="_blank">
  经济间谍
 </a>
 话动。
 <br/>
 <br/>
 全国广播公司NBC新闻报导，数名美国官员近日表示，日前被控告涉嫌儿童色情罪的前德州大学安德森癌症研究中心（MD Anderson Cancer Center）教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谢克平.html" target="_blank">
  谢克平
 </a>
 （Keping Xie），可能涉及间谍活动，向中共提供相关科技数据。联邦调查局等相关单位已对其展开调查。
 <br/>
 <br/>
 资料显示，今年55岁的谢克平于1995年获得休斯敦德州大学肿瘤生物学和免疫学双博士学位，他曾发表逾百篇学术文章，是肿瘤转移研究领域的知名学者。NBC称，谢克平有参与中国用作引进海外专才的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_千人计划.html" target="_blank">
  千人计划
 </a>
 〞。
 <br/>
 <br/>
 法庭文件显示，当局今年1月开始调查谢克平是否涉及〝篡改政府文件〞。该文件虽然没有具体说明他涉嫌欺诈的内容，但是记载了警察在搜查谢克平的住家及办公室时，查获了两个装有各种儿童色情图片的设备。
 <br/>
 <br/>
 谢克平的律师梅斯（Nathan Mays）告诉NBC，执法部门在谢家总计没收了88个设备，包括手机、硬盘、记忆棒和各种类型的电脑。两个硬盘被标记为〝龙虎一〞及〝龙虎二〞。梅斯说，并不清楚联邦当局是否展开调查涉及谢克平的间谍活动。
 <br/>
 <br/>
 谢克平今年4月辞去年薪25万美元的工作，8月被休斯敦联邦南区法院以涉嫌〝持有儿童色情图片〞罪名提告。他上周三（3日）出庭时交出护照，保释候审。谢克平被逮捕后，安德森癌症中心已在其网站上移除了与其有关的信息。德克萨斯大学休斯顿警察局没有回应NBC新闻的询问，安德森癌症中心的一位发言人仅证实了谢克平已离职，以及有关其涉嫌儿童色情的调查。
 <br/>
 <br/>
 大纪元此前曾独家报导，随着川普（特朗普）政府反击中共盗窃知识产权行动升级，FBI最近将德州大学安德森癌症研究中心当作打击重点。据知情人告诉大纪元，该中心有18个人或被抓、或被革职、或被要求辞职，包括4名华人教授，谢克平是其中之一。
 <br/>
 <br/>
 据中共官方媒体报导，谢克平通过〝千人计划〞受聘于同济大学上海东方医院，也曾参加2009年在北京举行的〝第一届海外高层留学人员座谈会〞。
 <br/>
 <br/>
 资料显示，所谓〝千人计划〞是中共引进海外高端人才的战略，从2008年执行至今，已有近8000位海外专家入选，他们多数在美国高等教育机构任教或就职于联邦政府科研机构，也有服务产业界的特殊科研人才。
 <br/>
 <br/>
 之前，曾有美国媒体引述美国国防和情报官员称，中共启动的引进海外人才〝千人计划〞，试图以合法或非法的手段，将美国技术、知识产权带入中国，因此呼吁美国政府应像对抗苏联一样来对付中共。
 <br/>
 <br/>
 美国政府对此也提高了警惕，近期，一些加入了中共〝千人计划〞的学者接二连三被调查，甚至已被捕判刑。
 <br/>
 <br/>
 今年8月，在谢克平被指控前不久，联邦调查局休斯顿办公室举办前所未有的公开说明会，与会者包括休斯顿的医学、科学和学术机构的领导人。FBI官员在会议上表示，FBI与私有合作伙伴和政府机构密切合作，以确保联邦政府资助的研究补助金得到保护，并遏阻肆无忌惮的外国敌人。
 <br/>
 <br/>
 〝谢克平的被捕正值FBI在按千人计划名单抓人〞的消息在华人社区盛传，因而引起广泛关注。报导中说，华人微信圈流传〝FBI在休斯顿医学中心按名单找人〞、〝参与国内千人计划的人被美国政府视为卧底〞等。
 <br/>
 <br/>
 曾参与马里兰州第43区州众议员竞选的华裔肿瘤治疗专家沈栋，在接受美国媒体采访时提醒说：〝华人同胞必须了解，任职企业或政府，领的是人家给的薪资，做出的研究成果不是属于个人。拿到国际会议或海外学术单位分享，都必须先征得雇主的同意。〞
 <br/>
 <br/>
 沈栋还表示，在美国做事，只要守法守纪就无需惧怕是否会成为〝标靶〞。如果不守法，不管你是哪个族裔的美国人，都同样会落入法网。
