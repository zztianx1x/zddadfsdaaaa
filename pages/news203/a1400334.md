### 涉贪遭控洗钱 巴拿马前总统两子在美被捕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】美国移民暨海关执法局（ICE）证实，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴拿马.html" target="_blank">
  巴拿马
 </a>
 前总统马丁内利两名儿子因遭控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗钱.html" target="_blank">
  洗钱
 </a>
 ，21日被证实已在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈阿密.html" target="_blank">
  迈阿密
 </a>
 被逮捕，现面临引渡回国命运。
 <br/>
 <br/>
 超市大亨出身的马丁内利（Ricardo Martinelli）于2009至2014年期间担任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴拿马.html" target="_blank">
  巴拿马
 </a>
 总统。他被控担任总统期间动用公款，对150多名政敌进行监视，今年6月自美国引渡回国后，即被关押在巴拿马市郊监狱。
 <br/>
 <br/>
 美国检察总长波塞尔（Kenia Porcell）21日表示，总统马丁内利的两名儿子路易斯．恩立格．马丁内利．林纳瑞斯（Luis Enrique Martinelli Linares）和瑞卡多．马丁内利．林纳瑞斯（Ricardo Martinelli Linares），因遭控
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洗钱.html" target="_blank">
  洗钱
 </a>
 ，已在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迈阿密.html" target="_blank">
  迈阿密
 </a>
 被逮捕，现面临引渡回国命运。
 <br/>
 <br/>
 巴拿马检方去年11月指控这两人在父亲担任总统时，收受巴西建设公司Odebrecht共4900万美元。
 <br/>
 <br/>
 马丁内利两名儿子的委任律师指出，他无法证实他们已遭到逮捕，对检方指控也未进一步置评。
 <br/>
 <br/>
