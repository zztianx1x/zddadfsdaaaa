### 南北加州同陷火海 毁屋2000栋 5人遇难
------------------------

<div class="wysiwyg">
 【新唐人2018年11月09日讯】观众朋友大家好，欢迎收看全球新闻。受干燥强风天气的影响，美国北
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴特县.html" target="_blank">
  巴特县
 </a>
 和南加州的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文图拉县.html" target="_blank">
  文图拉县
 </a>
 周四同时爆发大范围野火，已有至少5人在大火中丧生。
 <br/>
 <br/>
 南北
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加州.html" target="_blank">
  加州
 </a>
 同陷火海，出现入冬以来最糟糕的自然灾害。当局接连下达
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_紧急撤离令.html" target="_blank">
  紧急撤离令
 </a>
 ，呼吁民众远离火场。
 <br/>
 <br/>
 加州紧急事务管理署主任 Mark S. Ghilarducci：〝目前，在南加州有10.5万人撤离；在加州大约有5.2万人撤离。〞
 <br/>
 <br/>
 经过两天一夜，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴特县.html" target="_blank">
  巴特县
 </a>
 的〝营地野火〞（Camp Fire）在时速40英里的大风的助力下延烧到7万英亩，仅有5%的火势得到控制。当地小城天堂镇（Paradise）的民宅和商业街被成片摧毁，初步估算，烧毁的建筑超过2000栋，另有1.5万建筑受到威胁。巴特县警局周五上午通报，在被大火困住的汽车内找到5具遇难者的遗体。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文图拉县.html" target="_blank">
  文图拉县
 </a>
 ，〝伍尔西〞（Woolsey Fire）和〝山丘〞（Hill Fire）野火同时燃烧，过火面积已超过1万英亩。猛烈的风势下，火花飞溅，迫使当地的马里布市（Malibu）居民逃离家园，有民众悲伤地表示自己的房子被烧了。
 <br/>
 <br/>
 南加州马里布居民：〝感谢上帝，每个人都活着逃出来了，这是最重要的。〞
 <br/>
 <br/>
 目前，加州共有6场野火同时燃烧，当局出动6000多消防员投入灭火。监于火情的严重性，侯任州长纽森已宣布巴特县和文图拉县进入紧急状体，并向联邦请求救援。
 <br/>
 <br/>
 国家气象局的预报显示，从周六开始，加州又将迎来一个新的干燥强风天气模式，容易爆发新的野火。
 <br/>
 <br/>
 新唐人记者林骁然旧金山报导
