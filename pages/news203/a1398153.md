### 内部问题加剧 媒体：习近平一边批美一边让步
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】5日的上海进口博览会开幕式上，北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 发表演讲，多处暗批美国，但同时又宣布继续开放，保护知识产权。有分析指，北京内部压力增大，或被迫对美国放软态度。也有人认为，北京承诺无新意，不代表要对美国妥协。
 <br/>
 <br/>
 在当日的演讲中，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 多次提及国际贸易要〝互利共赢〞，反对〝单边主义〞，并用〝手电筒只照别人不照自己〞来形容〝某些国家〞。
 <br/>
 <br/>
 但另一方面，习近平又宣布会扩大进口，未来15年将进口40万亿美元商品和服务。同时，他还表示会进一步降低关税，放宽投资限制及惩罚侵犯知识产权的行为。
 <br/>
 <br/>
 自由亚洲5日报导称，中美贸易战进一步白热化，习近平一边间接批评
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）政府，一边又做出妥协姿态。
 <br/>
 <br/>
 报导引述中国问题专家胡星斗的分析称，美国和中共内部都在批评北京背离邓小平的〝改革开放〞，习近平这是在表态〝继续开放〞。
 <br/>
 <br/>
 时事评论员刘锐绍则指出，中美贸易战的主动权在川普一方，北京如果继续态度强硬，不但会给经济带来更大伤害，还会积累更大的内部矛盾，从而引发北京内部问题。
 <br/>
 <br/>
 不过，习近平在演讲中也用大海来比喻中国经济，来说明中方不会被摧毁。这与此前〝自力更生〞的发言一脉相承，被指暗示在贸易战中顽抗到底的立场。
 <br/>
 <br/>
 对于中方〝开放市场〞的表态，外媒引述商界人士指，北京宣布的进口额度不过稍高于此前的预期。而且中方只字不提备受外界谴责的政府补贴政策，其它的承诺也只停留在口头上，并没有给出具体措施和时间表。
 <br/>
 <br/>
 因此，有分析认为，这次演讲和此前中方的立场相比，并无多少新意。
 <br/>
 <br/>
 自从中美贸易战以来，北京的一贯策略就是〝边打边退〞。此前，中共官方已经多次降低进口关税，并宣布了一些放宽市场准入的措施。
 <br/>
 <br/>
 不过，北京十余年来都一直宣称〝保护外企知识产权〞，但具体行动则恰恰相反。而北京近期出台的〝市场准入〞措施，均非〝一步到位〞，美方认为只是在拖延时间。
 <br/>
 <br/>
 而北京宣布〝扩大进口〞，也与此前中方的贸易谈判策略相合。美媒消息，北京多次向美方表示，可以通过大量采购美国商品来调节美中贸易逆差，以换取美国停战，但都遭美方拒绝。
 <br/>
 <br/>
 此外，中共官方一直对政府补贴政策只字不提。北京一再宣称，〝中国制造2025〞计划涉及中方〝核心利益〞，绝不让步。而〝中国制造2025〞实施的关键，就是利用庞大的政府财力和资源打造科技龙头企业，依靠不公平竞争垄断相关行业的国际市场。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
