### 美301调查：中共窃密变本加厉 新关税或随时开征
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国贸易代表署.html" target="_blank">
  美国贸易代表署
 </a>
 发表了一份详细报告，指中共迄今为止并未改变
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易冲突.html" target="_blank">
  中美贸易冲突
 </a>
 中核心的〝不公平〞做法，而且在强迫技术转移、侵犯智慧产权等方面〝变本加厉〞。舆论认为，美方在G20峰会前继续加压，是为了促使中方真正面对关键性问题。如果中共不能在这些关键方面改变，新关税将随时开征。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国贸易代表署.html" target="_blank">
  美国贸易代表署
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_USTR.html" target="_blank">
  USTR
 </a>
 ）这次发表的〝301条款〞调查更新报告长达53页，十分详细地记录了从今年3月美国政府发表对华〝301条款〞调查报告以来，中共政府在对外贸易方面的各种表现。
 <br/>
 <br/>
 USTR的这份报告指，中共不但没有〝建设性〞地回应最初的〝301条款〞调查报告，没有采取实质行动从根本上改变其强迫外企技术转让、侵犯智慧产权等问题上的政策和做法，甚至最近几个月还采取了进一步的不合理行动，例如：由中共政府支持的、针对美国企业发动的网络攻击正变得更密集，也更老练。
 <br/>
 <br/>
 报告称，虽然北京当局近期放宽了部分外资所有权限制，但中共政府仍不断利用外商投资限制，来要求或施压美国企业将技术转移给中国企业。
 <br/>
 <br/>
 美国贸易代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莱特希泽.html" target="_blank">
  莱特希泽
 </a>
 (Robert Emmet Lighthizer)同时也对外发表声明称，〝作为本届政府加强监督和执法工作的一部分，我们完成了这次更新，而这份更新的报告显示，中国（中共政府）并未从根本上改变不公平、不合理且扭曲市场的做法，这些都是我们2018年3月的301条款调查报告中的主题。〞
 <br/>
 <br/>
 〝301条款〞指的是美国《1974年贸易法》中的第301条，该条款明确规定，在国际贸易中，如果贸易伙伴有不公平行为，美方可单方面进行报复。
 <br/>
 <br/>
 今年3月份美国政府称发表了一份对华〝301条款〞调查报告，川普（特朗普）政府已依据这一条款做出决定，向总值2500亿美元的中国商品征收进口关税。
 <br/>
 <br/>
 针对日前USTR发布的这份报告，有港媒引述观察人士指，美方现在发报告严厉批评中共没有改变贸易不公平行为，有在〝川习会〞前向中方加大施压力度之意。
 <br/>
 <br/>
 一名美国贸易代表署官员则向彭博社表示，USTR发布这份长篇报告，只是为了记录中共持续不变的做法。他并表示，美国政府仍然希望在〝川习会〞上取得进展。
 <br/>
 <br/>
 白宫首席经济顾问库德洛（Larry Kudlow）20日上午也对外表示，美国总统川普对于与中国达成贸易协议仍持〝乐观态度〞。
 <br/>
 <br/>
 库德洛在接受福克斯电视台采访时称，中美在阿根廷的对话〝仍有机会〞。他表示，中美态势与此前两、三周甚至10月份相比有所改善。
 <br/>
 <br/>
 此前，中美双方都曾对外界证实，川普和习近平将利用11月底在阿根廷布宜诺斯艾利斯出席20国集团（G20）峰会的间隙举行双边会谈，讨论解决双方贸易战的可能方案。
 <br/>
 <br/>
 不过，在上周末举行的亚太经合组织（APEC）峰会上，中美双方在不公平贸易问题上又发生了激烈交锋，显示双方在一些关键问题上的看法分歧仍然十分严重，以至于最终导致APEC领袖宣言难产。
 <br/>
 <br/>
 美国副总统彭斯周六曾对外表示，除非中方满足美国的要求，否则美国政府不会在贸易冲突中让步，甚至还可能将关税提高一倍。
 <br/>
 <br/>
 而川普一直在各种场合的表态都很明确，他始终认为美中之间达成的任何协议都必须符合美国利益，中方必须解决知识产权、关税壁垒、技术转让等关键问题。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：李泉）
