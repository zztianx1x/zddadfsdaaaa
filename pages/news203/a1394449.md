### 赶赴婚礼遇车祸 纽约州逾20人死
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约州.html" target="_blank">
  纽约州
 </a>
 警方周日表示，周六晚间，一辆加长豪华
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_婚礼.html" target="_blank">
  婚礼
 </a>
 轿车和另一辆汽车在纽约上州相撞，造成至少20人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_遇难.html" target="_blank">
  遇难
 </a>
 。
 <br/>
 <br/>
 事发地点位于纽约奥伯尼（Albany）以西38英里的舒哈里（Schoharie）小镇，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约州.html" target="_blank">
  纽约州
 </a>
 30号高速公路上。至少6辆救护车和两架直升飞机赶到现场救援。目击者称，有很多救援人员打破了汽车的窗户，试图把车内的人救出来。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车祸.html" target="_blank">
  车祸
 </a>
 发生在当地一个商店门外，当豪华车冲下山坡时，店外停车场有多人被撞身亡。
 <br/>
 <br/>
 这是当地十多年来最严重的一起车祸。美国国家交通安全委员会（National Transportation Safety Board）已派人前往现场调查。
 <br/>
 <br/>
 新唐人电视综合报导
