### 中期选举揭晓 川普：希望两党合作
------------------------

<div class="wysiwyg">
 【新唐人2018年11月08日讯】观众朋友大家好，欢迎收看全球新闻。备受瞩目的美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 尘埃落定，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 扩大了在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 的多数席位，民主党则拿下了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 的主导权。周三中午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统召开记者会，阐述选情和后续政策。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 认为在参院取得的胜利更为重要，打破了过去两党席次接近的僵局。
 <br/>
 <br/>
 美国总统川普：〝我认为选举结果接近完胜。从未来的协商角度来看，从达成协议的角度看。如果我们所占的多数只多出1、2票，反而容易陷入僵局。我确信，可以和民主党很好的合作，可以提出很多好的提案，获得两党通过。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 前，川普曾参加了44场助选集会。他自豪的说，自己为之站台的11位参议员，有9位获胜。
 <br/>
 <br/>
 川普特别提到，佛州、俄亥俄州、和奥巴马前去助选的乔治亚州，选情极为激烈，但是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 最终赢得了参议员和州长的席位。
 <br/>
 <br/>
 美国总统川普：〝我们赢了乔治亚州，奥巴马总统在那里卖力助选，脱口秀主持人欧普拉也在电视上拼命助选，我看到有多艰难，我们这边只有我，没有别人。我去了乔治亚州，参加我们集会的人多的你们都没见过。我们的人赢了。〞
 <br/>
 <br/>
 他指出，民主党在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 获得了多数席次，是因为民主党金主花大钱助选；共和党一些资深议员退休的人数比较多；加上美国媒体不肯报导政府的成绩。
 <br/>
 <br/>
 尽管如此，川普希望两党合作，民主党如果有意杯葛议题，川普说，两年时间会很快消耗过去，他们现在是众院多数党，所以我就可以怪到他们头上。
 <br/>
 <br/>
 川普也希望在贸易问题上和中国达成协议。
 <br/>
 <br/>
 美国总统川普：〝很快会有数十亿美元流入我们的国库，那是中国缴纳的关税。如果你问顶尖的中国问题专家白邦瑞，他说他从未见过这样的事。还有谁没见过呢？中国也没见过。我会尽力和中国达成协议。因为我希望和习主席保持好的关系，和中国保持好的关系。〞
 <br/>
 <br/>
 记者会持续了两个半小时。期间一些美国媒体记者粗暴打断川普，而且在川普多次叫到别人后继续提问，不肯坐下。川普毫不客气的批评这些记者和他们代表的媒体，但仍放缓语气，保持克制。
 <br/>
 <br/>
 美国总统川普：〝他们对我不公，所以我有权反击。但是我的反击不是为我自己，我是为这个国家的人民。谢谢大家。〞
 <br/>
 <br/>
 新唐人电视综合报导
</div>
