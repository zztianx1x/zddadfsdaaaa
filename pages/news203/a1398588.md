### 川普喊话先发制人？学者：中共不可能放弃制造2025
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】日前，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）对媒体表示北京已〝放弃了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞，外界纷纷猜测中美是否已经达成协议。而港媒分析，这可能只是川普的舆论攻势，大陆学者则认为中共不会放弃〝中国制造2025〞。
 <br/>
 <br/>
 美东时间11月7日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在白宫记者会上说：〝中国放弃了2025计划，因为我发现这个计划太放肆。我对他们说这很放肆，因为这意味着到2025年他们会在经济上称霸全球。我说这不会发生。〞【相关链接：
 <a href="http://www.ntdtv.com/xtr/b5/2018/11/08/a1398487.html" target="_blank">
  川习通话成果？川普暗示北京已放弃制造2025计划
 </a>
 】
 <br/>
 <br/>
 值得注意的是，川普在这里使用了〝放弃〞一词的过去时态〝got rid of〞，或显示事情已经发生。媒体猜测，川普此言是否在暗示已经和北京达成某种协议。
 <br/>
 <br/>
 不过，新加坡《海峡时报》报导，川普是在一个〝激烈〞的新闻发布会上发表了上述讲话。在发布会上，川普和CNN等媒体的记者互相进行了〝暴躁的交流〞。
 <br/>
 <br/>
 而香港《经济日报》认为，川普〝抢先〞替北京宣告放弃〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞，又明言不准中共经济称霸全球，可能只是一种谈判策略，目的是制造既成事实及公众期待，逼迫中方在即将举行的川习会上让步。
 <br/>
 <br/>
 文章分析，美国中期选举刚刚结束，川普就释放上述言论，显示他将继续加大对中共施压。
 <br/>
 <br/>
 文章认为，北京一直宣称〝中国制造2025〞涉及中共的〝核心利益〞，因此无论中美角力如何激烈，中方对此都没有放弃。北京高层屡次提及核心技术要靠〝自力更生〞，已显示了不愿让步的意思。
 <br/>
 <br/>
 广州中山大学经济系教授林江8日也对自由亚洲电台表示，〝中国制造2025〞和中共十九大设立的政治目标息息相关，北京已经为〝中国制造2025〞投放了很多资源。如果放弃该计划，不但不利中共发展经济，对政治上亦有很大的影响，所以北京绝不会放弃这个计划。
 <br/>
 <br/>
 不过，天则经济研究所副所长蒋豪认为，中美贸易战中，中共经济遭遇重挫，今年年底将陷入更大困境，因此北京应该会和美国作出妥协。此前中共官方已经停止宣传〝中国制造2025〞。
 <br/>
 <br/>
 〝中国制造2025〞是北京2015年提出的经济计划，旨在利用政府巨额补贴和政策支持，扶植一些中国核心科技产业主导全球相关领域。
 <br/>
 <br/>
 目前还没有确切消息指中共决定放弃该计划。不过近期有报告指，北京可能为求与美国达成贸易协议，不惜暂缓〝中国制造2025〞。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
</div>
