### 班农激辩经济学人主编：中共每个毛孔都是邪恶的
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】中美对抗逐渐迈向新冷战，反映出美国朝野对中共强硬立场的一致。但美国内外仍不乏左派政客或媒体站在北京一边。网传前白宫顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_班农.html" target="_blank">
  班农
 </a>
 上月和《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济学人.html" target="_blank">
  经济学人
 </a>
 》主编的访谈视频，班农力证跟中共邪恶政权不可能有真正的自由贸易。
 <br/>
 <br/>
 日前，推特上流传
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_班农.html" target="_blank">
  班农
 </a>
 9月15日和《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济学人.html" target="_blank">
  经济学人
 </a>
 》女主编扎尼．贝多斯在纽约激辩的一段视频。
 <br/>
 <br/>
 《经济学人》坚持自由贸易和全球化，这和北京的主张相合。今年上半年，在北京的《经济学人》创刊庆祝活动上，扎尼．贝多斯表示，为推动《经济学人》经济主张的实现，北京〝将扮演非常重要的角色〞。
 <br/>
 <br/>
 在9月和班农的访谈中，扎尼．贝多斯也表现出反对中美贸易战的倾向，主张继续推动和北京的〝自由贸易〞。
 <br/>
 <br/>
 班农则针锋相对地进行反驳。他说，中国的问题不仅仅是〝市场有点不自由〞，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共政权.html" target="_blank">
  中共政权
 </a>
 的每个毛孔都是邪恶的，这是本质，就跟30年代日本法西斯政权一样。〞〝怎么可能跟一个邪恶极权国家谈自由贸易？〞
 <br/>
 <br/>
 班农还呼吁，〝那些认为中国崛起、市场就会开放的精英们醒醒吧！〞
 <br/>

 班农是美国一名媒体人，曾任白宫首席策略师兼美国总统顾问。
 <br/>
 <br/>
 班农认为，中共邪恶政权的崛起，将威胁美国和西方社会的安全，主张对中共采取强硬的遏制措施。因此，班农一直支持川普（特朗普）启动中美贸易战。
 <br/>
 <br/>
 今年7月，班农向CNBC表示，川普总统清楚地知道，他需要团结西方国家来对抗极权主义中共政权的崛起。他认为，美国与中共的交战只会有一个结局，那就是美国将成为胜利者。
 <br/>
 <br/>
 去年8月，班农离开白宫后宣称，还要〝为川普而战〞。
