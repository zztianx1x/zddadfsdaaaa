### 贪腐镇压破坏民主 美制裁尼加拉瓜第一夫人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】暴力示威和普遍存在的贪腐已导致数以百计的无辜
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼加拉瓜.html" target="_blank">
  尼加拉瓜
 </a>
 人丧生。美国总统川普（特朗普）周二（27日）颁布新的行政命令，准许相关部会锁定镇压反政府示威的尼国官员，并制裁尼国第一夫人和她亲信幕僚。
 <br/>
 <br/>
 美国财政部以破坏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_尼加拉瓜.html" target="_blank">
  尼加拉瓜
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主.html" target="_blank">
  民主
 </a>
 为由，宣布制裁尼加拉瓜总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_奥蒂嘉.html" target="_blank">
  奥蒂嘉
 </a>
 （Daniel Ortega）的妻子兼副总统穆利优（Rosario Murillo），以及她的亲近幕僚蒙卡达劳（Nestor Moncada Lau）。
 <br/>
 <br/>
 美国财政部长梅努钦（Steven Mnuchin）发表声明称：〝穆利优及其政治打手系统性寻求瓦解民主制度，并掠夺尼加拉瓜的财富，以巩固权力掌控。〞
 <br/>
 <br/>
 声明称，暴力示威和普遍存在的贪腐已导致数以百计的无辜尼加拉瓜人丧生，并摧毁了国家经济〞，美国〝致力于让奥蒂嘉的政权对此负起责任〞。
 <br/>
 <br/>
 至于蒙卡达劳，美方形容他在政府中虽没有正式职位，但据信他对警察有相当大的影响力。他的角色就像是奥蒂嘉和穆利优的国安顾问。
 <br/>
 <br/>
 路透社报导，美国的制裁措施将可以冻结穆利优和蒙卡达劳在美国司法管辖范围内的任何财产，并禁止美国个人、银行和其他实体与他们进行任何交易。
 <br/>
 <br/>
 不过，曾任国防部顾问的卡基那（Roberto Cajina）说，因为穆利优和蒙卡达劳都没有使用自己的名字进行投资，一切都是透过挂名负责人进行的。而因缺乏书面文档，这些制裁恐无法使任何一名被锁定对象的资产被冻结。
 <br/>
 <br/>
 一位没有透露姓名的川普政府高阶官员表示，没放警告奥蒂嘉，应想好退场策略，启动自由和公平选举的程序。这是一个为他们自己制造出来的危机寻求某种解决之道的机会。
 <br/>
 <br/>
 尼加拉瓜政府在当天稍后发表的声明中〝拒绝〞接受美国制裁，但没提到穆利优和蒙卡达劳的名字。
 <br/>
 <br/>
 尼加拉瓜今年4月开始爆发示威潮，抗议社会福利制度改革计划，后来逐渐演变成为反政府示威，警方和武装团体镇压示威群众，导致超过300人丧生、至少2000人受伤。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
