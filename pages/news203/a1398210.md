### 川普：有些事快做成了 将与习近平共进晚餐 (视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月06日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）日前在接受采访时表示，他与习近平将在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 期间会面并共进晚餐，届时会讨论中美贸易话题，相信可以和中方达成一个非常好的协议，〝我们越来越接近做成一些事情了〞。
 <br/>
 <br/>
 《彭博社》11月5日报导说，美国已经在作准备，如果
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 和习近平在阿根廷举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 会晤中未能调和目前的中美贸易争端，将会在12月对另外2670亿美元的中国进口商品加征25%的关税。
 <br/>
 <br/>
 同日，川普接受福克斯电视台〝英格拉姆角度〞（Ingraham Angle）节目专访时表示，〝我认为我们将与中国达成很好的协议，必须要很好。〞
 <br/>
 <br/>
 川普与习近平在11月1日进行了贸易战以来的首次通话。川普在推特中形容，这是一次〝长时间而且非常友好的谈话〞。习近平则称，两国要推动中美经贸问题达成一个双方都能接受的方案。
 <br/>
 <br/>
 2日川普在接受媒体提问时，被问及美中是否将在G20期间达成协议。川普回答，与中方进行了良好讨论，他们非常希望达成协议，自从发生贸易冲突后，中国经济下行。
 <br/>
 <br/>
 川普还证实，他将在G20峰会上与习近平共进晚餐。届时双方将就讨论如何解决美中贸易争端。他补充说，现在是中方急切想要达成协议，因此相信应该可以和中方达成对大家都非常公平，对美国好的〝公平协议〞。〝我们越来越接近做成一些事情了。〞
 <br/>
 <br/>
 川普此前曾表示，中共每年从美国拿走5000亿美元，美国帮助重建了中共。〝如果我们不这样做，中国就不在他们现在的位置。我们不能再这样做了。〞必须改变。
 <br/>
 <br/>
 川普也透露，如果他与习近平在G20的会面中不能达成协议，美方会对2500亿美元（应为2000亿美元）加征25%关税，且可能对另外2670亿美元商品加征25%关税，或更高税率。
 <br/>
 <br/>
 11月1日，川普的经济顾问库德洛（Larry Kudlow）在华盛顿的一个活动中，谈到即将到来的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习川会.html" target="_blank">
  习川会
 </a>
 〞时，他说，这次会谈并非G20峰会的副产品，可能会是一次相当正式的双边会谈。双方正考虑进行一次非常正式的双边会谈，川习届时还可能一起进餐。
 <br/>
 <br/>
 在被问到〝习川会〞议程时，库德洛仅表示：〝双方团队正在就议程进行讨论研究。我想经贸（问题）会被包括在内，但我不是百分百确定。〞
 <br/>
 <br/>
 库德洛也表明，如果双方不能在阿根廷就知识产权、网路安全、商品关税及其他问题达成任何协议，川普将继续〝积极（aggressively）〞推行针对中共的议程。
 <br/>
 <br/>
 同时，库德洛还延续了川普政府一直以来对中共的严厉论调，他批评许多来自他国的不公平贸易行为，给美国劳工及经济带来伤害，坦率的说，中共正是其中的罪魁祸首。
 <br/>
 <br/>
 在库德洛宣布彼此关系〝解冻〞的同一天，美国司法部宣布了一项针对中共的新执法行动，成立〝中国计划〞（China Initiative）项目，专门打击中共针对美国的经济贸易间谍活动。
 <br/>
 <br/>
 司法部长塞申斯（Jeff Sessions）说，盗窃美国的科技，严重威胁着美国国家安全，是非法行为，这种威胁尤其来自因盗窃知识产权，而在全球恶名昭着的中共政权。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：范铭)
 <br/>
