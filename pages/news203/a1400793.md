### 专家揭北京头号危机 并非贸易战
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】受美中贸易战影响，中国经济增长进一步放缓。有美媒提醒，北京面临的最大挑战，并非中国经济持续下滑，而是中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_债务危机.html" target="_blank">
  债务危机
 </a>
 已经濒临爆发的边缘，贸易战只是导火索。
 <br/>
 <br/>
 美国《福布斯》杂志11月25日的一篇评论说，中国经济面临的最大挑战是债务，而非贸易战。评论称，虽然中共官方的数字称国债占国民生产总值GDP的比重仅为47.6%，但国际金融协会非官方的数字却提到，中国国债占GDP则高达300%。
 <br/>
 <br/>
 到目前为止，国际信用评级机构穆迪（Moody's）和标准普尔（Standard &amp; Poor's）在评级过程中，还都没有将中共非官方债务规模考虑在内，这也是为什么中国的信用评级跟美国和日本等国家接近的原因。
 <br/>
 <br/>
 评论说，除了国际金融协会外，过去也有其他机构估计，中国债务总量与GDP的比率正在急剧飙升，有可能触发金融危机。考虑到中国在世界经济中所占的比重，这场危机不仅会重创中国经济，还会影响全球经济。
 <br/>
 <br/>
 由于统计口径不同，以及中国存在大量影子信贷的问题，很少有人能够说清楚中国的债务规模有多大。中共媒体曾称，中国人均负债13.34万元人民币，债务总额接近200万亿人民币。
 <br/>
 <br/>
 评论说，美中贸易战本身虽然不会对中国经济构成重大威胁，但将会加剧中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_债务危机.html" target="_blank">
  债务危机
 </a>
 。其中的核心问题之一，在于中共政府即是借贷人又是放贷人，国有银行不过是政府的代理人。
 <br/>
 <br/>
 据中共财政部的官方数据，截至今年6月末，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地方政府.html" target="_blank">
  地方政府
 </a>
 债务余额167997亿元人民币。财政部称，这个规模控制在中国人大批准的限额之内。
 <br/>
 <br/>
 标准普尔全球评级（S&amp;P Global Ratings）10月16日发布的报告称，中共地方政府隐瞒的债务可能将近6万亿美元，是中共对外公布的地方债务的数倍。
 <br/>
 <br/>
 标普分析师认为，造成这个巨大落差的原因是，中共中央政府容许地方政府对基础设施的投资金额与实际申报金额有所差异。
 <br/>
 <br/>
 标准普尔信贷分析师Gloria Lu在报告中说，这个潜在债务是一个〝充满巨大风险的冰山〞。据标普分析，中共去年政府债务总额占其GDP比重达到六成。如果中国经济未见好转，中共很有可能让地方政府融资平台违约。
 <br/>
 <br/>
 过去，中共依靠出口创汇维持庞大的政府债务。如今在美中贸易战冲击之下，中共政府债务只增不减，出口创汇能力越来越小，人民币价值越来越低。外界分析，中共即使印更多的人民币也无法偿还债务。如果情况未见改善，中共经济将面临整体崩盘的危险。
 <br/>
 <br/>
 在这种情况下，中共8月起决定鼓励银行增加放贷，并再次允许负债累累的地方政府上大型项目，以应对日益放慢的经济增长。修改后的中国金融监管规定，将允许银行几乎不受限制地持有地方政府债券。
 <br/>
 <br/>
 彭博曾刊文说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 的副作用会使中共债务危机濒临引爆边缘。评论人士陈思敏分析说，中共债务危机将引爆13亿民怨。欠债还钱天经地义。现在中共只能拖，一拖再拖。到最后拖不下去的时候，全中国民众也会对中共来一个总清算。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
