### 美期中选举预测 共和党稳住参院 民主党夺众院
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_期中选举.html" target="_blank">
  期中选举
 </a>
 6日登场，全美各地不少选民早早就赶去投票，出席一连串助选集会的川普总统，则选择在白宫观战。目前国家广播公司新闻网（NBC News）和福斯（Fox）都预测
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 或将夺下联邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 。
 <br/>
 <br/>
 美国联邦
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 共有435席，由各州按人口比例选出代表组成。这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_期中选举.html" target="_blank">
  期中选举
 </a>
 前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 掌握236席，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民主党.html" target="_blank">
  民主党
 </a>
 人占193席，另有6席空缺。众院每2年全面改选，这回当选众议员的任期将从明年1月开始。
 <br/>
 <br/>
 北京时间12：15战况，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_参议院.html" target="_blank">
  参议院
 </a>
 改选方面，共和党50席：民主党41席。众议院方面，共和党152席：民主党146席。州长改选，共和党21席：民主党17席。
 <br/>
 <br/>
 现任众议院议长是代表威斯康辛州的共和党籍莱恩（Paul Ryan），这回他未寻求连任。民主党籍的前议长裴洛西（Nancy Pelosi），目前则是众院少数党领袖。
