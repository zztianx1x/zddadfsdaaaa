### 截堵大篷车 川普：可派1万5000美军赴边境
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】为截止
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 入境美国，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）政府近日宣布派驻5200名现役军人到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 。周三（10月31日），川普再次强调，华府可派遣更多美军前往墨西哥边界。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周三离开白宫准备参加佛罗里达州的集会时告诉记者，美国可以向墨西哥边境派遣1万5000名军人。
 <br/>
 <br/>
 川普表示，〝没有人能进来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_非法移民.html" target="_blank">
  非法移民
 </a>
 是一个非常大，非常危险的话题。我们不会允许人们（非法移民）进入我们的国家而不考虑到我们国家的福祉。〞
 <br/>
 <br/>
 川普当天还发推文强调，〝我们的军队正在南部边境行动起来，还有更多的军队来了。我们不会让这些由一些非常糟糕的暴徒和团伙成员组成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车.html" target="_blank">
  大篷车
 </a>
 进入美国。我们的边境是神圣的，必须合法地进入。请回去！〞
 <br/>
 <br/>
 川普在推文中说，大篷车由一些非常强硬的暴徒和团伙成员组成。在突破北部边境时与墨西哥警方对阵，导致墨西哥士兵受伤，（他们）无法或不愿意阻止大篷车。应该在他们到达美国边境之前阻止他们，但可能不会！
 <br/>

 <br/>
 川普表示，派更多士兵加强边境安全很有必要。
 <br/>
 <br/>
 五角大厦截至目前授权逾5200名美军赴南疆，另外2000名美军处于待命状态。之前已经有2000名美国民兵（National Guard）在边境支援当地执法人员。
 <br/>
 <br/>
 美国国防部长马蒂斯（Jim Mattis）周三称，派遣数以千计士兵前往美墨边界是根据海关与边境保护局局长的要求，对国土安全部长提供务实支持。这是马蒂斯近两年前上任以来最大规模兵力部署。
 <br/>
 <br/>
 这些美军也在积极筹备，美军将首先集中精力〝加强〞德州的边界安全，其次是亚利桑那州和加州。派驻边境的军人已经将军事设备装载到肯塔基州的运货飞机上，并在德克萨斯州卸载。同时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美墨边境.html" target="_blank">
  美墨边境
 </a>
 的德州本地平民，也拿起枪枝，准备协助军队等边防人员守卫他们的土地。
 <br/>
 <br/>
 大篷车难民目前在南墨西哥，据报已从7000人减少到了4000人。他们还在用各种方式向美国行进，有的乘坐卡车，有的步行。周三，另一批约2000人的大篷车队伍从中美洲国家萨尔瓦多出发，准备前往美国。
 <br/>
 <br/>
 更为严重的是，10月28日在墨西哥特昆乌曼和危地马拉的接壤地段，约千人的一批中美洲难民突破了边境门，同防卫警察发生冲突，他们向工作人员和联邦警察投掷石块，鞭炮，玻璃瓶。导致6名警察受伤，一名洪都拉斯难民死亡。
 <br/>
 <br/>
 上周末，墨西哥政府宣布向已进入墨西哥的中美洲移民提供庇护、工作、教育和医疗服务，但他们拒绝并继续向北进发，誓言设法入境美国。
 <br/>
 <br/>
 10月29日，川普在推文中呼吁大篷车移民〝请回去〞，他说，〝许多黑帮成员和一些很坏的人混在大篷车队伍中，向我们的南部边境进军。请回去，不会允许你们进入美国的——除非通过合法程序。这是对我国的入侵，我们的军队正等着你们！〞
 <br/>
 <br/>
 今年稍早，国土安全部部长尼尔森（Kirstjen Nielsen）及海关及边境保护局（U.S. Customs and Border Protection，CBP）局长麦卡莱南（Kevin K. McAleenan），鼓励想要寻求庇护的人在入境口岸申请庇护及等待结果，并表示这是申请庇护的唯一合法途径，任何非法越过边界的行为都是犯罪行为。
 <br/>
 <br/>
 （记者李红报导/责任编辑：戴明）
