### 美中〝知识产权战〞 实为制度战？
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 检方再对中共江苏国安厅的商业间谍提出起诉。专家认为，美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 分歧，实质上是两国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_社会制度.html" target="_blank">
  社会制度
 </a>
 的根本冲突。
 <br/>
 <br/>
 从9月以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国.html" target="_blank">
  美国
 </a>
 检方在短短2个月内，3次起诉江苏国安厅的情报人员，以详细证据，指控他们试图窃取欧美飞机发动机技术细节等重要商业机密。
 <br/>
 (9月25日  1人被起诉：Chaoqun Ji
 <br/>
 10月10日 1人被起诉：Yanjun Xu
 <br/>
 10月30日 2人被起诉：Zha Rong，Chai Meng)
 <br/>
 <br/>
 唐靖远：这一系列的案子，是第一次有中共情报部门的商业间谍受到美国司法追究，展现了美国防止商业窃密的决心。而这些商业机密极大可能是被交给了中共国有军工企业，我觉得下一步，可能会有更多中共国企因此受到美国制裁。
 <br/>
 <br/>
 不只是国安部门，中共军队也扮演商业和技术间谍的角色。
 <br/>
 <br/>
 根据美方2014年指控，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共军方.html" target="_blank">
  中共军方
 </a>
 61398等骇客部队，从2006年起窃取了大约115家美国大企业的数据，涉及通信、航空、能源等重要产业。造成美国企业年损失约3000亿美元。
 <br/>
 <br/>
 横河：在一般国家都不会这样做的，不会由军队出面来盗取经济情报。为什么能够动用军方呢？就是因为军队就是党的工具。盗取情报得益的都是中国的国营企业。国营企业就是中共利益集团的利益。这都是为了利益集团干的事情。
 <br/>
 <br/>
 唐靖远：很多都是军民两用的科技，也被用来强化中共军力。从根本上来说，共产党长期面临内忧外患的生存危机，试图加强〝硬实力〞以维持统治；它觉得最便捷的做法就是超限战，用偷抢来实现弯道超车。
 <br/>
 <br/>
 美国印太司令部司令戴维森(Philip S. Davidson)早前说，中共正在研发一系列新式武器，〝如果研发不出来，就会用偷的〞，通常是通过网路窃取。
 <br/>
 <br/>
 美国媒体《华盛顿自由灯塔》2015年曾引述美国国安机密简报，称过去几年，中共军方总参三部的骇客，通过数万次网络攻击，盗窃了50TB的美国军事机密数据。武器专家分析，中共随后研发的歼-31战机，基本就是仿制美国〝F-35〞战机。
 <br/>
 <br/>
 唐靖远：西方国家很难理解，作为一个执政党怎么能以举国之力去偷？他们可能忘了，共产主义从马克思开始，就宣称要打破一切传统道德和价值约束。它们也不承认私有产权。所以当前围绕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 的冲突，从根本上来讲是两种制度、两种文化、两种价值观的冲突。
 <br/>
 <br/>
 上周，美国研究人员最新披露，中共央企中国电信，透过劫取美国和加拿大的网络流量，从事间谍活动和窃取知识产权。中共相关做法，已引起越来越多的关注。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
