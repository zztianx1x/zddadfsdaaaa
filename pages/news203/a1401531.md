### FBI突袭希拉里铀矿门举报人 搜查6小时动机遭质疑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】近日，美国联邦调查局（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 ）突然闯进一名司法部公认的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_举报人.html" target="_blank">
  举报人
 </a>
 家里
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_搜查.html" target="_blank">
  搜查
 </a>
 。该举报人曾举报有关前国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希拉里.html" target="_blank">
  希拉里
 </a>
 、克林顿基金会及美俄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_铀交易.html" target="_blank">
  铀交易
 </a>
 的丑闻，而FBI特工却在举报人交出文件后仍对其住所搜查了6小时，动机非常可疑。
 <br/>
 <br/>
 据美国保守派网站〝每日电讯〞（The Daily Caller）11月29日报导，19日上午，16名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_FBI.html" target="_blank">
  FBI
 </a>
 特工突袭了司法部认可的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_举报人.html" target="_blank">
  举报人
 </a>
 丹尼斯‧内森‧凯恩（Dennis Nathan Cain）位于马里兰州的住所。该举报人持有关于克林顿基金会（Clinton Foundation）和〝铀壹公司〞（Uranium One）的文件。
 <br/>
 <br/>
 凯恩的律师索卡拉斯（Michael Socarras）称，凯恩在为一名FBI承包商工作时，发现了一些令人震惊的文件，而且他已把这些文件交给司法部总监察长霍洛维茨（Michael Horowitz）和参众两院的情报委员会。
 <br/>
 <br/>
 这些文件披露，时任FBI局长穆勒（Robert Mueller）没有调查关于时任国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希拉里.html" target="_blank">
  希拉里
 </a>
 （Hillary Clinton）和克林顿基金会的一项被指控的犯罪行为，即推动掌握大量美国铀矿资源的〝铀壹公司〞（Uranium One）将自己的一大部分卖给俄罗斯国家原子能公司（Rosatom），从而让俄罗斯掌握了美国20%以上的铀供应、危害美国国家安全。
 <br/>
 <br/>
 索卡拉斯告诉〝每日电讯新闻基金会〞（The Daily Caller News Foundation），FBI突袭凯恩的住所是为了获取凯恩已合法交给国会的文件，他们的行为蛮横地漠视了法律提供给举报人的保护。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_搜查.html" target="_blank">
  搜查
 </a>
 令是11月15日由美国巴尔的摩地区法院（U.S. District Court for Baltimore）的联邦法官加拉格尔（Stephanie A. Gallagher）签发的，该法官是前总统奥巴马（Barack Obama）提名的。
 <br/>
 <br/>
 索卡拉斯说，一名领导搜查凯恩住所的FBI特工指控凯恩拥有盗窃来的联邦财产，并要求进入凯恩的私人住所进行搜查。
 <br/>
 <br/>
 索卡拉斯还披露，凯恩在受到惊吓后很快就交出了FBI索要的文件，但16名特工仍然搜查了6小时之久。
 <br/>
 <br/>
 索卡拉斯指出，FBI在突袭凯恩之后并没有通知其律师，尽管他们明知绕开当事人的律师是严重的错误行为。
 <br/>
 <br/>
 FBI早在2009年就掌握了俄国核工业部门通过行贿等方式换取具有战略价值的铀矿、扩大俄罗斯在美国的原子能业务的情况。但2010年，美俄还是达成了交易。证据显示，是时任国务卿并负责〝外国投资委员会〞的希拉里帮助达成了这项对俄方极为有利的协议，而俄国官员则为此向克林顿基金会输送了几百万美元。
 <br/>
 <br/>
 由于这项协议加强了俄罗斯对美国铀工业的掌控，因此在签订之初就倍受争议，而奥巴马政府却并未加以任何阻止。作为利益回报，希拉里一家从中获益约145,000,000美元。
 <br/>
 <br/>
 然而，虽然FBI早就掌握了俄国行贿的证据，但司法部却继续调查了将近4年，其实是故意对美国国会和公众隐瞒这一黑幕。
 <br/>
 <br/>
 2015年，该事件曾被部分披露。但在奥巴马执政下，相关调查和媒体报导并没有跟进。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
