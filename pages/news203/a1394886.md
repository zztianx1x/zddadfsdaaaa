### 美媒揭谷歌计划6至9个月推出审查版引擎
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】互联网巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 为重返中国，秘密展开一个名为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蜻蜓计划.html" target="_blank">
  蜻蜓计划
 </a>
 〞的项目，配合
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 过滤搜索结果。美国媒体9号披露，谷歌试图在6到9个月内推出该项目，与此前声明内容背道而驰。
 <br/>
 <br/>
 联邦参议员 克鲁兹(09/26)：有没有蜻蜓计画？如果有，是什么内容？
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 首席隐私官 Keith Enright：有蜻蜓计画。
 <br/>
 克鲁兹：它的内容是什么？为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 开发一款
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_搜索引擎.html" target="_blank">
  搜索引擎
 </a>
 ？
 <br/>
 Keith Enright：我们尚未接近在中国推出一款搜索产品。
 <br/>
 <br/>
 今年8月，媒体曝光谷歌〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蜻蜓计划.html" target="_blank">
  蜻蜓计划
 </a>
 〞后，谷歌多名高管在不同场合表示，该计划尚处于早期探索阶段，离正式推出还很远。
 <br/>
 <br/>
 但美国新闻网站The Intercept，10月9号披露了一份谷歌内部会议纪录，谷歌搜索引擎负责人戈麦斯7月18日对员工说，他希望在6到9个月之内，或者更快，就能推出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审查版.html" target="_blank">
  审查版
 </a>
 搜索引擎。
 <br/>
 <br/>
 戈麦斯称，该项目〝非常重要〞，因为中国有庞大的10亿潜在用户。谷歌管理层已要求工程师加快开发，一旦中共批准，希望能迅速推出。
 <br/>
 <br/>
 联邦参议员 Maggie Hassan(09/26)：你们是否起码在考虑，打造一个工具，可以给极权政府用来压制人权。
 <br/>
 <br/>
 联邦参议员 克鲁兹(09/26)：在你看来，中共是否在审查中国人民？
 <br/>
 谷歌首席隐私官 Keith Enright：作为谷歌的隐私代表，对这个问题，我没有合适的观点。
 <br/>
 <br/>
 The Intercept报导说，谷歌审查版搜索引擎，将配合中共过滤屏蔽〝人权〞〝学生抗议〞等搜索结果。并可能将用户搜索纪录，和其电话号码绑定。
 <br/>
 <br/>
 而中共去年实施的《网安法》规定，互联网公司必须将用户数据储存在中国，如果当局以执法名义索取用户数据，网络商必须配合。
 <br/>
 <br/>
 联邦参议员 Maggie Hassan(09/26)：这会让中共政府更容易跟踪和监控民众，并且便于中共发展所谓〝社会信用评级〞，它能任意禁止民众旅行或入学。
 <br/>
 <br/>
 谷歌2017年秘密启动〝蜻蜓计划〞，约有300名员工参与，对核心团队以外的员工严格保密。
 <br/>
 <br/>
 直到媒体曝光后，1000多位
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌员工.html" target="_blank">
  谷歌员工
 </a>
 致函公司高层，另有7名谷歌员工辞职，抗议谷歌缺乏道德责任和透明度。
 <br/>
 <br/>
 辞职抗议的谷歌前科学家Jack Poulson：谷歌的人工智能原则，承诺公司将不会设计或使用，那些与国际公认令人担忧的人权问题相抵触的技术。目前已明确跨过了这些红线。
 <br/>
 <br/>
 14个人权组织也致信谷歌说，中共是对互联网审查和监视最严重的政权之一，当局常常只凭网上言论，就逮捕或拘押维权人士和记者。呼吁谷歌继续恪守〝不作恶〞的原则，取消蜻蜓计划。
 <br/>
 <br/>
 美国副总统 彭斯：谷歌应立即停止开发〝蜻蜓〞计划，该计划将加强共产党的审查，并损害中国消费者的隐私。
 <br/>
 <br/>
 新唐人记者林澜纽约报导
</div>
