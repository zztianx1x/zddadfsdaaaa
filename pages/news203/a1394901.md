### 华邮记者失踪引关注 川普要沙特交代
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】沙特阿拉伯知名记者兼专栏作家卡舒吉上周前往沙特驻土耳其领事馆办理结婚文件，其后神秘失踪，引起各界关注。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）周三（10日）表示，他已就此事要求沙特作出答覆，并要美方对此事追根究柢。
 <br/>
 <br/>
 路透社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周三表示，美国除与土耳其密切合作，以确认到底发生什么事，也要求沙特提供讯息。
 <br/>
 <br/>
 川普告诉记者：〝我们要求知道全部的内情，我们想知道发生什么事。对我们、对白宫而言，这个情况非常严重。我想知道发生什么事，我们跟土耳其密切合作，我认为我们会查个水落石出。〞
 <br/>
 <br/>
 川普形容情况不是很乐观，但拒绝透露与沙特高层的对话内容。他指没有人知道确实发生了什么事，但希望卡舒吉（Jamal Khashoggi）还活着。他还致电慰问卡舒吉的土耳其籍未婚妻成吉兹（Hatice Cengiz），希望邀请她到访白宫。
 <br/>
 <br/>
 与此同时，白宫表示，国务卿蓬佩奥（Mike Pompeo）与沙国王储沙尔曼（Mohammed bin Salman）通了电话，要求其提供卡舒吉的讯息。
 <br/>
 <br/>
 美国国家安全顾问波顿（John Bolton）、白宫高层幕僚库什纳（Jared Kushner）9日也分别与沙尔曼进行了通话。白宫发言人桑德斯（Sarah Sanders）声明说：〝蓬佩奥接着也致电王储，重申美国要求讯息的立场。〞
 <br/>
 <br/>

 </center>
 <br/>
 59岁的卡舒吉毕业于美国印第安纳州立大学，一向大胆公开批评沙特阿拉伯政府的政策，一直担心自己会遭受报复，一年前逃离了沙特。
 <br/>
 <br/>
 上周二（10月2日），卡舒吉前往沙特驻土耳其领事馆办理结婚文件，其后神秘失踪。土耳其当局相信，卡舒吉在领事馆内被绑架、谋杀；沙特方面则表示绑架指控毫无根据。沙特官员表示，卡舒吉已离开领事馆，但在领事馆外等他的成吉兹表示，卡舒吉后来就不曾再出现。
 <br/>
 <br/>
 亲土耳其政府传媒《萨巴赫报》周三（10日）亦公开由15名沙特人组成的〝神秘暗杀队〞成员相片，再把矛头指向沙特。
 <br/>
 <br/>
 《萨巴赫报》称，暗杀团队于卡舒吉失踪当日，分乘两架私人飞机在伊斯坦布尔入境，然后分别入住两间酒店，同日离国。当地电视台还拍到一辆黑色小型货车离开领事馆，驶往沙特领事住处，猜测卡舒吉可能身处其中。

 <br/>
 <br/>
 美国《纽约时报》则引述土耳其官员指，是沙特最高级的皇家法院下令杀死卡舒吉，沙特为此派出暗杀队，当中包括解剖专家，或负责肢解卡舒吉。该官员透露，暗杀队带着骨锯入境。
 <br/>
 <br/>
 但英国《每日邮报》报导，有沙特皇室消息人士称，卡舒吉只是被4名沙特官员以黑色房车及白色小型货车掳到机场，再由私人飞机经迪拜飞往沙特首都利雅得，目前扣留在当地。
 <br/>
 <br/>
 卡舒吉的未婚妻成吉兹周二（9日）撰文，恳请川普夫妇介入调查。
