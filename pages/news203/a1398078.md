### 最新民调：库默仍领先 但比分缩水
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】根据11月6日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 前最新的一次民意调查，在争夺纽约州长位置的两个主要候选人之间，民主党的竞选连任的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库默.html" target="_blank">
  库默
 </a>
 仍然领先，但是领先比分却大大缩水，和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马利纳罗.html" target="_blank">
  马利纳罗
 </a>
 的距离为：49比36。
 <br/>
 <br/>
 根据〝锡耶纳大学研究所〞（Siena College REserch Institue）4日的民调结果，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_库默.html" target="_blank">
  库默
 </a>
 的支持率大大下降，和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马利纳罗.html" target="_blank">
  马利纳罗
 </a>
 的支持率相比，从上个月的50比28下降到49比36，也就是说，后者的支持率上升了9个百分点。
 <br/>
 <br/>
 该调查发现，马利纳罗在本党的支持率上，以及在独立党的选民以及下州农村白人的选民中赢得了更多的好感。而且，他在纽约上州以46比36的支持率高于库默；而在纽约市以14比77的比例输给库默。
 <br/>
 <br/>
 同时，在库默任职的八年中，民调第一次出现了支持他的人数和反对他的人数反转的情况，即更多选民（49%）不看好他，而只有45%看好他。这项调查在上个月还是以支持他的人占大多数（50%比46%）。另外，多数人（47%）纽约人认为，现在纽约州的方向是错误的，只有40%的人认为纽约州处在正确的轨道上。
 <br/>
 <br/>
 ──转自《大纪元》
 <br/>
 <br/>
