### 18岁中国留学生纽约跳轨死亡 原因不明
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约大学.html" target="_blank">
  纽约大学
 </a>
 一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国留学生.html" target="_blank">
  中国留学生
 </a>
 ，日前突然跳轨，当场身亡。当地警方8日称，案件目前仍在调查中，具体原因不明，这是今年公开报导中已知的纽约大学第2起中国学生自杀案。
 <br/>
 <br/>
 据美媒10月9日报导，这名跳轨死亡的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国留学生.html" target="_blank">
  中国留学生
 </a>
 今年18岁，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纽约大学.html" target="_blank">
  纽约大学
 </a>
 一年级本科新生，10月2日中午，在曼哈顿东村地铁L线第一大道，向北行驶的列车入站时，该学生突然从月台上跳入轨道，被列车迎面撞击当场身亡。
 <br/>
 <br/>
 纽约大学发言人碧咸（John Beckman）发表声明称，该学生于2日中午12时9分去世，校方会提供学生心理辅导，为有需要学生提供帮助和支持，鼓励学生注意释放压力、排解消极情绪。
 <br/>
 <br/>
 纽约警方10月8日表示，事发时，这名留学生正在与人通电话，目前此案仍在调查中，具体原因不明。
 <br/>
 <br/>
 5月份，纽约大学医学院也有一名华裔女生，在宿舍内上吊自杀。
 <br/>
 <br/>
 该校亚裔健康研究中心教授权西蒙娜（Simona C. Kwon）表示，亚裔家长对学生的期望很高，导致亚裔学生学业压力大，留学生面临文化冲击时也易产生消极情绪。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
