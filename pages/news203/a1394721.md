### 飓风迈克尔周三登陆 佛州到北卡均波及
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】观众朋友大家好，欢迎收看新唐人全球新闻。上个月，美国东岸的南卡、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北卡.html" target="_blank">
  北卡
 </a>
 等州刚刚遭受飓风〝佛罗伦斯〞的袭击，这周三，飓风〝迈克尔〞又要从
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 的墨西哥湾一侧登陆。当局已下令疏散沿岸居民。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佛州.html" target="_blank">
  佛州
 </a>
 州长斯科特周二表示,
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_飓风迈克尔.html" target="_blank">
  飓风迈克尔
 </a>
 破坏性极强，该州67个县中，有35个县已进入紧急状态。
 <br/>
 <br/>
 佛州州长斯科特：〝飓风迈克尔是一场可怕的风暴。飓风迈克尔预计将成为几十年来袭击佛罗里达州狭长地带最具破坏性的风暴。风暴将危及生命，极其危险。〞
 <br/>
 <br/>
 美国国家海洋和大气管理局官员表示,目前为二级飓风的迈克尔风速是100英里，周三登陆时，迈克尔可能演变为三级飓风，风速达到110英里，并往北扫过其它地区。
 <br/>
 <br/>
 美国海洋和大气管理局预报员 Ken Widelski：〝飓风引发的风暴潮将带来重大影响，特别是佛州大弯地区。然后飓风将迅速向北穿过乔治亚州，吹倒树木，毁坏输电线，供电中断。然后飓风将于周四和周五穿过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北卡.html" target="_blank">
  北卡
 </a>
 和南卡。北卡将出现强降雨。〞
 <br/>
 <br/>
 阿拉巴马州和乔治亚州也相继宣布进入紧急状态。
 <br/>
 <br/>
 在佛州狭长地带和大弯地区，以及阿拉巴马州东南部和乔治亚州南部部分地区，约370万人收到飓风警报。另有310万人收到热带风暴警报。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周三批准了佛州进入紧急状态，随后联邦层面的救援力量将加强当地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抗灾行动.html" target="_blank">
  抗灾行动
 </a>
 。川普并发推文提醒当地居民说：〝你们必须留意州和地方官员的指令，要做好准备，小心并保证安全！〞
 <br/>
 <br/>
 新唐人记者王羽音综合报导
</div>
