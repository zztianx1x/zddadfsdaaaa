### 刘强东性侵后私求和解 路透曝案件久拖不决真正原因
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月22日讯】中国京东集团创始人刘强东在美国涉嫌性侵案细节不断曝光。日前路透社引述
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受害女生口供.html" target="_blank">
  受害女生口供
 </a>
 指，案发后刘强东曾试图通过中间人与她
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_私下和解.html" target="_blank">
  私下和解
 </a>
 。报导指，受害人案发初期的犹豫，成为检方是否提出公诉的一个障碍。
 <br/>
 <br/>
 路透社11月22日报导，根据受害女生提供的口供，8月30日晚的宴会结束后，这位女生乘坐刘强东提供的轿车离开，在车内刘强东就开始动手动脚，并试图脱下她的衣服，尽管她不断哀求他不要这样做。当时还有司机和刘强东的一名女助理爱丽丝．张(Alice Zhang)在车内。
 <br/>
 <br/>
 报导引述该女生和知情人士称，刘强东将女学生送回公寓后，也进入公寓并停留了大约4个小时，还洗了澡，赤裸地躺在女学生的床上。其间，刘强东还一度试图将女生拉进淋浴间一起洗澡。
 <br/>
 <br/>
 尽管遭到女学生竭力抵抗，刘强东最终还是将她压在床上实施了性侵。
 <br/>
 <br/>
 报导指，该女生保留了带有刘强东精斑的床单作为证据。
 <br/>
 <br/>
 该女生的口供还指，事发第二天，刘强东的另外一名助理薇薇安．杨(Vivian Yang)联系了她，希望谈谈〝昨天发生的事情〞。
 <br/>
 <br/>
 女生对薇薇安・杨表示，自己希望寻求法律的保护。而薇薇安．杨提议，让该女生先和卓越集团董事局主席李华谈一谈，因为双方都信任〝李老板〞。
 <br/>
 <br/>
 报导指，李华和刘强东一起参加了明尼苏达大学针对中国高管的工商管理博士课程，但没有提及该女生和李华是什么关系。
 <br/>
 <br/>
 而根据此前港媒消息，该女生是一名中国籍留美学生，其父是湖北省一名富商。
 <br/>
 <br/>
 该女生口供指，她拒绝了薇薇安．杨的提议，表示只想在有朋友在场的情况下和刘强东谈谈。薇薇安．杨最终同意在当晚10:30分让刘强东在明尼苏达大学卡尔森管理学院与女生会面。但刘强东抵达会面地点后，就被等待的警察逮捕。
 <br/>
 <br/>
 报导还称，美国亨内平县检察官办公室仍未决定是否对刘强东提起公诉，检方正在考虑的问题包括：双方就当晚事件存在分歧的解释，警方最初认为没有犯罪发生，受害女子一开始对指控刘强东感到犹豫。
 <br/>
 <br/>
 刘强东律师吉尔．布利斯波利斯(Jill Brisbois)坚持刘强东的〝清白〞。而受害女子的律师威尔．弗罗林(Wil Florin)则表示，刘强东实施性侵〝显而易见〞，并称一名强奸案受害者最初出现不情愿、恐惧以及困惑的情绪波澜很正常。
 <br/>
 <br/>
 一些法律专家称，受害者最初的犹豫可能会令检察官放弃起诉。另一些专家则认为，检察官应该仍可以提出指控，因为在专家证人的帮助下，完全可以让陪审员相信，受害者最初的犹豫是因为害怕报复、逃避和感到羞耻等原因。
 <br/>
 <br/>
 根据此前美媒的报导，受害女在案发初期一度不愿意对刘强东直接提出指控。
 <br/>
 <br/>
 在案发当晚，她的一名朋友报警后，警察赶到她的公寓中。但她的第一反应是〝息事宁人〞，导致警察没有将当时还在公寓的刘强东逮捕，而只是让刘离开公寓。
 <br/>
 <br/>
 随后，校方获悉事件后感到事态严重，并再次报警，导致刘强东在次日晚上被抓。但当时该女生还在犹豫是否指控刘强东。直到刘强东获得假释后，该女生才在朋友劝说下决定诉诸法律。不过，刘强东已经迅速离开了美国。
 <br/>
 <br/>
 根据媒体披露的微信信息，该女生害怕遭到刘强东的报复。她对朋友说：〝不要低估他（刘强东）的能量〞。她表示，自己当时只想得到刘强东的一个道歉。
 <br/>
 <br/>
 据港媒披露，案发后中共外交部介入，威逼该女生回国。刘强东回国后，曾在湖北省委高层陪同下高调露面，外界猜测或藉此向女方家族施压。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
