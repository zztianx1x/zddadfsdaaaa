### 阿拉斯加7级强震仅2房塌 法规确保建筑牢固
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】上周五，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿拉斯加.html" target="_blank">
  阿拉斯加
 </a>
 南部距离首府安克拉治不远地区发生7级大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 后，当地道路被地震扭曲断成碎片的图片在网络上热传，触目惊心。连日来该地区仍不断有余震发生，但迄今为止仅有2栋房屋报案倒塌，尚无人员伤亡的报告。地震危害安全委员会人员解释说，这全靠当地执行了一套严格的建筑兴建标准，让所有的房屋都建得够牢固。
 <br/>
 <br/>
 强震对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿拉斯加.html" target="_blank">
  阿拉斯加
 </a>
 道路造成的破坏令人触目惊心。尽管当地人早已对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地震.html" target="_blank">
  地震
 </a>
 习以为常，但11月30日上午8点30分左右大地开始剧烈摇晃时，仍让老一辈人直呼〝好害怕〞。
 <br/>
 <br/>
 在地震的强烈摇晃中，主要联外道路瞬间断成好几节，树木倒塌、超市货架上商品散落一地，不过除此之外，并无特别灾情。迄今当地市政府仅接获4起火警、2起房舍倒塌报案。
 <br/>
 <br/>
 美国阿拉斯加州（Alaska）7.0的大地震上发生后，在之后的4天里，该地区又陆续发生了上千起大大小小的余震。震区大马路严重扭曲损毁，网络上贴出的画面令人震惊。
 <br/>
 <br/>
 尽管这次地震的震级很高，但实际给当地居民带来的损失伤害却相当轻微。警方称，迄今为止仅接获2起房屋倒塌的报警和4起火警消息。
 <br/>
 <br/>
 安克雷奇市市长伊森（Ethan Berkowitz）也表示，强震过后，虽然造成道路严重坍塌，却没有发生灾难性的建筑物倒塌情况。一般出现的是房屋的墙壁开裂，或者水管爆裂等。迄今没有传出有人员死亡的消息。
 <br/>
 <br/>
 发生了高达7级的地震，通常都会带来十分严重的损失和伤亡情况，为什么这次的灾情却如此轻微呢？有专家指出，在这次的强震中，损失不严重，全因当地实施了严格的建筑法令规范。
 <br/>
 <br/>
 阿拉斯加大学土木工程系教授杨（Joey Yang）在接受《纽约时报》的采访时表示，自从1964年该地区发生超级大地震以来，阿拉斯加早对地震有所准备。当地政府机构严格把关建筑物建造流程，建商必须经过严苛的标准检验才能在这里修建房屋。
 <br/>
 <br/>
 国家地理杂志（National Geographic）也引述自美国地质调查所地球物理学者卡鲁索（Paul Caruso）指出，〝严格的建筑法规将地震灾害降到最低。〞
 <br/>
 <br/>
 据《旧金山纪事报》的报导，阿拉斯加地震危害安全委员会成员斯特林（Sterling Strait）也指出：〝阿拉斯加州使用国际建筑规范是最标准的，该规范严格要求建筑物的设计，加强结构连接，例如梁和柱，以抵抗振动造成的损坏。〞
 <br/>
 <br/>
 据公开的资讯，阿拉斯加位处太平洋板块与北美洲板块交界地带，板块运动频繁，导致这个区域经常会发生规模大小不同的地震。而1964年曾发生高达9.2级的超强地震，并伴随发生了海啸，总共夺走131条人命，让老一辈阿拉斯加居民至今仍记忆犹新。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：范铭）
