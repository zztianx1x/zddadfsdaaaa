  美国司法部
 </a>
 周四针对马来西亚主权基金〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一马发展公司.html" target="_blank">
  一马发展公司
 </a>
 〞贪污案发布声明，宣布起诉涉及该案的两名前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高盛银行家.html" target="_blank">
  高盛银行家
 </a>
 。
 <br/>
 <br/>
 布鲁克林美国检察官办公室通告，前高盛亚洲合伙人、蒂姆・莱斯纳承认
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共谋洗钱.html" target="_blank">
  共谋洗钱
 </a>
 ，及违反美国海外反腐败法的指控，同意偿还约4,400万美元的非法所得。
 <br/>
 <br/>
 另一名被指控的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高盛银行家.html" target="_blank">
  高盛银行家
 </a>
 黄罗杰已在马来西亚被捕，据悉将引渡到美国。
 <br/>
 <br/>
 被指控的金融家刘特佐尚未被捕，但马来西亚当局已对其发出逮捕令。
 <br/>
 <br/>
 美国检方表示，刘特佐在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一马发展公司.html" target="_blank">
  一马发展公司
 </a>
 〞和高盛及其他公司的多起交易中，起到中间人作用。刘特佐发言人称，他坚持自己无罪。
 <br/>
 <br/>
 美国检方指控刘特佐和其他两名前高盛高管共谋洗钱，用来为高盛谋取生意时作贿赂，以及个人奢侈消费。
 <br/>
 <br/>
 检方还表示，高盛为〝一马发展公司〞工作时，收取了约6亿美元费用。
 <br/>
 <br/>
 马来西亚前总理纳吉布2009年创立〝一马发展公司〞。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国司法部.html" target="_blank">
  美国司法部
 </a>
 表示，该公司挪用资金约45亿美元。目前至少六个国家在调查此事。
 <br/>
