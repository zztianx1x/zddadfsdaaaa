### 应川普要求 美司法部长终于辞职
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】周三（11月7日），美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部长.html" target="_blank">
  司法部长
 </a>
 塞申斯（Jeff Sessions）应
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统的要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 。川普宣布这一消息并委任司法部长的临时人选。这是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 后川普对内阁的首次重大调整。
 <br/>
 <br/>
 周三下午，塞申斯向白宫幕僚长凯利（John Kelly）递交了 辞呈。他承认，他是应
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 的要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辞职.html" target="_blank">
  辞职
 </a>
 的。川普周三上午向他提出了辞职要求。
 <br/>
 <br/>
 塞申斯递交辞呈后，川普很快在推特上发布了这一消息，同时宣布代理人选。
 <br/>
 <br/>
 川普决定，任命塞申斯的幕僚长马修．惠特克（Matthew G. Whitaker）接替塞申斯，出任代理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_司法部长.html" target="_blank">
  司法部长
 </a>
 ，并表示，之后会提名正式担任司法部长的人选。
 <br/>
 <br/>
 虽然对塞申斯极为不满，川普仍礼貌性地对他的工作表示感谢，并祝他今后一切顺利。
 <br/>
 <br/>

 <br/>
 <div style="clear:both;display:block;">
 </div>
 <br/>
 周三中午，川普就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 问题在白宫举行了长约90分钟的新闻发布会，期间虽然一再被问及内阁的人事安排及塞申斯的去留问题，但川普没有给予明确回答，只表示在中期选举后调整内阁人事是很正常的，具体安排须待决定后再向外界通报。
 <br/>
 <br/>
 不过，新闻发布会结束后刚刚一小时左右，川普便宣布了塞申斯辞职的消息。
 <br/>
 <br/>
 塞申斯是川普2016年竞选团队的成员，川普当选后，任命他担任自己的第一任司法部长。但在后来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_通俄门.html" target="_blank">
  通俄门
 </a>
 调查中，塞申斯出于自身利益的考虑而宣布回避，因而监督通俄门调查的任务便落到了司法部副部长罗森斯坦（Rod Rosenstein）的头上。
 <br/>
 <br/>
 罗森斯坦后来任命反对川普的前联邦调查局（FBI）局长穆勒（Robert Mueller）出任特别检察官，继而针对川普选战开始了旷日持久的调查，极大地干扰了川普的施政。
 <br/>
 <br/>
 川普对于塞申斯回避通俄门调查深感不满，他多次批评塞申斯，并表示自己实质上没有司法部长。
 <br/>
 <br/>
 马修．惠特克曾在小布什总统（George W Bush）任内供职司法部，担任爱荷华州检察官。2014年，他还曾以共和党人身份参加爱荷华州的参议员竞选。
 <br/>
 <br/>
 （记者李佳欣综合报导）
