### 重返中国生变？谷歌CEO:不清楚是否需做中国搜索产品
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】美国互联网搜索引擎巨头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 的CEO
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桑达．皮采.html" target="_blank">
  桑达．皮采
 </a>
 (Sundar Pichai)日前接受了美国媒体专访，针对该公司近期遭遇的一系列挑战作出了回应。在谈及谷歌公司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重返中国.html" target="_blank">
  重返中国
 </a>
 市场的话题时Pichai称，自己现在也不清楚是否需要做中国搜索产品。
 <br/>
 <br/>
 近期，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷歌.html" target="_blank">
  谷歌
 </a>
 遭遇了一系列的重大挑战：员工对性骚扰感到愤慨，罢工潮规模浩大；秘密打造配合中共审查搜索引擎的消息引发外界的质疑与担忧，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重返中国.html" target="_blank">
  重返中国
 </a>
 市场迷雾重重；社交网络谷歌+数据洩露；被迫停止发展军事级AI能力。针对这些问题，《纽约时报》11月9日对谷歌首席执行官Sundar Pichai进行了一次专访。
 <br/>
 <br/>
 在专访过程中，《纽时》的记者问及谷歌是否正在考虑通过搜索引擎重返中国市场的问题。Pichai回答说：〝我们其实在许多有审查制度的国家开展业务。当我们遵循‘被遗忘权’法律时我们需要审查搜索结果，因为我们遵守法律。〞
 <br/>
 <br/>
 Pichai这里所说的法律应该指的是欧盟1995年在有关数据保护的法律中提出了〝被遗忘权〞的概念，承认任何公民可以在其个人数据不再需要时提出删除要求。
 <br/>
 <br/>
 针对有关谷歌正在研究可以配合中共审查的搜索引擎‘蜻蜓’的传闻，Pichai回应称：〝谷歌致力于为中国用户提供服务，无论采取何种形式。实际上我都不知道答案，我甚至不清楚在中国搜索是我们今天需要做的产品。〞
 <br/>
 <br/>
 上述问答的英文原文如下：How do you approach this in China, where Google is considering returning to the market with a search engine?
 <br/>
 <br/>
 Pichai：One of the things that’s not well understood, I think, is that we operate in many countries where there is censorship. When we follow 〝right to be forgotten〞 laws, we are censoring search results because we’re complying with the law. I’m committed to serving users in China. Whatever form it takes, I actually don’t know the answer. It’s not even clear to me that search in China is the product we need to do today.
 <br/>
 <br/>
 据公开的资讯，今年8月初，美国新闻网站〝信息〞(The Information)的一篇报导称，谷歌正在开发一个适合中共政府审查要求的新闻应用程序。该程序基本上是克隆在中国非常受欢迎的〝今日头条〞，目的是重新返回拥有7亿手机用户的中国市场。
 <br/>
 <br/>
 据称，谷歌正在开发的这个应用程序使用人工智能和算法来选编新闻，但是会采取额外的步骤，屏蔽那些不符合中共政府严格的新闻审查标准的新闻。
 <br/>
 <br/>
 网络媒体〝拦截〞（The Intercept）也爆料称，谷歌计划在中国推出代号为〝蜻蜓〞（Dragonfly）的可以过滤政治敏感内容的搜索引擎，而首席执行官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桑达．皮采.html" target="_blank">
  桑达．皮采
 </a>
 2017年12月与中共政府官员会面后加速了这一计划。
 <br/>
 <br/>
 该报导并进一步透露说，谷歌已经给中共政府的有关官员进行了演示，而最终版本预计会在未来6到9个月内推出。
 <br/>
 <br/>
 不过，中共官方媒体《证券日报》8月2日的一篇报导则否认了类似的传闻，声称有关谷歌或重返中国的传闻并不属实。
 <br/>
 <br/>
 谷歌公司的一名发言人当时曾向《美国之音》提供一份声明称，〝我们在中国提供了一些移动应用程序...以帮助中国的开发人员，并在京东等中国公司进行了大量投资。但我们不会对有关未来计划的推测加以评论。〞
 <br/>
 <br/>
 自从2000年以来，〝不作恶〞一直是谷歌的公司行为准则。2010年，谷歌因不满中国政府的审查而高调退出中国市场。谷歌的主要搜索平台与视频平台YouTube等都被中国封锁。
 <br/>
 <br/>
 然而时过境迁，谷歌现在似乎正在改变它的初心，甚至在今年4、5月份的时候，悄悄从该公司的行为准则里去掉了〝不作恶〞的说法。
 <br/>
 <br/>
 近年来，谷歌公司在新的高层领导下进行了一系列重返中国市场的尝试，包括去年在北京开设一个人工智能中心，与中共公司京东达成5亿5千万美元的交易，并与腾讯建立伙伴关系。
 <br/>
 <br/>
 今年5月，谷歌更在中国推出了一款新的智能手机文件管理器 〝Files Go 文件极客〞。这款针对中国市场开发的安卓版软件，提供了在百度、华为、腾讯、小米四家应用商店的下载链接。
 <br/>
 <br/>
 以至于来自佛罗里达州的共和党参议员鲁比奥（Marco Rubio）在看到谷歌计划帮助中国设立审查版的搜索引擎的报导后，与另外五名来自两党的议员联署了一封信函，要求谷歌公司就相关传闻做出解释。
 <br/>
 <br/>
 不过，近期有消息说，美国的〝脸书〞在杭州注册了一家独资附属公司，但不久后，网上登记资料便消失，有关新闻也遭审查屏蔽，显示中共当局撤回了批准注册的决定。
 <br/>
 <br/>
 因此也有舆论认为，美中贸易战的日益恶化，似乎也使谷歌重返中国市场的计划变得复杂化了。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：秦锋)
