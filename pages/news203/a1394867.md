### 美国会报告：中共宗教迫害  文革以来最严重
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】美国国会及行政当局中国委员会)，10月10日发布了《2018年度报告》。国会议员认为，过去一年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教信仰.html" target="_blank">
  宗教信仰
 </a>
 团体的镇压是文革以来最严重的。
 <br/>
 <br/>
 CECC主席 美国联邦参议员Marco Rubio：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 统治下，中国政治系统的根本专制属性没有改变，实际上变得更野蛮。中国人权状况今年再度恶化。〞
 <br/>
 <br/>
 美国国会及行政当局中国委员会发表2018年度报告，谈到中国人权法制状况在去年的变化趋势，包括对新疆维吾尔自治区少数民族进行前所未有镇压；共产党对于政府、社会和商业的控制大幅增加；以及越来越多地利用技术作为压制工具。
 <br/>
 <br/>
 报告还记录中共加剧对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宗教信仰.html" target="_blank">
  宗教信仰
 </a>
 自由的侵犯、恐吓维权人士和律师、镇压言论自由等恶行。
 <br/>
 <br/>
 CECC共同主席美国联邦众议员Chris Smith：〝中共政府加强对所有宗教信仰的镇压，是文革以来最严重的。〞
 <br/>
 <br/>
 报告谈到，中共继续严酷打压法轮功，并对他们施以骚扰、任意拘留、起诉、虐待及酷刑。报告中还说，国际组织担忧，法轮功学员器官被继续用于器官移植。对于中共卫生官员声称，器官获取系统已按照国际标准进行改革，医疗专业人士和国际组织也提出质疑，理由是中国器官移植等待时间很短以及移植数据存在差异。
 <br/>
 <br/>
 CECC共同主席美国联邦众议员Chris Smith：〝我们把强摘器官纳入人口贩卖报告，这意味着可对这一邪恶政策施加制裁。这很猖獗，尤其是针对中国的法轮功学员。〞
 <br/>
 <br/>
 报告向美国国会和行政当局提出多项行动建议，包括把人权纳入整体双边关系，向践踏人权官员问责；及把宗教自由作为外交优先等。
 <br/>
 <br/>
 CECC共同主席美国联邦众议员Chris Smith：〝我们呼吁美国政府，我们已有《全球马格尼茨基人权问责法》，可以针对犯仇恨罪的个人，不会给你签证，你不能来美国，你也不能和美国做生意。〞
 <br/>
 <br/>
 上周，美国副总统彭斯针对美中关系发表重要演讲，全面揭露中共对内压制、对外渗透扩张的行径。议员们相信，目前状况已达转折点，川普政府将会采取有效行动。
 <br/>
 <br/>
 新唐人记者王凯迪美国首都华盛顿报导
</div>
