### 卡瓦诺确认后还有中期 左右之争更激烈
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】10月6日，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在堪萨斯州举行的集会上，庆贺
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 成功就任最高法院大法官的同时，继续提醒选民
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 十分关键。目前已有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左派.html" target="_blank">
  左派
 </a>
 人士提出，如果民主党未来控制国会多数，将考虑弹劾最高法院保守派法官。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （10.06.2018）：今天我站在你们面前，是满载着胜利，为我们的国家，为我们的人民，以及我们挚爱的宪法。
 <br/>
 <br/>
 周六，美国参议院以50-48投票确认川普提名的最高法院大法官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡瓦诺.html" target="_blank">
  卡瓦诺
 </a>
 ，卡瓦诺并随即在高院宣誓入职。投票几乎以党派划线。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共和党.html" target="_blank">
  共和党
 </a>
 中，除了一人缺席，只有阿拉斯加州参议员穆尔科斯基(Lisa Ann Murkowski)投票反对（后来改成投〝出席〞票）；而西弗吉尼亚州参议员乔・曼钦(Joe Manchin)，成为唯一一个投票支持卡瓦诺的民主党人。他来自川普选区，即将在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 时面对共和党候选人的强力挑战。
 <br/>
 <br/>
 美国总统 川普（10.06.2018）：我也想感谢了不起的共和党参议员们，拒绝在民主党无耻的，针对（美国）政治和（卡瓦诺）个人的破坏活动。从卡瓦诺被提名之日起，激进民主党就发起了一场无耻的运动，来抵抗、阻碍、推迟、破坏和摧毁。
 <br/>
 <br/>
 超过两周以来，卡瓦诺因为充满争议的性侵指控而使对他的提名表决一再推迟，还伴随着一些
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_左派.html" target="_blank">
  左派
 </a>
 团体在国会山内外激烈的喧嚣抵制，但对他的指控最终没有被证实。然而这过程中遇到的种种抵制，共和党认为是民主党故意制造的麻烦。
 <br/>
 <br/>
 共和党联邦参议员 林吉・葛瑞安姆（09.27.2018）：这是我从政以来遇到的最不道德的骗局。
 <br/>
 <br/>
 不过，左派的抵制对共和党来说也许是好事。
 <br/>
 <br/>
 参议院多数党（共和党）领袖 麦康奈尔（10.06.2018）：昨天我跟我的两个政治顾问谈话，我说（左派）这些人，他们的那些伎俩，为共和党州的激烈选战点燃了斗志，我们为此非常兴奋。他们做到了我们还不知道怎么做的事，就是点燃了我们选民的（投票）热情。
 <br/>
 <br/>
 据《纽约时报》报导，一些左派已开始动员，几乎要毫无底限地冲击最高法院保守派。他们提出两点：1）如果民主党在今年中期选举后能控制国会，他们将考虑弹劾卡瓦诺；2）增加最高法院大法官人数，以选入更多左派法官，当然实现这一点，还要在2020年拿下白宫。但以上任何一点要做到都相当难。
 <br/>
 <br/>
 川普总统并在多次选战中，提醒共和党选民，积极参与投票，阻止左派的政治野心。
 <br/>
 <br/>
 美国总统 川普（10.06.2018）：11月6日，你将有机会阻止激进的民主党人，我们需要更多共和党，需要更多共和党。
 <br/>
 <br/>
 卡瓦诺替代的是今年6月27日宣布退休的最高法院大法官肯尼迪，此人一直被认为是保守派法官中的摇摆票。而卡瓦诺是坚定保守派，他的入职，标志着最高法院9名大法官形成稳定的、保守派占多数的5-4格局，最高法院正式〝右转〞。外界认为，这对未来的堕胎权、同性恋权利、拥枪权等案件的判决，都有重大影响。
 <br/>
 <br/>
 美国总统 川普（10.06.2018）：你们还会有另一个最高法院大法官空缺，如果你们选了错误的人当选，事情就可能被改变。
 <br/>
 <br/>
 目前，最高法院有两名左派法官年龄已经相当大，他们的职位有可能出现空缺。一名是85岁的金斯伯格(Stephen Breyer)，另一名是80岁的布雷耶(Ruth Bader Ginsburg)。
 <br/>
 <br/>
 新唐人记者李大宇综合报导
</div>
