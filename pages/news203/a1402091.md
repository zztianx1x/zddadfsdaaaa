### 中共关押200万维族人 美参议员吁贸易谈判提人权
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】美国参议院下属机构再度举行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_听证会.html" target="_blank">
  听证会
 </a>
 ，讨论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权问题.html" target="_blank">
  中国人权问题
 </a>
 。来自两党的多名参议员呼吁关注新疆大规模关押维族人情况，有参议员呼吁美中贸易谈判应加入人权议题。
 <br/>
 <br/>
 当地时间12月4日，美国参议院外交事务委员会下属的亚太地区委员会举行了第三场有关中国问题的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_听证会.html" target="_blank">
  听证会
 </a>
 。这次的听证会焦点议题是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人权问题.html" target="_blank">
  中国人权问题
 </a>
 。
 <br/>
 <br/>
 在这次听证会上，美国国务院人权暨劳工局副助理国务卿巴斯比（Scott Busby）表示，从去年4月以来，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新疆再教育营.html" target="_blank">
  新疆再教育营
 </a>
 中被关押的维族人和哈萨克族人，可能比外界估计的100万人还更多，最终数字或多达200万，而其中大部分人都没被刑事起诉。
 <br/>
 <br/>
 巴斯比特别提到中国大陆维权律师王全璋、江天勇以及创办〝六四天网〞的黄琦等人被关押和虐待的案件，并指出他们是为维护其他人的权利而遭中共迫害。他希望川普（特朗普）政府能在最高领导人层面就人权问题向中方施压。
 <br/>
 <br/>
 亚太地区委员会主席、共和党参议员贾德纳（Cory Gardner）也在听证会上表示，中国的人权和宗教信仰自由状况，在过去数年里明显恶化。
 <br/>
 <br/>
 此外，民主党参议员马基（Ed Markey）也特别对新疆维族人被关押在再教育营的问题表示关切。
 <br/>
 <br/>
 香港东网报导称，有多名美国议员要求川普政府在中美贸易谈判中加入人权议题。
 <br/>
 <br/>
 为此，美国务院负责东亚事务的副助理国务卿石露蕊（Laura Stone）对媒体表示，不清楚川普总统有否在会面中向习近平提出过相关议题，但他保证国务院会在适当场合向中共施压。
 <br/>
 <br/>
 （记者李明报导/责任编辑：秦锋）
</div>
