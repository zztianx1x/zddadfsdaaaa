### 留墨西哥等身分？川普严限〝大篷车〞入境
------------------------

<div class="wysiwyg">
 【新唐人2018年11月25日讯】先来关注中美洲〝大篷车〞移民问题。近期，即将在12月1日上任的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 内政部长科德罗，提出了前后矛盾的说法，让外界对美墨两国如何处置中美洲〝大篷车〞移民产生不解。目前，已经有约5000人抵达墨西哥边境城市蒂华纳。
 <br/>
 <br/>
 在12月1号即将走马上任的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 内政部长科德罗（Olga Sanchez Cordero）周六曾向《华盛顿邮报》表示，墨西哥新政府与美国达成协议，将允许大篷车移民在等待美国的身分申请结果时，暂时留在墨西哥。但是中期和长期目标是终结这种〝大篷车〞式的移民流动。
 <br/>
 <br/>
 但很快，科德罗又矢口否认，称没有与美国达成任何相关协议，他的表态前后矛盾，但还没有做出解释。
 <br/>
 <br/>
 美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 周日连发推文，重申移民要通过合法途径进入美国。他说：南部边境的大篷车移民不允许进入美国，直到他们个人的移民申请获得法院批准。
 <br/>
 <br/>
 川普并指出，将执行〝抓到就拘留〞的政策，而不是〝抓了就放〞，而且等待身分要再墨西哥等。此外，如果有极端情况发生，川普说他会关闭美国南部边境。
 <br/>
 <br/>
 周日，大批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 从墨西哥边境城市蒂华纳（Tijuana）攀越栅栏、墙壁、山丘等，试图穿越美墨边境。
 <br/>
 <br/>
 美国海关与边境保护局（US Customs and Border Protection）周日关闭了蒂华纳到美国圣地亚哥圣伊西德罗（San Ysidro）的边境入口，包括公路和步行桥。墨西哥警方出动防暴警察，帮助维持边境秩序。
 <br/>
 <br/>
 新唐人电视安心综合报导
