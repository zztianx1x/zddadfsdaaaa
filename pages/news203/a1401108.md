### 北京有牌不敢出？中共大使罕见承认：非常危险
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】自
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 开打以来，中共节节败退，局势发展远超预期。外界关注中共是否动用〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融核弹.html" target="_blank">
  金融核弹
 </a>
 〞抛售
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国债.html" target="_blank">
  美国国债
 </a>
 。中共驻美大使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔天凯.html" target="_blank">
  崔天凯
 </a>
 近日罕见的承认，这款武器过于危险，副作用太大，不会被中共提起。
 <br/>
 <br/>
 11月27日，路透社报导说，中共驻美大使
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔天凯.html" target="_blank">
  崔天凯
 </a>
 在接受采访时，当被问到中美贸易关系持续恶化的情况下，会不会卖出手中持有的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国债.html" target="_blank">
  美国国债
 </a>
 ，或是减少对美国国债的购买时，他说：这非常危险，像是在玩火，中共方面应该不会有人考虑这种方式，它可能会引起反效果。
 <br/>
 <br/>
 他还说，中共持有的巨额美国国债是两国经济相互依存的标志，想切断这种关系几乎是不可能的。
 <br/>
 <br/>
 据美国统计，截止到9月末，中国持有的美国国债的规模达到了11,500亿美元，是所有海外持有者中最多的一个，比去年的11,900亿减少了一些。
 <br/>
 <br/>
 目前持续近一年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 还在持续升温中，川普（特朗普）政府此前已对2,500亿中国商品课征10%到25%的惩罚性关税，未来征税的规模还可能扩大一倍。
 <br/>
 <br/>
 中美首脑11月30日将在阿根廷二十国集团（G20）峰会期间会面，就中美贸易战进行谈判。美国总统川普26日在接受采访时说，预料华府将如期于明年1月1日起，把总值2000亿美元中国进口商品的关税税率，由10%增至25%。
 <br/>
 <br/>
 面对美方的持续施压，中国经济全面下滑，美国经济呈现强劲态势。有关中共是否将动用〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融核弹.html" target="_blank">
  金融核弹
 </a>
 〞，抛售手中持有的美国国债讨论不断。
 <br/>
 <br/>
 而崔天凯在这一问题的言论更是前后矛盾，此次承认抛售美国国债太危险之前，崔天凯3月份曾暗示，必要时中共会减少购买美国国债的可能性。
 <br/>
 <br/>
 《纽约时报》当时报导说，北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抛售美债.html" target="_blank">
  抛售美债
 </a>
 的〝核选项〞没有试验场所，没有可以预测的计算结果及比例模型。如果中共采用〝核选项〞，而市场不作出反应，北京将在众目睽睽下失去影响力，如果产生超过预期的效果，又将给中国经济带来伤害。
 <br/>
 <br/>
 经济学家、国民经济研究所所长樊纲表示，中共作为美国最大的债券持有国，如果抛售美国国债，贸易战争势必将扩大成为金融战争，鉴于中国在金融方面太过脆弱，一旦贸易战发展为金融战争，中国蒙受的损失将比美国更大。
 <br/>
 <br/>
 而美国财政部长姆钦则表示，如果中共决定抛售手中持有的美债，还有很多买家，但中共将因此付出很大的代价。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
