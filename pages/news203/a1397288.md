### CNN记者诱以色列大使批川普 回答出人意料
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】周日（10月28日），美国有线电视新闻网（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN.html" target="_blank">
  CNN
 </a>
 ）在采访以色列驻美国大使德默尔（Ron Dermer）时，试图诱使他就周六发生的宾州匹兹堡（Pittsburgh）
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹太教堂.html" target="_blank">
  犹太教堂
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_枪击案.html" target="_blank">
  枪击案
 </a>
 指责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ，但结果适得其反，德默尔反而称颂川普对反犹太主义的强烈谴责。
 <br/>
 <br/>
 匹兹堡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_犹太教堂.html" target="_blank">
  犹太教堂
 </a>
 枪击事件发生后，美国自由派媒体试图如对待上周〝炸弹邮包〞案一样，将案件责任推向总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 。周日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CNN.html" target="_blank">
  CNN
 </a>
 记者布莱克威尔（Victor Blackwell）在采访德默尔时，上演了一部尴尬的丑剧。
 <br/>
 <br/>
 布莱克威尔以诱导的方式询问德默尔，川普是否对美国反犹太主义的兴起负有责任。他问德默尔，他认为在何种程度上，川普应该对其所〝滋养的〞犯罪活动负责。
 <br/>
 <br/>
 但布莱克威尔没想到，德默尔立即赞扬川普在事件发生后所发表的言论，并且指出，川普的一些家人也是犹太人。
 <br/>
 <br/>
 川普的女婿库什纳（Jared Kushner）和几个外孙都是犹太人。第一千金伊万卡（Ivanka Trump）为与库什纳结婚，也皈依了犹太教。
 <br/>
 <br/>
 德默尔不客气地告诉布莱克威尔，他成年后一生都在关注反犹太主义，但〝我从未听到过比美国总统昨天所做的声明更为强硬的〞。
 <br/>
 <br/>
 川普周六晚在伊利诺伊州竞选集会上演讲时表示：〝这种邪恶的反犹太主义攻击是对我们所有人的攻击。这是对人类的攻击。这需要我们所有人共同努力，从世界上抽出反犹太主义的可恨毒药。〞
 <br/>
 <br/>
 周六上午，一名反犹太人的白人枪手在匹兹堡的〝生命之树〞犹太教堂向参加宗教活动的人群开枪射击，并大喊〝所有犹太人必须死亡〞，造成11名人死亡，6人受伤，其中包括4名警察。
 <br/>
 <br/>
 事件发生后，美国主流媒体立即挑起又一轮仇恨，把责任推向川普，指责他鼓励暴力。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
</div>
