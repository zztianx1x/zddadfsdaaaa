### 中南海〝后院起火〞  陆企暗挺川普内幕曝光
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】美中贸易战持续升温之际，中南海〝后院起火〞。近日英媒披露，面对中共近年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革.html" target="_blank">
  改革
 </a>
 停滞不前，不少
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国民营企业.html" target="_blank">
  中国民营企业
 </a>
 纷纷倒戈，支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府对中共的贸易战
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 ，并将此视为推动中共改革的唯一外部压力来源，暗自支持川普向中共施压。
 <br/>
 <br/>
 英媒《路透》专栏作家考克斯（Rob Cox）21日撰文称，美中贸易战表面看，招致了中共官员及官媒谴责华盛顿的贸易施压行为，但令人惊讶的是，大陆民间企业界却有一部分人暗暗支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 向北京当局施压并促其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革.html" target="_blank">
  改革
 </a>
 。
 <br/>
 <br/>
 这是因为这些大陆企业家，期盼大陆市场更自由开放，但国内却乏人能向最高层施压，加速这方面改革。
 <br/>
 <br/>
 文章说，尽管川普开出的贸易要求，对大多数中国企业来说未必有利，基于对大陆市场改革动力停滞不前，部分民间企业家对川普采取的贸易
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_制裁.html" target="_blank">
  制裁
 </a>
 手段有所期待，希望此举能够迫使北京向民营企业进一步开放市场，以提高民企的竞争力，尤其是跟大陆国企竞争。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国民营企业.html" target="_blank">
  中国民营企业
 </a>
 家的眼里，川普几乎可以说是北京高层唯一的改革压力来源。中国改革派人士也都期待，川普能成为刺激中国经济改革的外部驱动力。
 <br/>
 <br/>
 文章说，过去一年来中共宣布的一些自由化措施，以及近期习近平做出的关于支持民营企业的承诺，都好似对美国施压的回应，而这种支持民企的新政治氛围，或是因为美国压力促成的，中共希望通过此举为避免民营企业〝变心〞。
 <br/>
 <br/>
 文章说，无论如何，大陆企业主管私下正默默寄望川普带来改变。
 <br/>
 <br/>
 持续了半年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 火持续升高，中国经济持续下滑，美国经济呈现强劲态势。目前，川普政府已对2,500亿中国商品课征10%到25%的惩罚性关税。
 <br/>
 <br/>
 美国副总统彭斯在APEC峰会上表示，中共设置的贸易壁垒史无前例，包括关税、盗用知识产权、强逼技术转让等，美国已对中共采取决定性的措施，对2500亿美元的中国产品征收关税，并可能将征税规模扩大一倍。
 <br/>
 <br/>
 彭斯还警告中共，必须全面改变其行为作法，才能避免与美国发生新冷战。
 <br/>
 <br/>
 彭斯早前在华盛顿发表长篇演说时，从政治、经济、军事、外交等方面，全方位指控中共。他说，美国在过去25年帮助〝重建〞了中国，但中共没有像美国希望的那样，让自己的人民迈向更大的自由，而是朝着控制和压迫中国民众的方向急转弯。
 <br/>
 <br/>
 彭斯说，现在美国真的觉醒了，〝历史已经证明，那些压迫本国人民的国家，很少就此住手。北京还试图将其势力扩展到全世界各地。〞
 <br/>
 <br/>
 因此，川普总统在经济、政治、外交、军事等领域展现出的全新姿态，绝非中共诋毁的〝单边保护主义〞，而是反击邪恶共产主义的进取之姿。
 <br/>
 <br/>
 在这场正邪对抗赛中，全世界的人都已经看清，川普政府对中共的这轮贸易战，不仅是在为美国和美国人民的利益而战，也是在为中国和中国人民的利益而战。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
