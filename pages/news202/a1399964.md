### 中共外交攻势受挫 北京骑虎难下
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】日前，北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和中共总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 在两场外事活动中接连遭遇挫折。美媒引述分析指，北京受困于国内政治，已进退两难，从目前中美双方的强硬态度来看，月底的中美首脑会晤很难取得成果。
 <br/>
 <br/>
 在上周的东盟峰会上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 未能推动〝区域全面经济伙伴关系协定〞（RCEP）在年底达成协议。
 <br/>
 <br/>
 随后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在亚太经济合作会议（APEC）峰会上，也遭遇美国副总统彭斯的激烈批评。多家外媒指，与会的多数国家都对中共贸易政策表示不满，因为中方强烈抵制会议联合公报中对北京的谴责，导致本届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 的联合公报有史以来第一次流产。
 <br/>
 <br/>
 而习近平在峰会上也用强硬口吻指责美方，并高调喊话〝无论冷战、热战还是贸易战都没有赢家〞。
 <br/>
 <br/>
 19日，自由亚洲电台引述华人学者分析称，近年来北京一直在煽动〝民族情绪〞，让中方在美国压力下低头，这个弯太大，一时转不过来，因此北京会和美国继续对抗。
 <br/>
 <br/>
 分析指，虽然中方已经有所妥协，但只限于局部领域，而回避关键问题。因此美方并没有接受中方提交的让步清单，下一步可能还会对北京继续加大制裁。
 <br/>
 <br/>
 另一名华人学者指出，北京外交接连遇挫，说明中方主导的所谓〝多边体制〞已被美方击破，北京已经陷入被动。
 <br/>
 <br/>
 但是，从目前北京继续放〝狠话〞来看，中方直到目前还很难放低姿态去寻求解决贸易战的方法，本月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 取得成果的可能性很低。
 <br/>
 <br/>
 北京上周向美方提交了涵盖142个项目的让步清单，但被指老调重弹，缺乏诚意。虽然美国总统表示〝可能不再需要继续加征关税〞，但也指出中方清单应该补上漏掉的几个〝重要项目〞。
 <br/>
 <br/>
 而美国商务部长罗斯则透露，因为短期内很难和中方达成全面协议，美国政府将在2019年1月按原计划把对2,000亿美元中国商品加征关税的税率从10%增加到25%。美国贸易办公室也公开表态，称继续推进对剩余的2,670亿美元中国商品加税的计划。
 <br/>
 <br/>
 美国副总统彭斯此前警告，如果北京拒绝在贸易政策和南海问题上做出显著和实质的让步，美方将在经济、外交和政治上对北京施加更大压力，新冷战就将开启。
 <br/>
 <br/>
 随后，彭斯在APEC峰会上严词抨击北京贸易不公，并辛辣讽刺中方的〝一带一路〞。彭斯在演讲中还再次明确了美方将联合盟友，在印太地区与〝一带一路〞竞争，削减北京在该地区的影响力。
 <br/>
 <br/>
 法媒指，川普（特朗普）和彭斯在合演〝红脸白脸〞，目的是让中方做出重大让步，这是川普的一贯谈判手法。
 <br/>
 <br/>
 而美媒分析指，北京目前的经济政策，包括〝中国制造2025〞等，已经和当局确立的政治路线捆绑在一起，如果做出实质改变，最高层将面临重大政治风险。因此，北京很难对华盛顿真正妥协，在劣势下只能采取拖延战术。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
</div>
