### 韩媒：金正日第四任妻子全家被抓
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】在失踪6年后，近日，朝鲜已故领导人金正日的第四任夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金玉.html" target="_blank">
  金玉
 </a>
 有了新消息。据韩媒报导，金玉现在仍然活着，因受朝鲜二号人物张成泽事件牵连，她与家人自从2014年以来一直被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关押.html" target="_blank">
  关押
 </a>
 在政治犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_收容所.html" target="_blank">
  收容所
 </a>
 。
 <br/>
 <br/>
 11月22日，韩国《中央日报》引述韩国政府消息人士称，金正日的第四任夫人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金玉.html" target="_blank">
  金玉
 </a>
 最后一次公开露面是在2012年7月，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 一起到基层视察。其后6年来一直踪迹全无。
 <br/>
 <br/>
 熟知朝鲜内部情况的韩国政府官员表示，金正日2011年去世后，金玉的动向一直备受韩国当局关注。她曾短暂出现在公开活动中，后来销声匿迹。有传闻她遭到了清肃，也有说她在德国治病。
 <br/>
 <br/>
 这位韩国政府官员说：综合各种信息，韩方认为她受张成泽事件牵连，已在2014年送往政治犯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_收容所.html" target="_blank">
  收容所
 </a>
 。现在她仍然活着，但无法透露收容所的位置。
 <br/>
 <br/>
 此外，金玉的父亲金孝、弟弟金均、妹妹金玉京等家人全部都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关押.html" target="_blank">
  关押
 </a>
 在收容所。
 <br/>
 <br/>
 金正日在世时，金玉不仅陪同其到国内各地访问，还会陪同其出访国外，负责照顾金正日的起居出行。 金正日死后，金玉的命运因受张成泽牵连走上了不同的方向。
 <br/>
 <br/>
 金正恩于2013年12月以反革命罪行将自己的姑父、朝鲜二号人物张成泽处决。朝鲜当局当时发现金玉和张成泽一家存在〝交易〞。
 <br/>
 <br/>
 金玉的父亲金孝，在金正日死亡前担任劳动党财政经理部长，负责为劳动党看护金库，安放金日成遗体的锦绣山纪念宫（现锦绣山太阳宫）的财政也由他负责。
 <br/>
 <br/>
 曾任金日成综合大学教授的金均，在2000年代40多岁时就担任了学校的第一副校长。金玉的妹妹金玉京没有公开的官方职务。但金玉京在张成泽的独生女留法学习时，从2006年到她自杀前，一直与其同住负责照顾工作。
 <br/>
 <br/>
 张成泽被处决后，张的亲戚和部下等很多人受到株连，金玉全家也未能幸免，所幸被免除了死刑。
 <br/>
 <br/>
 去年2月，逃离朝鲜投奔韩国的一名脱北者对韩媒披露，金正恩在肃清张成泽的过程中，公开处决了近千人，还株连至少2万人。
 <br/>
 <br/>
 现代史研究所所长郑昌贤（音）说，金玉之所以被免除了死刑，是因为她曾经照顾过金正日。
 <br/>
 <br/>
 金玉出生于1964年，毕业于平壤音乐舞蹈大学，主修钢琴。自80年代初一直到金正恩的生母高容姬过世，担任金正日技术书记。高容姬死后。金玉跃升为朝鲜〝第一夫人〞，并介入人事重组工作。2006年金正日健康开始恶化后，还替他批审各种文件。
 <br/>
 <br/>
 金玉也是金正日所有的女人当中，唯一参加过外交活动的人，曾随金正日出访过中国6次，俄罗斯3次。2011年5月，金正日访华时，她与金正日同乘一辆汽车，并坐在他旁边。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿）
