### 俄罗斯修炼交流会 法轮功学员分享体会
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】周末在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫斯科.html" target="_blank">
  莫斯科
 </a>
 ，第19届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 法轮大法交流会召开。200多名俄罗斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 ，从各地赶来参加盛会。
 <br/>
 <br/>
 12岁的艾丽娜和母亲一起修炼，她交流了自己在学校的修炼体会。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 艾丽娜：〝我改变很多，变得更和善了，让自己表现得更温和。〞
 <br/>
 <br/>
 母亲叶卡捷琳娜修炼四年了，修炼前担忧未来，现在有了人生目标。
 <br/>
 <br/>
 法轮功学员叶卡捷琳娜：〝大法彻底改变了我的生活，现在我拥有目标，生活中有了里程碑，我不再害怕了。〞
 <br/>
 <br/>
 63岁的米哈伊尔・埃雷明是飞机制造厂的测试工程师，修炼19年了，按照法轮功〝真、善、忍〞的原则生活，感觉神清气爽。
 <br/>
 <br/>
 法轮功学员米哈伊尔・埃雷明：〝我的工作很不容易，需要非常专注。一个小错误就可能导致大量人死亡。下班后回家我很累了，但炼完功后，我感觉像刚睡醒一样神清气爽。〞
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 法轮功学员还在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_莫斯科.html" target="_blank">
  莫斯科
 </a>
 各地展开活动，演示功法，讲述中共迫害法轮功的真相。
 <br/>
 <br/>
 伊琳娜・奥希罗娃来自圣彼得堡，在前共产国家的生活经历，让她明白共产主义给人类传统价值带来的破坏。
 <br/>
 <br/>
 法轮功学员伊琳娜・奥希罗娃：〝（共产主义）在人们之间传播斗争（哲学）和仇恨。我希望中国民众觉醒过来，知道法轮大法好，这对整个民族都很好，这是他们的未来、来源和传统。〞
 <br/>
 <br/>
 2000年俄罗斯圣彼得堡第一次召开法轮大法法会，这次已经是第十九届。
 <br/>
 <br/>
 新唐人俄国记者欧萨娜莫斯科报导
</div>
