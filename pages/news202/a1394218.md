### 诺贝尔和平奖揭晓 关注战争中的性暴力
------------------------

<div class="wysiwyg">
 【新唐人2018年10月05日讯】周五，2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诺贝尔和平奖.html" target="_blank">
  诺贝尔和平奖
 </a>
 揭晓。刚果医生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆克维格.html" target="_blank">
  穆克维格
 </a>
 和伊拉克的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆拉德.html" target="_blank">
  穆拉德
 </a>
 ，因为关注和打击恐怖分子的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性暴力.html" target="_blank">
  性暴力
 </a>
 ，共同赢得诺贝尔和平奖。
 <br/>
 <br/>
 诺贝尔委员会表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆克维格.html" target="_blank">
  穆克维格
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_穆拉德.html" target="_blank">
  穆拉德
 </a>
 ，为终止使用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_性暴力.html" target="_blank">
  性暴力
 </a>
 作为战争武器做出的贡献，因此受到表彰。
 <br/>
 <br/>
 诺贝尔委员会主席 BERIT REISS-ANDERSEN：〝两位获奖者关注和打击此类战争犯罪，做出了突出贡献。〞
 <br/>
 <br/>
 穆克维格是一名妇科医生，1999年在刚果东部城市布卡武开设诊所，每年救治数千名妇女，其中许多人遭受过武装分子的性暴力伤害。
 <br/>
 <br/>
 穆克维格为此受到武装分子威胁。2012年，武装分子闯入他家，他本人受到枪击。
 <br/>
 <br/>
 穆克维格曾获联合国人权奖。
 <br/>
 <br/>
 伊拉克雅兹迪族妇女穆拉德今年25岁。2014年，她被伊斯兰国恐怖组织囚禁，遭受酷刑和性侵害。
 <br/>
 <br/>
 她获救后，公开讲述自己的痛苦经历，呼吁国际社会援助。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_诺贝尔和平奖.html" target="_blank">
  诺贝尔和平奖
 </a>
 获得者 穆拉德：〝我们但愿像男人一样被杀死，也不愿被卖掉，被叙利亚、伊拉克、突尼斯和欧洲人强奸。〞
 <br/>
 <br/>
 随后，她成为联合国亲善大使，呼吁国际社会铲除恐怖组织。穆拉德是第17位获得诺贝尔和平奖的女性。
 <br/>
 <br/>
 国际社会希望这次诺贝尔和平奖，能够加快结束战争中的性暴力问题。
 <br/>
 <br/>
 联合国发言人ALESSANDRA VELLUCC：〝我相信诺贝尔和平奖，有助于推进结束将性暴力作为战争武器的进程。〞
 <br/>
 <br/>
 今年，美国总统川普也成为诺贝尔和平奖热门人选。
 <br/>
 <br/>
 新唐人记者毕心慈、张琪综合报导
</div>
