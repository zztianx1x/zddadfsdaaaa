### G20峰会川普发声：中共害了我们25年了！（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月01日讯】全球关注的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 已在阿根廷登场，美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）在峰会的间歇接受采访时，再次发声指责中共〝害了我们25年了！〞川普与习近平于12月1日共进晚餐并展开会谈。双方是否达成〝停火协议〞，引起各界关注。
 <br/>
 <br/>
 G20领导人峰会在阿根廷首都布宜诺艾利斯已经拉开帷幕，正在布宜诺斯艾利斯出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 的美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 ，在峰会间歇接受了美国之音(VOA)特约撰稿人的专访，谈到即将举行的美中峰会以及美中贸易等多项话题。
 <br/>
 <br/>
 视频采访片段可见，川普总统被记者问到〝您认为在不久的将来，全球经济所面临的最大现实威胁是什么？〞时，他回答说：〝我们有很多威胁，但美国被这么多的国家坑了。我说‘中国’的时候，并不是在告诉你什么秘密，中国害了我们这么多年、25年，我这话已经说了很久了。〞
 <br/>
 <br/>
 按照原定安排，川普与习近平将于12月1日下午G20峰会结束后，举行双边会谈并共进晚餐，会谈的重点预料是贸易问题。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞前，川普11月22日表示，对于〝川习会〞他已做好〝充分准备〞。〝我毕生都在为会谈准备。我不需要坐在书桌前读书，我比任何人都了解任何内容，任何数据，我的直觉一直都是对的。我们做得很好，而且中国非常想和我们达成协议。〞
 <br/>
 <br/>
 为了让中共改变不公平贸易行为，美国已经对价值2500亿美元的中国商品加征关税。2019年1月1日，美国计划将对2000亿美元商品的关税从10%增加到25%。
 <br/>
 <br/>
 自中共2001年加入世贸组织后，25年来美中贸易逆差一直居高不下，去年达到了3,725亿美元的历史最高点。川普在公开场合多次为此指责中共。
 <br/>
 <br/>
 〝川习会〞前夕，海外媒体不断释出中方将加强保护知识产权、扩大开放市场等让步讯息，川普11月29日在搭机飞往阿根廷前也透露，美中双方〝很接近〞达成协议，但他不确定是否应该这么做。
 <br/>
 <br/>
 G20峰会间歇，川普告诉其它媒体记者说：〝我们正在加紧努力工作，如果我们能达成协议，那会是很好的结果。〞〝我想他们（中方）想要（达成协议）。我认为我们也愿意。等着瞧吧。〞
 <br/>
 <br/>
 当日早间，美国贸易代表莱特希泽也表示，他预计川普总统和习近平将举行的晚宴会谈会是成功的。他确信，这次〝川习会〞后，美中官员都会有〝积极的感觉〞，〝如果川习会不成功，我会感到惊讶。〞
 <br/>
 <br/>
 据南华早报报导，川普此次率领的9人顾问团队全属于对北京鹰派人物，被认为G20〝川习会〞川普与习近平谈判志在必得。
 <br/>
 <br/>
 最近一段时间来，习近平不断宣示中共开放市场决心，准备在G20上和美国总统取得贸易协议。
 <br/>
 <br/>
 然而中国知名金融专家巩胜利认为，美中贸易战不太可能停止，因为中共想继续赚取贸易顺差。澳洲中国问题专家冯崇义认为，在涉及中共命根子的方面，如开放媒体、互联网等这些关键问题上，中共不会让步；但在涉及民族经济利益方面，它完全可以让步。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
