### 印尼灾后已致1763死 帕卢2区几乎从地球消失
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏拉威西岛.html" target="_blank">
  苏拉威西岛
 </a>
 9月28日发生规模7.5强震，引发6公尺高
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海啸.html" target="_blank">
  海啸
 </a>
 及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土壤液化.html" target="_blank">
  土壤液化
 </a>
 ，造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_帕卢.html" target="_blank">
  帕卢
 </a>
 （Palu）等地灾情严重，当局今（7）日公布最新死亡人数，已上升至1763人，同时约有5000人下落不明。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 国家灾害应变总署（BNPB）发言人苏托波（Sutopo Purwo Nugroho）今日在雅加达的记者会上表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏拉威西岛.html" target="_blank">
  苏拉威西岛
 </a>
 强震发生一周后，已知罹难人数已上升至1763人。
 <br/>
 <br/>
 另
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_帕卢.html" target="_blank">
  帕卢
 </a>
 的2个重灾区〝贝托波（Petobo）和巴拉罗亚（Balaroa）政府负责人的报告，大约有5000人尚未被发现〞。据了解，因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土壤液化.html" target="_blank">
  土壤液化
 </a>
 导致地区地层下沉，原本坚实的地面变成松散不稳，泥流一瞬间将数千间房屋吞没，这两个地区几乎已从卫星图中消失。
 <br/>
 <br/>
 苏托波表示，目前难以得知确切数字，因为这2个地区因强震导致土壤液化，大量房屋被化为液体的土地吞噬，难以从土石、瓦砾堆找到尸体。
 <br/>
 <br/>
 他说，〝没有被发现的受害者将被宣布为失踪〞。
 <br/>
 <br/>
 苏托波也宣布，印尼搜救人员将在10月11日停止投入大量人员和重设备大规模寻找受灾者遗体，但仍可能继续进行有限的搜索。
 <br/>
 <br/>
 印尼政府日前表示，土壤液化严重损毁灾区交通系统，并增加了尸体搬运的难度，且灾情发生至今已超过一周，部分尸体已经出现腐烂现象，恐引爆卫生危机。
 <br/>
 <br/>
 官方表示，会为救援人员提供伤寒及霍乱等疾病的接种疫苗，极力避免卫生危机的爆发。
 <br/>
