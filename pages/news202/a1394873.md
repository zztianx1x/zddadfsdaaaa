### 违反联合国禁令？金正恩被抓包换劳斯莱斯防弹车
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】朝鲜在2006年遭联合国禁止运输奢侈品。但日前外界发现，朝鲜独裁者金正恩近日所乘坐的专车，疑似已经从原先的奔驰换成了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_劳斯莱斯幻影防弹车.html" target="_blank">
  劳斯莱斯幻影防弹车
 </a>
 。这辆高档轿车是如何突破禁令送到金正恩手中的？这个问题引起外界的关注。
 <br/>
 <br/>
 据韩国《中央日报》报导，金正恩日前与美国国务卿庞皮欧会面后，外界从美国政府官方网站上传出的现场照片中发现，金正恩这次所乘专车的轮框上印有R的字样，这正是劳斯莱斯的正字标记。
 <br/>
 <br/>
 韩国大林大学汽车学系金必洙教授接受韩国《中央日报》采访时指出，金正恩搭乘的应是2015年款的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_劳斯莱斯幻影防弹车.html" target="_blank">
  劳斯莱斯幻影防弹车
 </a>
 ，价格约为20亿韩元。
 <br/>
 <br/>
 该报导表示，金正恩是汽车迷，尤其喜爱奔驰。金正恩先前在板门店与韩国总统文在寅，以及今年6月12日在新加坡与美国总统川普会面时，搭乘的都是奔驰豪华加长型防弹轿车S600 Pullman Guard，而当时川普还差投其所好，在川金会时让金正恩参观了身为美国总统所搭乘的凯迪拉克一号〝野兽〞。
 <br/>
 <br/>
 报导提出质疑：金正恩的新款劳斯莱斯豪华轿车是如何突破联合国的制裁禁令而运入朝鲜的？
 <br/>
 <br/>
 报导引述金必洙指出，劳斯莱斯对防弹汽车的客户采取严格保密措施，国际社会恐怕很难查出金正恩新车的走私管道。
 <br/>
 <br/>
 由于朝鲜一再违反联合国禁令进行弹道导弹等核武器的研发试验，作为对朝鲜政府的制裁措施之一，联合国从2006年起禁止成员国向朝鲜输入包括高级轿车在内的奢侈品。
 <br/>
 <br/>
 （记者欣然报导/责任编辑：东野）
