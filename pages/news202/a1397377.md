### 澳智库:数千中共军方人员隐瞒身份留学 攫取军事技术
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】本周一，有澳洲智库发表研究报告称，中共军方多年来派遣了数千名研究人员前往美国、英国、澳大利亚等西方国家，以留学或参与合作研究等方式，搜集能促进中国军事科技进步的知识和技术，他们当中的不少人隐瞒了自己的军方背景。而中共军方人员参与的国际研究合作项目通常都是〝硬科学〞，尤其集中于新兴科技和军民两用的科技领域。
 <br/>
 <br/>
 澳洲战略政策研究所（ASPI）周一发表的一份报告指出，在过去的10年间，中共军方派遣了多达2500名隶属于军方的科研人员前往美国等西方国家的大学学习或参与合作研究项目，逐步建立起了一个〝可大大促进中国军事科技发展的研究合作网络〞。
 <br/>
 <br/>
 澳洲战略政策研究所的这份报告指称，中共军方大学派往西方大国的这些研究人员，通常不会与东道国的军方人员直接接触，而是专心参与诸如量子物理、密码学、导航科技、人工智能等军民两用科技领域的研究课题，暗中搜集相关的科学技术和知识，他们当中有部分人员隐瞒了自己与军方的关系。
 <br/>
 <br/>
 这份报告并列举了一些例子，比如：有一些来自中国的研究人员，使用了〝郑州测绘学院〞（Zhengzhou Institute of Surveying and Mapping）或〝郑州信息科学技术学院〞（Zhengzhou Information Science and Technology Institute）的名称来模糊自己的真实身份。其实这两所学院指的都是中国人民解放军资讯工程大学( PLA Information Engineering University )。东道国的接待机构如果不熟悉中文，将很难识别这两所学院与中共军队的联系。
 <br/>
 <br/>
 报告称，中共军方的外派科研人员，主要集中在美、英、澳、加拿大和新西兰等国家，而在过去5年以来，中国军方所属研究人员与英、美科学家联合发表研究报告的数量，比与其他任何国家学者共同发表的研究报告的数量还多。现在，部分西方国家正在辩论如何控制尖端科技、尤其是军民两用科技（dual-use technology）流往中国的问题，估计由此引发的辩论将更加激烈。
 <br/>
 <br/>
 据报告，在上述时期内，中国军方科学家和海外学者合作完成的同行评审文章数量，从2007年的95篇已激增至2017年的734篇，增加了将近7倍。而针对上述这些现象，美国政府正采取措施，欲阻止中国（中共政府）获取美国公司开发的关键技术。
 <br/>
 <br/>
 不过，报告同时也指出，以美国为代表的西方大国的学术体系是开放性的，因此要在学术层面来防止中共军方人员摄取关键技术的难度很大，尤其许多来自中共军方的学者参与西方国家的科技研究项目时，带来的不仅是其专业知识，还带来了资金。
 <br/>
 <br/>
 《美国之音》针对上述报告指出，美中关系近期急剧恶化，川普（特朗普）政府当前也正在竭力制止美国尖端科技流往中国，在此背景下，澳洲智库机构发布的这份报告可促使相关的议题更加引人瞩目。
 <br/>
 <br/>
 据公开的资讯，中国的《神州学人》月刊2015年曾刊文披露，中共军方所属的国防科技大学和英国剑桥大学正在进行一项跨学科的研究计划。文章直言不讳地称，这项合作可为中共军方培养新一代的〝超级电脑专家〞，并可以〝大大强化我国在国防、通讯、成像抗扰和高精准导航领域的能力〞。
 <br/>
 <br/>
 此外，英国里兹大学（University of Leeds）的一名研究人员也曾对英国《金融时报》透露说，中共国防科技大学空天科学学院副研究员、中共军总装备部飞机设计专家黄伟在攻读博士学位时，就曾于2008到2010年访问里兹大学；而中国国防科技大学另一名超音速燃烧冲压发动机研究人员罗文雷，2014年也在里兹大学学习并撰写了以超音速燃烧冲压发动机为主题的博士论文。
 <br/>
 <br/>
 据称，黄伟、罗文雷以及罗的博士论文指导教授，都曾和王振国一起发表有关超音速燃烧冲压发动机的研究报告。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：明轩）
