### 韩国召开亚洲法轮大法心得交流会
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】两年一度的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚洲地区.html" target="_blank">
  亚洲地区
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮大法.html" target="_blank">
  法轮大法
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_心得交流会.html" target="_blank">
  心得交流会
 </a>
 ，10月14日，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国首尔.html" target="_blank">
  韩国首尔
 </a>
 隆重举行。来自亚洲各国的部分法轮功学员，近2000人参与。
 <br/>
 <br/>
 交流会现场庄严肃穆，韩国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮大法.html" target="_blank">
  法轮大法
 </a>
 佛学会首先传达了李洪志师父发来的贺词。
 <br/>
 <br/>
 随后来自韩国、台湾、印尼、香港等地各行各业的法轮功学员上台发言。
 <br/>
 <br/>
 他们讲述了如何在各自生活、工作和讲真相中，按照〝真、善、忍〞提升自己和做好三件事。
 <br/>
 <br/>
 与会的学员纷纷表示，修炼法轮大法使自己身心收益。
 <br/>
 <br/>
 国立台湾大学新闻研究所教授 张锦华：〝修炼之后真是实在身体和心灵上变化都非常的大，身体的部分就是以前的一些胃痛失眠等等的问题就不翼而飞了，心性上面对学生和教学方面真的是效率更好，更有耐心，品质也更好〞
 <br/>
 <br/>
 日本媒体人 广田宽光〝身体变的很健康，以前有哮喘，对人生感到非常不安，但现在这些都没有了，对自己的将来也感到安心了。〞
 <br/>
 <br/>
 学员们也表示参加这次交流会有很大的收获。
 <br/>
 <br/>
 国立台湾大学新闻研究所教授 张锦华：〝同修在讲真相啊过病业关啊，找到自己不足的地方我觉得非常的感动，尤其是我们看到不同国家的学员，同样修炼这部大法，同样是在这样一个修炼过程中不断的提高，觉得非常感谢师父所传这部大法的洪大慈悲。〞
 <br/>
 <br/>
 日本媒体人 广田宽光〝我是日本人，从媒体报导来看我觉得日韩关系称不上良好，我自己也多少受到影响了。但是这次看到大家在修炼上那么谦虚的态度，让我感受到在大法修炼上是不存在这些问题的，大法修炼没有敌人，我是这样认为的。〞
 <br/>
 <br/>
 交流会圆满结束后，学员们比学比修，找到自己的差距，铭记师父的嘱咐，决心更加精进。
 <br/>
 <br/>
 新唐人记者杨云
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国首尔.html" target="_blank">
  韩国首尔
 </a>
 报导
</div>
