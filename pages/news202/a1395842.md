### 传华邮记者遭活肢解 沙特王储再次否认知情
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】据报《华邮》记者卡舒吉在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 领事馆遭活肢解，美国国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 周二（10月16日）先去会见了沙特国王和王储，17日又和土耳其总统及外长会面，希望尽快解决此事件引发的危机。沙特王储再次否认知情。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
   土耳其
  </a>
  ：卡舒吉被肢解
 </h4>
 <br/>
 美联社引述土耳其一名高级官员称，当局已经发现一定证据，确认卡舒吉（Jamal Khashoggi）是在领事馆内遇害。当局已经带走泥土样本及花园中一个金属制的门跟进调查。
 <br/>
 <br/>
 英国独立传媒《中东之眼》引述土耳其官员称，调查人员在听完卡舒吉在领馆内的录音，确认卡舒吉是遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特.html" target="_blank">
  沙特
 </a>
 派出的15人小组中的法医活生生肢解，过程长达7分钟，期间他一直惨叫，直至被注射不明药物后才停止。官员称，沙特并没有想要盘问卡舒吉，而是早有预谋要杀死他。
 <br/>
 <br/>
 而美国有线新闻网络（CNN）则称，卡舒吉案是由与王储穆罕默德关系密切的情报人员主使，他们怀疑卡舒吉与沙特断交一年多的卡塔尔有联系，决定在不知会王室下，擅自行动。
 <br/>
 <br/>
 <h4>
  美国务卿访沙特
 </h4>
 <br/>
 美国总统川普（特朗普）15日派国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蓬佩奥.html" target="_blank">
  蓬佩奥
 </a>
 前往沙特了解情况并敦促沙国解决问题。
 <br/>
 <br/>
 中央社报导，蓬佩奥16日先和沙特国王会谈，然后分别和沙特外长朱贝尔，以及掌握沙国实权的王储穆罕默德．萨勒曼会面。穆罕默德．萨勒曼在王宫接待蓬佩奥时表示：〝我们是关系牢固的老盟友，一起面对共同挑战。〞两人稍后共进晚餐。
 <br/>
 <br/>
 卡舒吉失踪案所引发的争议令沙特的传统西方盟友备感忧虑，这些西方国家也是沙特的主要武器供应国。另外，本案也削弱了穆罕默德．萨勒曼试图树立现代统治者形象的努力。
 <br/>
 <br/>
 蓬佩奥17日转抵土耳其首都安卡拉与总统埃尔多安会面，讨论卡舒吉失踪案。
 <br/>
 <br/>
 <h4>
  沙特王储否认知情
 </h4>
 <br/>
 与此同时，川普在其社交网站推特发文称，他和沙特王储透过电话会谈，王储称会扩大调查卡舒吉失踪案。他说，〝我刚刚和沙特王储说了话，他完全否认对（沙特驻）土耳其领馆内发生的事情知情。通话时他和国务卿蓬佩奥在一起，他告诉我他已开始并将快速深入对事件进行全面和彻底的调查。〞川普并称很快会有结果。
 <br/>
 <br/>
 川普稍后在接受美联社采访时表示，在获得证实前，不应对沙特当局下有罪结论。
 <br/>
 <br/>
 川普17日表示，他将等待蓬佩奥归来后的全面报告。他表示，已要求土耳其提供任何有关卡舒吉失踪的音频或视频证据。
 <br/>
 <br/>
 <h4>
  卡舒吉失踪案时间线
 </h4>
 <br/>
 卡舒吉是沙特多家知名日报主笔，过去他与沙国领导阶层关系良好，后来批评沙国政策，还曾撰文评论沙国王储穆罕默德．萨勒曼。
 <br/>
 <br/>
 <h4>
  10月2日
 </h4>
 <br/>
 卡舒吉为了与土耳其未婚妻结婚，造访沙国驻伊斯坦布尔领事馆领取相关文件，之后离奇失踪。
 <br/>
 <br/>
 <h4>
  10月6日
 </h4>
 <br/>
 土耳其警方初步研判卡舒吉在领事馆内遭杀害，沙国当局及领事馆否认指控。
 <br/>
 <br/>
 <h4>
  10月9日
 </h4>
 <br/>
 亲土耳其政府传媒《晨报》（Sabah）列出15名沙国男子姓名和照片，土耳其认为他们的任务是谋杀卡舒吉。
 <br/>
 <br/>
 <h4>
  10月13日
 </h4>
 <br/>
 传卡舒吉的Apple Watch录下遭严刑拷打和杀害的录音，但无法证实。沙国内政部长否认杀害卡舒吉并谴责传言都是〝谎言〞。美国、欧盟都呼吁沙国展开调查。
 <br/>
 <br/>
 <h4>
  10月15日
 </h4>
 <br/>
 沙国国王萨勒曼下令进行内部调查，同时，与土耳其成立的共同调查工作小组也准备前往沙国领事馆搜索。
 <br/>
 <br/>
 <h4>
  10月16日
 </h4>
 <br/>
 美国国务卿蓬佩奥敦促沙特澄清事件，沙特王储称将彻底调查；土耳其扩大调查，官员称卡舒吉一进沙特领馆就被杀害且分尸。
 <br/>
 <br/>
 <h4>
  事件影响
 </h4>
 <br/>
 失踪案冲击外资对沙国信心，造成沙国股、汇市大跌。愈来愈多西方企业退出23日起在利雅得举行的未来投资计划会议。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
