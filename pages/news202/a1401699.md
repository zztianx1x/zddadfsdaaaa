### 墨关闭原移民庇护所 新址远离美边境
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 蒂华纳政府周六（12月1日）宣布，已
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关闭.html" target="_blank">
  关闭
 </a>
 设于运动场内的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_庇护所.html" target="_blank">
  庇护所
 </a>
 ，该庇护所靠近美国边境，之前曾容纳约6,000名希望入境美国的中美洲移民。
 <br/>
 <br/>
 蒂华纳（Tijuana）当局声明，原先滞留在边境体育场内的数千大篷车
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 正搬往一个前音乐会场，但是距美墨边境的距离要比原来远得多。声明解释说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关闭.html" target="_blank">
  关闭
 </a>
 原来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_庇护所.html" target="_blank">
  庇护所
 </a>
 是因为那里卫生条件太恶劣了。
 <br/>
 <br/>
 设在边境体育场内的庇护所之前被水淹过，专家对那里的卫生条件表示担忧。该体育场原本只能容纳1,000人，但实际容纳的移民数量已高达6,000人左右，且正有多种传染病在移民中泛滥。
 <br/>
 <br/>
 据福克斯新闻（Fox News）记者詹金斯（Griff Jenkins）数日前从边境发来的消息，蒂华纳市一名卫生部门发言人确认，他们已经给2,267名中美洲移民实施了传染病治疗，其中60%为呼吸道传染，3人确认患有肺结核，4例艾滋病，101例头蝨感染，4例水痘，而且肝炎爆发的隐患很大。
 <br/>
 <br/>
 蒂华纳政府正用大巴运送移民，新庇护所距离美国圣地亚哥最南处的Otay Mesa入境口岸大约有10英里远，距移民排队向美国申请庇护的圣伊西德罗入境口岸（San Ysidro）14英里。
 <br/>
 <br/>
 周六，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥.html" target="_blank">
  墨西哥
 </a>
 新总统欧布拉多（Andres Manuel Lopez Obrador）在宣誓就职当天，即与3位中美洲国家元首签署了协议，同意制定一项发展计划，以阻止寻求庇护的移民流向美国。
 <br/>
 <br/>
 墨西哥外交部长表示，该计划包括一笔支持洪都拉斯、萨尔瓦多、危地马拉3国创造就业就会的资金，旨在攻克导致移民外流的结构性因素。因为很多移民表示，他们离开本国的原因是，那里充斥着贫穷和暴力。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
</div>
