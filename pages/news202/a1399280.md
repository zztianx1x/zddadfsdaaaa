### 亚太峰会习近平上头条 照片却变〝中华民国〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】2018年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚太峰会.html" target="_blank">
  亚太峰会
 </a>
 12日在巴布亚新几内亚登场，习近平预计15日访问该国并出席峰会。14日，该国发行量最大的报纸〝国民报〞头版头条刊登习近平来访新闻，却爆出乌龙，误将习近平照片的来源，标注为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中华民国.html" target="_blank">
  中华民国
 </a>
 大使馆〞。
 <br/>
 <br/>
 据台媒中央社11月14日报导，亚太经济合作会议（APEC）本周在巴布亚新几内亚首都莫士比港开幕，作为该国第二大援助国的中国大陆，为本次峰会提供了大量资金援助。
 <br/>
 <br/>
 中共外交部公布，习近平15至16日将先对巴布亚新几内亚进行国事访问，17至18日出席在该国首都莫士比港（Port Moresby）举行的亚太经济合作会议（APEC）第26次非正式领袖峰会。
 <br/>
 <br/>
 14日，巴布亚新几内亚最大报刊〝国民报〞(The National)，在头版头条刊登习近平将来访的相关报导。国民报每日发行量约6.5万份，读者群涵盖各阶层人士。
 <br/>
 <br/>
 然而，该报却爆出一个大乌龙，在搭配报导的照片图说中，误将习近平照片的来源，标注为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中华民国.html" target="_blank">
  中华民国
 </a>
 大使馆〞（Embassy of the Republic of China）。
 <br/>
 <br/>
 在〝国民报〞爆出习近平照片乌龙的同时，还刊登了以〝台湾：创新成长的伙伴〞为题的宣传广告，宣传台湾政府正在推动的5+2产业创新计划。
 <br/>
 <br/>
 本次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚太峰会.html" target="_blank">
  亚太峰会
 </a>
 ，台湾政府在当地两大报国民报及信使报(Post Courier)都刊登了广告。据相关官员表示，此次峰会，台方会尽量不与中方在同一场地举办活动，同时，也会使用非中方资助的车辆。
 <br/>
 <br/>
 目前，太平洋岛国巴布亚新几内亚是全球关注的焦点，本周一些世界上最有影响力的领导人将聚集在这里，参加一年一度的亚太经合组织（APEC）峰会。美国副总统彭斯将出席峰会，据悉，他将在峰会上再次严厉抨击中共的威权及侵略行为。
 <br/>
 <br/>
 《纽约时报》11月13日报导称，本次峰会的领导人会议预计将有7,000人参加。由于酒店短缺，大部分人将住在由澳大利亚嘉年华（Carnival Australia）提供的三艘停靠的邮轮上。澳大利亚是巴布亚新几内亚的最大援助国。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
