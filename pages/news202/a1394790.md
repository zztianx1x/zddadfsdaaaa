### 专抢巴黎华人 法警拘捕11名青年嫌犯
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国.html" target="_blank">
  法国
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎.html" target="_blank">
  巴黎
 </a>
 近几个月来接连发生多起针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人.html" target="_blank">
  华人
 </a>
 的暴力
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抢劫.html" target="_blank">
  抢劫
 </a>
 案，令当地华人人心惶惶。警方周二（9日）公布，抓获了一个专门暴力抢劫华人的犯罪团伙，目前正在对11名涉案人员进行讯问。
 <br/>
 <br/>
 警方称，这11名嫌犯年龄介于17～19岁，他们涉嫌在8月19日至10月2日期间，针对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华人.html" target="_blank">
  华人
 </a>
 实施了13起暴力
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抢劫.html" target="_blank">
  抢劫
 </a>
 。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎.html" target="_blank">
  巴黎
 </a>
 警方8月接到数起报案，其作案手法极其相似，这些案件随后交由巴黎北郊的奥贝维利耶市警察局调查。
 <br/>
 <br/>
 警方分析其作案手法、地点及时间后，最终锁定并抓捕了这个由11人组成的犯罪集团。
 <br/>
 <br/>
 警方表示，嫌犯通常会在受害人驾车离开工作地点或餐厅时开始跟踪，当目标抵达住所后便进行抢劫。被抢财物主要是手表、名牌手袋及现金，受害者均为当地华人。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
</div>
