### 美军锁定索马利亚盖达分支 空袭至少60死
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】美军12日开始对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_索马利亚.html" target="_blank">
  索马利亚
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_盖达.html" target="_blank">
  盖达
 </a>
 组织（Al-Qaeda）分支
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年党.html" target="_blank">
  青年党
 </a>
 发动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空袭.html" target="_blank">
  空袭
 </a>
 ，造成青年党约60名成员丧生。这是自2017年11月以来最大规模的空袭。
 <br/>
 <br/>
 美国总统川普去年放宽对驻索美军的限制，授权指挥官认为有必要时，毋须取得白宫的特别批准即可对武装嫌疑分子采取行动，因此近几个月来，锁定
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青年党.html" target="_blank">
  青年党
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_空袭.html" target="_blank">
  空袭
 </a>
 和飞弹攻击行动有增无减。
 <br/>
 <br/>
 美军非洲司令部（US Africa Command）16日发声明表示，美军在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_索马利亚.html" target="_blank">
  索马利亚
 </a>
 中部海岸的哈拉迪尔地区（Harardhere）发动空袭，并与当地联合国支持的非洲联盟驻索马利亚代表团（AMISOM）合作。
 <br/>
 <br/>
 声明表示：〝这次针对青年党的精确空袭行动，是2017年11月21日以来最大规模的行动〞。2017年的行动锁定青年党其中一所训练营空投炸弹，造成约100名极端分子丧生。
 <br/>
 <br/>
 青年党当前正极力推翻国际支持的索马利亚政府，美军非洲司令部于声明中表示：〝空袭削弱青年党策划未来攻击的能力、破坏领导阶层网络，并重挫青年党在此地活动的自由度及机动性。〞
 <br/>
 <br/>
 〝这个组织利用索马利亚中南部若干地区策划及指挥恐怖攻击、窃取人道援助物资，勒索当地民众为其行动提供资金及掩护激进的恐怖分子。〞
 <br/>
