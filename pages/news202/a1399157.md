### 美日协商双边贸易 施压朝鲜实现无核化
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】周二，正在日本访问的美国副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 ，与日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 会谈，协商美国与日本的双边贸易合作，同时也商讨继续对朝鲜全力施压，直到朝鲜实现真正的无核化。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯.html" target="_blank">
  彭斯
 </a>
 表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍晋三.html" target="_blank">
  安倍晋三
 </a>
 与美国总统川普在九月份会晤时，就决定展开双边贸易协定谈判，预计会谈将在明年初开始。
 <br/>
 <br/>
 美国副总统 彭斯：〝实现自由、公平、互惠贸易的最佳机会将是双边贸易协定。〞
 <br/>
 <br/>
 彭斯要求日本必须削减对美国的贸易顺差。
 <br/>
 <br/>
 日本首相 安倍晋三：实现印太地区经济繁荣、自由与开放，并建立在公平贸易原则上。
 <br/>
 <br/>
 另外，美国与日本一致同意，朝鲜核武对世界的威胁仍是紧迫要解决的问题。
 <br/>
 <br/>
 美国副总统 彭斯：〝继续对朝鲜施压。制裁将继续有效，直到证实朝鲜取得最终、彻底的无核化。〞
 <br/>
 <br/>
 日本首相 安倍晋三：〝我们一致认为，有必要继续全面落实联合国对朝鲜的决议，实现朝鲜半岛完全无核化。〞
 <br/>
 <br/>
 彭斯周二离开日本，代表美国总统川普出席在新加坡举行的东盟(ASEAN)会议，以及在巴布新几内亚举行的亚太经合组织(APEC)峰会。
 <br/>
 <br/>
 新唐人记者陈依心，明玉综合报导
</div>
