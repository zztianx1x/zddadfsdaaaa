### 日媒：习近平晚宴表情突变 安倍安慰〝川普信任你〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】日本首相安倍晋三已结束访华行程返回日本。日媒28日披露安倍与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在晚宴中的一些内幕，习谈到贸易战时表情阴郁沮丧，安倍则劝说，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特朗普.html" target="_blank">
  特朗普
 </a>
 ）总统信任你。〞他还指，川普虽然大力施压中共，但对习个人基本没有指责，仍然保持善意。
 <br/>
 <br/>
 安倍此次访华之行被称为〝中日破冰〞之旅，中共罕见高规格接待，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 还特设晚宴款待安倍一行。
 <br/>
 <br/>
 日经新闻网10月28日报导称，26日在北京钓鱼台国宾馆举行的晚宴上，习近平祝贺安倍再次当选，语气轻松。但随后，习谈到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 问题，表情就变得阴郁、沮丧。
 <br/>
 <br/>
 习近平告诉安倍，他对华盛顿在贸易谈判中的强硬策略，感到失望。安倍则劝说：〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统信任你。〞
 <br/>
 <br/>
 安倍还指出，即使川普加大对中共的施压，但他基本上没有指责习近平个人。
 <br/>
 <br/>
 有分析认为，作为美国重要盟国的领袖，安倍对习近平的一番安抚颇有意味。显然，安倍也很清楚，川普虽然一方面强硬抨击中共政权，但另一方面仍对习近平本人保持善意和尊重。
 <br/>
 <br/>
 事实上，川普总统与历届美国总统一个很大的不同之处是，他对于中共的认识非常清晰。川普在9月底的联合国大会上，严厉抨击社会主义或共产主义导致了苦难、腐败和衰变。呼吁世界上所有国家都来抵制社会主义及其给每个人带来的苦难。
 <br/>
 <br/>
 但川普一直对外界称，习近平与他有良好的个人关系。他曾表示，希望能和习近平共同解决这场贸易纠纷。虽然贸易战目前未见有缓解的迹象，但即将在11月举行的G20峰会，川普还将与习近平举行元首会面。
 <br/>
 <br/>
 针对安倍此次访华，以及中日关系的回暖，许多西方媒体认为，这是因为美国方面的压力导致的。《纽约时报》刊文称，受与华盛顿关系迅速降温，尤其是贸易战打击的影响，习近平正在寄希望于一个更友好的日本作为一种对冲。
 <br/>
 <br/>
 而港媒评论文章称，断定中日两国不会友好，只是互相利用。中共试图借日抗美，而日本则借中图利。
 <br/>
 <br/>
 文章还指，中共的如意算盘不可能打响，因为日本在各项层面都走不出美国，而且本身对中共也不信任。
 <br/>
 <br/>
 日媒也对中日关系未来走向大泼冷水，称中日新时代只不过是〝同床异梦〞、〝沙上城堡〞，中日不可能建立真正友谊。
 <br/>
 <br/>
 近期，美日已多次就共同解决中共贸易问题举行会谈，日本明确支持美国针对中共现有的不公平贸易做法，对WTO规则进行调整。很显然，中共试图联日抗美的设想，难以实现。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
</div>
