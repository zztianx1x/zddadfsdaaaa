### 记者失踪案 土耳其官员：在沙特国领事馆遭分尸
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特阿拉伯.html" target="_blank">
  沙特阿拉伯
 </a>
 记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 两周前在沙特驻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 领馆失踪。土耳其警方15日晚对沙特领馆进行了9个小时的搜查后，土耳其官员周二（16日）告诉美媒，卡舒吉在领馆遭杀害后，遗体可能已被切成碎片。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_土耳其.html" target="_blank">
  土耳其
 </a>
 一名要求匿名的高阶官员告诉美联社，土国警方在搜索
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沙特阿拉伯.html" target="_blank">
  沙特阿拉伯
 </a>
 驻伊斯坦布尔领事馆过程中发现〝某些证据〞，显示日前失踪的沙国记者
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_卡舒吉.html" target="_blank">
  卡舒吉
 </a>
 （Jamal Khashoggi）是在领事馆内遇害。该官员没有透露〝某些证据〞的具体细节，此次搜索行动持续到周二清晨。
 <br/>
 <br/>
 卡舒吉是《华盛顿邮报》特约记者、沙特阿拉伯公民，他经常批评沙特时政，因害怕自己有危险，于一年前逃离了沙特。他在10月2日进入沙特阿拉伯驻伊斯坦布尔领事馆，领取和未婚妻成吉兹（Hatice Cengiz）结婚的文件后失踪。
 <br/>
 <br/>
 数名土耳其官员表示，多名沙特特工2日在沙特驻伊斯坦布尔领事馆内杀害了卡舒吉，并将他分尸。
 <br/>
 <br/>
 在土耳其数家电视台播放的影片中显示，一辆黑色轿车在卡舒吉失踪2小时后离开领事馆，并停在了沙特领事艾尔欧塔比（Mohammad al-Otaibi）的住处。
 <br/>
 <br/>
 土耳其警方表示，周二稍晚将会搜索艾尔欧塔比的官邸。不过，外传艾尔欧塔比周一已搭机返回沙特，但他没有说明返国原因。
 <br/>
 <br/>
 沙特目前面临强大的国际压力，西方国家要求沙特解释卡舒吉的死亡，这让沙特和西方国家面临了外交问题。这次的调查结果可能对沙特不利，据悉，沙特可能将承认审讯卡舒吉导致其意外死亡，是误杀。
 <br/>
 <br/>
 美国国务卿蓬佩奥周一赶往沙特，会见了沙特国王、王储和沙特外交部长。
 <br/>
 <br/>
 （记者李红报导/责任编辑：程以仁）
