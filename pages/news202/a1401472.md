### 20国领导人齐聚阿根廷 G20争议与机会并存
------------------------

<div class="wysiwyg">
 【新唐人2018年11月30日讯】为期两天的20国集团峰会，聚焦全球经济。〝川习会〞是否会给美中贸易摩擦带来转折变化，是最大焦点。川普取消了和俄罗斯总统普京的会晤，在俄罗斯和乌克兰冲突升级的背景下，两人会有什么样的互动，也是看点之一。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 僵持不下之际，川普和习近平将于周六在20国峰会期间进行会晤。这将是贸易战爆发后两国领导人首次会面，外界期待此次会晤有助于打破僵局。
 <br/>
 <br/>
 美国至今已向2500亿美元中国进口商品加征了惩罚性关税。中共则对大约1,100亿美元美国商品实施报复性关税。
 <br/>
 <br/>
 明年1月1日，美国将提高2,000亿美元中国商品的关税。而剩下的2670亿美元中国商品增加关税也已经是箭在弦上。
 <br/>
 <br/>
 美方一再施压，目的是迫使中共取消现有的关税及非关税壁垒，并停止盗窃美国的知识产权。
 <br/>
 <br/>
 这些问题多年来也一直引起欧盟和日本的抱怨。白宫经济顾问柯德洛周二表示，美国与欧盟、日本达成协议，合作处理对中国的经济问题。
 <br/>
 <br/>
 和另外一个大国领袖俄罗斯总统普京的会谈被川普取消。
 <br/>
 <br/>
 上周日，俄罗斯扣留了三艘乌克兰海军舰艇，造成该地区紧张局势升级。船隻和水手至今仍没有被释放。因此川普周四宣布取消和普京在峰会上的双边会晤。
 <br/>
 <br/>
 沙特王储穆罕默德也将出席此次峰会，他有可能涉及沙特记者卡舒吉谋杀案，其他出席峰会的领袖将如何与他互动，受到各国媒体关注。
 <br/>
 <br/>
 除此之外，英国与东道主
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阿根廷.html" target="_blank">
  阿根廷
 </a>
 之间的马岛争端，也成为会议的另一种焦点。
 <br/>
 <br/>
 阿根廷希望在英国脱欧之际，可以在英属马岛主权问题上取得进展。尤其是，上周日英国已经在压力之下承诺，脱欧后，让西班牙有权参与决定英属直布罗陀的未来。
 <br/>
 <br/>
 但英国首相特雷莎・梅周五接受阿根廷媒体采访时表示，脱欧一事不会影响英国本土和马岛之间的联系。
 <br/>
 <br/>
 新唐人记者王羽音、天羽综合报导
