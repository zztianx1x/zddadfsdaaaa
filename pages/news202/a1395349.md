### 土耳其载移民卡车 翻落沟渠酿22死13伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】土耳其西部伊兹米尔省一辆载着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_移民.html" target="_blank">
  移民
 </a>
 的卡车，14日清晨冲出路障，翻落到20公尺下方的灌溉运河，造成22人罹难，其中包含孩童，另有13人受伤。
 <br/>
 <br/>
 从土耳其国营安纳杜鲁新闻社（Anadolu）发出的影片和照片显示，灌溉运河横躺着一辆损坏上下翻转的车体。
 <br/>
 <br/>
 报导说，这辆卡车自爱丁（Aydin）来到伊兹米尔（Izmir）。
 <br/>
 <br/>
 路透社报导，这辆无顶的卡车在高速公路上冲出围栏，直接从高架桥掉下去撞进沟渠，残骸散布在沟渠内，许多人在事故中身亡、受伤，救护车接获通报后抵达现场救援，并试图救出被压在卡车下方受困的死伤者。

 稍早，民营新闻社Demiroren报导，这起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_车祸.html" target="_blank">
  车祸
 </a>
 造成18人丧生，另有多名伤者已送医救治。
 <br/>
 <br/>
 安纳杜鲁新闻社报导，经过救援后，车祸共造成22人丧生，13人受伤，死者包括2名婴儿、2名儿童和一名孕妇。目前尚不知道这些移民的国籍。
 <br/>
 <br/>
 多安通讯社（DHA）报导说，35岁土耳其司机生还，说他是为了躲避迎面而来的白色车辆才失控。警方将于司机出院后将他逮捕，且地方检察官已展开调查。
 <br/>
 <br/>
 据了解，每年有不少非法移民渡海到达土耳其，为了要从土耳其就近转往希腊，进入欧盟。但欧盟为了遏止纷涌而至的移民，2016年与土耳其达成协议，将会把所有非法入境的难民遣返，但目前仍有许多移民前往土耳其。
