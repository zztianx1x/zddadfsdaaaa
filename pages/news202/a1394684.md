### 信号？金正恩解除禁美令 国营店出售美国电脑
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】美媒消息，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 当局近日已经正式解除
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国商品禁令.html" target="_blank">
  美国商品禁令
 </a>
 ，国营商店也开始推销美国二手笔记本电脑。目前中美对抗日渐升级，北京明显处于劣势。朝鲜和美国逐渐靠近，中共手中的朝鲜牌越来越失去效力。
 <br/>
 <br/>
 自由亚洲电台9月28日引述
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_朝鲜.html" target="_blank">
  朝鲜
 </a>
 消息来源称，朝鲜北部南平安省（South Pyongan）平松市（Pyongsong）百货公司开始公开销售美国制造的笔记本电脑。
 <br/>
 <br/>
 消息人士称，这家朝鲜国营商店之所以推销美国笔记本，是因为朝鲜劳动党中央委员会为了改善美朝关系，已经正式允许销售美国产品。此前销售或使用美国产品会遭到惩罚。
 <br/>
 <br/>
 消息称，该百货公司销售的笔记本品牌包括惠普、戴尔和T400，是从中国走私而来。在此之前，平松百货商店也出售中国组装的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国笔记本电脑.html" target="_blank">
  美国笔记本电脑
 </a>
 ，但会伪装成中国产品，以避免被当局处罚。
 <br/>
 <br/>
 与中国接壤的北平安（North Pyongan）省的消息来源也证实，平松市许多学院和行政办公室的学生和官员都需要笔记本电脑，因此平松百货经理让中朝边境新义州市（Sinuiju）的商人运来二手惠普和戴尔笔记本电脑，并以300美元的价格出售。
 <br/>
 <br/>
 朝鲜的日常用品此前大多依赖中国市场。现在当局打破禁令，接受美国商品，显示向美国靠拢的意向。
 <br/>
 <br/>
 8月下旬，美国总统川普曾宣布取消国务卿蓬佩奥的访朝计划，并在推特明言将访朝推迟到解决中美贸易战之后，显示出先解决中美问题、再解决朝核问题的意图。这使得北京想利用朝鲜牵制贸易战的计划难以再发挥效用。
 <br/>
 <br/>
 之后，朝鲜态度开始软化，承诺废弃洲际导弹试验场，并不再强硬要求签署停战宣言。日前，蓬佩奥再次访朝，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金正恩.html" target="_blank">
  金正恩
 </a>
 又表态接受无核化核查。
 <br/>
 <br/>
 10月4日，美国副总统彭斯发表演讲，对中共进行全方位谴责，中美对抗再次升级，海内外惊呼新冷战时代开启。
 <br/>
 <br/>
 在这种情况下，金正恩在中美之间左右逢迎、搏取好处的空间，正在逐渐缩小。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
