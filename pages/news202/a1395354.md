### IMF世行峰会闭幕 承诺对印尼更多援助
------------------------

<div class="wysiwyg">
 【新唐人2018年10月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界银行.html" target="_blank">
  世界银行
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际货币基金组织.html" target="_blank">
  国际货币基金组织
 </a>
 ，为期一周的年度峰会，10月14日在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 巴厘岛努沙杜瓦落下帷幕。在最后一天的会议上，世行表示，将为近几个月来饱受灾难侵袭的印度尼西亚，提供高达10亿美元的贷款，用于该国遭受地震和海啸袭击的两个岛屿的救济和重建工作。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_世界银行.html" target="_blank">
  世界银行
 </a>
 行长金墉：〝我们回应
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印尼.html" target="_blank">
  印尼
 </a>
 政府请求，提供10亿美元帮助印尼苏拉威西岛和龙目岛进行灾难后恢复和重建。〞
 <br/>
 <br/>
 9月28日的地震和海啸在苏拉威西岛造成2000多人死亡，其中大部分人死于沿海城市帕卢。
 <br/>
 <br/>
 世界银行表示，其初步报告显示，近一段时间的灾难给苏拉威西岛的基础设施，住宅和非住宅财产带来的实际损失约为5.31亿美元。
 <br/>
 <br/>
 金墉还称赞印度尼西亚〝人团结，并决心确保不让任何一个人，甚至最穷的人被遗忘。〞
 <br/>
 <br/>
 印尼超过80,000人生活在临时避难所或其他地方。地震的力量将土壤液化并将房屋吸入地下。可能有5000人被埋葬。
 <br/>
 <br/>
 世行的贷款融资方案将向最贫困的15万个受灾家庭提供为期6个月至一年的现金支持，这将有助于灾后的恢复阶段。
 <br/>
 <br/>
 新唐人记者严枫、天羽综合报导
</div>
