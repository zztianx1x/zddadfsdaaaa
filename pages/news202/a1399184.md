### 川普回击马克龙： 二战投降  贸易不公
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】周二（11月13日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统发表一系列推文，回击法国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 （Emmanuel Macron）近来对美国及他本人的不当言论。
 <br/>
 <br/>
 11月9日至11日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 携第一夫人梅拉尼娅（Melania Trump）出访巴黎，会同全球70多个国家的元首或政府首脑，参加纪念第一次世界大战停战
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_100周年.html" target="_blank">
  100周年
 </a>
 系列活动。但在川普访法前后，法国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马克龙.html" target="_blank">
  马克龙
 </a>
 一再发表对美国蕴含敌意的言论，并在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一战停战.html" target="_blank">
  一战停战
 </a>
 纪念活动中抨击〝民族主义〞（Nationalist）是〝对爱国主义的背叛〞，藉此公开批评川普政府的外交政策。
 <br/>
 <br/>
 周二早上，已返回华府的川普连发多篇推文，批评并嘲讽马克龙的诸多不当观点和言辞。
 <br/>
 <br/>
 川普指出，马克龙建议建立自己的军队，以保护欧洲不受美国、中共和俄罗斯侵犯。但是，挑起两次世界大战的都是德国，法国又是如何解决的？
 <br/>
 <br/>
 川普暗讽法国在二战中向纳粹德国投降：在美军到达法国之前，法国人就开始在巴黎学德语了。
 <br/>
 <br/>
 川普接着敦促，法国须在支付或不支付北约（NATO）军费间做出选择。
