### 西班牙叠人大赛 加泰隆尼亚要求独立
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 两年一度的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叠人大赛.html" target="_blank">
  叠人大赛
 </a>
 ，周日在西班牙北部举行，吸引了6千人观看。
 <br/>
 <br/>
 今年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_叠人大赛.html" target="_blank">
  叠人大赛
 </a>
 （Castell）在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_加泰隆尼亚.html" target="_blank">
  加泰隆尼亚
 </a>
 （Catalonia）的塔拉哥纳市（Tarragona）举行，共42个队伍参加，吸引6000人观看。
 <br/>
 <br/>
 比赛紧张刺激，要叠六到十层，具备相当的难度。
 <br/>
 <br/>
 最底下的基层，通常需要上百人支撑和稳固，到越顶端人数越少，顶层的人需要具备轻盈和灵活的条件，通常由小孩担任。
 <br/>
 <br/>
 各参赛队伍成员组成复杂的人塔后，要再回到地面的原先位置才算完成。
 <br/>
 <br/>
 西班牙叠人塔类似于华人的叠罗汉，不但需要具备良好的团队合作、体力、平衡感，还需要具备勇气。
 <br/>
 <br/>
 今年，加泰隆尼亚的独立诉求，成为比赛的一大主题。
 <br/>
 <br/>
 加泰隆尼亚独立运动的旗帜，星旗，在大赛会场中展开，参赛者和观众合唱加泰隆尼亚国歌。
 <br/>
 <br/>
 西班牙两年一度的叠人大赛，已有三百年历史，被联合国教科文组织列为非物质文化遗产。
 <br/>
 <br/>
 新唐人记者 王愉贺 综合报导
</div>
