### 李克强东亚经贸协定谈判遇挫 中共组团抗美或泡汤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】中美贸易战改变世界格局。原本今年有望达成的〝东亚区域全面经济伙伴关系协定〞（RCEP）遭无限期拖延。中共本拟借RCEP组建排除美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_亚洲经济圈.html" target="_blank">
  亚洲经济圈
 </a>
 ，但目前亚太各国靠近中共的意愿已经明显降低。
 <br/>
 <br/>
 11月14日，中共国务院总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 在新加坡出席第二次RCEP领导人会议时表示，自从去年RCEP第一次领导人会议以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_RCEP谈判.html" target="_blank">
  RCEP谈判
 </a>
 任务完成度从去年的50%提升到今年的近80%。现在谈判已进入〝关键时期〞，各方要〝再接再厉〞，〝力争明年达成协定〞。
 <br/>
 <br/>
 不过，据新加坡《联合早报》14日报导，RCEP的18个章节的谈判中，目前只完成了7个。报导指，如果谈判再久拖不决，RCEP将面临失去可信度与支持的危险。
 <br/>
 <br/>
 RCEP谈判从2012年开始，是中共力推的贸易协议。其成员国包括中国、日本、韩国、印度、澳大利亚、新西兰和10个东南亚国家，涵盖全世界45%的人口，经济产出占全球的三分之一。
 <br/>
 <br/>
 《华尔街日报》15日报导称，在川普政府遏制全球化的时代，中共试图把自己塑造成〝自由贸易和多边主义的卫士〞。但是，要把这个口号转化为具体结果很难。RCEP谈判无法在今年完成，意味着北京还拿不出一项刻上自己标记的重要协议。
 <br/>
 <br/>
 报导指，目前中美贸易战的影响已经显现，北京的一些亚洲贸易伙伴开始抵制中方投资，以避免自己因此背上沉重的债务或其它负担。
 <br/>
 <br/>
 日本《时事通信》15日报导，参与RCEP谈判的16国未能统一步调，因而放弃了年内达成实质协议的目标。报导指，印度对取消关税后中国商品冲击印度市场的担心，是达成协议的最大障碍。
 <br/>
 <br/>
 不过，正在和印度、美国一起推进〝印度洋太平洋战略〞的日本，在决定是否和中共一起站队时，似乎也同样会考虑美国的感受。
 <br/>
 <br/>
 此前，因为中共的军事威胁、政治操纵和经济捆绑，亚太周边国家不得不承认中共在该地区的霸权，对中共做出不同程度的妥协。而随着美国公开反击中共，这些国家将获得一次摆脱中共阴影的机会。
 <br/>
 <br/>
 日前，正在亚洲访问的美国副总统彭斯对印太国家喊话称，美国将会在各方面支援该地区，包括反恐、对朝鲜施压、捍卫航行及飞行自由等。彭斯并表示，〝帝国主义及侵略行为〞在印太地区没有立足之地，明显有警告中共之意。
 <br/>
 <br/>
 此外，彭斯还和日本首相安倍晋三发表联合声明，宣布美日将投入700亿美元资金，支持印太地区的社会基础设施开发等。外界认为，此举旨在打击中共的〝一带一路〞计划。
 <br/>
 <br/>
 （记者综合穆峰宇报导/责任编辑：明轩）
