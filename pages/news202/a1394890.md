### 英国脱欧不顺 苏格兰再次发出独立呼声
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】脱离
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 ，让英国政府两头失火。一方面，和欧盟的谈判陷入僵局，另一方面，因为脱欧不顺利，引起内部纷争，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏格兰.html" target="_blank">
  苏格兰
 </a>
 再次提出，要争取
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_独立.html" target="_blank">
  独立
 </a>
 。
 <br/>
 <br/>
 英国治下的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_苏格兰.html" target="_blank">
  苏格兰
 </a>
 和北爱尔兰，在2016年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 公投时，选择留在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欧盟.html" target="_blank">
  欧盟
 </a>
 。
 <br/>
 <br/>
 周二，苏格兰地方政府首席部长斯特金表示，对苏格兰来说，英国脱欧对苏格兰来说是个严重的问题。
 <br/>
 <br/>
 苏格兰首席部长 斯特金：〝我们的未来不在我们自己手中，苏格兰的未来在（英国政府）手中，唯一的解决方案就是（苏格兰）成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_独立.html" target="_blank">
  独立
 </a>
 国家。〞
 <br/>
 <br/>
 脱欧谈判进行两年，商贸关系和北爱边境这两个核心问题仍没有解决方案，这让苏格兰人失去耐心。
 <br/>
 <br/>
 连线本台欧洲记者 王羽音：〝苏格兰300年前正式成为英国的一部分，要求独立的呼声一直没有断过。2014年苏格兰进行了第一次独立公投，没有成功。现在英国脱欧陷入混乱，可以说帮了苏格兰独立派一个大忙。上周日的民意调查显示，苏格兰人支持独立的比例已经升高到了50%。但是英国首相认为脱欧谈判过程中举行公投〝不是时候〞，所以多次拒绝了苏格兰二次独立公投的要求。脱欧问题再加上苏格兰独立，英国政府现在可以说是焦头烂额，预计会有更多的英国人从支持脱欧，转而希望留在欧盟，结束这些麻烦。〞
 <br/>
 <br/>
 新唐人电视综合报导
</div>
