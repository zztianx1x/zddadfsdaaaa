### 印度钢铁厂煤气管道爆炸 已知9死14伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度.html" target="_blank">
  印度
 </a>
 恰蒂斯加尔邦一处钢铁厂，当地时间周二（9日）上午发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_煤气管道.html" target="_blank">
  煤气管道
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 事故，造成至少9死14伤。初步调查，事发原因是连接焦炉的煤气管道爆炸，但详细原因还在进一步调查中。
 <br/>
 <br/>
 据《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_印度.html" target="_blank">
  印度
 </a>
 报》（Hindu）报导，事发于周二上午10时45分左右，印度最大的钢铁公司，恰蒂斯加尔邦（Chhattisgarh）比莱（SAIL Bhilai）钢铁厂突然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爆炸.html" target="_blank">
  爆炸
 </a>
 ，该厂距离恰邦首府赖普尔约30公里，隶属于国有印度钢铁管理局。
 <br/>
 <br/>
 当地警方表示，警方和救援人员立即赶赴事发工厂，9人当场遇难，另外14人则送医急救，其中5人伤势严重。
 <br/>
 <br/>
 钢铁厂称，事发时工人正在做定期维护。
 <br/>
 <br/>
 印度工会发言人称，过去两天天然气管道的压力不均衡，需要维护。维护工作按照协议进行，但当管道中的接头打开时，发生爆炸导致火灾。爆炸发生在离地面约130米的地方。
