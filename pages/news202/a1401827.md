### 纳瓦罗透露川习会细节：前30分钟中方只有习近平说话
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】周一（12月3日），白宫贸易顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 （Peter Navarro）接受美媒采访，透露了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20峰会.html" target="_blank">
  G20峰会
 </a>
 后〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞的一些细节，并指出，美方除了给中共90天时间去完成其早该做的事情之外，并没有做出任何让步。
 <br/>
 <br/>
 周一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纳瓦罗.html" target="_blank">
  纳瓦罗
 </a>
 接受了美国全国公共广播电台（NPR）的电话采访，回顾了在布宜诺斯艾利斯（Buenos Aires）举行的晚宴上，川普总统与中共领导人习近平及其他中方代表谈话的一些细节。
 <br/>
 <br/>
 纳瓦罗表示，双方就整个结构性问题进行了讨论，包括中共强制技术转让、知识产权保护、非关税壁垒、网络入侵及网络盗窃、服务和农业等诸多方面。
 <br/>
 <br/>
 对于NPR主持人询问——晚宴上中方是否重视解决美中贸易争端问题，纳瓦罗回答，一开始主要是习近平在谈，他是前30分钟里中方唯一在谈话的人。
 <br/>
 <br/>
 纳瓦罗说，习近平就加大购买美国商品、保护知识产权、强制技术转让等许多问题做出了承诺。对于美中媒体对会谈内容的报导为何不太一致，他表示这并不奇怪，因为中共是弱势的一方，中共媒体传递的消息是有差异。但当天会谈结束时，双方互相传递的信息已非常明确，即到90天结束时，美方需要看到中方在贸易问题上做出的实质性、结构性的改变。
 <br/>
 <br/>
 纳瓦罗表示双方讨论了汽车关税问题，并指出当前两国的汽车贸易是不公平的——中共向美国征收40%的汽车进口税，而美国对进口汽车只征收2.5%的关税。
 <br/>
 <br/>
 纳瓦罗说，关于汽车关税的问题，将由美国贸易代表莱特希泽（Robert Lighthizer）负责谈判，他是最难对付的谈判对手，将负责与中共进行关于降低关税方面的谈判。
 <br/>
 <br/>
 最重要的是，纳瓦罗指出，美方在〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞上没有做出任何让步，只是给了中共90天时间，让他们去做过去20年来早就该做的事情而已。
 <br/>
 <br/>
 （记者李佳欣编译报导/责任编辑：东野）
