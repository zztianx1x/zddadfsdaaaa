### 习访西班牙 法轮功展〝真善忍〞吁制止迫害
------------------------

<div class="wysiwyg">
 【新唐人2018年11月28日讯】中国领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在G20峰会前到访
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 ，停留两天。周三在马德里议会外，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员展开〝法轮大法好〞和〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_真善忍.html" target="_blank">
  真善忍
 </a>
 〞的横幅，呼吁停止在中国发生的迫害。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_西班牙.html" target="_blank">
  西班牙
 </a>
 议会外，当地的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员高举展板，讲述在中国发生的残酷迫害。一些打着中共血旗的人试图阻挡。
 <br/>
 <br/>
 当地警方在现场维持秩序。
 <br/>
 <br/>
 法轮功学员还打出〝法办迫害元凶江泽民〞的横幅，要求结束1999年以来持续近二十年的迫害。
 <br/>
 <br/>
 西班牙法轮大法协会发言人CARMEN ECHEVESTE：〝的确
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 没有下令迫害法轮功。但是他是有责任的，因为他是现在的执政者。我们在这里呼吁，他应该给良心犯自由，尤其是法轮功学员、藏人、还有基督徒。〞
 <br/>
 <br/>
 法轮功学员：〝他需要停止对法轮大法修炼者的迫害。〞
 <br/>
 <br/>
 当地学员也呼吁西班牙政府，不要因为利益而放弃普世的准则。
 <br/>
 <br/>
 新唐人电视综合报导
