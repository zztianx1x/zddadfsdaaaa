### 合游戏庆典“G-STAR 2018&#39;开幕
------------------------

 【新唐人2018年11月16日讯】15日，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G-STAR 2018.html" target="_blank">
  G-STAR 2018
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际游戏展.html" target="_blank">
  国际游戏展
 </a>
 在韩国第二大城市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_釜山.html" target="_blank">
  釜山
 </a>
 开幕，有来自36个国家的企业参展，请看韩国发来的报导。
 <br/>
 <br/>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G-STAR 2018.html" target="_blank">
  G-STAR 2018
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际游戏展.html" target="_blank">
  国际游戏展
 </a>
 在韩国最美的港口城市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_釜山.html" target="_blank">
  釜山
 </a>
 会展中心BEXO开幕。
 <br/>
 <br/>
 釜山市自2009年开始举办的〝G-STAR 2018〞，今年已迎来了第10届。
 <br/>
 <br/>
 釜山市市长吴巨敦出席了开幕式，并称，将不遗余力的推动釜山发展成一个游戏产业城市。
 <br/>
 <br/>
 本次展示会将持续到18日，来自36个国家的共689家企业参展，开幕第一天人山人海，盛况空前。
 <br/>
 <br/>
 游戏如今已成为一种受到各年龄层欢迎的娱乐文化，广泛地进入了我们的生活。
 <br/>
 <br/>
 金錀溢釜山市文化福祉振兴室室长：〝我们计划将展会发展成，游戏商务和民众参与为一体的综合性庆典。〞
 <br/>
 <br/>
 这是一个综合性的游戏展示会，参展者可接触到多种多样的大型游戏产品。了解国际游戏趋势，开展商务，了解
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_游戏文化.html" target="_blank">
  游戏文化
 </a>
 。这一展会受到了游戏产业企业及游戏爱好者的广泛关注。
 <br/>
 <br/>
 新唐人记者孔英和韩国釜山报导
