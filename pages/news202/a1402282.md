### 加总理对逮捕孟晚舟事先知情 否认牵连政治（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】加拿大总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特鲁多.html" target="_blank">
  特鲁多
 </a>
 （Justin Trudeau）周四（12月6日）表示，他在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司首席财务官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 被逮捕前几天即已知情，但加拿大政府与该案并无政治牵连，这是有关部门做出的司法决定。
 <br/>
 <br/>
 据加拿大广播公司（Canadian Broadcasting Corporation, CBC）6日报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_特鲁多.html" target="_blank">
  特鲁多
 </a>
 周四表示，加拿大执法部门逮捕
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 只是基于司法原因，没有政治目的。这是特鲁多首次就孟晚舟被捕做出评论。
 <br/>
 <br/>
 上周六（12月1日），加拿大当局在温哥华国际机场逮捕了孟晚舟。保释听证会将于周五召开，但由于孟晚舟要求签署了〝禁止报导令〞，加拿大司法部依法拒绝透露案件细节。
 <br/>
 <br/>
 特鲁多在蒙特利尔（Montreal）告诉记者，关于这一案件的处理是由有关当局决定的，但他们没有参与或干预政治。他们提前几天通知特鲁多说，这是工作。
 <br/>
 <br/>
 当被问及他是否向中共的总理和大使谈及此案时，特鲁多表示，他没有。他指出，周五就要举行孟晚舟的保释听证会，所以他拒绝进一步提供细节。
 <br/>
 <br/>
 周四，中共外交部发言人对孟晚舟被逮捕一事表示谴责，但在听证会召开前，美加似乎都不准备就案件细节进行解释。
 <br/>
 <br/>
 按时间推算，在阿根廷参加G20峰会时，特鲁多就已对该案知情。而正是在逮捕孟晚舟的12月1日，美国总统川普与中共领导人习近平共进晚餐并达成了暂时性贸易战停火协议。
 <br/>
 <br/>
 根据早前的报导，逮捕身为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司创办人任正非的女儿、华为副董事长的孟晚舟，可能是因为她与美国调查违反对伊朗的制裁有关。美国要求引渡孟晚舟。
 <br/>
 <br/>
 美国方面，国家安全顾问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_博尔顿.html" target="_blank">
  博尔顿
 </a>
 （John Bolton）周四在接受美国全国公共广播电台（NPR）采访时也承认，他预先知道这一将要发生的逮捕事件。但他拒绝应记者的要求做出评论。
 <br/>
 <br/>
 相反，他针对华为的情况表示，他对中共企业的商业行为及其作为政府〝武器〞可能实行的运作非常担忧。
