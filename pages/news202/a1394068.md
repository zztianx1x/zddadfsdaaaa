### 南海军舰对峙时机蹊跷   美智库：中共军委直接下令
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】美国军方表示，一艘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军舰.html" target="_blank">
  军舰
 </a>
 日前在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海.html" target="_blank">
  南海
 </a>
 逼近一艘美国军舰，最近距离仅约40米，此举迫使美国军舰采取紧急措施，以避免相撞。美智库认为，中共军舰敢做出这么危险的动作，是因为收到了高层的直接命令。
 <br/>
 <br/>
 据美国有线电视新闻网（CNN）报导，上周日（9月30日），美国海军驱逐舰〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迪凯特号.html" target="_blank">
  迪凯特号
 </a>
 〞（USS Decatur DDG-73）正在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海.html" target="_blank">
  南海
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南沙群岛.html" target="_blank">
  南沙群岛
 </a>
 附近行使自由航行权。
 <br/>
 <br/>
 根据美方公布的消息，〝迪凯特号〞在驶入南沙群岛的南薰礁（Gaven）和赤瓜礁（Johnson）12海里内海域时，一艘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 驱逐舰驶近，双方距离仅45码 （约41米）。中共此举迫使美舰改变航道。
 <br/>
 <br/>
 CNN报导引述美国太平洋舰队发言人布朗（Charles Brown）的声明说：〝一艘（中共）旅洋级驱逐舰在南海南薰礁附近以不安全、不专业的调动方式，接近迪凯特号。〞
 <br/>
 <br/>
 布朗指出，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军舰.html" target="_blank">
  军舰
 </a>
 〝进行一系列越来越具有挑衅性的动作，同时警告迪凯特号离开该海域〞。这艘军舰逼近到迪凯特号前方45码处，迪凯特号因而采取紧急措施以避免碰撞。

 布朗在声明中也表示：〝美国海军船舰和飞机定期航行于印太地区，包括南海。正如过去几十年来一样，我们的部队将在国际法允许的任何地方继续飞行、航行和行动。〞
 <br/>
 <br/>
 中共国防部10月2日已经承认，中方曾派出军舰靠近美舰，以〝警告〞其离开。
 <br/>
 <br/>
 <h4>
  美智库：中共军委直接下令
 </h4>
 <br/>
 拥有12年航海经验的前美国海军舰长、现任夏威夷太平洋大学（Hawaii Pacific University）教授舒斯特尔（Carl Schuster）表示，船舰如此接近，〝这是非常危险的。〞
 <br/>
 <br/>
 他说：〝当船隻进逼千码时，船长们就会绷紧神经〞，何况上述距离只容许美舰长在数秒内做出反应，改变航道。他认为，中共似乎正对美舰采取更〝具挑衅性的政策〞。
 <br/>
 <br/>
 美国智库战略与国际问题研究中心的葛来仪（Bonnie Glaser）针对中共〝兰州舰〞驶近美舰〝迪凯特号〞表示，〝如果没有直接的命令，任何舰长不会做出这样的动作——靠近其它战舰到45码距离。〞
 <br/>
 <br/>
 她指出，中方舰艇的接触规则是由中共中央军委直接决定。此前，中共军委要求舰长在跟随美舰的时候，不要靠得太近，而此次事件显示，中共的〝规则〞似乎发生了变化。
 <br/>
 <br/>
 葛来仪说：〝我只能理解为，中共决定借此机会向川普（特朗普）政府发出一个信号；这也是一个地区的紧张态势扩散到另一个区域的证据。〞
 <br/>
 <br/>
 她认为，这次事件的时机很〝巧合〞，是在美舰正在进行〝自由航行〞靠近到中国岛礁12海里后正在驶离的过程中。而如果中共军舰是在美舰靠近12海里范围之前就采取行动，那么美舰很可能会被迫放弃此次自由航行的剩余部分，但也有可能会出现更危险的情况。
 <br/>
 <br/>
 也就是说，中共选择的出舰时机，既能够达到挑衅美军的目的，又自认不会导致中美关系进一步恶化，还可以制造出美舰被〝驱离〞的假象。
 <br/>
 <br/>
 <h4>
  中美关系近期持续陷入僵局
 </h4>
 <br/>
 伴随贸易战的升级，中美关系也日趋步入僵局。此前的9月27日，美国海军〝黄蜂号〞两栖攻击舰（USS Wasp）以及舰上的海军陆战队第31远征部队，在南海实施两栖部队防卫训练；
 <br/>
 <br/>
 9月26日，美军B-52轰炸机飞越南海和东海；
 <br/>
 <br/>
 9月24日，美国开始对价值2000亿美元中国商品加征10%关税；同天，美国国务院批准总额3.3亿美元对台军售案，这也是川普总统上台后宣布的第二笔对台军售；
 <br/>
 <br/>
 9月20日，美国国务院宣布，中共国防部因购买俄罗斯苏霍伊Su-35战斗机和S-400地对空导弹系统，因此对中共军委装备发展部部长李尚福（中将）祭出金融制裁，禁止其使用美国金融系统和进行外汇交易，冻结其在美国管辖范围内的任何财产或利益，以及禁止持有美国签证。
 <br/>
 <br/>
 对于美军的上述动作，中共国防部和外交部发言人均反复表态：〝坚决反对〞、〝严正抗议〞。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
