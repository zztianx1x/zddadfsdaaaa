### 中共红歌《洪湖赤卫队》在澳遭抵制
------------------------

<div class="wysiwyg">
 【新唐人2018年10月31日讯】宣传中共暴力文化的红色政治歌剧《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪湖赤卫队.html" target="_blank">
  洪湖赤卫队
 </a>
 》计划到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 上演，当地一个华人团体为守护澳洲传统价值，发起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抵制活动.html" target="_blank">
  抵制活动
 </a>
 ，得到很多人的响应，请看报导。
 <br/>
 <br/>
 红色歌剧《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_洪湖赤卫队.html" target="_blank">
  洪湖赤卫队
 </a>
 》，由中共湖北省政府独资拥有的〝湖北省演艺集团〞主办，由澳丰文化承办，计划于11月上旬在澳大利亚悉尼和墨尔本两个城市演出，受到〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲价值守护联盟.html" target="_blank">
  澳洲价值守护联盟
 </a>
 〞团体的抵制。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_澳洲.html" target="_blank">
  澳洲
 </a>
 价值守护联盟发起人钟锦江：〝现在大概有2000多人签名，联名反对洪湖赤卫队的演出。我们这次站出来，就是向更多的人表明我们是不接受他们那样的东西的。
 <br/>
 <br/>
 原首都师范大学副教授、历史学者李元华：〝在中国大陆大家都把它称为‘红色经典’，也就是宣传中共趁国难篡权这个过程当中的一段历史。中共其实把一个很丑陋的，就是普世价值或者任何一个正常人不能容许、不能容忍的一个恶事，就是杀人放火、强奸绑架、绑票，这种活动以艺术形式包装一下。实际上是一种共产文化在全球的散布，在毒害自由社会的百姓。〞
 <br/>
 <br/>
 悉尼麦考瑞大学中国问题专家Kevin Carrico认为《洪湖赤卫队》是中共在输出意识形态，侵蚀澳洲的价值观。
 <br/>
 <br/>
 悉尼麦考瑞大学中国研究讲师Kevin Carrico：〝实际上，这部歌剧本身传递了很深刻的政治信息，这些信息在我看来令人不安。并且不符合澳洲这里构成核心自由价值观的自由民主理念。〞
 <br/>
 <br/>
 澳洲价值守护联盟发起人种锦江：〝他要想把这套东西对全世界输出，那将威胁到世界的和平与发展，这是我们坚决要抵制的。〞
 <br/>
 <br/>
 原首都师范大学副教授、历史学者李元华：〝歌剧院是澳洲文化的一个代表，那么任何有悖澳洲价值观的，或者违背澳洲利益的演出一定要杜绝，不管它以何种形式伪装进来。〞
 <br/>
 <br/>
 澳洲价值守护联盟还表示，如果演出按计划进行，他们会在悉尼歌剧院外举办抗议活动。
 <br/>
 <br/>
 新唐人悉尼记者站报导
</div>
