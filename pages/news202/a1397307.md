### 意大利强风暴雨酿6死 威尼斯7成地区淹大水
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 近期受到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强风暴雨.html" target="_blank">
  强风暴雨
 </a>
 侵袭，大部份地区处于警戒状态，学校停课，至今造成6人丧命；有水都之称的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威尼斯.html" target="_blank">
  威尼斯
 </a>
 ，逾70%地区淹水，水位更涨至历史罕见的156公分。
 <br/>
 <br/>
 综合媒体报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强风暴雨.html" target="_blank">
  强风暴雨
 </a>
 侵袭下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_威尼斯.html" target="_blank">
  威尼斯
 </a>
 水位连架高的人行道也被淹，游客和居民须涉水而行，著名景点圣马可广场（San Marco Square）禁止全身湿透的游客进入，水上巴士服务也暂停。
 <br/>
 <br/>
 虽然威尼斯市民对淹水已经习以为常，但这次淹水非同小可。威尼斯当地政府表示，威尼斯周遭水域的〝高水位〞在29日下午稍早前涨至156公分。据了解，威尼斯历来只有5次高过150公分。1966年，洪水侵袭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 全国，当时威尼斯水位高达194公分。
 <br/>
 <br/>
 <center>
  <a href="http://imgs.ntdtv.com/pic/2018/10-30/p9102863a780537137.jpg" target="_blank">

   </font>
  </a>
 </center>
 <br/>
 而受到暴风影响，意大利北部几乎所有地区都处于警戒状态，强风风速高达每小时100公里，部分地区的降雨量相当于下了好几个月。
 <br/>
 <br/>
 威尼斯所在的威尼托大区（Veneto），以及贝卢诺省（Belluno）、特雷维索省（Treviso），近16万户断电。威尼托地区首长柴亚（Luca Zaia）表示，灾情可能比2010年的大洪水还糟。
 <br/>
 <br/>
 当局表示，距离首都罗马不远的地方，有树木倾倒且压到一辆汽车，造成车内1男1女死亡，另外2名罹难者分别位于罗马南方与那不勒斯（Naples）地区。
 <br/>
 <br/>
 同时在意大利西北部城市沙伏纳（Savona）附近，一块建筑物的檐口掉落砸死1名老妇人。在南部卡拉布里亚区（Calabria），1人在暴风中驶船入卡坦札罗港时，船撞向码头后失踪。
 <br/>
 <br/>
 另外，热那亚（Genoa）、罗马和威尼托，还有北部数个城镇及西西里岛墨西拿港市（Messina）的学校都已停课。
 <br/>
 <br/>
 布伦纳山口（Brenner Pass）附近来往意大利与奥地利的铁路及公路，28日、29日都暂时关闭。米兰与博洛尼亚之间的公路局部淹水。
