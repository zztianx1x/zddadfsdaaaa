### 川普突然〝唱红脸〞 北京心里没底做最坏打算
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞即将在本月底举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会上演。川普表示，他对会谈准备充分，暗示美中双方达成协议的可能性，并再次强调他与习近平之间的私人关系。同时有消息传出，中共近日疑向全国各主要政府单位发布指令，要求各级政府做好最坏打算应对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 带来的经济和社会问题。
 <br/>
 <br/>
 川普今年在佛罗里达州的海湖俱乐部（Mar-a-Lago）度感恩节假期，他上午透过电话向驻扎海外的美军致意后，回答随行记者的问题。川普谈到下周将赴阿根廷出席20国集团（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 ）峰会，并与习近平会谈，他说：〝我一生都在为此做准备。〞
 <br/>
 <br/>
 川普在谈论他的准备工作时表示，他已作好充分准备：〝我知道每项要素、每个统计数字。我比所有知道这些东西的人更加了解。我的直觉向来正确。〞
 <br/>
 <br/>
 他说：〝中国希望（就关税大战）达成协议。如果我们能找到共识，我们会这么做。〞
 <br/>
 <br/>
 川普还强调，他和习近平〝关系很棒〞，〝我非常喜欢他。我认为他喜欢我。可能现在喜欢我的程度，不若以前我们相处时那样。〞
 <br/>
 <br/>
 外界预计，川普和习近平将在阿根廷举行的G20峰会期间举行中美贸易峰会，就中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 是否休战进行谈判，但目前传出的消息似乎对北京不利。
 <br/>
 <br/>
 美国贸易代表莱特希泽20日表示，中共没有改正其不公正，不理性，引发市场违规的贸易政策。相反，美方最新报告显示，中共〝在最近几月，明显地采取了非理性的举措〞。
 <br/>
 <br/>
 《法广》的报导则披露，美国商务部工业和安全局于19日发布了一份技术出口管制先期通知。这个方案被认为是历来最严格的技术出口管制方案，方案拟管制包括人工智慧、晶元、量子计算、机器人、面印和声纹技术在内的14项涉及国家安全和前沿科技的技术出口，并就这一方案向公众征询意见。中共对这一方案的潜在性后果十分担心。
 <br/>
 <br/>
 另据《自由亚洲电台》报导，随着大陆经济持续恶化，加上亚太经合会（APEC）峰会未能有所斩获，中共中央压力倍增。近日北京向全国厅级以上政府机关发布文件，指接下来，国内或将出现一波企业倒闭潮，股市和房市也会有所波动，进而引发社会矛盾，要求全国厅级以上政府机关在中共的领导下，自力更生战胜困难。
 <br/>
 <br/>
 报导引述多名来自中国国营企业、媒体界和法律界的观察人士指出，倘若消息属实的话，就代表当前的情况相对严峻，而中共中央对月底的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 〞抱持悲观态度。
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：戴明）
