### 孟宏伟妻受恐吓？法国宣布调查结果
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】国际刑警组织前主席、中共公安部前副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 返回中国被捕后，身在法国的孟妻报警说，自己受到恐吓，警方随后介入调查。14日， 一名匿名法国官员披露，警方调查已结束，但找不到有人曾在法国犯罪的任何证据。
 <br/>
 <br/>
 11月14日，美联社引述一名不愿透露姓名的法国官员的话说，里昂警方已完成有关
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 妻子受恐吓的调查，但找不到任何证据证明有人曾在法国犯罪，目前调查已结束。
 <br/>
 <br/>
 孟宏伟9月29日从法国前往中国后神秘消失，孟妻10月5日向法国警方报案后，立即引发国际舆论界的哗然。
 <br/>
 <br/>
 在国际舆论的压力之下，中纪委10月7日深夜通报，孟宏伟涉嫌违法，正接受调查。
 <br/>
 <br/>
 中共公安部官网8日即于头条位置刊文称，公安部部长赵克志当日凌晨，紧急召开公安部党委会议，就孟宏伟被调查一事进行通报，并宣布成立工作组，查处孟的同党。
 <br/>
 <br/>
 值得关注的是，孟宏伟被调查后，身在法国里昂的孟妻葛瑞丝‧孟（Grace Meng）不但高调召开记者会为孟发声，她还接连接受美联社、CNN等媒体采访。
 <br/>
 <br/>
 孟妻说，孟宏伟失踪后，她接到陌生男子的威胁电话。该男子说，他们有〝两组人马〞已经过来，目标就是她。
 <br/>
 <br/>
 孟妻还透露，她和孟宏伟有两个7岁的双胞胎儿子，她现在和媒体联系，是因为担心一家人的安全，她需要得到帮助等。
 <br/>
 <br/>
 就在她接受采访期间，她自称自己接到3个来自中共驻法国领事馆的电话，其中一个电话是副总领事打来的，但她拒绝单独与他们见面。
 <br/>
 <br/>
 随后，法国警方就事件展开调查，并派员对Grace Meng母子进行了安全保护。
 <br/>
 <br/>
 而陆媒随后对孟妻大起底，挖出她真名叫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高歌.html" target="_blank">
  高歌
 </a>
 ，1972年出生，以〝小三〞身份坐正，与孟宏伟在年龄上相差整整一代，以及其〝吃空饷〞日均挥金过万人民币，年赚至少500万元人民币等丑闻也被曝光。
 <br/>
 <br/>
 据悉中共官方以涉贪为由，要求法国引渡孟妻。流亡法国的大陆民运人士张健曾在推特披露，中共把孟宏伟夫妇涉嫌贪腐的资料，传给法国警方和外交部，要求将孟妻引渡回国。中共还要求孟妻〝闭嘴换平安〞。
 <br/>
 <br/>
 孟妻10月份对媒体表示，她不确定丈夫还活在人世，因为对方（中共当局）的手段非常〝残忍及肮脏〞。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
