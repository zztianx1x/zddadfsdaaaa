### 46名中国人突然在日本北海道失踪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月03日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_日本北海道.html" target="_blank">
  日本北海道
 </a>
 知内町
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地盘.html" target="_blank">
  地盘
 </a>
 ，约有10名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国黑工.html" target="_blank">
  中国黑工
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被捕.html" target="_blank">
  被捕
 </a>
 。3日。地盘相关人士表示，在同一地盘工作过的共有58名，除1人死亡外，还有46名中国人，目前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下落不明.html" target="_blank">
  下落不明
 </a>
 。
 <br/>
 <br/>
 12月3日，日本共同社报导说，日本警方在11月26日对北海道一建筑工地工作的11名中国人实施了逮捕，这11名年龄由27至62岁的中国人，涉嫌逾期逗留或没持有护照，目前警方正在调查雇主是否得知疑犯非法滞留日本。
 <br/>
 <br/>
 日本警方称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被捕.html" target="_blank">
  被捕
 </a>
 的11名中国人涉嫌违反《出入境管理及难民认定法》。
 <br/>
 <br/>
 相关人士透露，这11名中国人于2018年6月以后，持短期滞留签证等从成田机场或长崎港等口岸入境日本，并经由千叶县一家公司派遣，前往与北海道木古内町相邻的知内町的太阳能发电建筑工地工作。
 <br/>
 <br/>
 该工地原承包公司负责人说，还有46名中国人被派往同一工地，但这46人都已经都不在宿舍了，去向不明。
 <br/>
 <br/>
 该负责人续说，〝我们从千叶县的公司那里得到，并检查过这些人的滞留资格等证明的复印件〞。警方认为，此事或涉及有组织非法工作的违法行为，正在开展调查。
 <br/>
 <br/>
 日媒称，事发的太阳能发电设施建设
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_地盘.html" target="_blank">
  地盘
 </a>
 ，共有58名中国人打工，除了有一人于11月25日病故11人被捕后，余下46人突然失踪。
 <br/>
 <br/>
 报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_游客失踪.html" target="_blank">
  游客失踪
 </a>
 的案件在日本九州岛十分常见。每年有大量外国游客在登陆后相继失踪，据日本出入境管理局和海上保安厅数据称，福冈与长崎截至上个月，半年里已有61名外国邮轮行游客失踪，其中有50人是中国人，11人是越南人。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
</div>
