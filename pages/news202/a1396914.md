### 大篷车继续北上 组织者改口不去美国
------------------------

<div class="wysiwyg">
 【新唐人2018年10月26日讯】首先来关注洪都拉斯
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 潮。越过
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_墨西哥边境.html" target="_blank">
  墨西哥边境
 </a>
 后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 日夜兼程继续
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北上.html" target="_blank">
  北上
 </a>
 ，人数已经过万。周五，大篷车组织者突然改口，声称他们的目的地不是美国，与难民之前的说法不同。川普周五再发推文，希望民主党不再阻挠，共同出台新的移民法。
 <br/>
 <br/>
 周四晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大篷车难民.html" target="_blank">
  大篷车难民
 </a>
 从墨西哥南部小镇马帕斯特佩克出发，连夜
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北上.html" target="_blank">
  北上
 </a>
 。
 <br/>
 <br/>
 在大篷车队的帮助下，部分
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_难民.html" target="_blank">
  难民
 </a>
 周五早上抵达皮希希亚潘小镇，在当地政府提供的庇护所休息。
 <br/>
 <br/>
 大篷车组织者对媒体表示，他们不是要去美国。
 <br/>
 <br/>
 大篷车难民协调人 Denis Contreras：〝我们离开洪都拉斯的时候，目标非常明确：到墨西哥寻求政治庇护。不知道为什么引起这么多恐慌。我不知道他们为什么要调遣军队到边境，唐纳德・川普，我们不是去美国。〞
 <br/>
 <br/>
 但难民表示，他们就是要去美国。
 <br/>
 <br/>
 洪都拉斯难民 Enrique：〝他们（墨西哥政府）应该准许我们通过，请支持我们。我们不是要来这里（举墨西哥国旗），我们是要去这里（举美国国旗）。〞
 <br/>
 <br/>
 洪都拉斯 难民 ：〝向川普问好。我们要去看望他。〞
 <br/>
 <br/>
 川普周四推文，呼吁大篷车难民返回家园，通过合法渠道取得难民身分。
 <br/>
 <br/>
 推文说：〝大篷车难民请转身回去。我们不会让人非法进入美国。回到你们的国家，如果你想来，就申请移民，像数百万其他申请人一样。〞
 <br/>
 <br/>
 川普还推文表示，将借鉴意大利的经验，关闭接收难民的港口，阻挡恐怖份子混在难民中入境。
 <br/>
 <br/>
 推文说：〝我百分百赞同他们的立场。美国也将对非法移民采取强硬立场。〞
 <br/>
 <br/>
 周五，川普再发推文说：〝美国在非法移民上每年花费数十亿美元。这不会再继续了。民主党必须投票支持更有力（但是公平）的法律。如果不行，我们将不得不采取更强硬的措施。〞
 <br/>
 <br/>
 据报，川普为阻止大篷车难民继续北上，考虑出台新的行政令，全面阻止任何人进入美国南部边境，包括难民申请者。
 <br/>
 <br/>
 新唐人记者安心、天羽综合报导
</div>
