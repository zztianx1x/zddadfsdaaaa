### 〝小齿轮〞也有罪责   95岁前纳粹看守被起诉
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】德国一名95岁的男子被查出年轻时曾在纳粹集中营担任看守，他涉嫌协助谋杀约3万6000名集中营里囚犯而被德国检方起诉。德国司法界认为，集中营的看守作为希特勒〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀人机器.html" target="_blank">
  杀人机器
 </a>
 〞上的〝小齿轮〞，同样对集中营里被杀害的人负有罪责。
 <br/>
 <br/>
 这名95岁的被告被指控，从1944年7月到1945年5月期间在纳粹集中营毛特豪森（Mauthausen）担任看守。他是该集中营〝党卫军骷髅头风暴队〞第16大队的成员，涉嫌协助、参与用毒气或射杀等手段残害犹太人，而在他担任看守期间，毛特豪森集中营至少杀害了36,223人。
 <br/>
 <br/>
 据德国媒体的报导，这名被告最初谎称自己当年是在距离毛特豪森集中营20公里的工厂当看守。不过调查人员经过仔细比对后发现，那家工厂历年的看守名单里都没有他的名字，却从毛特豪森集中营的看守名单中查出了他的名字，并在他的家中搜出了当年他穿着纳粹党卫军制服的照片
 <br/>
 <br/>
 虽然被告已95岁高龄，当年也只是担任集中营的看守，但从2011年以后，德国司法界对曾任纳粹工作者的审判原则达成了新的共识——即使是希特勒〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杀人机器.html" target="_blank">
  杀人机器
 </a>
 〞下的一个〝小齿轮〞，也应该被判有罪。
 <br/>
 <br/>
 德国司法界认为，这些看似微不足道的小角色对纳粹的支持，同样起到了帮助希特勒杀人机器顺利运转的作用，因此他们也必须担负其相应的罪责。而对于担任过集中营看守的纳粹军人，德国法院通常都会以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_协助谋杀罪.html" target="_blank">
  协助谋杀罪
 </a>
 〞来惩处量刑。
 <br/>
 <br/>
 据了解，就在今年8月21日，美国移民和海关执法人员经过14年的不懈努力，终于将最后一名生活在美国的前纳粹集中营的看守帕利吉（Jakiw Palij）从纽约带走，并驱逐回德国。
 <br/>
 <br/>
 这名95岁的男子被指控当年担任驻守波兰集中营的看守时，在成千上万的犹太人屠杀事件中扮演了〝不可或缺的角色。〞
 <br/>
 <br/>
 据公开的资讯，帕利吉是1949年移民来美国的，他隐瞒了自己的纳粹背景并于1957年成为美国公民。
 <br/>
 <br/>
 帕利吉2003年向美国司法部承认，他曾在德国占领波兰后建起的一个纳粹集中营接受训练。而法庭文件显示，在特拉尼基的党卫军训练营受训的人实施了纳粹政权杀害波兰犹太人的计划。
 <br/>
 <br/>
 2003年他的美国公民身份被联邦法官撤销，一年后法庭下达了驱逐令。他的上诉在2005年被驳回。但之后的遣返工作却进行得很困难，因为帕利吉不是德国公民，在失去美国公民身份后没有国籍，遣返德国缺乏法律上的依据。
 <br/>
 <br/>
 直到德国新任外交和内政部长上任后，才积极与川普政府合作，最终将帕利吉遣送回德国。
 <br/>
 <br/>
 (记者黎明综合报导/责任编辑：戴明）
