### 英相脱欧协议难过关 传40多名同党议员欲否决
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】距离正式
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧.html" target="_blank">
  脱欧
 </a>
 不到半年，英国将以何种方式退出欧洲联盟（EU）仍是未知数。英国国会议员贝克周二（9日）表示，如果首相特蕾莎．梅达成让英国〝半留半退〞欧盟的协议，她所领导的保守党至少有40名议员将投否决票。
 <br/>
 <br/>
 路透社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英国脱欧.html" target="_blank">
  英国脱欧
 </a>
 事务大臣拉布称，预计在下周欧盟领导人峰会上
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_脱欧.html" target="_blank">
  脱欧
 </a>
 磋商将取得一些进展。
 <br/>
 <br/>
 根据特蕾莎．梅的提案，英国将接受一套货物流通和参与欧盟货物授权机构的〝共同规则〞，藉此寻求和欧盟建立一个货物的自由贸易区。
 <br/>
 <br/>
 如果她能与欧盟敲定协议，还需要英国国会约320名议员支持才能通过。但保守党（Conservative Party）资深议员贝克（Steve Baker）周二告诉《BBC》，他估计，至少有40名同党议员不会接受‘半留半退’的契克斯协议（Chequers deal），或是一个确实让英国留在单一市场和关税同盟的保障。
 <br/>
 <br/>
 贝克过去曾任英国脱欧部政务次长，直到特蕾莎．梅（Theresa May）提出脱欧计划后辞去职务以示抗议。
 <br/>
 <br/>
 如果真有40名同党议员投下反对票，特蕾莎．梅的提案就只能靠在野的工党（Labour Party）支持。而工党之前就已表明，针对特蕾莎．梅可能达成的几乎任何协议，他们可能都将投反对票。
 <br/>
 <br/>
 英国脱欧公投已经过去两年多，脱欧谈判已经进入最后阶段。到目前为止，英国执政党和反对党均无法就如何脱欧给出清晰、有号召力的方案。
 <br/>
 <br/>
 特蕾莎．梅选择的脱欧之路，介于硬脱欧和软脱欧之间，她希望藉此统一国内不同声音，但却适得其反。而欧盟也强硬表示脱欧建议行不通。
 <br/>
 <br/>
 英国周一称，如果没有一个针对双方未来关系的框架协定，英国可能不会同意与欧盟的分手协议。而欧盟则表示，在英国没有行动前，欧盟不会有动作。双方都想要在10月17-18日的布鲁塞尔峰会上取得重大进展，但重点却截然不同。
 <br/>
 <br/>
 鉴于特雷莎．梅阵营内部对退欧条款存在严重分歧，本周欧盟可能会推迟提出他们有关退欧后的贸易关系的提议，转而关注自身的应对举措，包括对〝无协议退欧〞的应急计划。
 <br/>
 <br/>
 (记者李红报导/责任编辑：程以仁）
