### 斯里兰卡新总理不信任案 国会肢体冲突不断
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯里兰卡.html" target="_blank">
  斯里兰卡
 </a>
 国会14日通过对新总理拉贾帕克萨的不信任案，15日国会陷入暴力纷争，16日进行第2次不信任表决时，仍纷乱不断，支持新总理的议员向议长扔书本、辣椒酱和水瓶，试图阻挠。
 <br/>
 <br/>
 综合媒体报导，15日在议长贾雅苏瑞亚（Karu Jayasuriya）宣布，拉贾帕克萨（Mahinda Rajapaksa）和他的内阁遭不信任投票否决，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_斯里兰卡.html" target="_blank">
  斯里兰卡
 </a>
 现在没有政府。引发支持拉贾帕克萨的议员发难，将议长团团围住，最后爆发推挤冲突，过程中有人拿东西乱砸，造成10多人受伤，议会被迫中止。
 <br/>
 <br/>
 16日再次举行不信任表决，议长被一群议员阻挡，延迟近1小时进入国会，且必须在无武装警察和议会工作人员的护卫下才能入座。但议长仍遭拉贾帕克萨的支持投掷书本和文具，护卫人员试图保护议长。
 <br/>
 <br/>
 议员肆意破坏桌椅并攻击官员，拉贾帕克萨的部分支持者还向敌对议员和警察喷洒辣椒粉。
 <br/>
 <br/>
 遭罢黜的总理威克瑞米辛赫（Ranil Wickremesinghe）同党议员裴里拉（Gamini Jayawickrema Perera）的脸部遭喷洒辣椒水，被送到议会的医疗中心接受治疗。

 国会连续两天爆发全武行，政府无法运作，政治危机持续蔓延，加深国际社会的关注，德国驻斯里兰卡大使表示，这是〝不民主的〞。
 <br/>
 <br/>
 路透社报导，第2次表决结果再次对拉贾帕克萨（Mahinda Rajapaksa）及其新任政府不利，可能会为遭总统席瑞塞纳罢黜的总理威克瑞米辛赫开启复位之路。
 <br/>
 <br/>
 席瑞塞纳10月26日撤换威克瑞米辛赫，转而任命前强人拉贾帕克萨出任新总理，随后又宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_解散国会.html" target="_blank">
  解散国会
 </a>
 ，提早于明年1月举行大选。斯里兰卡最高法院13日裁定，此举违宪。
 <br/>
