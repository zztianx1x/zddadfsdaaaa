### 冰岛建鲸鱼休栖地 被虏鲸鱼回归自然
------------------------

<div class="wysiwyg">
 【新唐人2018年10月21日讯】世界第一座〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白鲸.html" target="_blank">
  白鲸
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自然保护区.html" target="_blank">
  自然保护区
 </a>
 〞在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰岛.html" target="_blank">
  冰岛
 </a>
 偏远的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克雷茨维克湾.html" target="_blank">
  克雷茨维克湾
 </a>
 〞开始动工。动物保护者对该项目表示称赞，并呼吁其它海洋公园，也能将鲸鱼和海豚都放归自然。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冰岛.html" target="_blank">
  冰岛
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_克雷茨维克湾.html" target="_blank">
  克雷茨维克湾
 </a>
 （Klettsvik Bay）位于冰岛南部海岸约7公里处的赫马岛（Heimaey），地理位置僻静，沿海水域凉爽，将成为世界第一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白鲸.html" target="_blank">
  白鲸
 </a>
 保护区。目的是将被人类捕获的鲸目动物安置在一个接近于它们天然生活的野生环境中。2019年春季，两头雌性白鲸——小白和小灰，将从9600公里远处的中国上海长风海洋世界，迁入这个完美的新家。它们在那里作为观赏动物已经被囚禁了9年。
 <br/>
 <br/>
 建造工程正在进行中。主要部分是有网隔开的32,000平方米、10米深的一个海湾。
 <br/>
 <br/>
 赫马岛知识中心负责人琼森（Palle Marvin Jonsson）：〝我们唯一需要做的就是设置一个小栅栏。当然我们想让（这个保护区）尽可能地接近自然，让它们感受到野生环境。所以最重要的就是尽量保持海湾干净，一直监控它们的安全。〞
 <br/>
 <br/>
 赫马岛是冰岛韦斯特曼群岛15个岛屿之一，居住着约4500岛民。当地居民对保护区可能带来的旅游热感到兴奋。
 <br/>
 <br/>
 韦斯特曼群岛市长（Iris Robertsdottir）：〝我们是个小岛，我们希望旅游业有新鲜的东西。我们有火山和美丽的大自然，所以我们认为，把（保护区）纳入韦斯特曼群岛这个整体当中，非常完美。〞
 <br/>
 <br/>
 冰岛的捕鲸业长期受到西方政府和国际非政府组织的批评。许多人希望保护区展示这个人口稀少的北大西洋岛屿的另一面。
 <br/>
 <br/>
 小白和小灰这两头白鲸将经陆海空3种途径，抵达它们的新家，整个旅程预计需要约30个小时。
 <br/>
 <br/>
 保护区是保护机构海洋生物信托（Sea Life Trust）和经营全球120个景点的梅林娱乐（Merlin Entertainments）合作的成果。
 <br/>
 <br/>
 海洋生物信托基金负责人安迪-布尔（Andy Bool）：〝我们希望确定其它水族馆中‘被囚的’白鲸，也让它们来保护区与小白和小灰作伴。目前世界各地的水族馆有300多头白鲸。〞
 <br/>
 <br/>
 在2013年反对圈养逆戟鲸（orca whales ）的纪记片〝黑鱼〞 （Blackfish）放映之后，以美国奥兰多为总部的海洋世界游客人数逐年下降。之后，〝海洋世界〞集团宣布不再繁殖虎鲸（killer whales），并停止表演节目。
 <br/>
 <br/>
 动物保护组织外联协调员艾欧娜（Theodora Iona）：〝通过发展海滨保护区，我们能够给这些动物提供一些他们被长期剥夺的自然生活。所以，对它们来説，绝对是一个更好的选择。〞
 <br/>
 <br/>
 冰岛保护区预计将在2019年3月完工。
 <br/>
 <br/>
 新唐人记者徐哲、雨薇综合报导
</div>
