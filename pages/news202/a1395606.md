### 传孟宏伟妻被秘密转移筹备发布会 北京欲引渡
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】前中共公安部副部长兼国际刑警主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 落马，其妻在法国里昂的公开活动成为舆论焦点。日前有未经证实的消息指，孟妻已被秘密转移到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎.html" target="_blank">
  巴黎
 </a>
 ，并准备下一场发布会，中共官方以涉贪腐要求法国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引渡.html" target="_blank">
  引渡
 </a>
 ，威胁孟妻闭嘴。
 <br/>
 <br/>
 10月16日，六四事件后流亡法国的民运人士张健，在其推特发布〝号外〞消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 妻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高歌.html" target="_blank">
  高歌
 </a>
 被欧美强力部门保护，正继续联系法国和欧美媒体，准备新的发布会。
 <br/>
 <br/>
 消息还指，中方把孟宏伟夫妇涉嫌贪腐的资料传给法国警方和外交部，要求将孟妻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引渡.html" target="_blank">
  引渡
 </a>
 回国。中共还威胁孟妻〝闭嘴换平安〞。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-17/p9083011a206381786.jpg" target="_blank">

  <br/>
 </center>
 <br/>
 10月14日，张健在其推特发布〝最新消息〞指，孟妻受到法国与美国强力情治机构保护，已离开国际刑警组织所在地里昂，被严密保护在法国首都
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_巴黎.html" target="_blank">
  巴黎
 </a>
 某处。孟妻还会有下一步行动。
 <br/>
 <br/>
 张健还感叹，法国最近总是国际大事的焦点。重庆市委原书记薄熙来别墅在这里，中国影视明星赵薇酒庄和范冰冰合伙人都在这里，海航原董事局董事长王健死这里。
 <br/>
 <center>
  <br/>

   <br/>

   </font>
  </a>
  <br/>
 </center>
 <br/>
 上述消息为张健独家发布，暂时未得到进一步证实。
 <br/>
 <br/>
 孟妻此前在里昂接受媒体采访，都不以真名和脸部示人。但日前陆媒揭露，其真名为高歌，小孟宏伟19岁，属于〝小三上位〞。和孟宏伟结婚后，高歌依仗孟的权势，在中国大陆、香港和海外担任多家企业高管，坐吃空饷，过着豪华奢侈的生活。
 <br/>
 <br/>
 港媒则披露，孟宏伟早已帮助妻儿获得海外国籍，成〝裸官〞多年。
 <br/>
 <br/>
 但孟妻接受采访时，极力替孟宏伟遮掩，称孟宏伟〝没有能力贪腐〞，并指孟〝毕生追求法治〞等。
 <br/>
 <br/>
 10月4日，孟妻在里昂向法国警方报案，称孟宏伟回中国后失踪。7日，孟妻向CNN公布了孟宏伟失联前通过手机发回的尖刀图案。同日，中共中纪委宣布孟宏伟落马。
 <br/>
 <br/>
 中共公安部称孟宏伟是〝周永康余毒〞。外界猜测，孟宏伟应该早已明白自身处境，事前已有安全预案，那个尖刀图案可能就是指示妻子下一步行动的代号。
 <br/>
 <br/>
 有猜测称，孟妻或手握中共高层海外资产的材料，或中共某些内部机密，以此作为和北京谈判的筹码。
 <br/>
 <br/>
 2012年，前重庆公安局长王立军只身闯入美国大使馆寻求庇护。美媒报导，王立军向美方提供了周永康和薄熙来密谋政变、以及中共活摘器官的大量证据。
 <br/>
 <br/>
 王立军是活摘器官的凶手，曾和辽宁医院共同发明用于活摘器官的脑干撞击机。而孟宏伟曾是周永康在公安部的直接下属，被指与中共镇压维权人士和法轮功、以及活摘器官脱不了干系。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
