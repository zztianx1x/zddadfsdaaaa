### 美首次引渡中共间谍开审 誓保美经济安全
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】美国司法部周三（10月10日）宣布，一名在比利时被捕的中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国安部.html" target="_blank">
  国安部
 </a>
 间谍9日已被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引渡.html" target="_blank">
  引渡
 </a>
 到美国，他被指控窃取包括美国国防部承包商在内的多家航空航天公司的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商业机密.html" target="_blank">
  商业机密
 </a>
 ，且周三已首次出庭受审。
 <br/>
 <br/>
 被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引渡.html" target="_blank">
  引渡
 </a>
 的间谍徐延军（Yanjun Xu，音译）是中共国家安全部下属的江苏省国安厅六局的一名副处长，曾以多个化名从事涉及美国军方商业情报的间谍活动。
 <br/>
 <br/>
 徐延军企图窃取通用电气航空公司（GE Aviation）的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商业机密.html" target="_blank">
  商业机密
 </a>
 ，于今年4月1日在比利斯被捕。
 <br/>
 <br/>
 通用电气航空公司是航空发动机研发的领先企业，也是美国国防部的承包商。
 <br/>
 <br/>
 法庭文件显示，从2013年起，徐延军以多种伪装身份接触美国航空界人士，并邀请这些专业人士赴中共参加〝学术交流〞。
 <br/>
 <br/>
 2013年12月26日，徐延军负责接待受邀赴华交流的美国航空界人士。在客人们抵达前，他发短讯告诉其他中共情报人员，他是以南京市科学技术协会副秘书长的身分与对方接洽的，要求中方人员帮他掩饰真实身份。
 <br/>
 <br/>
 去年年底，他不断邀约一名将因公赴欧的业界人士在比利时见面，并要求对方交付一些航空研发方面的机密资料。他之后还多次做出反常行为，引发了美方怀疑，美国联邦调查局（FBI）最终在今年4月从比利时把他诱补。
 <br/>
 <br/>
 FBI反间谍部门的助理局长普利斯塔普（William Priestap）在声明中指出，这次引渡中国情报官员事件说明，中共政府在直接监督中方人员在美国的经济间谍活动。
 <br/>
 <br/>
 周三，美国俄亥俄州辛辛那提（Cincinnati）的联邦法院已展开相关司法程序，徐延军已被一个大陪审团起诉并首次出庭。他面对4项指控，如果罪名成立，他可能面临至多可达25年的监禁及经济处罚。
 <br/>
 <br/>
 负责美国司法部国家安全事务的助理部长戴莫斯（John Demers）表示，这起案件并非孤立事件，而是中共以美国利益为代价发展经济的整体政策中的一环，但是，美国绝不会容忍其它国家窃取美国的实力和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_知识产权.html" target="_blank">
  知识产权
 </a>
 。
 <br/>
 <br/>
 中共商业间谍活动近来频频见诸美国媒体。多年来，中共一直利用间谍和黑客窃取美国的企业、学术和军事信息，并利用所获信息增强自身的经济实力和政治影响力。
 <br/>
 <br/>
 上个月， 美方在芝加哥逮捕了27岁的中国公民季超群（Chaoqun Ji，音译），指控他为一名中共情报官员秘密工作，帮其招募美国境内的工程师和科学家充当
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共间谍.html" target="_blank">
  中共间谍
 </a>
 。
 <br/>
 <br/>
 《华尔街日报》（The Wall Street Journal）报导，根据一份法庭文件，季超群和徐延军所属的江苏省国安厅有关联。
 <br/>
 <br/>
 FBI局长克里斯托弗‧雷（Christopher Wray）周三在参议院国土安全委员会的听证会上表示，在很多方面，中共是美国迄今所面临的最广泛、最复杂、最持久的反间谍威胁。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
