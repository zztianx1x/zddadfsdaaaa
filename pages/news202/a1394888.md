### 中共涉影响舆论 对美安全威胁超俄罗斯
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】美国国土安全部长尼尔森认为，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 正在以前所未有的力度来影响美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舆论.html" target="_blank">
  舆论
 </a>
 。FBI局长弗瑞指出，中国影响着美国经济的每个部门和各州。
 <br/>
 <br/>
 美国国土安全部长尼尔森：〝中国（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 ），绝对是进行一项前所未有的，或正在实施前所未有的努力来影响美国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舆论.html" target="_blank">
  舆论
 </a>
 。〞
 <br/>
 <br/>
 尼尔森指出，对美国选举安全存在两种威胁：一是威胁选举基础设施，包括对选民登记名单、骇客入侵投票机等；二是影响舆论。而证据显示中共的威胁存在于后者。
 <br/>
 <br/>
 联邦调查局局长 弗瑞：〝中国（中共）在很多方面，成为我们所面临的最广泛，最复杂，最长期的反间谍威胁。
 <br/>
 <br/>
 路透社报导指出，弗瑞的话表明，中共对美国的安全威胁要大于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_俄罗斯.html" target="_blank">
  俄罗斯
 </a>
 。
 <br/>
 <br/>
 新唐人记者李子玉综合报导
</div>
