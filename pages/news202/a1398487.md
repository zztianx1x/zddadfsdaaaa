### 川习通话成果？川普暗示北京已放弃制造2025计划
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 总统周三（11月7日）在白宫举行的新闻发布会上暗示，华盛顿或与北京达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易协议.html" target="_blank">
  贸易协议
 </a>
 。他还表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 企图到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_2025.html" target="_blank">
  2025
 </a>
 年从技术上主导世界，这对美国来说是种侮辱，也是不可能实现的。
 <br/>
 <br/>
 周三中午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在白宫召开新闻发布会，就中期选举的结果答记者问。
 <br/>
 <br/>
 新闻发布会持续了大约一个半小时，期间有记者问，在经历激烈的国会选举后，应该如何弥补美国的政治分歧。川普话锋一转，提到了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 ，指出有一种能帮助治愈美国的方式，那就是美国的成功。
 <br/>
 <br/>
 川普指出，中共本来可能会在两年后从经济上超越美国，但其现在〝放弃（got rid of）〞了《中国制造
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_2025.html" target="_blank">
  2025
 </a>
 》计划。
 <br/>
 <br/>
 川普认为，这是一项令人感到侮辱的计划，他也把这种看法告诉了中共。他之所以这样说，是因为《中国制造2025》意味着中共将在2025年从经济上接管世界，但〝我说过，那不会发生〞。
 <br/>
 <br/>
 《中国制造2025》是中共企图主导未来科技的一项野心计划，是当前美中贸易争端难以解决的关键因素。而美国之所以难以接受这个计划，是因为该计划建立在对美国知识产权的盗窃和强制技术转让基础上。
 <br/>
 <br/>
 目前还没有关于中共已放弃该计划的报导，不过近来有消息称，中共称它可能会搁置这项计划，以换取与美方达成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易协议.html" target="_blank">
  贸易协议
 </a>
 。
 <br/>
 <br/>
 川普说，他并不想让中共的经济下滑，双方将举行很好的会谈，美国也将看到自己能做什么。
 <br/>
 <br/>
 尽管川普没有明确说明，但他似乎是指本月底将在G20峰会上会晤中共领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 。
 <br/>
 <br/>
 川普表示：〝我们将尝试与中国达成协议，因为正如我所做的那样，我希望与习主席和中共拥有很好的关系。〞
 <br/>
 <br/>
 很多人猜测，〝川习会〞是解决美中关税问题的一个契机，美中双方可能就此达成贸易协议。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：东野）
 <br/>
