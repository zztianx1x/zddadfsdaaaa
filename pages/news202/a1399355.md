### 美国会发布重磅报告 吁设基金抗衡中共输出专制
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】美国国会下属的一个审查委员会周三公布了一份年度报告，提醒美国政府中共正向世界输出其专制模式，建议美国国会设立专项基金以便采取抗衡行动。
 <br/>
 <br/>
 <h4>
  中共正利用〝
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_一带一路.html" target="_blank">
   一带一路
  </a>
  〞对外输出专制模式
 </h4>
 <br/>
 据路透社报导，美国国会下属的〝美中经济安全审查委员会〞14日发布了该机构的年度报告。这份报告指出，中共政府正在通过向发展中国家提供基础设施建设和融资，向世界输出〝专制模式〞。
 <br/>
 <br/>
 报告表示，中共正在致力于推动所谓〝一带一路〞的倡议，为马来西亚、肯雅和俄罗斯等国的主要港口、高速铁路、高速公路和桥梁提供资金。中共试图通过这些活动来鼓励这些国家政府认可〝海外专制行为者〞，并利用这项计划使中共在部分国家植入军事势力的行为合理化。
 <br/>
 <br/>
 这份报告并指称，现在中方在5G无线技术方面的优势正在增强，其连接到互联网的设备数量也在激增，而这些连网设备可能被用于发动网络攻击。如果美国电信供应商继续依赖于中国进口产品，将给美国下一代电信基础设施的安全部署带来严重隐患。
 <br/>
 <br/>
 此外，中共正计划着要向世界输出其技术应用标准，而这些所谓的〝中国标准〞也会对美国的企业和全球市场准入构成威胁。
 <br/>
 <br/>
 为此，〝美中经济安全审查委员会〞建议美国国会成立一个基金，以便帮助世界各国，尤其是印太地区的国家，发展数字连接、基础设施和能源供应，以此来抗衡中共的专制输出行动。
 <br/>

 <br/>
 <h4>
  中共对外输出专制模式的对象也包括西方老牌民主国家
 </h4>
 <br/>
 事实上，中共对外输出专制模式的对象，不仅仅是那些接受其大笔资金〝援助〞的发展中国家或欠发达国家，中共对于西方发达国家的渗透于同样令人心惊。
 <br/>
 <br/>
 早在去年年底，西方传媒界已经开始集中讨论中共对外输出推销其专制模式的问题。英国的《金融时报》、《经济学人》和BBC；德国的《德国之声》以及部分美国媒体，几乎不约而同地集中发表文章，警告西方各国政府，中共正在不择手段地向西方，包括一些老牌的民主国家搞渗透，向世界推销其专制模式。
 <br/>
 <br/>
 《自由亚洲电台》今年1月初发表的《中国向世界推销专制模式，西方开始警觉并反击》一文指出，中共对西方国家进行渗透的手段包括：利用西方的言论自由与学术自由提供的空间，一边通过煽动、欺凌、虚假宣传等手段来压制西方的价值观，一边又暗中收买当地政客，说服西方精英为中共的合法性以及其专制统治的方式背书，甚至试图迫使被影响的国家政府实现自我审查，或迫使那些害怕失去中国资源或利益的国家政府无条件臣服。
 <br/>
 <br/>
 文章以德国和澳大利亚遭中共大肆渗透的情况为例，介绍了中共对西方国家输出专制模式的严重情形。
 <br/>
 <br/>
 德国情报机构宪法保卫局曾于2017年12月10日发出警告，指中共特工正利用社交媒体接触德国国会议员和其他政府雇员。
 <br/>
 <br/>
 据披露，为了赢得这些德国人的信任，中共的特工冒充智库领导人和猎头，通过互联网社群与他们试图拉拢利用的德国政要或政府人员建立其联系，并鼓励他们到中国去参加什么研讨会或游历，并主动为其提供去中国旅行的全部费用，以及与有影响力的客户见面的机会。目前已有上万名德国公民成为中国间谍的目标。
 <br/>
 <br/>
 至于澳大利亚，该国的安全情报机构发现，有两位与北京关系密切的中国亿万富翁向澳大利亚两大主要政党捐赠了至少670万美元的政治献金；还有大约十名奥州中央和地方政府的政治候选人与中国情报机构有密切关系。澳洲的国家安全情报机构明确发出警告说，澳大利亚正面临史无前例的威胁，比苏联特工在冷战时期渗透进政府时要严重得多。
 <br/>
 <br/>
 文章指出，西方现在似乎终于意识到，在过去四十年中，中共政府并没有因为与世界的接触加深而逐渐接受西方政治制度的影响，反而其目的是要通过各种渗透方式促使西方民主国家〝社会主义化〞，这四十年的接触恐怕只是中共的一场骗局。
 <br/>
 <br/>
 不过，西方民主国家现在已经开始觉醒，并陆续对中共的渗透展开反击。
 <br/>
 <br/>
 澳洲率先修改了有关间谍活动和外国影响的法律，禁止来自外国的政治捐赠，并强制游说者公布其境外机构工作的经历。
 <br/>
 <br/>
 美国政府则正在考虑扩大其外国代理注册法案，以限制中共官媒和智库在美国做宣传、散布虚假信息。
 <br/>
 <br/>
 北约秘书长拉斯穆森（Anders Fogh Rasmussen）则向欧盟提议，制定措施调查并可能限制中共在欧洲的投资。
 <br/>
 <br/>
 文章最后指出，中共政府和西方国家之间的这场渗透和反渗透，推销与反推销的战斗，会不会转化为一场新的大规模的冷战，非常值得关注。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
