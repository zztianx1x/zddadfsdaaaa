### 沙特外长谈记者之死：流氓行径 王储不知
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月22日讯】周日（10月21日），沙特阿拉伯外交部长阿德尔．阿．朱拜尔（Adel al-Jubeir）表示，沙特记者卡舒吉（Jamal Khashoggi）在驻土耳其
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_领事馆.html" target="_blank">
  领事馆
 </a>
 被杀一案系〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_流氓行径.html" target="_blank">
  流氓行径
 </a>
 〞，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王储.html" target="_blank">
  王储
 </a>
 并不知晓此事。
 <br/>
 <br/>
 朱拜尔是在接受福克斯新闻（Fox News）专访时做出上述表示的。
 <br/>
 <br/>
 朱拜尔告诉福克斯新闻主播拜尔（Bret Baier），沙特政府当前正调查卡舒吉之死的具体情况，涉案的18人都将受到相应惩罚。
 <br/>
 <br/>
 但朱拜尔对事件的评论明显比沙特官方先前的说法前进了一步，他谴责说，杀害卡舒吉的行为是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_流氓行径.html" target="_blank">
  流氓行径
 </a>
 〞。
 <br/>
 <br/>
 朱拜尔接着表示，行动者超越了个人权限和职责，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_领事馆.html" target="_blank">
  领事馆
 </a>
 内杀死卡舒吉、犯下错误，之后又试图掩盖自己的行径。
 <br/>
 <br/>
 他还特别补充说，上述18名嫌犯中无人与沙特
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王储.html" target="_blank">
  王储
 </a>
 穆罕默德（Mohammed bin Salman）有密切关系。
 <br/>
 <br/>
 朱拜尔指出，调查仍处于初期阶段，沙特官员目前也不知道卡舒吉死亡的具体原因以及他的尸体现在何处。
 <br/>
 <br/>
 川普总统周六在出席一场竞选集会时表示，关于卡舒吉之死，他还需要了解更多情况。至于美国政府对此将作出何种回应，他将与国会进行商议。
 <br/>
 <br/>
 川普还表示，他很快就会跟33岁的沙特王储谈论此事。
 <br/>
 <br/>
 卡舒吉是美国《华盛顿邮报》（The Washington Post）专栏作家，经常批评沙特王储，去年逃离沙特。
 <br/>
 <br/>
 10月2日，卡舒吉前往沙特驻土耳其伊斯坦布尔（Istanbul）领事馆领取文件，但之后离奇消失。
 <br/>
 <br/>
 土耳其媒体随后报导，卡舒吉已在领事馆遇害，尸体已转移，而〝暗杀小组〞来自沙特。
 <br/>
 <br/>
 沙特官方一开始否认知道该事件，但在美国及一些欧洲国家的施压下，20日发布公告，称卡舒吉进入领事馆后与沙特官员发生争执，并在搏斗过程中死亡。但这一解释遭到普遍质疑。
 <br/>
 <br/>
 （记者李佳欣综合报导/责任编辑：程以仁）
</div>
