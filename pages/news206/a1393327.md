### 台南成大医院开刀房砍人 女医勇敢挡刀受伤
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月28日讯】台南
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_成功大学.html" target="_blank">
  成功大学
 </a>
 (成大)医院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开刀房.html" target="_blank">
  开刀房
 </a>
 28日上午发生林姓技师疑不满遭陈姓护理师霸凌，涉嫌持刀砍杀陈，为了救护理师，心脏血管外科胡姓女医师见状，奋勇挡在林嫌面前，导致左手遭砍伤，缝11针。林嫌随后赴派出所投案。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_成大医院.html" target="_blank">
  成大医院
 </a>
 发言人 赵庭兴： 〝今天早上大约九点时候，本院手术室一位同仁持刀刺向另一位同仁腹部，受伤同仁马上在手术室里面接受缝合等等，目前生命现象稳定意识非常清楚，涉案同仁自行到派出所去投案。〞
 <br/>
 <br/>
 成大医院表示，嫌犯是医院心脏科技师，与受害者有官司诉讼纠纷，现场一位医师为了抢下凶嫌手上的刀械手部刮伤，目前全部程序已经在司法调查中。
 <br/>
 <br/>
 据目击者说，今天早上心脏
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开刀房.html" target="_blank">
  开刀房
 </a>
 正准备动手术，现场有正要接受开刀的病人与实习医师等医护人员，突然林姓技师持刀冲进开刀房，二话不说的就朝陈姓护理师腹部捅了一刀，大家惊声尖叫，全都吓坏了。这时，胡姓女医师冲上前去，勇敢的挡住林姓技师继续砍人，并拉住林嫌，不让他再砍下去，但林则继续向朝护理师戳了几刀，过程中也划伤了吴姓女医师后才离去。
 <br/>
 <br/>

 </center>
 <br/>
 据了解，胡姓女医师专精于心脏二尖瓣膜修补术，此次舍身救人，令人感到钦佩。因左手被砍受伤，缝11针，至少必须休养半个月，不能进开刀房开刀，她本人低调不愿受访。
 <br/>
 <br/>
 <h4>
  技师脸书留下〝终结霸凌〞 成大医院否认
 </h4>
 <br/>
 林姓技师持刀行凶伤及2名医护人员后，在投案途中于脸书留下〝终结霸凌〞4个字。
 <br/>
 <br/>
 赵庭兴下午接受媒体采访时说，〝没有职场霸凌〞，院方会对此事调查，有需要改进的地方也会检讨。
 <br/>
 <br/>
 赵庭兴说，在职场上发生这种事情，很遗憾，院方对林姓男技术师和同事间的相处也有掌握；林姓男技术师确实可能因个性上的问题，在团队中人缘不是那么好。
 <br/>
 <br/>
 也有医护人员说，林嫌平日与同事相处不睦，大家就比较不理他，可能误会陈护理师在背后中伤他，才发生此意外。
 <br/>
 <br/>
