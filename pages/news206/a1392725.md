### 烤肉利器 一秒生火、串肉利器登热搜
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月24日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中秋.html" target="_blank">
  中秋
 </a>
 3天连假，今天开跑，不少人家里都有准备要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烤肉.html" target="_blank">
  烤肉
 </a>
 ，看准中秋节烤肉商机，有业者推出号称一秒助燃的生火器具、一次能完成16串肉的烤肉小帮手，还有自动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烤肉架.html" target="_blank">
  烤肉架
 </a>
 ，要让民众优雅烤肉、不费力。
 <br/>
 <br/>
 短短一分钟，就能把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烤肉.html" target="_blank">
  烤肉
 </a>
 架给组装起来，携带方便，打开开关，不用动手，自动帮你旋转食材、不怕烧焦，旁边还有平底盘，想吃煎肉也没问题。
 <br/>
 <br/>
 民众：〝每个食材都可以均匀受热，然后我不用每个都去顾它，就可以熟了。〞
 <br/>
 <br/>
 片状
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_酒精膏.html" target="_blank">
  酒精膏
 </a>
 埋在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_木炭.html" target="_blank">
  木炭
 </a>
 堆里面，一秒就能让你轻松生火。还有这个像魔术盒的东西，一层肉片、一层青椒，依照顺序叠起来，再把竹签插进盒盖上的孔洞，四面各切一刀，16串的肉串就能同时完成，虽然节省不少时间，但成功率好像不到一半。
 <br/>
 <br/>
 像这样的烤肉小帮手，随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中秋.html" target="_blank">
  中秋
 </a>
 节脚步越来越近，询问度也跟着上升。
 <br/>
 <br/>
 网购平台业者 韩昕景：〝掀起一波优雅烤肉的商机，烤肉神气的热搜量，比过往上升了两倍，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_烤肉架.html" target="_blank">
  烤肉架
 </a>
 的销量比上个月成长1.5倍。〞
 <br/>
 <br/>
 业者脑筋动得快，大推〝烤肉利器〞创造话题，实不实用见人见智，但是新奇的商品已经为中秋节增添不少乐趣。
