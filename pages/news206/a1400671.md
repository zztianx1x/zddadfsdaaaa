### 九合一选举民进党大败 蔡英文发感言并辞党主席（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月25日讯】当地时间11月24日晚9时10分，台湾现任总统兼民进党主席蔡英文召开记者会，承认民进党在2018年九合一选举中失败，并宣布请辞民进党党主席一职，以承担这次败选的政治责任。
 <br/>
 <br/>
 据台湾《自由时报》报导，蔡英文在记者会上表示，自己身为执政党的主席，要对今天地方选举的结果负完全的责任。她说：〝我们努力不够，让所有和我们一起打拚的支持者失望，我也要在此表达我们非常诚挚的歉意〞。
 <br/>
 <br/>
 蔡英文强调，民进党相信民主价值，所以今天败选的结果，可以说是民主给民进党〝上了一课〞。
 <br/>
 <br/>
 她说：〝对于民意的展现，我们当然要深自检讨，但一场选举的成败，不能让我们停下脚步，我们更不能因此自怨自艾。〞
 <br/>
 <br/>
 蔡英文在记者会上还表示，在她辞去党主席之后，民进党将进行党务改组。她呼吁在民进党进行改革的关键期，政府的行政团队一定要保持稳定。
 <br/>
 <br/>
 她说，正是鉴于上述原因，刚才
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赖清德.html" target="_blank">
  赖清德
 </a>
 向她口头请辞时，她已请赖清德留下来继续打拚，以确保现政府的各项重大政策持续推动。
 <br/>

 据了解，就在24日晚间，现任中华民国行政院院长赖清德在自己的脸书上已经发文称，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_九合一大选.html" target="_blank">
  九合一大选
 </a>
 ，是人民对政府施政的检验，而投票的结果，显示人民并不满意。对此，我感到抱歉，已向蔡英文总统口头请辞，负起政治责任。〞
 <br/>
 <br/>
 4年前，国民党在2014年举行的九合一选举中大败时，时任行政院长的江宜桦也是在国民党败选的当晚即宣布辞职。
 <br/>
 <br/>
 据了解，在全台湾的22歌县市中，原本民进党执政的县市多达13个，国民党执政仅6个，无党派执政为3个。但这次台湾的九合一选举开票结果显示，民进党获胜选的市县只有6个，而且这6个原本都掌握4席，连高雄都失去，如今只剩桃园和台南两席。
 <br/>
 <br/>
 与此同时，国民党在这次的选战中执政优势逆转，可谓大获全胜。
