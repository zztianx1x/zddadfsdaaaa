### 美中贸易拉长战？王文渊：要看G20进展
------------------------

<div class="wysiwyg">
 【新唐人2018年11月10日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台塑.html" target="_blank">
  台塑
 </a>
 集团，今天举行第35届运动会，由总裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王文渊.html" target="_blank">
  王文渊
 </a>
 亲自主持。而关于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 ，王文渊说，目前看不到缓和迹象，但对集团来说，影响不大。后续的情况，还是要看美中两国，在11月底
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 峰会上的协商结果。王文渊也以美中两国，强调制造业治国的现象，来肯定台塑，在台湾制造业方面的努力。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台塑.html" target="_blank">
  台塑
 </a>
 集团总裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王文渊.html" target="_blank">
  王文渊
 </a>
 ：〝台塑企业长期坚持制造业作法，不但正确，而且对国家经济，也具有长远实际，实际的贡献。〞
 <br/>
 <br/>
 台塑集团运动会在明志科大举行，总裁王文渊以美中两国，强调制造业治国的现象，来肯定台塑在制造业的努力。也提到，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 自下半年开始，好像看不到缓和迹象，但对集团来说，影响不大。
 <br/>
 <br/>
 台塑集团总裁王文渊：〝现在看他们
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_G20.html" target="_blank">
  G20
 </a>
 以后看怎么样，看它进展会怎么样。因为那个美国选举过后，情况稍微，微妙的变化。〞
 <br/>
 <br/>
 台塑今年前3季营收，受全球经济景气持续好转，预计今年绩效和去年相当，王文渊还透露，年终奖金至少6个月，但加码金额尚未决定。王文渊也亲自率领一级主管，跑操场5000公尺，台塑生医董事长王瑞瑜、台塑集团三娘李宝珠，前总统马英九，也加入行列，不过副总裁王瑞华，则罕见缺席。
 <br/>
 <br/>
 台塑集团总裁王文渊：〝(接那个，三个学校董事长的职务吗？)还没安排，还没安排，(还没安排吗？)还没有还没，(那现在目前进度是怎么样？)我还没签到。〞
 <br/>
 <br/>
 台塑集团总裁王文渊：〝这张，6月3号还是4号，我忘记了。〞
 <br/>
 <br/>
 今年适逢台塑企业创办人王永庆辞世10周年、王永在辞世4周年，特别规划纪念特展，王文渊、王文潮兄弟俩一同参观，边看边讨论，唤起不少共同经历过的回忆。
 <br/>
 <br/>
 新唐人亚太电视胡宗翰、李晶晶台湾新北市报导
