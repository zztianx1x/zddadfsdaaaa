### 号称〝地表最强〞 美商在台造势第九代CPU
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】外商重视台湾市场，尤其台湾更是ICT科技重镇，美国最大半导体公司业者，周一晚间，大规模举办，国际新品发表会，我们带您抢先目睹，最新趋势，尤其不光锁定电竞产业，更看好，商用创作者高阶需求的〝新蓝海〞。
 <br/>
 <br/>
 美商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_英特尔.html" target="_blank">
  英特尔
 </a>
 台湾区电脑通讯事业群产品经理 余孝伦：〝一些烟尘的效果，那譬如有些星际大战，他有些电线短路走火，冒火花，有些时候也要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CPU.html" target="_blank">
  CPU
 </a>
 的配合，才能展现出这种，细微的一些效果。〞
 <br/>
 <br/>
 细致声光影像，除了显示卡(GPU)，细部光影，得靠电脑的心脏-CPU扮演关键脚色。外商大动作推出三款第九代桌上型电脑
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_处理器.html" target="_blank">
  处理器
 </a>
 (i5-9600K i7-9700K
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_i9.html" target="_blank">
  i9
 </a>
 -9900K)，还特别在新闻稿上打出〝地表最强〞称号，当然是有备而来。
 <br/>
 <br/>
 美商英特尔台湾区电脑通讯事业群产品经理 余孝伦：〝那我们因为制程上进步，所以增加两个核心呢，整个TDP(Thermal Design Power热设计功耗)，还是维持在95瓦，跟上一代处理器比，我们大概好大概10-11%，那跟三年前的比，大概是3成到4成的进步。〞
 <br/>
 <br/>
 市调机构GFK统计，2015到2018年，电竞PC产品销售，年复合成率达到29%，全球更有5亿规模的庞大用户市场。而高速运算效能，不是只能用在娱乐，还有更庞大市场，在后头。
 <br/>
 <br/>
 记者 沈唯同：〝以往相对难以处理的4K、8K影像，甚至3D渲染效果，现在有外商推出工作站专用CPU，号称效率提升30%以上，将所需要的时间，降到最低。〞
 <br/>
 <br/>
 美商英特尔台湾区业务暨行销部通路业务经理 陈存贤：〝36张照片运行的话，大概需要90分钟，才能够做到这样的情况，现在大概半小时就可以完成了。〞
 <br/>
 <br/>
 没看错，用照片就可以算出高品质的人脸3D模型，还有解析度高达8K的影像，不用压缩转档，直接进行剪辑处理，光时间等待就少掉不知多少。最耗费运算时间的3D光影模组，也不是困难事，利用高校即时运算，结果立刻呈现。
 <br/>
 <br/>
 美商英特尔台湾区桌上平台产品经理 李昆龙：〝可以做一个渲染，甚至补光，甚至是一个动画的一个画面。〞
 <br/>
 <br/>
 外商加码推出七款CoreX系列处理器，瞄准商用市场，Youtuber、影像工作者、软体工程师、绘图工程师在内，数量上看1.3亿用户，2015-2020年，成长潜力，高达四倍。
 <br/>
 <br/>
 美商英特尔台湾区桌上平台产品经理 李昆龙：〝creator(创作者)这一块其实我们看到成长的空间非常大的，因为这一群客户群，针对内容创作也好，或者是他们对电脑的运算需求是越来越高。帮助到这些领域，更多行业可以有更多创新，而不是在等待。〞
 <br/>
 <br/>
 看好新产品推出，将带动新一轮业绩成长，业者在台湾召开新品发表会还有线上直播，不难看出对台湾消费市场深具信心。
