### 中颱潭美风强速度快 将冲击东京首都圈
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年09月30日讯】今年第24号中度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风潭美.html" target="_blank">
  颱风潭美
 </a>
 的强风豪雨29日影响日本，首当其冲的冲绳群岛各地已淹水变成汪洋一片，随之30日将扫过九州、四国及本州南部，预期中午前后，中心将登陆四国一带。而当天在太平洋海面又形成的第25号中度
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_颱风康芮.html" target="_blank">
  颱风康芮
 </a>
 ，目前距离台湾超过2000公里，未来路径仍有不确定性。
 <br/>
 <br/>
 日本放送协会（NHK）报导，潭美(TRAMI)从29日到今天至少造成冲绳县及鹿儿岛县共51人轻重伤。
 <br/>
 <br/>
 据weather map气象预报士杉江勇次撰文表示，潭美在冲绳及奄美地方都刮起最大瞬间风速超过每秒50公尺的暴风。依照目前预测路径，潭美傍晚到晚间将从四国或纪伊半岛登陆，在晚间9时左右通过大阪附近。
 <br/>
 <br/>
 杉江说，因为潭美行进速度相当快，在通过东日本及东北地方时，也可以保有很强的威力，预测最大瞬间风速可维持每秒50公尺以上。并于深夜到明天凌晨，为东京首都圈带来前所未见的暴风。
 <br/>
 <br/>
 杉江解释说，潭美之所以会为首都圈带来强风的理由有3点，第一是潭美以强烈威力通过；第2是首都圈位于潭美行进方向的右侧，也就是所谓的危险半圆区域；第3是潭美行进速度高达时速85公里，这3个条件同时出现。
 <br/>
 <br/>
 他说，如果把从海上吹入东京湾周边的南风计算进来，东京地方在潭美过境时的最大瞬间风速可能达到每秒50公尺，神奈川县及千叶县也可能出现每秒45公尺的强风，其他地方的最大瞬间风速也有每秒35公尺。
 <br/>
 <br/>
 杉江说，东京都心过去被观测到最大的瞬间风速为每秒46.7公尺、横滨是48.7公尺、千叶为48.6公尺；潭美这次带来的强风，可能跟过去的纪录不相上下，有些地方甚至有机会出现更大的风速。
 <br/>
 <br/>
 他表示，最大瞬间风速若达到每秒45到50公尺，很多树木将被吹倒、看板被吹落、卡车被吹翻，人员在外行动将非常危险；因此在颱风行进路径上的民众要避免外出，也要做好停电的相关准备。
 <br/>
 <br/>
 他表示，特别是位于潭美行进路径右侧的纪伊半岛到东北地方南部的太平洋侧地区，可能出现前所未见的暴风，有必要严加戒备。
 <br/>
 <br/>
 至于东京首都圈，杉江说，在深夜到明天凌晨时，会是潭美最接近的时刻。并提醒民众，除了靠山的地区可能会出现大雨，这次特别需要严加防范强烈的南风。
 <br/>
 <br/>

 而目前由轻颱转为中颱的康芮(KONG-REY)下午2时的中心位置位于北纬14.50度、东经137.90度，以每小时17公里速度，向西北进行。颱风近中心最大风速每秒33公尺，瞬间之最大阵风每秒43公尺，七级风半径150公里，十级风半径50公里。
 <br/>
 <br/>
 台湾中央气象局预测，短期内将朝西北西移动，10月3日会接近琉球东南方海面，但后期路径变数大，仍有不确定性，是否会影响台湾仍待观察。
 <br/>
 <br/>
 另据天气风险 WeatherRisk脸书专页发文表示，〝康芮颱风〞受到太平洋高压导引，下周四(10月4日) 接近琉球东南方海面，此时位于高压尾端的康芮，偏北分量增加，各国路径开始分歧，若颱风偏东于台湾东方经琉球海域北上，则台湾仅受外围风场影响为东北至偏北风环境；若较偏西接近，则可能于下周五、六影响台湾。不过后期预测路径变动度仍高，预计要到下周三视北转角度及时间，才能知道对台湾的影响为如何。
