### 礼器到食器 看遍青铜器展的万千变革
------------------------

<div class="wysiwyg">
 【新唐人2018年09月29日讯】不知道电视机前的观众朋友们，是否了解，中国的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青铜器.html" target="_blank">
  青铜器
 </a>
 ，最早发现于四千年前的龙山文化？从上古的夏商周时期开始，青铜器逐渐成为，国家重器，依照天子和官位大小，可制造的青铜器多寡，也不相同。在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北故宫.html" target="_blank">
  台北故宫
 </a>
 ，收藏了不少精彩的商周青铜器，非常值得前去一看。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台北故宫.html" target="_blank">
  台北故宫
 </a>
 收藏的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_青铜器.html" target="_blank">
  青铜器
 </a>
 可说是世界之冠，多半来自紫禁城旧有的收藏，光是鼎就占了所有青铜器的六分之一。
 <br/>
 <br/>
 例如这只商朝晚期的〝乃孙作祖己鼎〞，原本放置于乾隆皇帝的重华宫，是故宫的第二重鼎，有着显著的兽面纹，也是故宫四大巨鼎之一。
 <br/>
 <br/>
 夏商周三代，青铜器除了用来制造兵器、乐器，在祭祀大典上，从青铜器的多寡，就能看出身份地位的高低，因此，才有楚庄王〝问鼎中原〞的典故。
 <br/>
 <br/>
 东周时期，因为诸侯的势力强大，不再遵从典章制度，开始铸造青铜器。
 <br/>
 <br/>
 而小国如河南的蓼国也有自己的青铜文化，例如这只鄝公鼎是用来煮水的。
 <br/>
 <br/>
 而这只春秋时期的宽儿鼎虽然是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_食器.html" target="_blank">
  食器
 </a>
 ，也刻上希望自己长寿、子孙能永远保用的铭文，字数虽少，但都是学者研究上古史的最佳材料。
 <br/>
 <br/>
 而相较于商朝、西周，东周时代的青铜器纹饰都十分简单，因为主要是在日常生活中使用。
 <br/>
 <br/>
 这次的商周青铜器常设展按照时代与不同的品类分别展出，且有清楚地说明，民众不妨前往观赏一下，各种青铜器在不同时代的演变，十分有趣。
 <br/>
 <br/>
 新唐人记者谢心欣综合报导
</div>
