### 美联储升息 亚洲央行接招 台湾不跟
------------------------

<div class="wysiwyg">
 【新唐人2018年09月27日讯】美联储周三宣布升息后，亚洲各国央行采取不同策略。香港12年来首次上调最优惠利率，结束超低息环境。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾央行.html" target="_blank">
  台湾央行
 </a>
 周四决定，维持基准利率不动。
 <br/>
 <br/>
 央行第三季理监事会，不出外界意料，利率维持〝9连冻〞，考量国际不确定性，通膨温和，不过总裁话中有话。
 <br/>
 <br/>
 央行总裁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杨金龙.html" target="_blank">
  杨金龙
 </a>
 ：〝利率这个东西，利率这个指标，是要看中长期，利率不会就是说永远不变的，到底要到第几〝冻〞，我想要看每一次的情况，每一次我们都会做一个预测。〞
 <br/>
 <br/>
 美国联准会八度升息，利率已经来到金融海啸后最高。各国采不同策略。菲律宾、印尼央行预期跟进升息，中国人行可能采取降准策略，南韩进退两难，台湾也是不跟进。不过货币政策已经有所收紧。
 <br/>
 <br/>
 央行总裁 杨金龙：〝如果跟上一次来比较的时候，我们这一次，应该是说，(货币政策)稍微紧了一点点。所以我们这一次就是〝适度宽松〞。我们新台币贬值的幅度相较其他的主要币别呢，来的小。所以从汇率的观点它是紧的。〞
 <br/>
 <br/>
 美中贸易战升温，央行关注后续发展，低利也有助台商回流。 去年新台币走强，央行也背上一兆多的汇损，好在今年不只赚回来，帐上收益还有1364亿。
 <br/>
 <br/>
 央行总裁 杨金龙：〝今年四月以来，美元是强劲。就是因为这汇率的变动，所以到八月底的时候，我们去年的这个，未实现的损失呢，在到八月底的时候，是全部冲销了，而且我们还甚至有，一千多亿的未实现利益。〞
 <br/>
 <br/>
 值得一提的是，央行报告，首次将中国P2P借贷乱象，列入会议讨论。央行上调今年经济成长率2.73％，预期明年经济成长率2.48％，内需为经济成长主力。
 <br/>
 <br/>
 新唐人亚太电视胡宗翰、沈唯同台湾台北报导
</div>
