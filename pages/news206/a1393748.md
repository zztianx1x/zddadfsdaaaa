### 酷！乐高创作登〝LEGO ideas〞台湾14岁男孩全球最年轻
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月02日讯】年仅14岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾男孩.html" target="_blank">
  台湾男孩
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑宇亮.html" target="_blank">
  郑宇亮
 </a>
 根据BMW R60/2实车创作
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乐高.html" target="_blank">
  乐高
 </a>
 作品，投稿到〝LEGO ideas〞，历经一年多的连署确定达标。作品周一（10月1日）还登上〝LEGO ideas〞脸书专页，成为台湾第一人，其作品还有望被量产。
 <br/>
 <br/>
 根据〝LEGO ideas〞网站介绍，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郑宇亮.html" target="_blank">
  郑宇亮
 </a>
 现在就读国二，他从3岁起开始玩
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乐高.html" target="_blank">
  乐高
 </a>
 ，当时爸爸的一位朋友送给他一组旧的乐高城市套装，开启了他的乐高之路。他的爸爸之后也为他投资超过百万元，让儿子玩乐高。
 <br/>
 <br/>
 郑宇亮说，他一直很喜欢
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_摩托车模型.html" target="_blank">
  摩托车模型
 </a>
 ，之前他的朋友建议他在生日时做点特别东西，于是他以BMW R60/2实车为原型，做出了一台精致的复古摩托车模型。因为还要上学，每天只能工作2～3小时，制造这个模型一共花了他3个月的时间。
 <br/>
 <br/>
 2016年12月，郑宇亮将这个摩托车模型设计投稿到〝LEGO ideas〞，在历时一年多的连署活动中获得1万票，昨(1)日作品成功登上〝LEGO ideas〞的脸书专页，成为台湾第一人，同时也是全世界通过连署的最年轻乐高玩家，而他的作品还有望被量产。
 <br/>
 <br/>
 LEGO ideas在留言中写道，本周10K俱乐部会员是俱乐部中年纪最小的，来自台湾的14岁的Maxime Cheng a.k.a。 maximecheng03 用了3个月的时间建造了BMW R60/2高度精致老式摩托车。
 <br/>
 <br/>
 作品登出后，网友纷纷留言表示：〝这个机车太酷了！希望能被量产〞、〝英雄出少年，好车自经典〞、〝只有14岁的设计师！太厉害了！〞、〝了不起的设计师啊！〞
 <br/>
 <br/>
 郑宇亮说，从小学四年级开始，就一直以未来进入乐高公司上班为目标，现在的目标还是没有改变，未来也会持续朝这个方向努力。
 <br/>
 <br/>
 为了让自己的作品走上官方量产之路，郑宇亮近几个月以来，都不断在尝试加强作品的稳固、可玩性等，希望能进一步达到量产的目标。
 <br/>
 <br/>
 不过，由于郑宇亮的作品是实车创作的乐高模型，乐高官方量产必须取得BMW的授权，有乐高玩家认为，这也将是郑宇亮的作品能否走上官方量产的难题之一。
 <br/>
