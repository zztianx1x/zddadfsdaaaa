### 四大美馆迎花博 花之礼赞联展开幕
------------------------

<div class="wysiwyg">
 【新唐人2018年10月06日讯】为配合11月，台中市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际花卉博览会.html" target="_blank">
  国际花卉博览会
 </a>
 ，联合了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国立故宫博物馆.html" target="_blank">
  国立故宫博物馆
 </a>
 、东京富士美术馆、奇美博物馆和国立台湾美术馆四大馆，6号在国立台湾美术馆，举行一场以
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花卉.html" target="_blank">
  花卉
 </a>
 为主题的，盛大联展，先为台中国际
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花博.html" target="_blank">
  花博
 </a>
 暖场。
 <br/>
 <br/>
 〝花之礼赞—四大美术馆联合大展〞，以〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花卉.html" target="_blank">
  花卉
 </a>
 〞为主题，展出国立故宫博物院、东京富士美术馆、奇美博物馆、国立台湾美术馆共148件典藏精品，另外还有20组台湾艺术家作品参展，藉由艺术的诠释、创作，呈现花的象征意蕴与不同的生命哲思。
 <br/>
 <br/>
 文化部政务次长 萧宗煌：〝故宫他们提供的是华夏文物，台南的奇美美术馆他们提供的是西洋文物，东京富士美术馆提供的是东洋文物，台湾美术馆的话提供是台湾艺术家的在地的，近现代跟当代的作品。〞
 <br/>
 <br/>
 东京富士美术馆，精选25组作品，包含屏风、画轴及浮世绘。国立故宫博物院则精选宋朝至清朝的器物、书画和图书文献。奇美博物馆本次精选30组作品，分为〝花开缘由〞、〝寓意抒情〞和〝工艺时尚〞三个单元，呈现西方花卉主题创作的多元与丰富。国立台湾美术馆则精选自日治时期至当代之典藏品53件。
 <br/>
 <br/>
 台中市长 林佳龙：〝我们也想呈现台中的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_花博.html" target="_blank">
  花博
 </a>
 ，不是只是一个花卉的展览，事实上它透过花卉园艺这一些主题，可以让我们聆听花开的声音，也可以结合绿色生产，自然生态跟人文生活，所以还是以人为本 ，透过花的主题进行馆际的合作，联合的策展，非常的精采。〞
 <br/>
 <br/>
 因部分展品极为珍贵，预计将在展览期间轮替展出，第一阶段限展作品展期为10月6日至12月9日，第二阶段为12月12日至明年2月10日，适逢台中花博期间，预计将带给观众一场繁花盛宴。
 <br/>
 <br/>
 新唐人亚太电视王媛、林莉台湾台中采访报导
</div>
