### 6款降血压药物含致癌物 药商在台卖出126万颗
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高血压.html" target="_blank">
  高血压
 </a>
 药又出包！2018年8月台湾才发现6款
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降血压.html" target="_blank">
  降血压
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_药物.html" target="_blank">
  药物
 </a>
 ，含有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_致癌.html" target="_blank">
  致癌
 </a>
 成分〝N-亚硝基二乙胺〞并紧急
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下架.html" target="_blank">
  下架
 </a>
 后，现在再爆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_致癌药.html" target="_blank">
  致癌药
 </a>
 ，〝平压妥膜衣锭〞，虽然公司已主动启动回收作业，但这款药在台湾卖出126万颗。
 <br/>
 <br/>
 市面上各式各样
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高血压.html" target="_blank">
  高血压
 </a>
 药，高血压患者得天天服用，但没想到惊传〝平压妥膜衣锭〞，原料药含
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_致癌.html" target="_blank">
  致癌
 </a>
 物〝N-亚硝基二乙胺〞，具动物致癌性。
 <br/>
 <br/>
 高血压患者：〝当然会怕，因为我全身都很多毛病，要是吃坏的药更麻烦。〞
 <br/>
 <br/>
 高血压患者：〝我刚刚有去看(我的药)，我老婆看到电视赶快拿药来对，我没有吃这款药。怕阿，会怕阿。〞
 <br/>
 <br/>
 食药署立清查所有该成分之制剂品许可证，发现健亚生物科技公司的〝平压妥膜衣锭300毫克〞使用该原料药，该公司已主动启动回收作业。根据统计，这款药已卖出4万多盒，约为126万颗。
 <br/>
 <br/>
 食药署
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_药品.html" target="_blank">
  药品
 </a>
 组科长 洪国登：〝健亚要在12月2号之前，全面
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_下架.html" target="_blank">
  下架
 </a>
 回收，回收的数量大概是126万颗，如果健亚没有按照期限(回收)，对它处20万到500万罚锾。〞
 <br/>
 <br/>
 药师公会全国联合会政策执行长 沈采颖：〝它的市占率，只有3%上下，最多不超过5%，(服用)这个部分没有立刻致命的危险，血压高会有立刻致命危险，我们希望民众不要立刻停药。〞
 <br/>
 <br/>
 食药署紧急要求全面下架，但高血压药却不是第一次出包，2018年8月台湾才发现6款
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_降血压.html" target="_blank">
  降血压
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_药物.html" target="_blank">
  药物
 </a>
 ，含有致癌成分〝N-亚硝基二乙胺〞，紧急下架后，现在再爆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_致癌药.html" target="_blank">
  致癌药
 </a>
 ，让民众吃药难安心。
 <br/>
 <br/>
 药师公会全国联合会政策执行长 沈采颖：〝食药署，能够针对这一类的原料药要来管理，必须要订定一个长远的标准跟计画，我们不希望又再有第三次，另外一种成分，也有这种致癌物。〞
 <br/>
 <br/>
 高血压问题药连环爆，药师希望食药署加强列管，别再发生，否则引起民众恐慌、药厂损失，也让药剂师们开药很头痛。
 <br/>
 <br/>
 (新唐人亚太电视 综合报导)
</div>
