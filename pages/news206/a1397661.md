### 一场意外险丧命 红龙果农锲而不舍创佳绩
------------------------

<div class="wysiwyg">
 【新唐人2018年11月01日讯】屏东有位种植
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红龙果.html" target="_blank">
  红龙果
 </a>
 青农，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_傅柏琮.html" target="_blank">
  傅柏琮
 </a>
 ，三年前的一场意外，竟脑部开刀，险些丧命，后来靠欣赏古曲音乐与田园治疗，再度投入热爱的红龙果种植，不但身体变好了，今年还荣获全国第三名佳绩。
 <br/>
 <br/>
 外型硕大一秤就有1斤半的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_红龙果.html" target="_blank">
  红龙果
 </a>
 ，切开后，深红色果肉上的黑籽，粒粒分明，甜度高达21度，吃起来的口感细致绵密，这就是俗称的〝大红〞品种。
 <br/>
 <br/>
 采买民众 郑小姐：〝汁很多果肉很细，就入口即化的感觉，没有害怕的那种生的那种味道。〞
 <br/>
 <br/>
 红龙果青农
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_傅柏琮.html" target="_blank">
  傅柏琮
 </a>
 ：〝大红为什么好吃，因为从平地到我这里，大概海拔170公尺，所以我这里大热天是35度，但是冬天只有20来度，差10度。〞
 <br/>
 <br/>
 他就是青农傅柏琮，种植在果园内的火龙果，刚获得全国第三名佳绩的大红，颗颗鲜红饱满，娇艳欲滴，当初因为看上火龙果无需授粉以及种植便利，于是将原本的荔枝等杂木砍除后，改种红龙果，但三年前却因一场意外，脑部开刀，险些让傅柏琮丧命。
 <br/>
 <br/>
 红龙果青农 傅柏琮：〝脑部开刀后，就无法工作了吗？是这么回事吗？所以要继续做，我每天都想要做事，这样你才有办法突破，才有办法，不要因为这样就不干活了，如果不继续干活，就会比较萎缩。〞
 <br/>
 <br/>
 手术后，虽然动作及记忆力大不如前，但并没有摧毁他热爱红龙果的热情，施肥、拔草、剪枝全不假手他人，在家人陪伴，经过三年的田园疗愈下，改变旧有的观念与方法，尤其在肥田管理上的突破，更让他得到今年的殊荣。
 <br/>
 <br/>
 红龙果青农 傅柏琮：〝这里每天都在下雨，为什么都没有烂掉呢，就是把你的有机肥改变，是中性还是硷性，自然就比较不会烂，所以我也不需要去洒农乐，这么简单的事，但是以前就没有办法突破。〞
 <br/>
 <br/>
 红龙果青农 傅柏琮：〝有一个想法在里面，它（红龙果）做出来和我的想法有符合，我觉得这个快乐原因就是在里面。〞
 <br/>
 <br/>
 凭着锲而不舍的毅力与坚持，傅柏琮不但在田园间找到了一份宁静与快乐，也将危机化为转机，尝到成功后甜美的果实。
 <br/>
 <br/>
 新唐人亚太电视李金菊台湾屏东报导
</div>
