### 韩国瑜〝挺进冈山〞造势  10万支持者挤爆会场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高雄市.html" target="_blank">
  高雄市
 </a>
 长候选人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 今(14)日在在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冈山.html" target="_blank">
  冈山
 </a>
 举行决战三山最终场造势大会，现场涌进10万支持者高喊〝冻蒜〞(当选)，整片国旗海飘扬，民众情绪沸腾。韩国瑜从舞台后方由护卫队陪同，几乎是寸步难行下，穿越台下群众，花了10分钟步上舞台。
 <br/>
 <br/>
 〝挺进
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冈山.html" target="_blank">
  冈山
 </a>
 〞造势大会，从今天中午就开始有摊商进驻，卡位抢商机，连知名速食店麦当劳也来摆摊。
 <br/>
 <br/>
 一名上午9点多就赶至会场的民众带妻小一起来相挺，他说，高雄人闷太久了，要改变、不能继续沉沦下去。
 <br/>
 <br/>
 警方下午2点进行第一阶段交通管制，高捷冈山站涌现许多从市区搭捷运到晚会现场的支持群众。高捷调派人员在现场维持秩序，并比照跨年晚会模式，广播呼吁出站乘客先购买回程车票。
 <br/>
 <br/>
 支持者一出站，即有义勇军组成的〝义勇计程车队〞与〝义勇机车队〞，免费载送民众至约三公里外的活动现场。
 <br/>
 <br/>
 民众陈先生说，他与朋友都请假，要到场支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩国瑜.html" target="_blank">
  韩国瑜
 </a>
 。一名王小姐与朋友说，她们都是请假到场支持韩国瑜，〝我们都是扎实有选票的高雄人。〞
 <br/>
 <br/>
 到了下午3时半，已有1万约名韩迷提早到场，还高唱军歌《夜袭》暖身。
 <br/>
 <br/>
 傍晚5点30前，现场摆放的2万5千张椅子，早就被坐满，之后才到场的民众，只能拿国旗站着呐喊，但热情不减。
 <br/>
 <br/>
 晚间6点10分，高雄乐团〝城市惟客〞乐团，在台上高唱〝把高雄还给我〞，台下群众随着主唱的歌词〝高雄需要新的希望…，回乡的我，会认真来打拼…〞，情绪沸腾大家一起高喊〝把高雄还给我〞、〝把高雄还给我！〞
 <br/>
 <br/>
 韩国瑜造势大会上由萧汉俊、立委许淑华、前立委林国正主持，前农委会主委陈保基、立委蒋万安、柯志恩、江启臣等人助讲，韩国瑜晚上7时30分左右到场，与市议员候选人会师大进场，为晚会掀起最高潮。
 <br/>
 <br/>
 晚间7时36分，主办单位宣布，现场人数已破10万人。当昂扬的〝夜袭〞军歌响起，韩国瑜和前立法院长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王金平.html" target="_blank">
  王金平
 </a>
 在手机灯光、旗队的导引下进场，两人在热情民众簇拥及高喊冻蒜声中缓步前进，约百公尺的走道花了近10分钟才走上台，汽笛声响、民众激情渲染，现场气氛达到最高潮。
 <br/>
 <br/>
 晚间7点40分，韩国瑜上台发表20分钟演说，全场气氛到达高潮，国旗海飞舞，〝冻蒜〞声不绝于耳，随后在领唱多首歌曲，包括 〝爱拚才会赢〞、〝你是我的兄弟〞、〝梅花〞、〝国旗歌〞等；过程中会场关灯，民众开起手机灯光形成星海，场面更是壮观。
 <br/>
 <br/>

 <br/>
 <h4>
  韩国瑜宣示全力拚经济
 </h4>
 <br/>
 <br/>
 韩国瑜演说时先用国、英和客语三声带感谢现场支持者，接着话锋一转，批评〝抹黑到了离谱〞境界，指有人说他当市长会卖掉高雄、会取消老人年金，还说未来会把林园大寮变殡葬区等是选举恶质。
 <br/>
 <br/>
 他说，民心思变是因为人民不满现况生活，渴望要富强美好生活，允诺当市长会全力拚经济。他形容高雄就像西装笔挺、口袋没钱的人。
 <br/>
 <br/>
 韩国瑜说，有人流泪说日子难过，〝民进党不是高雄人的爸爸，为什么每次要投民进党〞，〝人民才是政党的爸爸〞，吁政治人物、政党勿忘初衷及对选民的承诺；他说，民主原始的精神是〝人民做主〞。
 <br/>
 <br/>
 他说：〝丐帮有多少弟子，不是由我决定，而是由皇帝决定的〞。如果高雄景气好的话谁愿意离乡背井北漂呀！
 <br/>
 <br/>
 他表示，今年选举高雄人的脑袋要全部打开，重新看政治人物、政党，做不好就下台，呼吁人民一起写历史。
 <br/>
 <br/>
 韩国瑜强调，决心好好打一场选仗，让世界看到高雄支持有理念的人、选贤与能，努力招商不怠忽。
 <br/>
 <br/>
 全场气氛到达高潮，国旗海飞舞，〝冻蒜〞声不绝于耳，随后他领唱多首歌曲，包括 〝爱拚才会赢〞、〝你是我的兄弟〞、〝梅花〞、〝国旗歌〞等；过程中会场关灯，民众开起手机灯光形成星海，场面更是壮观，晚会在多首合唱曲中落幕。
