### 出版24年 42种语言 拥有全球读者的转法轮
------------------------

<p>
 【大纪元2019年01月05日讯】
 <span class="s1">
  （大纪元记者施萍采访报导）
  <span class="s1">
   这
  </span>
  是一本万古难遇的奇书，他能让人变好，让人健康；他回答人类百思不解的命运之谜，他阐述宇宙万物的运动规律；他令
 </span>
 <span class="s2">
  中共
 </span>
 <span class="s1">
  嫉恨发狂而倾举国之力镇压至今，他使修炼人百折不挠而坚定二十载有余。
 </span>
</p>
<p class="p1">
 <span class="s1">
  这本书问世短短24年，就被翻译成四十多种语言，拥有读者亿万之众，遍及全球一百一十
 </span>
 <span class="s1">
  多个国家和地区，受欢迎程度足以跃居几千年来人类流传下来的所有道德与智慧经典之首；他已成为无数人每日必读之书，有人甚至已背诵上千遍
 </span>
 <span class="s3">
  ……
 </span>
</p>
<p class="p1">
 <span class="s1">
  这本书就是《
  <a href="http://www.epochtimes.com/gb/tag/%E8%BD%AC%E6%B3%95%E8%BD%AE.html">
   转法轮
  </a>
  》。
 </span>
</p>
<p class="p2">
 <span class="s1">
  《
  <a href="http://www.epochtimes.com/gb/tag/%E8%BD%AC%E6%B3%95%E8%BD%AE.html">
   转法轮
  </a>
  》是佛家上乘大法法轮大法（
  <a href="http://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
   法轮功
  </a>
  ）指导炼功人修炼的主要著作，作者为法轮功创始人李洪志大师。
 </span>
</p>
<p class="p1">
 <span class="s3">
  1994
 </span>
 <span class="s1">
  年
 </span>
 <span class="s3">
  12
 </span>
 <span class="s1">
  月，《转法轮》由中国国家广播电视局下属中国广播电视出版社出版发行；
 </span>
 <span class="s3">
  1995
 </span>
 <span class="s1">
  年
 </span>
 <span class="s3">
  1
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  4
 </span>
 <span class="s1">
  日，李洪志大师在北京公安大学礼堂举行《转法轮》首发式。
 </span>
 <span class="s1">
  今年的
 </span>
 <span class="s3">
  1
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  4
 </span>
 <span class="s1">
  日是《转法轮》出版发行
 </span>
 <span class="s3">
  24
 </span>
 <span class="s1">
  周年纪念日，几位来自不同国家的《转法轮》的读者，分享了研读这本著作的感想以及该书对他们生活的意义。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  “一本无价的指导书”
 </span>
</h4>
<p class="p3">
 <span class="s3">
  4
 </span>
 <span class="s1">
  年前，
 </span>
 <span class="s3">
  24
 </span>
 <span class="s1">
  岁的爱尔兰小伙子格瑞尼（
 </span>
 <span class="s3">
  Paul Greaney
 </span>
 <span class="s1">
  ）正漫无目的地在旧金山大街上溜达着，他在都柏林学习商业和中文之后，正在美国短期工作。和很多当代青年一样，他酗酒、抽烟，天天百无聊赖地在空虚中度日。当时，他正感觉自己的身体和精神都脆弱得不堪一击。
 </span>
</p>
<p class="p3">
 <span class="s1">
  正当他路过唐人街的时候，一个年轻的中国女孩走到了他面前，微笑地跟他打招呼：“你好！”并递过来两份小册子。
 </span>
</p>
<p class="p3">
 <span class="s1">
  格瑞尼一看，一份小册子上印着三个金色的大字“真、善、忍”，他内心忽地一动。那个时候他正在读老子的《道德经》和一些佛教的书，而
  <a href="http://www.epochtimes.com/gb/tag/%E6%B3%95%E8%BD%AE%E5%8A%9F.html">
   法轮功
  </a>
  是一种涵盖了佛家和道家理论的古老修炼方法。
 </span>
</p>
<p class="p3">
 <span class="s1">
  他回家就上网找到了《转法轮》和其他法轮功书籍，“这就是我要找的。”他对自己说。他自学法轮功动作，立即感受到了强大能量，从此他开始了法轮大法的修炼。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “我到现在可能读了
 </span>
 <span class="s3">
  一百
 </span>
 <span class="s1">
  多遍《转法轮》了吧，他对我来说是一本无价的道德指导书。”格瑞尼说，“我过去读过很多不同的哲学书，读过很多民族的经典书籍，但是没有一本书像《转法轮》这样能够指导我，并让我用平和、理性的方式帮助我的家人度过艰难的时期。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  “对当代人来说，《转法轮》是唯一的阐述更高层次法理的书，其他典籍或者修炼方法都不如他博大精深，并具有如此大的影响力。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  格瑞尼表示，他现在天天读《转法轮》，越读越明白做好人的道理，越来越平静，越来越远离烦恼。
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  “一本救命的书”
 </span>
</h4>
<p class="p3">
 <span class="s1">
  美国姑娘梅尔（
 </span>
 <span class="s3">
  Tiffany Meier
 </span>
 <span class="s1">
  ）
 </span>
 <span class="s1">
  在大学学习小提琴和电影制作，虽然她学习成绩优异，表面也很开心，但是学习的压力经常让她有一种压抑的情绪。
 </span>
</p>
<p class="p1">
 <span class="s1">
  大四那年的三月份，纽约还很寒冷。一天她自己一个人待在宿舍里，朋友们都出去了，她忽然感觉内心压力非常大：老师在催她毕业后选什么职业，父母的经济压力那么大，朋友无故埋怨她
 </span>
 <span class="s3">
  ……
 </span>
 <span class="s1">
  她感觉眼前一片灰暗，她忽然有一种自杀的渴望。
 </span>
</p>
<p class="p3">
 <span class="s1">
  她打开自己的电脑，上了博客，通常她心中烦闷的时候都会写写博客来排解。可是这次却没有效果，她的手在键盘上麻木地敲击著，忽然她看见自己打出的一行字：“你是知道答案的。”她于是想到了自己在
 </span>
 <span class="s3">
  4
 </span>
 <span class="s1">
  岁半就从母亲那里看过的《转法轮》，就背诵其中的几段文字，顿时她感觉心灵的天空亮了起来。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “多年来，《转法轮》就像我心中的太阳，我知道无论我多么艰难，多难受，只要我有《转法轮》，我一定会过得去的。”她说，“他叫我为别人着想，帮助别人，不和别人竞争；这本书对我来说就是一本救命的书。”
 </span>
</p>

<h4 class="p3">
 <span class="s1">
  “一本解惑的书”
 </span>
</h4>
<p class="p3">
 <span class="s1">
  在三州地区拥有三家诊所的整合医学专家杨景端博士一生都在探索人体与生命的科学，但是在2
 </span>
 <span class="s3">
  0
 </span>
 <span class="s1">
  年之前他却一直没有找到答案。
 </span>
</p>
<p class="p3">
 <span class="s3">
  1998
 </span>
 <span class="s1">
  年的一天他在自家附近的小卖部中发现了一套法轮功的书籍，他从此开始了法轮大法的修炼，立即奇迹般戒掉了多年的烟瘾。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “看了《转法轮》之后，人和人体等等怎么回事，对我一下子就全清楚了。”他说，“他让我明白了中医的来源、中、西医是解决什么问题的、以及人的灵魂和人体的关系等等一切我以前不明白的事情。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  “所以这本书对于我来说是一本解惑的书，他解答了我对人生、生命和我的专业领域所有的问题，用极其浅白、通俗的语言解决了所有困惑。”
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  “这本书是渡船”
 </span>
</h4>
<p class="p3">
 <span class="s1">
  中国大陆的一个法轮功学员大维（化名）从
 </span>
 <span class="s3">
  1995
 </span>
 <span class="s1">
  年开始读《转法轮》，到现在他已经背诵
 </span>
 <span class="s3">
  一千
 </span>
 <span class="s1">
  多遍了。
 </span>
</p>
<p class="p3">
 <span class="s1">
  自从中国共产党
 </span>
 <span class="s3">
  1999
 </span>
 <span class="s1">
  年
 </span>
 <span class="s3">
  7月20日
 </span>
 <span class="s1">
  以莫须有的罪名开始镇压无辜的法轮功学员以后，大维被抓五次。在北方一家劳教所里，他每天在心里默背一遍《转法轮》，就是靠书中强大的正念支撑才度过了那段非人的日子。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “我回想着《转法轮》上的每一页，每一段。共
 </span>
 <span class="s3">
  9
 </span>
 <span class="s1">
  讲，
 </span>
 <span class="s3">
  60
 </span>
 <span class="s1">
  个问题，
 </span>
 <span class="s3">
  332
 </span>
 <span class="s1">
  页，
 </span>
 <span class="s3">
  601
 </span>
 <span class="s1">
  个自然段，一共背了
 </span>
 <span class="s3">
  两百
 </span>
 <span class="s1">
  多遍。”他回忆说，“有时候为了加深理解，我就从倒数第一段背，一直背到第一段。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  “我的身体整天都在能量场的包围中，有时候能感受到表面身体的变化。”那时候，再恶的警察在他面前也恶不起来，电棍一打到他的身上就没电了，最后劳教所把诽谤法轮功的材料从他面前都收走了。
 </span>
</p>
<p class="p3">
 <span class="s1">
  大维把《转法轮》比做“船”。“就像一个人飘洋过海，并不是这个人如何厉害，他是靠渡船的承载过去的，要是没有渡船的承载，人早就淹死了。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  出狱之后，大维发现默写
 </span>
 <span class="s2">
  比
 </span>
 <span class="s1">
  背诵效果还要好，因为精力更加集中，他现在正在默写第十遍。“《转法轮》能净化人的思想，当你溶于其中后，思想中那些不好的想法还没有形成就被消掉了。”
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  “一部佛法”
 </span>
</h4>
<p class="p3">
 <span class="s1">
 </span>
 <span class="s1">
  秘鲁小伙子特里哥索（
 </span>
 <span class="s3">
  Enrico Trigoso
 </span>
 <span class="s1">
  ）第一次接触的《转法轮》是西班牙文的，他后来碰到的一些问题促使他想看中文原文，于是开始学中文，结果发生了神奇的事情。他从一个只认识几个中文字的人，学了三个月的中文《转法轮》便能够通读全书，
 </span>
 <span class="s3">
  6
 </span>
 <span class="s1">
  个月就能和中国人用流利的汉语交谈了。
 </span>
</p>
<p class="p1">
 <span class="s1">
  到现在，已成为作曲家的特里哥索已经看过七百
 </span>
 <span class="s1">
  多遍《转法轮》了。他越看越入神，思想也越来越扩大，对音乐的理解也越来越加深了。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “最关键的是我现在能够为他人着想了。以前别人对我不好的时候，我是不会去想自己哪里不好的，但《转法轮》教人真善忍，我现在会忍住，缓一下，向内找自己的不足，这是和我以前巨大的不同。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  他说，“而且，我对专业的态度更加认真。原来我认为音乐是用来娱乐和表现自己以及追求名利的，现在我发现音乐是很神圣的事情，可以修自己又可以帮助别人，这对我来说是巨大的变化。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  特里哥索
 </span>
 <span class="s2">
  用娴熟的
 </span>
 <span class="s1">
  中文引用释迦牟尼佛的
 </span>
 <span class="s2">
  预言谈起他对《转法轮》的认识
 </span>
 <span class="s1">
  ：“‘人身难得，中土难生，正法难遇’，如果你想要你的人生未来有个好归宿的话，你就一定要看《转法轮》。”#
 </span>
