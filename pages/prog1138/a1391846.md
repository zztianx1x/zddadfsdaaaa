### 中共前政协常委 侨委会主任孙怀山被判14年
------------------------

<div class="wysiwyg">
 【新唐人2018年09月17日讯】中共官媒周一通报，全国政协前常委、港澳台侨委员会主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙怀山.html" target="_blank">
  孙怀山
 </a>
 ，被判刑14年。据报，孙怀山是江派前常委贾庆林的亲信。
 <br/>
 <br/>
 判决称，1994年到2016年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙怀山.html" target="_blank">
  孙怀山
 </a>
 任中共共青团中央办公厅主任、全国政协副秘书长、常委、中共中央委员会委员期间，在获取土地使用权、承办个人画展、企业经营、项目审批、职务提拔等事项上，为他人谋利；非法收受财物，共计折合人民币3975万余元。
 <br/>
 <br/>
 判决还称，孙怀山是〝全家腐的典型〞，其妻、其子及儿媳均涉案。
 <br/>
 <br/>
 据报，孙怀山被查可能与香港特首选举有关联。2016年8月31日，孙怀山任中共政协港澳台侨委员会主任，还是十八届中央委员。
 <br/>
 <br/>
 另外，孙怀山任中共全国政协副秘书长达17年，与江派前常委、前中共政协主席贾庆林共事10年，被指是贾庆林的亲信。
</div>
