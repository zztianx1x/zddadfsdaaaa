### 四中全会年内泡汤？政治局会议释信号
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】中共中央政治局在10月31日召开会议，没有提及召开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 的时间。分析认为，这意味着四中全会持续推迟，今年召开的可能性已经泡汤了。甚至有可能推迟到2019年9月。
 <br/>
 <br/>
 按照中共此前惯例，中共每届中央全会都在召开当年的7、8月间公布大致开幕时间，然后在正式召开前夕的政治局会议上再次敲定准确日期和细节。今年因为加开了一次确定修宪的二中全会，外界一度猜测本届
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 将沿用往届三中全会的旧例，最迟在10月召开。
 <br/>
 <br/>
 但上述惯例也被打破，预料中的四中全会被一再推迟。
 <br/>
 <br/>
 10月31日，党媒报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 主持召开了政治局会议，研究当前经济形势，部署了〝年末各项工作〞。但会议上并未提及召开四中全会的事项，因此广受媒体热炒的四中全会10月召开成为一场子虚乌有的事。
 <br/>
 <br/>
 香港《明报》10月30日报导，中共商务部在吹风会上说，习近平11月5日将出席进博会的开幕式，并致开幕辞。进博会从5日开到10日，参会的各国领导人，还将顺道访问中国，日期从10月31日起到11月8日。所以，四中全会最快也要11月10日才能召开。
 <br/>
 <br/>
 由于此次中共中央政治局会议没有公布四中全会的任何信息。有猜测，会议或推迟到11月下旬召开，甚至可能要循例于2019年9月召开。
 <br/>
 <br/>
 自中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 以来，全会开会时间最晚的一次，除了1989年开会时间定于6月之外，其他几届大都是在9月中下旬召开，只有2012年前相对定在10月下旬召开。
 <br/>
 <br/>
 至于四中全会一再推迟的原因，有报导称，北京政界有两个传言，一是因为有政变风险，传武警有一批人策划〝起事〞。不过，自习近平掌管权力后，各种政变传闻一直不断，习的安保也在不断加强。
 <br/>
 <br/>
 另外一个传言是，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 使得中国经济急剧恶化，习近平希望在四中全会上，能拿出一个有说服力的计划。但面对积重难返的国内经济问题，以及中美贸易战的压力，特别是在中共体制的束缚之下，北京很难再推出有效的应对之策。
 <br/>
 <br/>
 自由亚洲分析认为，中共权力斗争陷入胶着状态，而当前北京高层无力掌控局势，是四中全会推迟的原因之一。
 <br/>
 <br/>
 报导还预测，四中全会在美国中期选举有了结果之后召开。北京期盼民主党重新掌控美国国会参、众两院，到时会针对贸易战最新形势，确定后续方案。
 <br/>
 <br/>
 不过，原香港城市大学政治学教授郑宇硕认为，无论美国中期选举结果如何，对中共施压是美国民主共和两党的共识。美国的中期选举不应是全会延期的原因。他相信全会延期的真正原因，是中共领导层意见不一，就如何回应美国的要求仍举棋不定。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
