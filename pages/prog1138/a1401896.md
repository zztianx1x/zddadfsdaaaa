### 3上将落马传闻刚过 中共数名高级将领异动
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】中共军队新一轮大调整和大整肃正在同时进行中，继传出3名军中上将同日被抓后，陆媒3日称，又有多名军中高级将领被调整。
 <br/>
 <br/>
 12月3日，陆媒报导称，中共军队人事再现异动。近期中共军队高级将领调整密集。他们分别是：原西部战区副司令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_韩胜延.html" target="_blank">
  韩胜延
 </a>
 出任中部战区空军司令；原空军空降兵某军军长刘发庆跨军种出任陆军副司令；原北京军区空军副司令员杨卫东少将已调任南部战区副参谋长。
 <br/>
 <br/>
 武警部队副政委颜晓东中将兼任了武警部队党委政法委书记；原第14集团军副政委高伟少将已接下颜晓东原本担任的武警部队政治工作部主任职务；原第13集团军政治部主任、正军级少将刘诚已担任北部战区政治工作部副主任、战区党委政法委副书记。
 <br/>
 <br/>
 原西部战区陆军纪委书记姚永良出任南部战区副政委兼政治工作部主任、党委政法委书记；原吉林省军区副司令员刘维已任吉林省军区司令员；副战区级空军少将李勇已担任联勤保障部队司令员；江西省委常委、省军区政委杨笑祥调任浙江省军区政委；原新疆军区副司令员李发义转任新疆生产建设兵团副司令员。
 <br/>
 <br/>
 其中，韩胜延是中共空军的一名老将，历任原成都军区空军副参谋长、原兰州军区空军参谋长、原成都军区空军副司令、空军所属试验训练基地司令等。 2009年韩胜延晋升为空军少将，2017年晋升为空军中将。
 <br/>
 <br/>
 原任中部战区空军司令的是现年63岁的庄可柱。庄可柱历任兰州军区空军参谋长、广州军区空军参谋长。2011年，接替到龄退役的朱清益出任兰州军区空军司令员，跻身副大军区级将领之列。2013年7月，平调至北京军区空军司令员，此轮军改启动后，出任中部战区空军首任司令员。
 <br/>
 <br/>
 另外，原空军空降兵某军军长刘发庆已跨军种出任陆军副司令，跻身副战区级将领之列。刘发庆长期在空降兵部队服役，2014年晋升为空军少将军衔。2017年担任空军空降兵某军首任军长。
 <br/>
 <br/>
 据了解，2018年以来，中共高级将领跨军种调动已有多个案例。如海军原副政委兼政治工作部主任陈学斌中将转任空军副政委，北部战区空军政治工作部主任王征少将升任海军政治工作部主任，空军政治工作部原副主任刘青松少将升任东海舰队政委。
 <br/>
 <br/>
 中共军队最新一轮大整肃与此轮大调整正在同时进行中。
 <br/>
 <br/>
 11月30日，多家海外媒体报导称，11月29日同一天，3名中共军中上将被抓，他们都已退出现役，这3名上将是前海军司令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴胜利.html" target="_blank">
  吴胜利
 </a>
 、前南京军区司令
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_蔡英挺.html" target="_blank">
  蔡英挺
 </a>
 、前中央军委装备发展部政委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王洪尧.html" target="_blank">
  王洪尧
 </a>
 。
 <br/>
 <br/>
 据称，三人同日被抓的原因，均是清除中共前军委副主席徐才厚和郭伯雄的余毒的一部分。不过目前相关消息仍没得到中共官方证实。
 <br/>
 <br/>
 8月下旬起，就陆续传出有中共将领落马。其中包括原总参情报部部长、东部战区副司令兼参谋长杨晖中将被传落马，以及西部战区副司令刘小午被传已被拘查。还有南部战区政委魏亮和原广州军区司令徐粉林被降级退役等。
 <br/>
 <br/>
 而上述将领也都与已落马的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军老虎.html" target="_blank">
  军老虎
 </a>
 郭伯雄、徐才厚、中共前军委委员张阳和房峰辉等人有关联，被认为是清除他们在军中余毒的一部分。
 <br/>
 <br/>
 10月24日，多家媒体消息称，由于中共军队内部120名将领正在接受反腐调查，军中已出现极大内乱。虽然上述消息至今仍未获中共官方证实。但自从习近平又一次发出反腐不会〝变风转向〞信号之后，外界普遍分认为，军中新一轮的大整肃已拉开帷幕。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
