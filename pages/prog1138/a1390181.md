### 江西广播电视台副台长李建国落马
------------------------

<div class="wysiwyg">
 【新唐人2018年09月04日讯】中共广播电视系统又一名贪官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 。中共官媒报导，江西广播电视台副台长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李建国.html" target="_blank">
  李建国
 </a>
 ，因涉嫌严重违纪违法，目前正在接受调查。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李建国.html" target="_blank">
  李建国
 </a>
 1994年8月起，在江西电视台任职，曾任第二套节目编辑部主任、台长助理、台长助理兼都市频道总监，2012年5月升任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江西电视台副台长.html" target="_blank">
  江西电视台副台长
 </a>
 。
 <br/>
 <br/>
 广播电视系统被认为是中共的喉舌，也是腐败和淫乱的黑窝。
 <br/>
 <br/>
 中共十九大后，主管文宣的江派政治局常委刘云山退下，中宣部前副部长、网信办主任鲁炜
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 ，官员相继被判刑或逮捕，包括湖南电视台前副台长罗毅，广东广播电视网络公司前副总经理杨清仿，湖南广电网络控股集团有限公司前党委副书记、董事长龙秋云等。
 <br/>
 <br/>
 据《明慧网》统计，1999年7月20日以来，中共利用宣传机器抹黑法轮功，各级宣传部门进行全民洗脑，编造谎言，诋毁诬陷，为残酷迫害推波助澜，成为杀人工具。中共宣传部门的各级负责人，有着不可推卸的责任，已有百名官员遭恶报。
</div>
