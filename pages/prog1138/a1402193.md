### 谁让习近平〝检讨〞？习回国面临七件大事
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月06日讯】外访多日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，率领一众官员已于6日赶回北京。有分析认为，尽管美中双方宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 〝停火90天〞，但贸易战仍存变数。面对内忧外患的各种困境，在接下来的日子，习近平急需处理七件大事。其中包括，大外宣喊出的让习近平〝检讨〞，究竟释放出何种信号？以及北京如何应对〝缓刑〞90天的贸易战。
 <br/>
 <br/>
 美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 在阿根廷G20峰会间隙的双方首脑会晤后，宣布停火90天。美方多位官员表示，美中双方达成〝令人难以置信的协议〞，而中共媒体除了报导停火外，对美中双方达成的协议具体内容只字不提，对接下来的90天谈判更是留下许多未回答的问题。
 <br/>
 <br/>
 中共外交部是唯一每天对外媒进行简报的政府部门，就中美贸易协定却重复称，媒体应该向商务部询问细节，而商务部6日举行每周的新闻发布会，仍未提及任何内容。
 <br/>
 <br/>
 一名中共官员告诉路透社，中共官员们需要〝等待领导人回国〞后才能宣布细节。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 和多位高层官员结束对西班牙、阿根廷、巴拿马、葡萄牙国事访问，已于12月6日紧急赶回北京。
 <br/>
 <br/>
 有知情者分析，美中双方虽然宣布贸易战〝停火90天〞，但未来仍存变数，面对严峻的外患内忧，习近平在接下来的90天内，急需处理7件大事。
 <br/>
 <br/>
 一，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40周年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纪念大会.html" target="_blank">
  纪念大会
 </a>
 ；二，召开中央经济工作会议；三，召开中央农村工作会议；四，中美联合公报的系列纪念活动；五，召开
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四中全会.html" target="_blank">
  四中全会
 </a>
 ；六，谁让习近平〝检讨〞？七，90天后北京如何应对贸易战。
 <br/>
 <br/>
 而这七件大事，对习近平来说件件揪心。
 <br/>
 <br/>
 关于改革开放40周年纪念大会，此前北京已经举办多次纪念活动，预计在12月18日，中共十一届三中全会召开40周年之际，官方将会高规格纪念。11月13日，那次纪念〝改革开放40周年〞大型展览举行时，已经有3名中共一把手被消失。
 <br/>
 <br/>
 3人分别是在中共改革开放初期担任重要职务的前领导人华国锋、胡耀邦和赵紫阳。而三人消失的原因，被认为是因三人都属于被迫下台的，所以官方必须冷处理。
 <br/>
 <br/>
 至于中共第十九届四中全会，中南海知情者分析：仍存在变数，不过，为了避免海内外的种种质疑和猜测，习近平倾向在明年3月两会之前召开，以便进一步消除党内〝不同声音〞，尤其是部分〝杂音〞，从而确保两会的顺利召开。
 <br/>
 <br/>
 对于四中全会一再推迟的原因，此前就有报导说，北京政界传言，一是因为有政变风险，传武警有一批人策划〝起事〞；二是，中美贸易战使得中国经济急剧恶化，习近平希望在四中全会上，能拿出一个有说服力的计划。
 <br/>
 <br/>
 但面对积重难返的国内经济问题及中美贸易战的压力，特别是在中共体制的束缚之下，北京很难再推出有效的应对之策，中共权斗陷入胶着状态，导致北京高层无力掌控局势，因此四中全会一推再推。
 <br/>
 <br/>
 最为敏感的一件事是，在习出访期间，中共大外宣突然发表文章，要让习近平〝检讨〞，尽管文章随后被紧急删改，但这其中究竟释放出何种政治信号，背后是否涌动着一股凶险的政治暗流？习当局将如何处理这起事件，有待于进一步观察。
 <br/>
 <br/>
 最重要的一件事是，已经开始倒计时的90天贸易战〝缓刑〞时间内，中美将就双边贸易结构性改革达成协议，北京将如何应对？对习近平来说，也是当务之急。
 <br/>
 <br/>
 根据白宫在会晤后发布的声明，习近平承诺减少中美间的贸易不平衡，并将从美国采购大量农产品等产品。川普则同意2019年1月1日暂时对2,000亿美元中国进口商品的关税维持在10%。但如果双方不能在90天内就双边贸易的结构性改变达成协议，对这批商品的关税仍将提高到25%。
 <br/>
 <br/>
 这意味着对中方来说，虽然获得了90天的〝缓刑〞时间，但是仍然是祸福难料，因为美国方面已换上了一位更加强硬的官员——莱特希泽负责这90天的谈判。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
