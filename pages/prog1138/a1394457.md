### 中纪委宣布公安部副部长孟宏伟被调查
------------------------

<div class="wysiwyg">
 【新唐人2018年10月07日讯】中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 、担任国际刑警组织主席的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 失踪几天后，北京当局周日宣布他已被扣押调查。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中纪委.html" target="_blank">
  中纪委
 </a>
 官网10月7日通报，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 因为〝涉嫌违法〞，目前正接受中共监委监察调查。
 <br/>
 <br/>
 这是法国电视台10月5日率先披露孟宏伟失踪后，北京当局的首次回应。
 <br/>
 <br/>
 香港《苹果日报》报导，孟宏伟可能涉嫌参与2015年的北戴河政变未遂案。
 <br/>
 <br/>
 65岁的孟宏伟长期在中共公安部任职，被认为是中共前政法委书记周永康的亲信。
 <br/>
 <br/>
 10月4日，孟宏伟的妻子向里昂巿警方报案，声称自己生命受到威胁。周日她还向媒体宣读声明，称案件属于国际范畴和中国人民。
 <br/>
 <br/>
 外界认为，孟宏伟夫妇早已设计好被调查后的应对计划，目的是想让孟宏伟的妻儿能获得法国的庇护身分。
 <br/>
 <br/>
 法新社和路透社消息，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法国警方.html" target="_blank">
  法国警方
 </a>
 已展开自己的调查，并给予孟宏伟的妻子司法安全保护。
 <br/>
 <br/>
 新唐人电视综合报导
