### 曾任610主任 淮安前副市长倪兴余被判十年半
------------------------

<div class="wysiwyg">
 【新唐人2018年08月28日讯】中共官媒周二消息，江苏淮安市前副市长、公安局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_倪兴余.html" target="_blank">
  倪兴余
 </a>
 ，因受贿罪
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被判刑.html" target="_blank">
  被判刑
 </a>
 十年六个月。
 <br/>
 <br/>
 判决称，2007年到2014年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_倪兴余.html" target="_blank">
  倪兴余
 </a>
 担任淮安市公安局局长、副市长期间，利用职务便利，在承接工程、职务调整以及案件处理等方面，为他人谋取利益，收受财物，共计人民币925万余元。
 <br/>
 <br/>
 倪兴余1978年12月进入江苏省公安系统，2003年以来，曾任扬州市公安局副局长、610办公室主任；淮安市副市长、公安局长；镇江市副市长、公安局长；江苏警官学院副院长等职；2016年9月被免职，提前退休。
 <br/>
 <br/>
 〝610〞办公室，是江泽民迫害法轮功的专门机构。
 <br/>
 <br/>
 另据《明慧网》报导，倪兴余任职期间，积极参与迫害法轮功学员，多次被点名。其恶行还殃及家人。
 <br/>
 <br/>
 倪兴余还因对迫害法轮功学员负有责任，被〝追查迫害法轮功国际组织〞列为追查对象。
</div>
