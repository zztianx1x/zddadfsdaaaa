### 【禁闻】中共称孟宏伟收受贿赂 诸多谜团待解
------------------------

<div class="wysiwyg">
 【新唐人2018年10月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 9月底前往中国后失踪的消息，迅速传遍全球。虽然
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 官方宣称孟宏伟因收受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贿赂.html" target="_blank">
  贿赂
 </a>
 被查，但此案仍有重重
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谜团.html" target="_blank">
  谜团
 </a>
 未解。孟宏伟妻子在法国的记者会上声称要追求真相、正义。到底是什么样的真相和正义？从她展示的私信来看，孟宏伟此前极有可能早已预感自己会出事。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 公安部长赵克志10月8号凌晨主持会议披露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 涉及收受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贿赂.html" target="_blank">
  贿赂
 </a>
 ，并提到要坚决彻底肃清
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康流毒.html" target="_blank">
  周永康流毒
 </a>
 影响。
 <br/>
 <br/>
 孟宏伟妻子格蕾丝・孟，此前在向法国警方报案丈夫失踪，并受到法国警方司法安全保护后，7号在法国里昂举行记者会，但她背对摄影机，不愿露面。
 <br/>
 <br/>
 据美联社报导，格蕾丝提到孟宏伟9月25号用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 的手机给她发短信〝等我电〞，4分钟后，发来一把刀子的图像后，就失去联系。她认为这是孟宏伟示警的信号，表示身在危险中。
 <br/>
 <br/>
 她说：〝从现在开始，我已经从悲伤和恐惧转向追求真相、正义和历史责任。〞
 <br/>
 <br/>
 在格蕾丝发言后不到一小时，中共中纪委国家监委网站出现一则重磅短消息：公安部副部长孟宏伟涉嫌违法，目前正接受国家监委监察调查。
 <br/>
 <br/>
 而在孟宏伟受查消息传出后，国际刑警组织声称收到他的辞职通知，即时生效。
 <br/>
 <br/>
 时事评论员夏小强：〝这个辞职信当然是孟宏伟在被中共官方控制之后所发出的，那么自然也就是被辞职了。〞
 <br/>
 <br/>
 尽管中共官方发声，但这起事件目前仍有诸多
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谜团.html" target="_blank">
  谜团
 </a>
 。
 <br/>
 <br/>
 丈夫官至中共正部级，妻子却第一时间向法国警方报案，法国警方介入、中共的动作，都让人联想到2012年王立军逃亡美领馆案。
 <br/>
 <br/>
 熟悉中共内部事务的一位体制内人士向《新唐人》表示，孟宏伟肯定不是因为经济问题出事，否则中共一定会等他国际刑警组织主席任期满后才把他拿下。现在中共这个做法非常反常，完全不顾颜面，不惜制造国际丑闻都要把孟宏伟匆忙拿下，可以看出情况非常紧急，是偶发事件，很可能是威胁到了中共某位核心高层人物的安危。但具体内情目前还不清楚。而当局最终的处理，一定是使用经济罪名等惯用手法。
 <br/>
 <br/>
 至于孟宏伟太太的举动，这位人士表示，很显然是事先和孟宏伟约好。否则不可能一上来就去寻求国际帮助，因为这就相当于跟中共撕破脸了。
 <br/>
 <br/>
 原大陆法学博士张杰：〝（孟宏伟）他的影响力和他掌握的机密可能会远远超过王立军，（中共）可能采取诱骗的方式让他回国。但是孟宏伟显然是有准备的，因为他妻子、孩子都留在法国，并且他妻子在发布会上展现他的私信，我觉得孟宏伟是有交代的，就是说如果我回去失踪了，我会怎么样，那么你看到马上报警。因为一旦形成了国际事件，中共可能会投鼠忌器。〞
 <br/>
 <br/>
 孟宏伟2016年起担任国际刑警组织主席，任期到2020年止。2017年底，他被免去中共国家海洋局副局长和海警局局长职务，今年4月他又被免去公安部党委委员职务。
 <br/>
 <br/>
 原大陆法学博士张杰分析，孟宏伟妻子将来很可能申请政治庇护，加上孟宏伟在中国被抓，这对中共政权是一个重大挫败和讽刺。
 <br/>
 <br/>
 张杰：〝如果说国际刑警组织的主席都在中国没有安全，那么中国跟一个黑社会又有什么区别呢？这个（事件的）影响力不亚于王立军。因为后续他的妻子还有没有核弹级的爆料，或者其他东西，我们都不知道，可能还需要拭目以待。〞
 <br/>
 <br/>
 夏小强：〝在中共的体制下，从中共的党总书记、到中共各级的高官、企业家甚至普通民众，没有一个人是安全的。〞
 <br/>
 <br/>
 而对于孟宏伟妻子召开记者会要求真相正义。有网友说，〝孟宏伟官至公安副部长，不知道帮党国干了多少坏事呢？党国内斗，看戏就好。〞〝追求真理，正义，那说明她手上有货。〞
 <br/>
 <br/>
 还有网友表示：薄熙来和令计划被抓，也没看到其美国的亲人敢于站出来说No。如今有了第一个中共高官妻子站出来，就会有第二、第三个站出来，这是中共最害怕的。
 <br/>
 <br/>
 采访/常春、周玉林 编辑/王子琦 后制/陈建铭
