### 高僧细说黄山〝蛤蟆洞〞与江泽民结局
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】乙酉年初春，韩国京畿道仪旺市冠岳山龙珠寺及全罗南道顺天市海龙面之须弥山禅院佛像上有优昙婆罗花盛开。此花非同小可，自青铜佛像脸上生成。中国有成语曰〝昙花一现〞，此昙花实为优昙婆罗花也，三千年开放一次。
 <br/>
 <br/>
 按佛经记载：优昙婆罗花开乃法轮圣王，或曰转轮圣王驻世说法之征兆。
 <br/>
 <br/>
 消息传入中国安徽祁门县，县里有一农户姓文名传灯。这文传灯家无长物，孑然一身。闻知有优昙婆罗花异事，自思道：〝人道优昙婆罗花开，应在弥勒佛出世。某虔心向佛已有多时，新近死了父母，不如趁此机会云游四海，寻个得道高僧剃度了，从此青灯礼佛，了却一生。若有缘听弥勒佛讲法，则是天大幸事。〞
 <br/>
 <br/>
 检点家中，实在无甚值钱之物，唯有祖上传下宝剑一口，长不过一尺二寸，锋利异常，可做晋见之物。传灯携了宝剑，自思囊中羞涩，便就近往黄山上寻得道之人。
 <br/>
 <br/>
 行至光明顶，见一导游正向一群人解说道：〝几年前，在黄山发现一天然石窟，内有大厅、石柱、石室、水池、及壁画，皆是天然生成，各位有空可以随我前去赏玩。〞这群人游兴正浓，听导游如此说，便轰然答应，随导游下山去了。传灯左右无事，便尾随着跟在后面。
 <br/>
 <br/>
 进了石窟，导游道：〝这石窟十分灵异，1999年方被人发现。当时发现洞口处皆是碎石，遂不断向内挖。孰料越挖越多，在清理碎石时亦发现碎石之中夹杂着很多牛骨，令工人们百思不解。若本来为洞，无论人工或天然，无人会有力有闲在废弃此洞后再以碎石牛骨一点不差地填满，当地人亦十分费解，故名此洞为‘千年迷窟’。洞内壁画更是无人能解。2001年，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 听说此事，还专程来过一次。〞
 <br/>
 <br/>
 众人听说了，便四散去看那洞中的壁画，皆不知所云。当时虽是初春，洞内十分阴寒。传灯正看之际，忽觉腰间宝剑嗡嗡作响，不断跳动，心知有异，取剑出来，但见剑身上文如流水，自柄至尖，连绵不断，嗡嗡之声越来越响，直欲脱手飞去。传灯连打了几个寒噤，怕人看到他宝剑之异状，连忙还剑入鞘，急步出洞。
 <br/>
 <br/>
 传灯走得忙了，出洞口时向右转，却拌在什么东西上摔了一跤，爬起来回身一看，见一老僧身披黄色袈裟，正贴着洞口趺坐在地上，单手立掌于胸前。传灯连忙道歉。那老僧也不恼，淡淡问道：〝施主何故如此慌张？〞传灯道：〝无它，洞内阴寒彻骨，出来晒晒太阳。〞
 <br/>
 <br/>
 老僧沉吟了一下道：〝难得、难得。世人庸碌者甚多，浑浑噩噩，不知此洞之险，将其当作胜地游玩，唯有施主根基非凡，能见微知着，及早抽身出洞。〞传灯见老僧话中透着玄机，知其非常人也，便道：〝某亦不知凶险，惟家传宝剑跃动鸣响，恐被他人看见，故此出洞。〞老僧道：〝且借剑一观。〞
 <br/>
 <br/>
 传灯捧出宝剑，交与老僧。那老僧拔剑出鞘，反覆看了片刻，问道：〝施主可知此剑来历？〞传灯道：〝不知，此为我家世代单传之物，据说已经两千五百年了。〞
 <br/>
 <br/>
 老僧点点头，问道：〝施主可姓文？〞传灯大奇，道：〝我便姓文，师父怎知？〞
 <br/>
 <br/>
 老僧道：〝这把剑有些来历，春秋年间有铸剑师欧冶子，为楚王铸了三柄宝剑，一曰龙渊、二曰太阿、三曰工布，后历经楚平王、伍子胥与吴王夫差之手。吴越争霸之际，越王勾践曾被夫差俘虏，在石屋养马三年，回国后卧薪尝胆，欲报此仇。勾践手下有两个大夫，曰范蠡、文种。文大夫献七策，勾践只用三条便灭了吴国。范蠡领兵灭吴时，夫差以工布剑自刎，临死前传语范蠡，曰：‘狡兔死，走狗烹；飞鸟尽，良弓藏；敌国破，谋臣亡。’
 <br/>
 <br/>
 范蠡得了宝剑，又听了夫差遗言，便劝文种道：‘越王为人，长颈鸟嘴，忍辱妒功，可以共患难，不能同安乐。你我大事已完，不如归隐。’文种不听，范蠡遂遗工布剑与文种，自己携了西施，出三江，泛舟五湖。那勾践果是个心胸狭窄之辈，妒忌文种大才，赐死了文种。老僧从宝剑上推断，若为家传，施主便该姓文了，此剑即工布也，颇有灵气。〞
 <br/>
 <br/>
 传灯听老僧侃侃而谈，心生敬仰，便问道：〝此剑方才为何在洞中跃动鸣响，师父适才言道‘世人庸碌者甚多，浑浑噩噩，不知此洞之险’，此为何意？〞
 <br/>
 <br/>
 老僧道：〝此为天机，唯虔心向佛者方可语之。〞传灯双掌合十，道：〝不瞒师父，此次在下携剑出游，便要找一方外高僧开示根本，指解源流。若师父不弃，愿剃度在师父门下。在下别无贵重之物，谨以此宝剑为晋见之礼。〞
 <br/>
 <br/>
 老僧笑道：〝若你果有此心，也是个不贪富贵不爱财之人。如此，说与你倒也不妨，只是我做不了你师父，世间只有一人能够度你。〞老僧抬眼望着天边出了一会儿神，继续说道：〝韩国佛像上有优昙婆罗花开，应在转轮圣王以弥勒佛乘下世传法。那转轮圣王无人知其来历，释迦佛当年称他为‘万王之王的无上王’。
 <br/>
 <br/>
 释迦佛在古印度传法之时，称印度为‘五毒恶世’，如今两千五百年过去，世间宗教皆走入末法，当代社会十恶俱全。人间不过是表象而已，从人、神界到法界，再层层往上皆在宇宙苍穹之末劫之中。欲挽救宇宙，内部任何生命皆无能为力，惟有来自宇宙之外之神方有这再造乾坤之洪大法力。这转轮圣王便是宇宙获得拯救之唯一希望了。〞
 <br/>
 <br/>
 传灯听得出神，问道：〝那转轮圣王莫非如释迦牟尼一般将要转生人间了吗？〞
 <br/>
 <br/>
 老僧道：〝非也。世人见优昙婆罗花开，只道是转轮圣王尚未出世，实则他已在世间讲法十三年了。〞传灯掐指算去，失惊道：〝师父说的莫非是法轮大法？〞
 <br/>
 <br/>
 〝孺子可教，〞那老僧笑道，〝其中详情还要你自己去看才会知道。如今且说说这‘千年迷窟’。法轮圣王传法，宇宙中败坏的生命形成了一股极恶的旧势力。为阻碍正法，造出了一个江泽民，在世间疯狂对大法行恶。
 <br/>
 <br/>
 这江泽民元神本是个蛤蟆，刚才所进的迷窟其实是个蛤蟆洞。迷窟的入口扁扁平平，像个倒扣的盘子，就是蛤蟆嘴；迷窟本身就是蛤蟆的肚子；迷窟顶部圆鼓凸出，上有一圈圈绿色，那便是蛤蟆后背。刚才施主打寒噤，乃是因为洞中阴气极重，聚集了不少肉眼凡胎看不见之烂鬼，宝剑跳动便是要斩妖邪啊。〞
 <br/>
 <br/>
 传灯恍然大悟，回思适才在洞中所见壁画，方知每一幅皆与蛤蟆有关，便问道：〝导游说江泽民来此游玩，在洞中待了很久。〞老僧道：〝不错。江泽民镇压法轮功元气大伤。2001年5月，江泽民之皮囊奄奄一息，于是来到黄山这蛤蟆洞中补充黑色能量。
 <br/>
 <br/>
 暴君不顾民之疾苦，这江泽民游黄山乃五月下旬之事，花费了两亿元人民币。黄山地区失业者甚众，食不果腹，却专为他来修筑公路，空运水果花费数十万元。上山动用保安武警不计其数，连他上山坐的滑竿儿皆是用进口的竹子和绳子特别定制的。〞
 <br/>
 <br/>
 传灯对江泽民本无好感，便问道：〝这江泽民结局如何？〞
 <br/>
 <br/>
 老僧道：〝果报已在目前。那转轮圣王是个慈悲无量的佛主，虽然江迫害法轮功，最初还是给了他一段悔过自新的机会。直到2000年9月，江恶贯满盈，罪业弥天，这机会被他自己断送了。要知道虽然佛法慈悲，但是威严同在！
 <br/>
 <br/>
 江泽民元神在2000年9月间既彻底被打入无间地狱了，现在世间上活动的，不过是江的人皮，以及操纵它的烂鬼而已。法轮功弟子洞彻天机，称之为‘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江鬼.html" target="_blank">
  江鬼
 </a>
 ’，准确贴切。这江鬼皮囊亦现黄斑病、黑腿病，眇一目、跛一足，更大业报源源不绝。自古迫害佛法，将入无生之门，永不得超生。〞
 <br/>
 <br/>
 传灯道：〝这江鬼却好似聊斋志异中披着画皮的女鬼。〞
 <br/>
 <br/>
 老僧呵呵笑道：〝那鬼也要披张美女人皮，只可惜江丑体态臃肿，肚囊肥大，活似直立蛤蟆，倒难为那些烂鬼了。〞沉吟片刻，老僧又道：〝这些烂鬼各有分工，有能说的、能唱的、会弹奏乐器的，平时无事，烂鬼也不理它。因江鬼魂魄俱丧，故素常神思恍惚，似随时撒手人寰，一旦需要会见外宾、或为镇压法轮功造势，烂鬼就会摆布江鬼肉身，令其伶牙俐齿，又唱又跳。〞
 <br/>
 <br/>
 传灯问道：〝留此祸胎是何用意？〞
 <br/>
 <br/>
 老僧道：〝人见目前，天见久远。修炼有素之人，智慧朗照，掌上千秋史，胸中万卷书，那时回顾人类历史，如看一场大戏。现在这丑角还不到谢幕之时，只怕还有最后一场好戏上演。〞
 <br/>
 <br/>
 传灯好奇，便问道：〝好戏何时上演？〞
 <br/>
 <br/>
 老僧道：〝天道微妙高远，到时施主自然知道。〞
 <br/>
 <br/>
 传灯又问道：〝适才师父提到无间地狱，请问何为无间地狱？〞
 <br/>
 <br/>
 老僧道：〝无间地狱极广极大。无间者，地狱之刑罚永不间断者也。
 <br/>
 <br/>
 无间有五：其一为‘时无间’，即时间没有间断，日夜受罪，永无停止之时；其二为‘空无间’，受刑者全身每一处皆在受刑，毫无间断；其三为‘罪器无间’，即刑罚的器具没有间断，不停用各式各样的刑具用刑；其四为‘平等无间’，无论男女，亦不论前世身份，同样平等无间，同样要受刑；其五为‘生死无间’，受刑人每一刻既痛苦而死，但又活过来继续受刑。生命在层层灭尽中的痛苦，永无停息。此为宇宙中最可怕之事了。〞
 <br/>
 <br/>
 传灯还要再问，忽然一股冷气从体内直冲出来，打了个寒噤。那老僧道，〝施主刚才中了阴气，此间人员嘈杂，不如到莲花峰顶休息一下。今夜老僧权做护法，明日施主便下山找那法轮佛法去吧。〞
 <br/>
 <br/>
 传灯次日醒来已是旭日初升之时，老僧早已不知去向，极目四顾，但见群峰挺秀，卓立云海，天都绝顶，霞光万道。传灯知此为天地复明之兆。对老僧言语深信不疑，下山后，便找了本《转法轮》，依法修行去了。
 <br/>
 <br/>
 ※※※
 <br/>
 <br/>
 某年月日，全球公审江泽民大联盟联络各国大法官组成陪审团，在天安门广场对江泽民进行公审，其时法轮功真相已大白于天下，陪审团宣读江泽民罪状达上千页，最后以叛国罪、贪污罪、酷刑罪、反人类罪、群体灭绝罪等等判处江泽民极刑。
 <br/>
 <br/>
 话音刚落，半空中忽然降下来一根绳索将江泽民从头到脚牢牢捆住，一只钩子将江泽民倒挂在半空中，片刻之间，风雷大起。万千闪电同时击在江泽民的每一寸肌肤上，烟雾缭绕之际，江泽民的衣物、头发、肌肤、内脏、骨骼同时起火，整个身躯全被雷火消灭殆尽，未留一点残余。
 <br/>
 <br/>
 彼时，文传灯已修行了一些时日，亲往天安门见证江泽民下场，愈加庆幸自己得法修行，不妄在法轮圣王传法时得了人身，口占一诗以言志：
 <br/>
 <br/>
 转世人间不计春，几回沧海变成尘，
 <br/>
 幸结法缘朝佛主，忍苦精进喜归真，
 <br/>
 修得长生千古秀，炼就金刚不坏身，
 <br/>
 助师转轮了洪愿，扫灭群魔正乾坤。
 <br/>
 <br/>
 吟罢，心中甚喜，但听得大街上爆竹响成一片，锣鼓喧天。人人喜气洋洋。传灯至此方才明白那
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄山老僧.html" target="_blank">
  黄山老僧
 </a>
 所言〝丑角谢幕〞之意。正思索间，忽然半空中飘下一幅黄绢，刚好在传灯面前盘旋，传灯接绢在手，见上面有六句诗，其文曰：
 <br/>
 <br/>
 丑角
 <br/>
 <br/>
 三脚蟾蜍现世丑
 <br/>
 十恶俱全尽阴谋
 <br/>
 此邪尽做丢人事
 <br/>
 无德无才装风流
 <br/>
 中华五千文物齐
 <br/>
 只欠此丑戏中秀
 <br/>
 <br/>
