### 一件让中共极惊惧的事 在中国民间快速传播
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月16日讯】十多年以来，大陆的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 一直在用写上真相的人民币揭露中共谎言，劝民众退出中共党团队保平安，而且数量惊人，这让中共极度恐惧却又无计可施，只能眼睁睁看着真相在民间快速传播。
 <br/>
 <br/>
 中共自1999年迫害法轮功后封闭了国内外一切能揭示法轮功真相的舆论工具，用一言堂的媒体嫁祸、抹黑法轮功，挑起中国人对法轮功的误解和仇恨。从开始的公开迫害，到背地里见不得人的阴毒迫害。
 <br/>
 <br/>
 2012年王立军出逃美领馆之后，在国际上曝光了中共活摘
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功学员.html" target="_blank">
  法轮功学员
 </a>
 的身体器官，高价出售做器官移植，牟取暴利，干出了〝这个星球上从未有过的罪恶〞，19年的残酷迫害，使无数法轮功学员致残、致死，家破人亡，甚至失去了生命。
 <br/>
 <br/>
 然而，法轮功学员坚守着〝真、善、忍〞的信念，不断的向世人讲清着法轮功被迫害的真相。
 <br/>
 <br/>
 由于一切宣传机器都掌控在中共的手里，法轮功学员只能面对面的讲清真相，还有使用写有真相的人民币（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_真相币.html" target="_blank">
  真相币
 </a>
 ）等方式传播真相，揭露共产党的邪恶本质，让人们明辨是非，选择善良，给自己选择美好的未来。
 <br/>
 <br/>
 <h4>
  真相币从〝零售〞转〝批发〞
 </h4>
 <br/>
 据《明慧网》报导，一位大陆老年法轮功女学员讲述了其将真相币换给商家从〝零售〞到〝批发〞的过程。
 <br/>
 <br/>
 以前这位大陆老年法轮功女学员都是从其他法轮功学员那里拿来真相币，自己或家人买东西时把它花出去。一次她回农村老家，给亲戚们讲法轮功真相，谈到真相币，她的两个弟妹各拿出一张写有〝诚心敬念法轮大法好、真善忍好天赐平安〞的真相币让她看。她们还说舍不得花，得留着它保护自己。自此，这位女学员决心做更多的真相币，让它救度更多的世人。
 <br/>
 <br/>
 她经过思考后，就开始到经商者那里换零钱，第一次就换回来七、八百元。然后把换回来的零钱进行整理，把旧的、残的剔出去，用消毒液和洗衣粉清洗干净，用毛巾裹好在洗衣机里甩去水分，再一张一张铺在干净的地板上晾干，用熨斗熨平，送到其他法轮功学员那里打上字，然后再到商业区分给做经商的法轮功学员或明白真相的经商者。
 <br/>
 <br/>
 经过一段时间的实践，发现这个办法非常奏效，明显提高了真相币的流通范围和流通速度。一位法轮功学员的丈夫对她开玩笑说：〝这下好了，零售改批发了！〞
 <br/>
 <br/>
 在换零钱的过程中，这位法轮功女学员严格按真、善、忍要求自己，有时人家给多钱了，第二天她就送回去，打交道多了，商家对她非常信任，甚至让她自己从他们的钱盒子里数钱，而他们放心的去忙生意。
 <br/>
 <br/>
 还有一件事让她的记忆最为深刻。一次一个人换给她一张一百元的假钞，她在不知道的情况下拿着去换零钱，那个商家一看是假钞就嚷嚷起来：你怎么用假币骗我们！另一个人喊道：去报警！这位法轮功女学员一点也不慌，她说：〝我不知道是假的，是别人刚给我的，你退给我吧。〞
 <br/>
 <br/>
 后来，无法搞清这假钞是谁给的，她便拿着钱返回到嚷嚷着要报警的商贩那里，当着他们的面把钱撕碎。这几个人看到后竖起了大拇指。
 <br/>
 <br/>
 <h4>
  每天兑换两千张真相币
 </h4>
 <br/>
 另一位老年法轮功员从2002年开始就在家里做真相币，到现在每天都兑换两千张真相币。
 <br/>
 <br/>
 刚刚开始做真相币时，她就从家人、亲属、朋友处开始换，一张、十张，最后成百成百的兑换，用小四号字打在纸币上，然后花出去。
 <br/>
 <br/>
 她家开了一个炸串店，上货时都用真相币。她从在小卖店换纸币，到大商店去换，最后到公交公司换，想换多少就有多少，再后来，商场里的人几乎都上她这成百成百的换真相币。现在，她几乎每天都要做两千多张真相币并及时换出去，使真相币遍地开花。
 <br/>
 没有拿到真相币的店铺不高兴
 <br/>
 <br/>
 还有一位法轮功学员在用真相币购物时向甲店老板讲真相，劝她进出货物时多用真相币会功德无量，生意也会好，并从该店换回无字的纸币。
 <br/>
 <br/>
 之后有一天在经过甲店铺时，隔壁有一家夫妻店（以下简称乙店）店主看到这位法轮功学员就问：隔壁店（甲店）为什么她生意好的都忙不过来呢？我为什么都没有生意呢？
 <br/>
 <br/>
 这位法轮功学员就给他们讲了真相并帮他们做了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三退.html" target="_blank">
  三退
 </a>
 （退出党团队），又告诉他们使用真相币有福报，生意也会好。接着就用五十张写有〝法轮大法好〞的一元票额换给她使用。从此，这位法轮功学员每个星期都大量的用真相币换回无字纸币。
 <br/>
 <br/>
 <h4>
  翻墙破网真相币受欢迎
 </h4>
 <br/>
 有一次，这位法轮功学员到乙店换钱时，乙店老板对她说：有的买物品的人不要真相币，有的找给对方了又退回。这位法轮功学员就告诉他们说，她那里还有印翻墙破网的纸币，可以从国内上网看国内看不到的新闻。
 <br/>
 <br/>
 过几天，法轮功学员又去乙店换无字纸币时，他们告诉她说，现在没有出现买货人退换真相币了，并说他们一天能用100到200张，要她每月提供5000张一元纸钞给他们使用。其它店铺看乙店生意这么好也纷纷向法轮功学员换真相币使用。
 <br/>
 <br/>
 这一来这位法轮功学员手上真相币周转不过来，向她要到真相币的店主很高兴，要不到的店主很生气，她只好对他们说：下回多拿点给你们。
 <br/>
 <br/>
 有一天，这位法轮功学员到另一家（简称丙店）买物品，店老板找零钱时她一看是真相币，她就笑着对她说：〝这钱都是我送给别人的救命钱，请你换没字的给我好吗？〞
 <br/>
 <br/>
 店老板指着一个桶里说：〝这桶里的钱，都是真相币。〞过几天，法轮功学员又到她店购买物品，并请她看一看真相币还多不多？她一看感到好奇，今天怎么都没有了？都用完了。
 <br/>
 <br/>
 从此这位法轮功学员一个月都要换上万元真相币给各店铺使用。
 <br/>
 <br/>
 不仅仅上述这些法轮功学员，而是数千万法轮功学员，十几年如一日地利用真相币，或面对面等方式，讲真相、劝〝三退〞（即退出中共少先队、共青团、党员）。
 <br/>
 <br/>
 无论是在北京、上海，还是偏远的小村庄；无论是各国的景点、出名的商场或者大学校园；不管是网路上、传真上、电话上，还是媒体上，只要是有中国人的地方，〝三退〞的消息都有法轮功学员们传递到那里。
 <br/>
 <br/>
 不少中国人都在说，他们的邮箱里、车筐里、家门口或者马路边上，或者从电子邮件里、电话上，或者翻墙，都能看到真相资料或者接触到法轮功学员。
 <br/>
 <br/>
 截至2018年11月16日，在大纪元网站上声明退出中共党、团、队的〝三退大潮〞，人数已经超过3.19亿人。
 <br/>
