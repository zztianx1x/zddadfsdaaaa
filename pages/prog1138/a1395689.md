### 政变控制习近平传闻成真？房峰辉张阳三罪引关注
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月17日讯】中共前军委委员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张阳.html" target="_blank">
  张阳
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房峰辉.html" target="_blank">
  房峰辉
 </a>
 两名上将，16日被开除了中共党籍，褫夺上将军衔。值得关注的是，当局在通报二人的罪名时，措辞罕见严厉，在两人的罪名中，有三项是一致的，如〝搞两面派做两面人〞，〝不忠诚不老实〞和〝政治上蜕变、经济上贪婪〞等。似乎印证两人曾谋划
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政变.html" target="_blank">
  政变
 </a>
 控制习近平的传闻。
 <br/>
 <br/>
 中共新华社10月16日通报，中共中央军委联合参谋部原参谋长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房峰辉.html" target="_blank">
  房峰辉
 </a>
 、政治工作部原主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张阳.html" target="_blank">
  张阳
 </a>
 双双被开除党籍，褫夺上将军衔。
 <br/>
 <br/>
 其中房峰辉被指控，严重违反中共的政治纪律和政治规矩，涉嫌行贿、受贿、巨额财产来源不明犯罪，情节极为严重，数额特别巨大，影响极其恶劣。
 <br/>
 <br/>
 房峰辉还被指控，不忠诚不老实，搞两面派做两面人，政治上蜕变、经济上贪婪。因此被开除党籍，侦查终结，移送审查起诉。此前中共军委已决定给予房峰辉开除军籍处分，取消了其上将军衔。
 <br/>
 <br/>
 张阳则被指控，严重违反中共的政治纪律和政治规矩、组织纪律、廉洁纪律和国家法律法规规定，涉嫌行贿、受贿、巨额财产来源不明犯罪，情节极为严重，影响极其恶劣。
 <br/>
 <br/>
 另外，张阳也被指控，不忠诚不老实，搞两面派做两面人，政治蜕变、经济贪婪、生活腐化、品行低劣，且在十八大后不收敛不收手、顶风违纪等。张阳还被党媒声讨为，以自杀方式企图逃避惩处。也被开除党籍及追缴涉案财物。而且，此前已被开除张阳军籍，取消其上将军衔。
 <br/>
 <br/>
 值得关注的是，两人都被指控〝搞两面派做两面人〞，〝对党不忠诚不老实〞和〝政治上蜕变、经济上贪婪〞三宗罪，似乎印证了此前的传闻：即两人曾谋划用1976年抓捕〝四人帮〞的手段控制习近平。
 <br/>
 <br/>
 中共官方去年11月28日披露，张阳被〝约谈〞期间在家自缢身亡。当时有美媒称，张阳、房峰辉等人被抓的背后是一场
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政变.html" target="_blank">
  政变
 </a>
 ，他们计划用1976年抓捕〝四人帮〞的手段控制习近平等，结果这个计划被提早察觉。
 <br/>
 <br/>
 报导说，以抓捕〝四人帮〞模式而言，房峰辉、张阳等人只是具体行动者，布局策划者包括海外舆论的准备且另有〝高人〞。而张阳的离奇死亡，更大的可能是被灭口，这显示更高层的主谋仍未坐实。房峰辉应该是突破口。
 <br/>
 <br/>
 多家港媒此前也曾披露，房峰辉、张阳涉嫌谋划针对习近平的军事政变而被抓，但版本不同，一种说法是，两人在习近平军改中被削权，而心怀不满，因此密谋策划军事政变。
 <br/>
 <br/>
 曝料说，两人曾暗中抵制习近平〝肃清
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐才厚.html" target="_blank">
  徐才厚
 </a>
 遗毒〞，多方保护他们的老战友、老同学和老乡，早已引起习当局不满，本打算在军委换届时让他们退休，但两人心有不甘，于是密谋发动军事政变，没想到机密泄露，被习近平下令抓捕。
 <br/>
 <br/>
 张阳被指是徐才厚的嫡系，房峰辉被指是郭伯雄的头马，两人关系密切。
 <br/>
 <br/>
 郭、徐二人则是江泽民的亲信，盘踞军队近20年，架空胡锦涛的军权，卖官鬻爵，批发将军军衔，江系党羽遍及中共军队各个层级。
 <br/>
 <br/>
 郭伯雄于2015年4月落马，2016年7月被判处无期徒刑；徐才厚于2014年3月落马，2015年3月因患癌死亡。
 <br/>
 <br/>
 2017年8月底，张阳、房峰辉突然被免职、调查，11月23日，张阳在家自缢身亡；今年1月9日，当局首次宣布房峰辉已被调查，并移送军事检察机关处理。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
