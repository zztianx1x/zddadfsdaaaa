### 分析：孟宏伟案牵出中南海危局  两股力量正较劲
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】国际刑警组织原主席、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 自法国返回中国后被抓捕。评论人士说，在遭到以美国为首的国际性政治、经济、军事围剿之下，北京仍不顾顔面的抓人，表明中共决策层两股力量正在较劲。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 妻子葛瑞丝‧孟（Grace Meng）10月初向法国警方报案其丈夫失踪，并于7日召开记者会，向世界公布其丈夫回到中国境内与家人失联。立即引发全世界关注，使得中共不顾及国际法和颜面，秘密关押孟宏伟的计划暴露。
 <br/>
 <br/>
 中共官方7日深夜通报：孟宏伟涉嫌违法，目前正接受中共监委监察调查。
 <br/>
 <br/>
 香港苹果日报分析说，在遭到美国为首的世界性政治、经济、军事围剿之下，北京仍不顾一切进行党内大清洗，表明中南海高层当前的矛盾激化已到了不可想像的程度。像孟宏伟这样人，当了一辈子共产党刀把子的酷吏，掌握着重大机密，中共当然要对其格外小心了。
 <br/>
 <br/>
 自由亚洲电台引述一名北京匿名知情人士说，美国副总统彭斯（Michael Pence）4日刚刚在演讲中抨击中共，但北京仍不顾国际社会压力，对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 下手，表明中共决策层两股力量正在较劲。
 <br/>
 <br/>
 中国民主活动人士王军涛分析认为，孟是十九大以后的权利斗争中逐步下来的，共产党内部的两股力量，正在围绕孟宏伟案件较劲。
 <br/>
 <br/>
 王军涛分析说，如果孟宏伟事件是一个政治事故的话，按照共产党的规矩，他的举荐人，这事的负责人会被问责。跟他一派的人也会受牵连，在孟妻高调发声问题上做过安排的人也会被问责。
 <br/>
 <br/>
 共产党对高官的家人都有边控，他们出去时家人留在国内做人质，但是孟妻不在这个安排之列，这就意味着有人可能要对此承担责任。对孟事件给中共造成被动局面和负面影响也要有人负责。
 <br/>
 <br/>
 孟宏伟是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 的亲信，曾在中共安全机关担任过多项职务。值得关注的是，周落马后，孟却能全身而退，甚至于2016年当选国际刑警组织主席。但在十九大，赵克志担任中共公安部部长后，多次要求继续清洗周永康的〝余毒〞。
 <br/>
 <br/>
 孟宏伟此次被拿下后，公安部部长赵克志8日凌晨紧急召开公安部党委会议，就孟被调查事件进行通报，要求坚决清除周永康的〝余毒〞影响，并宣布成立工作组，查处孟的同党。
 <br/>
 <br/>
 与公安部形成鲜明对比的是，至今中共政法委仍未就此事发声。
 <br/>
 <br/>
 据报，现任中共政法委书记郭声琨，与孟宏伟关系密切。两人即是湖南中南工业大学校友，同时又都曾担任公安部正副部长。
 <br/>
 <br/>
 有港媒分析认为，从中共公安部连夜召开会议，通报孟宏伟的案情，以及政法委书记郭声琨失声来看，此案是由中纪委直接操刀，公安部和政法委事前很可能都蒙在鼓里。
 <br/>
 <br/>
 目前中共政法委〝一把手〞郭声琨，以及委员周强、傅政华等人，都属周永康的亲信。孟宏伟也曾是周的亲信，此次孟被查，赵克志宣布查处孟的同党。分析认为，接下来，公安系统甚至整个政法系统或将大震荡，背后很可能涉及中南海两个派系之间的生死博弈。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：赵云）
