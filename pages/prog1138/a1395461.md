### 王立军再传猝死 台媒曝家属报丧信息
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】今年6月，海外网络一度流传前重庆公安局长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 狱中猝死传言。日前，王立军心脏病突发去世的消息再次流传，台媒独家爆料其子报丧信息。不过消息存在多个疑点。正值孟宏伟落马、北京整肃政法系，王再传猝死被指时间点巧合。
 <br/>
 <br/>
 10月15日，台湾互联警政新闻网报导称〝独家掌握〞一名自称〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 儿子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王辰.html" target="_blank">
  王辰
 </a>
 〞的网民对海外亲友发出的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_讣文.html" target="_blank">
  讣文
 </a>
 。报导并指〝惊传〞王立军已于14日晚间因心脏病发去世。
 <br/>
 <br/>
 王立军2012年被判15年徒刑，目前服刑期未满。
 <br/>
 <br/>
 报导刊登一张疑似手机屏幕的截图，画面显示出一则〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吊唁通知.html" target="_blank">
  吊唁通知
 </a>
 〞。通知称：〝我是王立军儿子王辰。我爸因心脏病突发去世，望各位叔叔伯伯做最后告别，定于农历初九家中举行告别仪式。跪谢，王辰。〞
 <br/>
 <br/>
 通知后，有人发讯息质疑消息真假，并询问对方是否真是〝王立军的公子〞。
 <br/>
 <br/>
 报导指，时值前国际刑警组织主席孟宏伟妻子在海外曝光其丈夫被失踪一事，此时传出王立军猝死，两者间存在巧合，难免引发外界联想王立军真正死因。
 <br/>
 <br/>
 孟宏伟妻子向海外爆料并寻求法国警方保护，外界将其与当年王立军闯美国使馆寻求庇护相比，称其为〝女版王立军〞。
 <br/>
 <br/>
 不过，上述〝王立军猝死〞消息尚未得到其他渠道的进一步证实。
 <br/>
 <br/>
 报导没有指明上述〝讣文〞是在何处发布，消息是何人提供。而且，报导称这是王立军之子给海外亲友发送的讣文，但对方似乎质疑其身份。
 <br/>
 <br/>
 本台记者翻查此前海内外报导，王立军有一个女儿叫王迪，至于该消息中的〝儿子王辰〞是头一次出现在媒体报导中。
 <br/>
 <br/>
 另外，这份〝吊唁通知〞也出现在海外一个注册为〝@AlexLIU1991〞的推特账号。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-16/p9081351a972008244.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-16/p9081351a972008244-ss.jpg"/>
   <br/>
   <font size="-1">
    网传所谓〝王立军之子〞发布的吊唁通知。（推特图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 早在今年6月底，海外推特就曾热传王立军狱中猝死的消息。因此，这两次〝王立军死亡〞的消息，至少有一次不实。
 <br/>
 <br/>
 虽然这次王立军死亡消息尚未得到进一步证实，但其传出的时机确实与孟宏伟事件存在〝巧合〞。
 <br/>
 <br/>
 孟宏伟被指〝周永康余毒〞。当局突然不顾国际影响，拿下孟宏伟，外界普遍认为北京出现〝紧急情况〞。有分析指，或与习近平整肃政法系有关。
 <br/>
 <br/>
 而王立军是前重庆市委书记薄熙来的打手，薄熙来和周永康当年是阴谋政变、推翻习近平的同党。在这一点上，王立军和孟宏伟存在一定关联。
 <br/>
 <br/>
 此前，薄熙来的另一名亲信徐明曾被传狱中猝死。后来该消息得到多方证实，指徐明也是死于突发心肌梗死。徐明是薄熙来的大管家，帮薄熙来家族掌管财产，被指掌握薄熙来和谷开来夫妇在大连活摘器官、开办尸体工厂的证据。
 <br/>
 <br/>
 而王立军也是活摘器官的凶手，早在辽宁锦州时就参与活摘法轮功学员器官，并和当地医院合作，发明了专为活摘设计的所谓〝脑干撞击机〞。韩媒揭露，该机器唯一的功能是制造〝脑死亡〞，为活摘提供〝合法供体〞。
 <br/>
 <br/>
 而孟宏伟曾是周永康在公安部的直接下属，也被指与政法系统活摘器官摆脱不了干系。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
