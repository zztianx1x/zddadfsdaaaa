### 北京准备作重大妥协? 刘鹤贸战讲话释信号
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】中国副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 接受党媒采访时，提及中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 下滑的原因，有中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的因素，他还透露，中美正在接触。有港媒分析，这表明中共将在贸易战的后续发展中，准备作出重大妥协。
 <br/>
 <br/>
 身兼中共国务院金融稳定发展委员会主任的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 ，10月19日接受中共三大官媒采访时，谈及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国经济.html" target="_blank">
  中国经济
 </a>
 、中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_股市.html" target="_blank">
  股市
 </a>
 、中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 ，以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国进民退.html" target="_blank">
  国进民退
 </a>
 等议题。
 <br/>
 <br/>
 刘鹤表示，最近一段时间以来，中国股市出现了明显的波动和下滑，这是诸多因素造成的。首先是外部因素。
 <br/>
 <br/>
 〝中美贸易摩擦对市场也造成了影响，但坦率地说，心理影响大于实际影响，目前中美正在接触。〞刘鹤说。但他并没有说明中美接触的具体情况。
 <br/>
 <br/>
 在港媒《明报》看来，刘鹤的讲话，意味着中共将在贸易战的后续发展中，准备作出重大妥协。
 <br/>
 <br/>
 报导说，自中美贸易战以来，中国的经济学人、商人、股市投资者均属于主和派，而军方、外交，甚至部分网友，以主战派居多。
 <br/>
 <br/>
 报导认为，北京直到现在仍用〝中美贸易摩擦〞一词，说明北京不愿承认中美处于〝战争状态〞，〝这与当年国民政府蒋介石面对日本侵略所言‘和平未到完全绝望时期，决不放弃和平’有异曲同工之处。〞
 <br/>
 <br/>
 对于刘鹤有关〝心理影响大于实际影响〞以及〝中美正在接触〞说法，报导认为，这是在告诉中国民众和投资者，中美有接触有沟通，打不起来。
 <br/>
 <br/>
 英国《金融时报》报导说，中国今年第三季度经济增速，低于许多经济学家和分析师的预测，创近10年来最慢增速，世界第二大经济体面临着经济挑战。
 <br/>
 <br/>
 根据中共官方公布的数据显示，中国今年第三季度国内生产总值（GDP）增长仅6.5%，这是自2008～09年全球金融危机最严重时期以来最低的季度增速。
 <br/>
 <br/>
 今年以来，中国股市也连续下挫，追踪沪深两市大型上市企业的沪深300指数（CSI 300）的跌幅约为25.5%。
 <br/>
 <br/>
 GDP增速也不断下滑，从第一季度的6.8%，下滑到第二季度的6.7%，再到第三季度的6.5%。
 <br/>
 <br/>
 根据美国财政部10月17日的报告，自6月中旬以来，人民币兑美元汇率下跌超过7%。
 <br/>
 <br/>
 英媒认为，刘鹤接受三大党媒高调发表讲话，就是为了提振人们对中国经济、股市和改革计划的信心。
 <br/>
 <br/>
 在刘鹤接受专访之前，白宫首席经济顾问库德洛（Larry Kudlow）表示，川普（特朗普）总统也许会在G20国集团峰会上会晤习近平，届时将会看看两国领袖之间的私人外交是否可以产生一些有益的后果。
 <br/>
 <br/>
 库德洛还表示：他非常担心中共正走在一条非常错误的路上，回到一个封闭的、不流通的经济环境中。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：戴明）
