### 原薄熙来秘书被没收全部财产执行完毕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月10日讯】前中共重庆市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘书.html" target="_blank">
  秘书
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴文康.html" target="_blank">
  吴文康
 </a>
 在判刑3年多后，没收个人财产执行完毕，已于今年7月结案。吴文康遭收缴、没收财产总计超过2000万元人民币（下同）。
 <br/>
 <br/>
 近期，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大陆.html" target="_blank">
  大陆
 </a>
 裁判文书网发布的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴文康.html" target="_blank">
  吴文康
 </a>
 没收个人全部财产结案通知书》公布，吴文康案审理期间，扣押在案的吴文康及其亲友上缴的财产共计2020.664436万元，收缴其中2020.3348万元，其余3296.36元予以没收。
 <br/>
 <br/>
 2015年3月，吴文康因受贿罪被中共吉林省高院判处无期徒刑，剥夺政治权利终身，并处没收个人全部财产。
 <br/>
 <br/>
 吴文康曾任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秘书.html" target="_blank">
  秘书
 </a>
 。历任中共大连市政府秘书、大连市口岸委员会主任助理、辽宁省政府办公厅副主任，中共重庆市委员会副秘书长、办公厅主任等职。
 <br/>
 <br/>
 在王立军出逃事件后，吴文康曾检举、揭发薄熙来滥用职权的犯罪事实。
 <br/>
 <br/>
 时评家姜维平题为《吴文康是薄熙来的软肋》一文披露，吴文康是薄熙来的心腹，薄熙来任中共大连市长时，吴文康是他的秘书，后薄熙来调任辽宁省长，吴又到沈阳出任省政府办公厅副主任。2007年，薄熙来出掌中共重庆市委书记，吴文康也到重庆任职。
 <br/>
 <br/>
 姜维平透露，吴文康投薄熙来所好，为其拉皮条，公开选美。为讨薄熙来欢心，他在大连富丽酒店留出豪华标准间，提供数十名美女供薄熙来淫乐，其中有包括海内外影星、歌星和服装模特。每每出事，吴文康就花钱摆平。为其卖命，不遗余力。
 <br/>
 <br/>
 姜维平说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贪腐.html" target="_blank">
  贪腐
 </a>
 的权欲把薄、吴二人捆绑在一起。吴文康是薄熙来的死党，他几乎一步不落地跟随着薄熙来，长达十几年之久，只是在90年代后期，他在大连口岸委担任过很短一段时间的副主任，而那也只是为了掩人耳目，过渡而已。
 <br/>
 <br/>
 吴文康紧跟薄熙来，狐假虎威，其权力在大连一度炙手可热，被称为〝大吴〞。但薄熙来的倒台结束了他狗仗人势，横行无忌的历史。
 <br/>
 <br/>
 （记者李蒨蒨报导/责任编辑：戴明）
</div>
