### 传孟宏伟毒杀十世班禅  太子党称邓李薄三人策划
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月14日讯】国际刑警组织前主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被北京秘密抓捕事件，引发了国际舆论的持续关注。流亡海外的中国自由派法学家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_袁红冰.html" target="_blank">
  袁红冰
 </a>
 近日在社交网站推特上披露，自己早年间经过多年的查访发现，十世班禅系为中共毒杀而死，而孟宏伟正是当年实施毒杀的操刀手之一。
 <br/>
 <br/>
 近半个月来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被北京抓捕事件是国际舆论持续关注的焦点话题。北京时间10月8日，中国自由派法学家
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_袁红冰.html" target="_blank">
  袁红冰
 </a>
 曾在其推特（Twitter）帐号上发出一条推文称，〝在孟宏伟官运亨通过程中，我曾通过《杀佛》和《被囚禁的台湾》两本书披露孟宏伟参与谋杀十世班禅大师和参与制造台湾三一九枪击案等罪过。〞
 <br/>
 <br/>
 藏传佛教的领袖、第十世班禅额尔德尼．确吉坚赞，1989年1月28日在西藏日喀则市的新宫猝然离世，终年51岁。当时外界就有一种说法，认为十世班禅是因为试图摆脱中共的控制而遭中共毒杀，但中共官方的说法是〝心脏病〞突发而身亡。
 <br/>
 <br/>
 据查询，中共官方公开的孟宏伟的个人履历中，并没有他1989年担任何职的记录。受中共官方控制的百度上在介绍孟宏伟的履历时，只比较明确地罗列了他2004年担任中共公安部副部长以后的职务变迁，而对其此前的履历仅笼统简称〝历任公安部部长助理、交通管理局局长〞。
 <br/>
 <br/>
 2013年流亡海外的中国作家与法学家袁红冰，与藏汉双语诗人、藏学研究者安乐业共同完成的著作《杀佛——十世班禅大师蒙难真相》一书中称，经过他们多年的查访，包括采访到中共太子党的相关成员，发现当年
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毒杀十世班禅.html" target="_blank">
  毒杀十世班禅
 </a>
 的决策出自当时掌控着中国最高权力的中共元老邓小平以及另两名中共元老李先念和薄一波。
 <br/>
 <br/>
 按照袁红冰等人当时的说法，孟宏伟就是当年具体执行毒杀班禅行动的操刀手之一。而当年孟宏伟从西藏回北京后即受到重用，官至公安部助理。其后又一度出任交通局局长，2004年担任中共公安部副部长，参与过迫害法轮功信仰者。
 <br/>
 <br/>
 有消息指袁红冰与孟宏伟同于1979年入北大法律系学习，也就是说，他们二人应该曾经是〝同学〞。
 <br/>
 <br/>
 不过，以上说法仅仅是袁红冰等人的私人观点，目前尚无其他来源的佐证。
 <br/>
 <br/>
 据公开的资讯，1989年1月28日凌晨，十世班禅在西藏日喀则在主持历代班禅灵塔落成仪式中突然晕倒，随即被送医抢救。中共官方的说法是，十世班禅的心电图显示为急性下壁和广泛前壁心肌梗塞。次日（29日）晚20时16分，十世班禅逝世。
 <br/>
 <br/>
 十世班禅的突然去世在当时就引发了许多传言和猜测。当时西藏司政洛桑桑盖对外提及十世班禅过世的情况时称：〝51岁的十世班禅仁波切在西藏日喀则扎西伦布寺的德庆格桑颇彰新宫，不明不白地突然圆寂。〞
 <br/>
 <br/>
 第八世阿嘉呼图克图阿嘉．洛桑图旦．久美嘉措（阿嘉仁波切）则在其自传《顺水逆风》中表示，当时人们对班禅大师于1989年〝突然离去〞早就疑惑重重， 而包括第十四世达赖喇嘛在内的许多人认为，十世班禅是被其医护人员毒死的。
 <br/>
 <br/>
 袁红冰这次在推特上再次提及孟宏伟毒杀十世班禅的话题后，署名〝古玉文〞的作者日前发表评论文章称，孟宏伟的妻子应该明白一个道理，作为中共体制内的官员及家属，如果只是效仿王立军、令完成等着眼于与中共权斗，企图利用西方社会的自由与舆论来自保，也许能暂缓中共的迫害，但却难逃中共无处不在的威胁与消音。
 <br/>
 <br/>
 文章指出，中共体制内真正实行自救和救人的行动楷模，是原中共驻澳洲大使馆一秘陈用林先生、天津国保局警察郝凤军先生和沈阳司法局局长韩广生先生等英雄。
 <br/>
 <br/>
 因不齿于与中共为伍，陈用林先生2005年带着夫人和年幼的孩子逃离中共使馆，在澳洲公开揭露中共迫害法轮功及异议人士，并向全世界揭露中共在澳洲有一千多个间谍。
 <br/>
 <br/>
 古玉文在文章中表示，如果孟妻真像自己表述的那样〝追求真相、正义和社会责任〞，就应效仿陈用林先生，真正的站出来揭露中共，以孟宏伟家属的身份现身说法，公开孟本人所犯的罪行，并替其表示忏悔，才能真正的摆脱中共的威胁而实现自救救人。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
