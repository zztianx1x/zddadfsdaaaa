### 习近平〝进博会〞或发重要信息 川普拒绝捧场
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国首届.html" target="_blank">
  中国首届
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际进口博览会.html" target="_blank">
  国际进口博览会
 </a>
 （简称：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 ）将在上海登场，在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 激化的背景下，外界分析
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将参加
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开幕式.html" target="_blank">
  开幕式
 </a>
 ，可能会发表扩大开放等重要信息。然而，欧洲、日本等发达国家的领导人几乎无人捧场，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 也拒绝派员出席。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 导致中国经济受到重创的大背景下，11月5日至10日，中共将在上海举办首届中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际进口博览会.html" target="_blank">
  国际进口博览会
 </a>
 。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_开幕式.html" target="_blank">
  开幕式
 </a>
 。
 <br/>
 <br/>
 韩联社近日引述消息指，习近平可能在开幕式演讲中，发表具体的扩大对外开放的信息。
 <br/>
 <br/>
 有观点认为，从中共官宣来看，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_进博会.html" target="_blank">
  进博会
 </a>
 〞可能被习近平视为今年最重要的主场外交活动之一，在中美贸易战背景下，习近平不仅希望藉此纾解外部压力，也想展示当局在外交方面并未被孤立，藉此说服中共高层，缓解来自内部的对抗。
 <br/>
 <br/>
 中共商务部于10月底举行的中外媒体吹风会，商务部国际贸易谈判代表兼副部长傅自应称，习近平多次强调，〝进博会〞不是一般性的会展，要求办成国际一流的博览会。
 <br/>
 <br/>
 尽管中共官方高调宣传此次博览会，但海外舆论并不看好。从中共外交部发言人陆慷发布的应邀出席的外国领导人名单看，参加者几乎清一色来自发展中国家。
 <br/>
 <br/>
 而英国、法国、德国、意大利、日本等绝大多数发达国家领导人都将缺席，华盛顿也拒绝捧场。
 <br/>
 <br/>
 美国驻华大使馆发言人对路透社说，美国不会派出高层官员参加此次〝进博会〞。英媒引述中共官方的说法称，对于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）政府拒绝派员与会的决定，中方表示〝难以理解〞。
 <br/>
 <br/>
 10月31日，韩国《朝鲜日报》称，〝进博会〞还没有开，就注定会遭到〝冷遇〞。
 <br/>
 <br/>
 报导称，即使中共方面信心满满打算〝向世界展示全面开放〞的意志，但是中美贸易战下，美国等主要西方国家的不参加，会让这次会议的效果大打折扣。
 <br/>
 <br/>
 时政评论员吴铭表示，从西方绝大多数国家政要缺席来看，这〝国际一流的博览会〞，很可能也象〝厉害了，我的国〞一样，是中共关起门来的〝厉害〞。
 <br/>
 <br/>
 自从中美贸易战开打后，中共在国际上的影响力不断下降。4月8日至11日，中共在海南省举办了2018年博鳌亚洲论坛年会。与往年相比，不仅绝大多数西方国家领导人缺席，甚至连中共拚命撒钱的非洲，竟然也没有一位领导人前来捧场。
 <br/>
 <br/>
 中美贸易战爆发以来，美国与其它国家的贸易摩擦正在逐步化解，美国总统川普正在通过一对一的谈判，构筑新的世界经济贸易体系。
 <br/>
 <br/>
 美国与墨西哥、加拿大、日本、欧盟、韩国、印度等国家之间，有的已经达成贸易协定，有的正在加紧谈判。
 <br/>
 <br/>
 川普总统表示，美国与这些国家之间有望在2019年全部达成〝自由公正〞的协议。川普对中共的指责得到越来越多的西方国家的认同和支持，对于中共盗窃知识产权问题，甚至连中共竭力拉拢的俄罗斯也感同身受。
 <br/>
 <br/>
 目前，中美贸易战火不断升高，已经延伸到其它领域。国际社会在美中之间存在着选边站的问题。随着美国对中共全方位的围堵，可能会有更多的国家选择站在美国一边。
 <br/>
 <br/>
 美国白宫国家经济委员会主任库德洛此前表示，美国将很快组建一个联盟，以应对中共〝不公平贸易行为〞。有分析称，不难看出，川普的华贸易战的核心目的，就是联合全球孤立中共、围剿中共。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
