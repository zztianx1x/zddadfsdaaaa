### 习近平来访前夕 上海会场附近惊传大爆炸
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海进口博览会.html" target="_blank">
  上海进口博览会
 </a>
 将于北京时间11月5日开幕，北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将赴会演讲。正在会场所在地区紧张布置安保措施之际，临近会场的高架桥上发生化学品车辆大爆炸事件。官方迅速消声，反而引发各种猜测联想。
 <br/>
 <br/>
 网传视频指，11月1日晚间，一辆满载的货车正在上海一处高架桥路段起火，并不断发生爆炸，猛烈火势在几公里外均能看到。
 <br/>
 <br/>
 上海松江区政府官方微博在当晚7点20分左右通告称，晚6点25分左右，在位于九亭镇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_沪松公路.html" target="_blank">
  沪松公路
 </a>
 高架桥上（9号线九亭站3号出入口上方），一辆载有〝泡沫类物品〞的8吨货车〝起火〞，火势已被控制，〝无人员伤亡〞。
 <br/>
 <br/>
 通告回避了现场视频中频频传出的爆炸声，也对事故原因只字不提，甚至都没有出现〝事故原因正在调查中〞的字眼。
 <br/>
 <br/>
 网上还传出一段微博视频显示，松江高架桥上的爆炸现场，满地都是油气罐子。
 <br/>
 <br/>
 <br/>
 <br/>
 不过，有关此次爆炸的消息很快在网上被全部屏蔽，官媒也集体噤声。网友纷纷猜测，爆炸是否涉及某种恐怖活动。
 <br/>
 <br/>
 事发地点松江区九亭镇，是举办中共首届进口博览会的重镇。博览会会场就位于该镇辖区的上海虹桥空港。爆炸路段离会场只有数公里之遥，相信也是部分厂商或官员赴会途径的路线。
 <br/>
 <br/>
 10月30日，九亭镇刚刚召开虹桥空港开幕式及周边情况协调工作会议。
 <br/>
 <br/>
 此前，中共官方已经宣布，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 将出席11月5日的进口博览会开幕式，并发表演讲。
 <br/>
 <br/>
 数日前的10月23日，习近平参加了在广东珠海市举办的珠港澳大桥开通仪式。此前外界预测，在陷入经济危局之际，习近平或将在开通仪式上发表有关〝改革开放〞的〝重要讲话〞。不过，当天习近平只简单宣布大桥开通之后，就匆匆离场。
 <br/>
 <br/>
 就在习近平抵达珠海的3天前，中共官方突然宣布澳门中联办主任郑晓松19日晚因〝抑郁症〞堕楼身亡。外界多有猜测称，习近平此次考察广东行程低调隐秘，可能和郑晓松离奇猝死有关。
 <br/>
 <br/>
 内忧外患之下，中共内部也陷入残酷的权力和利益之争。官员自杀和离奇猝死频率明显增加。有哈佛大学学者引述中共高官透露北京内部气氛：不是你死，就是我亡。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：云涛）
