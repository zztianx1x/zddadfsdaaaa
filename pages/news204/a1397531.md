### 贸易战没影响外贸？航运数据戳破中共谎言
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月01日讯】中共一直宣称贸易战〝短期内对中方影响不大〞，官方数据甚至显示近期中国进出口都大幅增加。但海外航运公司公布的数据显示，受关税战影响的中国商品出口暴跌。
 <br/>
 <br/>
 中共官方经济数据，目前仍然显示〝形势一片大好〞，中美贸易战7月爆发以来，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国外贸.html" target="_blank">
  中国外贸
 </a>
 似乎〝丝毫未损〞。
 <br/>
 <br/>
 其中，中共海关总署公布的9月外贸数据显示，当月出口按年增速反弹至14.5%，虽然进口增速有所放缓，仍然维持在14.3%的水平。
 <br/>
 <br/>
 不过，专业人士更为关注的航运业内数据显示的结果，与中共官方数字截然不同。全球大约90%的货物都是通过海运输送，因此中国的货柜输出量，最能反映外贸实际状况。
 <br/>
 <br/>
 美国媒体CNBC日前报导，全球最大货柜船租赁公司西斯班（Seaspan，SSW）联合创办人王友贵（Gerry Wang）指出，早在六个月前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_航运数据.html" target="_blank">
  航运数据
 </a>
 就显示中国经济放缓的信息。
 <br/>
 <br/>
 根据西斯班全球海洋贸易数据库（Seabury Global Ocean Trade Database）的资讯，自川普政府今年3月23日开始对进口钢铝加征关税，到今年9月为止，中国钢铝材出口美国的数量比去年同期下降了53%。
 <br/>
 <br/>
 同样情况，美国7月6日开始对340亿美元中国商品加征关税后，直到今年9月，受影响的中国商品对美出口按年下降了21%。
 <br/>
 <br/>
 西斯班公司海运主管罗顿（Michel Looten）表示，海运货柜数据显示，其中一部分中国商品可能已转运到其他国家。
 <br/>
 <br/>
 据西斯班公司估计，11月即将发布的航运数据，将能反映川普政府8月23日开始对160亿美元中国商品加税的影响；12月的数据则将显示9月24日对2,000亿美元中国商品加税的冲击。其中，12月的数据是观测中国明年经济走向的重要窗口。
 <br/>
 <br/>
 此外，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关税战.html" target="_blank">
  中美关税战
 </a>
 影响的不只是中国短期的进出口数据，还将严重打击制造业和外资投入。
 <br/>
 <br/>
 根据华南美国商会（The American Chamber of Commerce in South China）10月29日公布的调查报告，在中国南部设厂的美国企业中，有超过7成正在考虑推迟或取消投资，并将部份或全部生产线迁出中国。报告亦指出，有一半的中国企业也持相同的想法。
 <br/>
 <br/>
 按照既定计划，两个月后，美国对2,000亿美元中国商品加征的税率将从10%增加到25%。另外，川普政府发出警告，如果北京拒绝回应美国对公平贸易的要求，美方12月将开始推动对剩余的2千多亿中国商品全部加征关税。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
