### 习近平:你们经济数据有多少水分？辽宁:最少1千亿
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】中共官场经济数据作假，几乎已是国际共识。东北三省更是其中典型代表。日前，北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 考察东北时，当面询问
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁.html" target="_blank">
  辽宁
 </a>
 经济数据有多少水分，辽宁高层回答至少1千多亿。
 <br/>
 <br/>
 据新华社报导，9月28日下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在沈阳召开〝推进东北振兴座谈会〞。
 <br/>
 <br/>
 在座谈会上，当
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁.html" target="_blank">
  辽宁
 </a>
 省委负责人谈到〝主动做实经济数据〞时，习近平插话：〝你们在统计数据上压了多少水分？〞
 <br/>
 <br/>
 这位负责人回答：〝仅财政收入就有1000多亿（人民币）。〞
 <br/>
 <br/>
 报导还说，座谈会上，辽宁、吉林、黑龙江、内蒙古四省区负责人都谈到所谓〝反对弄虚作假等不正之风〞。
 <br/>
 <br/>
 上述报导是将该细节当作〝纠正虚假作风〞的正面例子来报导的。但报导没有提到，在经过辽宁当局的〝挤压〞之后，呈交给中共中央的数据是否还有水分。
 <br/>
 <br/>
 中共上至国家统计局，下到基层村镇，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济数据造假.html" target="_blank">
  经济数据造假
 </a>
 已是常态。中共官场流传一句〝著名〞的顺口溜：〝政绩不够，数字来凑，官出数字，数字出官〞。
 <br/>
 <br/>
 今年6月，中共国家统计局也曾公布2017年度核查报告，自曝家丑。报告指，去年全国共有1195家企业、2775个投资单位编造、虚报统计数据。
 <br/>
 <br/>
 其中，报告提到，辽宁省、吉林省、黑龙江省都承认财政数据大幅造假。如果按照各地汇报的产业成长性计算，东北一些县域经济规模甚至超过香港。
 <br/>
 <br/>
 东北三省此前长期被江派人马掌控，吉林帮、辽宁帮近年来成为习近平当局反腐打虎的重点目标。而黑龙江省，在上一任省委书记王宪魁时期，还是和北京公开对抗的〝独立王国〞。
 <br/>
 <br/>
 对这三省官员来说，似乎比其它省份更有欺骗北京现当局的动机。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
