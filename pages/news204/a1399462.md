### 中方迈出妥协第一步 按美要求提交谈判书面文件
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】本月底的中美首脑会晤在即，占据主动的美方态度坚决，中方坚持无果后，终于首先提交书面文件回应美方贸易要求。不过美媒指，文件内容只是老调重弹，回避美方的核心关切。外界分析，双方谈判的路依然漫长。
 <br/>
 <br/>
 路透社11月15日引述三名知情人士的消息指，中方已经就美国早前提出的贸易改革要求，向美方提交
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_书面回应.html" target="_blank">
  书面回应
 </a>
 。这有可能令双方在中断三个月后重启贸易谈判。
 <br/>
 <br/>
 知情人士并没有透露中方回应的具体细节，但表示中方目前提交的文件不过是重申此前做出的承诺，与美方要求依旧相距甚远。
 <br/>
 <br/>
 其中，一名华盛顿消息人士说，离达成协议还很遥远，双方〝完全不在同一个世界〞（not in the same universe）。
 <br/>
 <br/>
 彭博社同日消息亦指，虽然中方书面回应被视为对美方作出让步，包括承诺放宽某些行业的外资股权上限，但内容只是老调重弹，并未回应美方关键要求，包括改变产业政策，放弃〝中国制造2025〞计划等。
 <br/>
 <br/>
 日前，美方副总统彭斯警告，中方如果不能在保护知识产权、强迫技术转移、限制外企进入中国市场、尊重国际规则及在国际水域航行自由等问题上做出显著的、实质的让步，美方准备在经济、外交和政治上对北京施加更大压力，包括将加征关税的中国商品数额翻倍。
 <br/>
 <br/>
 彭斯称，本月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ，是中方避免和美方陷入新冷战的〝最佳机会〞（如果不是最后机会的话）。
 <br/>
 <br/>
 据《华尔街日报》消息，中方首席谈判代表
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 11月9日曾和美国财政部长梅努钦通话。美方依旧坚持中方必须首先提交具体方案回应美方要求，而中方则主张先谈判后提交文件。通话无果而终。
 <br/>
 <br/>
 报导指，中方之所以不同意先提交方案，是因为担心这样会失去筹码，在谈判中先输一筹。
 <br/>
 <br/>
 港媒13日报导，刘鹤将亲自赴美谈判，为川习会铺路。随后美国官方也证实了该消息。不过刘鹤具体行程尚未确定。
 <br/>
 <br/>
 中方让步向美方提交书面回应，或在为刘鹤访美做试探性准备。有猜测认为，为保留谈判筹码，中方提交的文件可能并未透露其所有底牌，未来或将有更多贸易让步。
 <br/>
 <br/>
 不过，目前中方尚未做出实质性让步的任何迹象，做势重启对话被指只是拖延战术。
 <br/>
 <br/>
 而美方依然指派梅努钦和中方保持接触，也令外界对这一对话过程产生怀疑。梅努钦一直主张和中方达成和解，5月时和刘鹤达成的协议框架在共和党内备受质疑，随后被川普政府推翻。分析人士认为，梅努钦现在根本已无力和中方达成协议。
 <br/>
 <br/>
 美媒曾引述美国官员消息指，如果中方回应不能满足美方要求，川习会将剔除贸易议题。
 <br/>
 <br/>
 也有评论指，此次川习会的〝最好结果〞是中方承诺做出实质性让步，双方达成一个临时性的〝停火〞协议，然后就中方承诺展开进一步的谈判。不过可以预见，果真如此，后续谈判将会非常艰难。
 <br/>
 <br/>
 白宫强硬派、贸易高级顾问纳瓦罗强烈反对和中方和解。他表示，中共一贯言而无信，所以即使美中达成协议，也无法相信中共能够真正执行。
 <br/>
 <br/>
 今年6月，川普一度和朝鲜党魁金正恩共同发表〝无核化〞声明。但是朝鲜之后依然敷衍推脱无核化进程，并不断进行挑衅。川普政府不但从未放松制裁，目前还再次搁置美朝谈判，并重启了美韩联合军演。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
