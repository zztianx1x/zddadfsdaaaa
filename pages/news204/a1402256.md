### 华为高层被捕引发中共恐慌 大批五毛围攻加国使馆微博
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】中国华为集团副董兼财务长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大被捕引发中共恐慌。在中共官方连连发出声明要求加、美放人的同时，大批中共培养的五毛水军围攻加拿大驻中国大使馆的官方微博，将此事渲染为〝民族事件〞，并扬言要组团去砸烂大使馆。台媒评论称，中共麾下网军〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集体崩溃.html" target="_blank">
  集体崩溃
 </a>
 〞。
 <br/>
 <br/>
 华为创办人任正非长女
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟晚舟.html" target="_blank">
  孟晚舟
 </a>
 在加拿大被执法机构逮捕，并将被引渡到美国接受审判的消息传出后，中共官方的反应异乎寻常地强烈。
 <br/>
 <br/>
 中共驻渥太华大使馆首先对加拿大政府提出了〝强烈抗议〞，不由分说并要求加国政府立即放人。
 <br/>
 <br/>
 随后，中共外交部就孟晚舟被捕事件分别向加、美两国政府发表所谓〝严正声明〞，要求加、美两国政府立即对拘押孟晚舟的理由作出澄清。然后不等美方对孟晚舟所涉案情做出公开回应，中共官方就抢先要求〝立即释放被拘押的人员〞。
 <br/>
 <br/>
 与此同时，大量中国网军涌入〝加拿大大使馆官方微博〞留言洩愤，将此事定调为，还〝下最后通牒〞扬言组团去砸大使馆。
 <br/>
 <br/>
 据台湾《自由时报》报导，截至12月7日零时，有关孟晚舟在加拿大被逮捕的一篇贴文下，已涌入9553则留言，其中大批明显为中共官方培养的五毛网军叫嚣着〝赶紧释放孟晚舟〞，否则将发起
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抵制.html" target="_blank">
  抵制
 </a>
 运动。
 <br/>
 <br/>
 有人留言威胁要拒买所有来自加拿大的商品；有人扬言要〝坚决抵制去加拿大旅游〞；甚至还有人留言邀约其他人〝有没有人一起去北京砸了这个大使馆？〞
 <br/>

   </font>

 <br/>
 台媒的报导形容中共这种五毛气势汹汹的表演是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_集体崩溃.html" target="_blank">
  集体崩溃
 </a>
 〞。
 <br/>
 <br/>
 亦有网友感叹：〝中共这次的反应简直就像被捅翻了‘马蜂窝’，不顾一切地摆出了‘不管孟究竟触犯了美国的哪条法律，先放人再说’的架势，连基本的道理都不讲啦。〞
 <br/>
 <br/>
 中共的海外党媒则发文称，孟晚舟是华为创始人任正非的女儿，而任正非是中共〝在战略技术方面实现自给自足的全国领军人物〞，美国逮捕〝像这样的中国重要企业高管〞的做法实属罕见。
 <br/>
 <br/>
 路透社的报导称，任正非虽然只持有华为1.4%的股权，但该公司内部人士指出，任正非仍然牢牢地控制着这个公司，参与公司的各种重大决策。
 <br/>
 <br/>
 据华为官网上有关孟晚舟的简介，孟晚舟毕业于华中理工大学，有硕士学位，1993年加入华为。从2005年起，孟晚舟主导在全球建立了五个共享中心；从 2007年起，她负责实施了与IBM合作的、长达八年的华为IFS（集成财经服务）变革。
 <br/>
 <br/>
 舆论界普遍认为，孟晚舟虽然进入华为后一直行事低调，但她在华为集团所扮演的角色其实不可等閒视之，〝俨然有任正非接班人的态势〞。所以孟女被抓捕的消息一出，就引发了中共官方的恐慌。
 <br/>
 <br/>
 孟晚舟被捕后申请了媒体禁制令，加拿大司法部因此对外表示不能披露更多有关该案的内情；美国司法部及提起指控的纽约东区联邦检察官办公室的新闻代表也拒绝就此事件发表置评。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：明轩）
