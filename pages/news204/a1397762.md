### 重庆公交坠江原因曝光 乘客与司机激烈冲突酿祸（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】11月2日，中共官方公布重庆公交车坠江事故原因。据公交车内黑匣子监控视频显示，事发时，车内一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_乘客与司机.html" target="_blank">
  乘客与司机
 </a>
 发生激烈争执、互殴，导致公交车失控坠江，造成重大人员伤亡。
 <br/>
 <br/>
 据官方发布的车内视频显示，10月28日上午10点多钟，出事公交车内有一名中等身材、着浅蓝色牛仔衣的女乘客刘某，因错过下车地点与驾驶员冉某发生激烈争吵，刘某用手机攻击冉某头部，冉某则将右手放开方向盘还击刘某，后又用右手格挡刘某的攻击，并与刘某抓扯。
 <br/>
 <br/>
 随后冉某收回右手并往左侧急打方向盘，导致车辆失控向左偏离越过中心实线，与对向正常行驶的红色小轿车相撞后，冲上路沿、撞断护栏坠入江中，造成重大人员伤亡事故。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 据进一步调查核实，刘某48岁，是万州区人。当地警方表示，乘客刘某和驾驶员冉某之间的互殴行为，导致事故发生。两人的行为严重危害公共安全，涉嫌犯罪。
 <br/>
 <br/>
 事发80多个小时后，10月31日晚23时28分，坠江公交车被打捞上岸。官方通报，事故共导致15人遇难（包括驾驶员1人）。
 <br/>
 <br/>
 目前，救援队已先后打捞出13名遇难者遗体，相信仍有数人失踪。
 <br/>
 <br/>
 不过，对于官方通报的死亡数字，大陆网友普遍存疑。据事发时的目击者此前透露，出事时公交车上满载乘客，数字远超官方通报的数字。
 <br/>
 <br/>
 另外，在事故发生后第一时间就有网民爆料称，出事公交车车牌为F27085，22路车，驾驶员叫冉勇，车上可能有29人。
 <br/>
 <br/>
 事发后，官方一直未公布公交车的行驶监控视频，直到公交车被打捞上来，才对外界公布视频和事故原因。
 <br/>
 <br/>
 不过，外界发现，官方公布的车内视频仅有司机冉某和乘客刘某互殴的场景，车内究竟有多少乘客，仍是未知数。
 <br/>
 <br/>
