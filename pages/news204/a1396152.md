### 中共前军委副主席外访秘闻 洋人以为煤老板来了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月20日讯】被视为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 在军中的最爱〞的前中央军委副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐才厚.html" target="_blank">
  徐才厚
 </a>
 ，港媒曾曝光其贪婪无度，甚至
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出国外访.html" target="_blank">
  出国外访
 </a>
 时丑态百出，横扫名牌不眨眼，商场内的洋人都以为来了个中国大款或山西煤老板。
 <br/>
 <br/>
 在中国，军队成了中共谋取一党私利的工具。中共前军委主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 ，因为在军中毫无威望，〝腐败牌〞成了他收买官、商、学、兵各界的工具，致使中共军队腐败日益猖獗，军心涣散、缺乏战斗力。
 <br/>
 <br/>
 据香港中文杂志报导，中共前中央军委副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐才厚.html" target="_blank">
  徐才厚
 </a>
 ，掌军十多年，战没打过一场，枪没放过一响，毫无威信可言，而且贪财贪酒玩女人五毒俱全，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_出国外访.html" target="_blank">
  出国外访
 </a>
 又丑态百出，各层级官兵没人看得起他。
 <br/>
 <br/>
 报导说，更令众将士咬牙切齿的，是他参照江泽民〝贪官治国〞模式，发明创造并〝发扬光大〞了军队贪腐文化，把一支号称世界〝最强大〞（至少在人数上）的军队，蜕变成若有战争必将一击即溃的〝北洋水师〞。
 <br/>
 <br/>
 报导特别提到徐才厚外访出丑的事。
 <br/>
 <br/>
 2009年10月，徐才厚以军委副主席身份，应美国防长盖茨邀请访美。其间要使馆人员陪他逛购物中心，又不好意思要美方人员陪同，徐作贼似地换了便服而去。大家发现徐对国际名牌很熟，一边走一边用手往东一指：这个不错；往西一指：那个不错。
 <br/>
 <br/>
 随行人员立刻把不错的〝这个〞和〝那个〞都买下，一看，果然不错，都是国际顶级名牌。
 <br/>
 <br/>
 徐的兴致很高，扫了许多名牌货物，无论多贵眼睛都不眨，不眨的原因是不用他掏一分钱，全由随行人员买单，相信最后都入了军费数，一个堂堂中央军委副主席，公款买名牌。
 <br/>
 <br/>
 商场内的洋人都以为来了个中国大款、山西煤老板之类。
 <br/>
 <br/>
 众所周知，徐才厚与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 两人均是江泽民的〝军中最爱〞，〝军中代言人〞。江泽民退下前，安排郭徐二人任职军委副主席，架空胡锦涛同时钳制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 。
 <br/>
 <br/>
 习近平十八大上台后，大肆清洗中共党政军中的江系人马。2014年6月，徐才厚落马，2015年3月死于爱滋病；同年7月30日，郭伯雄也正式被抛出，现在秦城监狱服刑。
 <br/>
 <br/>
