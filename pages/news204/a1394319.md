### 国际刑警组织主席中国失踪内幕深  由周永康一手提拔
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，9月从法国返回中国后失踪。10月5日，港媒爆料，孟宏伟已被中纪委控制。其涉及的内幕很深，当初由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 一手提拔，此次做为周的〝余毒〞被拿下也不令人意外。
 <br/>
 <br/>
 10月5日，前中共高级安全官员、现任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 的妻子，向法国里昂警方报案说，孟宏伟9月29日返回中国后音讯全无。
 <br/>
 <br/>
 香港《南华早报》引述消息说，孟宏伟抵达中国后立即被中纪委〝带走〞调查。目前正接受调查，但不清楚关押在什么地方。据知他涉及经济问题及其他贪腐问题，其中包括孟宏伟违规在香港购置房产。
 <br/>
 <br/>
 5日晚，国际刑警组织表示，已关注到有关报导，并称这是法国和中共有关当局的事，又称国际刑警组织的日常运作是组织的秘书长。
 <br/>
 <br/>
 今年64岁的孟宏伟，曾在中共安全机关担任过多项职务，2016年当选国际刑警组织主席后，依然保留中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 的职位。
 <br/>
 <br/>
 有关孟宏伟落马此前就有预兆。去年12月，孟宏伟被免去中共国家海洋局副局长、中共海警局局长职务。他在2012年3月兼任上述两个职位的。
 <br/>
 <br/>
 今年4月份，中共公安部官网〝领导信息〞栏目再次出现更新。孟宏伟被免去公安部党委委员的职务。早在2004年4月，孟宏伟就已升任中共公安部副部长、党委委员。
 <br/>
 <br/>
 除了接连被削权外，孟宏伟还连续数次缺席公安部高层会议。
 <br/>
 <br/>
 分析认为，孟宏伟长期在公安部任职，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 交集不少，如果此次作为周永康的〝余毒〞被拿下也不令人意外。
 <br/>
 <br/>
 周永康在2002年的中共十六大上升任政治局委员，2003年3月兼任公安部部长、国务委员后，孟宏伟在第二年被提拔为公安部副部长、党委委员，成为周的副手之一。
 <br/>
 <br/>
 中共公安部过去20年来，一直被江派人马把持。从1998年3月开始，公安部部长先后由江派大员贾春旺、周永康、孟建柱、郭声琨担任。
 <br/>
 <br/>
 去年中共十九大上，郭声琨升任政法委书记后，其空缺被中共河北省委书记赵克志接替。赵克志上任后，多次要求公安部继续清洗周永康的〝余毒〞。
 <br/>
 <br/>
 中共十八大至今，当局已拿下了周永康，以及周的多名心腹，包括公安部前副部长李东生、副部长杨焕宁、政治部主任夏崇源、公安部政治部主任夏崇源，重庆市公安局长何挺等多名部级官员
 <br/>
 <br/>
 3月中共两会至今，中共公安部高层持续大调整。外界预计，习阵营接管了公安部高层后，将继续在公安部清洗江派人马。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
