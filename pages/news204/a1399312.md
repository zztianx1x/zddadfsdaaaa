### 基辛格临别私宴吐真言 一句话点穿中美关系
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月15日讯】日前，北京邀请〝老朋友〞、美国前国务卿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基辛格.html" target="_blank">
  基辛格
 </a>
 访华，期望他出面斡旋贸易战。但据港媒披露，基辛格在离开北京前的私宴上表示，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系.html" target="_blank">
  中美关系
 </a>
 不可能回到从前了，似乎透出让北京〝好自为之〞的含义。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基辛格.html" target="_blank">
  基辛格
 </a>
 11月6日出席新加坡的彭博新经济论坛时，获中共国家副主席王岐山邀请访华，在北京先后得到习近平、王岐山和刘鹤等人的连番接见。港媒指，北京希望基辛格向川普传递中方愿意和解的信息。
 <br/>
 <br/>
 不过，香港《苹果日报》及社交网络日前登出消息指，基辛格在北京的最后晚宴上称，时光太快，两国关系回不到从前了。
 <br/>
 <br/>
 消息指，在基辛格离开北京前的晚上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈元.html" target="_blank">
  陈元
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_唐闻生.html" target="_blank">
  唐闻生
 </a>
 宴请了基辛格。唐闻生曾任毛泽东的秘书，是当年秘密邀请基辛格访华的四人中唯一在世的。而陈元是邓小平时代的元老陈云的长子。据说，基辛格提出要见陈元、何理良、唐闻生、傅莹和辛旗。
 <br/>
 <br/>
 在晚宴上，基辛格回答傅莹说，两国都在变，未来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美关系.html" target="_blank">
  中美关系
 </a>
 不会再回到川普以前的状态。
 <br/>
 <br/>
 基辛格认为，中美不会走向全面冷战，而是一种〝新的范式〞，但他没有具体说明是什么样的范式，只是让中方去寻找〝新的外交模式〞，〝决不能走向对抗升级〞。
 <br/>
 <br/>
 11月6日的新加坡论坛上，基辛格在演讲中已经告诫北京，〝中国需要超越自己的旧制度，才能成为引领亚洲的大国〞。他明言，美中关系正在从合作转为对抗，双方都应该知道冲突的严重后果。
 <br/>
 <br/>
 法广11日评论文章指，基辛格的发言很尖锐，熟读托克维尔的王岐山应该知道〝旧制度〞的含义。基辛格表面上在说〝美中两国〞，其实恐怕只是在警告北京。
 <br/>
 <br/>
 另外，对于中美是否走向
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_新冷战.html" target="_blank">
  新冷战
 </a>
 ，美国副总统彭斯的说法，似乎和这位〝北京老朋友〞明显不同。
 <br/>
 <br/>
 彭斯13日对美媒表示，中方要想避免与美国及其盟友陷入新冷战，必须〝从根本上〞改变其行为，在经贸和南海问题上认真对待美方的要求。
 <br/>
 <br/>
 彭斯并警告，11月底的川习会是北京避免冷战的最好机会（如果不是最后机会的话），如果北京拒绝做出显著、实质的让步，美国已经准备好在经济、外交和政治对中方施加更大压力。
 <br/>
 <br/>
 外界普遍认为，美国中期选举的结果，加重了美国政府对中共的强硬立场。中共对美国的〝经济侵略〞以及称霸世界的野心，让西方社会感受到实实在在的威胁，遏制中共已是美国朝野和两党的共识。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
