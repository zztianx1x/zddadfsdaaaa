### 辽宁610秘密文件曝光 中共迫害法轮功再度升级
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】伴随着所谓的〝扫黑除恶〞，中共骚扰和绑架法轮功学员的行动也同步展开。日前海外曝光一份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_辽宁省.html" target="_blank">
  辽宁省
 </a>
 〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_610办公室.html" target="_blank">
  610办公室
 </a>
 〞内部文件，内容为布置
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迫害法轮功.html" target="_blank">
  迫害法轮功
 </a>
 的所谓〝专项行动〞。文件显示，从10月开始，中共已开始系统性地升级迫害。
 <br/>
 <br/>
 日前，西方杂志《寒冬》（Bitter Winter）率先公布了这份
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共机密文件.html" target="_blank">
  中共机密文件
 </a>
 。该文件由北京下达给各个有关部门，其中包括辽宁各地的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_610办公室.html" target="_blank">
  610办公室
 </a>
 〞、法院、检察院、公安局、司法局等。
 <br/>
 <br/>
 这个文件被称为〝防范打击法轮功的专项行动方案〞，由公安局具体负责实施，互联网信息办公室（网信办）等相关机构配合，行动时间是10月到12月。
 <br/>
 <br/>
 这份机密文件详细列出了这轮迫害行动的具体措施，包括要求使用〝主动进攻〞、〝高压恐吓〞、〝建立专案〞等打压方式。
 <br/>
 <br/>
 文件显示，国内法轮功学员的讲真相活动、学法小组、真相资料点成为中共这轮打击的重点。文件并列出了法轮功学员讲真相的具体活动，包括悬挂横幅、涂写标语、制作散发〝真相币〞、发真相短讯、拨打真相电话等。
 <br/>
 <br/>
 为阻止真相消息的传播，网络、媒体是中共这轮打击的重点之一。
 <br/>
 <br/>
 文件列出了对微信、QQ等社交平台的监控、打压方式，包括关闭或屏蔽法轮功学员为讲真相建立的群组和账号，严打群组管理员等。
 <br/>
 <br/>
 海外媒体大纪元、新唐人等网站也在文件中被列为打击重点之一。同时，向海外媒体曝光当局恶行的国内民众也将遭打击。
 <br/>
 <br/>
 文件还要求对法轮功学员办的明慧网、〝明慧期刊〞地方版，以及明慧网的信息员、记者、编辑进行重点打压。
 <br/>
 <br/>
 文件还特别提到继续进行去年10月开始的〝敲门行动〞，并称要对上次没有覆盖的法轮功学员进行〝补敲〞，并收集法轮功学员的个人信息，包括流离失所的法轮功学员的信息。
 <br/>
 <br/>
 文件要求法院、检察院提前介入，协助定罪、批捕、起诉、审判等，并将为法轮功学员辩护的律师列为打击对象，要求律师按当局的意志接案和辩护。
 <br/>
 <br/>
 文件还要求强化对法轮功学员的所谓〝转化〞（多采用酷刑方式逼迫法轮功学员放弃信仰）。
 <br/>
 <br/>
 〝610〞是中共内部的重点保密单位，1999年6月10日由中共前党魁江泽民成立，是专职
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迫害法轮功.html" target="_blank">
  迫害法轮功
 </a>
 的〝盖世太保〞式机构，各级〝610办公室〞拥有调动和组织同级所有党政部门镇压法轮功的特权，其中包括实施系统性活摘法轮功学员器官的恶行。
 <br/>
 <br/>
 虽然中共中央〝610办公室〞在今年3月被并入中央政法委，但上述文件显示，它依然作为独立机构在政法委体系内存在和运作。
 <br/>
 <br/>
 11月26日，曝光该文件的《寒冬》杂志主编Massimo Introvigne向大纪元记者表示，〝向国际社会公布这些信息，非常重要。〞他建议，各国政府、国际社会〝发挥重要作用，发声谴责〞。
 <br/>
 <br/>
 Introvigne说，〝中共设立610办公室的目的是为了迫害法轮功，现在也延伸到其它宗教团体〞， 〝‘610’是中共镇压这些团体的工具。〞
 <br/>
 <br/>
 时政评论员横河表示，这份机密文件内容提到很多具体的措施，这表明中共〝610〞办公室、公安部门也在假借〝扫黑除恶〞的名义，一再迫害法轮功学员。
 <br/>
 <br/>
 他说，上述文件明显违反中国宪法和法律，〝尤其是对于打压宗教、信仰迫害、打压人权，它没有按照法律或者哪怕是公开的规定，而只能用秘密的方式来进行。因为如果要把法律公开了，它就不需要任何文件了，但是它也就没有办法打击宗教信仰了，所以它只能用这种方式。〞
 <br/>

 <br/>
 2018年8月27日至9月1日，中共9个中央督导组进驻山西、辽宁、福建、山东、河南、湖北、广东、重庆、四川九省（市），标志着中共所谓的〝扫黑除恶〞全面铺开。与此同时，中共政法委、〝610办公室〞对法轮功的大规模骚扰、绑架活动，也借〝扫黑除恶〞全面展开。
 <br/>
 <br/>
 而辽宁是中央第一轮〝督导〞的省份之一。
 <br/>
 <br/>
 其实，早在今年7月，中国多个省份已经陆续出现假借〝扫黑除恶〞，针对法轮功学员的大规模绑架行动。
 <br/>
 <br/>
 法轮大法明慧网11月17日消息，据不完全统计，2018年1月至10月，中共政法委、610、公安部假借〝扫黑除恶〞之名，在大陆各地绑架法轮功学员3,944名，骚扰3,783名学员。
 <br/>
 <br/>
 2017年，中共在全国实施所谓〝敲门行动〞，在全国范围内挨家挨户地骚扰、绑架法轮功学员。
 <br/>
 <br/>
