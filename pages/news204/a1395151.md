### 中国女律师控遭警围殴脱光检查 警称〝查无此事〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】中国广州维权女律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙世华.html" target="_blank">
  孙世华
 </a>
 近日对当地某派出所的警员提出控诉。她指控这些警员在她带委托人到派出所办事时藉故
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_围殴.html" target="_blank">
  围殴
 </a>
 自己，更以搜查是否携带武器为由逼迫她脱光衣服来羞辱于她。日前，涉事警局对外声称〝查无此事〞，却拒不对外公布案发时的监控录像，中国网友质疑警方护短。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙世华.html" target="_blank">
  孙世华
 </a>
 在维权网上发出的文章，孙世华今年9月20日与委托人一起到广州华林派出所，想为委托人的丈夫办保释。不料在派出所内，当孙询问办案人员警号后，该警官却突然把证件甩向她。
 <br/>
 <br/>
 孙世华当时本能举手遮挡，竟然被警员指控抢证以及袭警，接着便遭到该派出所的一群警察
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_围殴.html" target="_blank">
  围殴
 </a>
 。孙世华称，〝我被掐得满面通红，白眼上翻，双脚也几乎离地，瞬间失去知觉〞。
 <br/>
 <br/>
 直到警方发现随行的一名女民众正拿手机录下现场状况时，警方才停止殴打行为。
 <br/>
 <br/>
 孙世华随后被警员带到办案区，被迫缴出所有随身物品后，又被喝令脱光衣服接受〝检查〞，理由是怀疑她身上藏有〝武器〞。居然遭到这样的羞辱，孙几乎崩溃，直到她泪崩哀求，才被允许穿上衣服，整个过程约20分钟。
 <br/>
 <br/>
 随后孙世华又被警方拍照、指纹建档，然后又以怀疑她吸毒为藉口，强制她验尿。之后还被讯问了6个小时才获得释放。她表示，在自己被讯问的6小时中，警方提问的内容大多跟〝袭警〞毫不相干。事后，她得知，当天与她同行的女委托人与另一名女民众也曾被带到办案区脱光搜身。
 <br/>
 <br/>
 孙世华指出，警员以〝碰瓷〞的方式诬陷她袭警，并以此为藉口对她施暴并羞辱，这是她近20年律师生涯中最黑暗、恐惧、屈辱的遭遇。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-13/p9077801a924876015.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-13/p9077801a924876015-ss.jpg"/>
   <br/>
   <font size="-1">
    图为中国广州维权女律师孙世华和她的丈夫广州维权律师隋牧青。（网络图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 孙世华在网络上公开控诉警方的上述暴行后，警方反控孙〝以查看民警身份为由拉扯民警胸前的警察证〞，并指这种行为扰乱了警局的〝秩序〞。
 <br/>
 <br/>
 对于孙世华指控，警方宣称〝督察部已对此展开调查，通过调阅监视器、询问相关人员后，不存在孙世华等3人遭民警殴打和羞辱状况。〞
 <br/>
 <br/>
 据公开的资讯，孙世华是中国广州维权律师隋牧青的妻子，她本身也是一名资深律师。
 <br/>
 <br/>
 围殴侮辱事件发生后，孙世华曾对外表示，有关案子的委托人夫妻，因为对拆迁不满而多次到北京上访抗议，并多次遭警方抓捕。这一次是委托人的丈夫在北京上访被拘留后，委托孙去帮助寻求保释的方法，没想到警方会将对委托人夫妻的仇恨心理转嫁到前往申请保释的律师身上。她称，这是一起〝恨屋及乌〞的〝集体构陷〞事件。
 <br/>
 <br/>
 孙世华表示，因此遭遇这样匪夷所思的事情，自己一度怀疑自己是否仍有勇气继续执业。而据她的了解，不少女律师都碰过类以的警察碰瓷事件，但当事人大都选择沉默。她感叹自己可能没有资格同情弱者，因为女律师是〝弱者中的弱者〞。
 <br/>
 <br/>
 不过，对于办案现场的监控视频这一最关键的证据，警方却一直拒绝公开，因此网络上不少人质疑警方的上述说辞只是在〝护短〞。许多中国网友包括部分媒体都在呼吁，负责调查的警方应该公开监视器画面，以便让真相水落石出。有网友评论称〝真相不能靠脑补，法治不能靠煽情，请拿出证据。〞
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：秦锋）
