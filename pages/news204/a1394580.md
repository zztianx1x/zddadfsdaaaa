### 与组织〝讲条件讲价钱〞？公安部声明藏孟宏伟罪名玄机
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】中共公安部网站周一（8日）通报，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 收受贿赂、涉嫌违法，正接受国家监委监察调查。有分析指，这份声明同时透露出一些不寻常的讯息。
 <br/>
 <br/>
 8日凌晨，中共公安部部长赵克志主持召开公安部会议，通报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 因收受贿赂、涉嫌违法接受国家监委监察调查情况。
 <br/>
 <br/>
 中共公安部官网上发布的这份声明中还称：〝坚定不移反对腐败，坚决彻底肃清
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_流毒.html" target="_blank">
  流毒
 </a>
 影响〞，〝绝不允许跟组织
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_讲条件.html" target="_blank">
  讲条件
 </a>
 、讲‘价钱’〞等。但声明中并没有提及孟提出过哪些〝条件或价钱〞。
 <br/>
 <br/>
 对此，有分析人士认为，这份声明也透露出一些耐人寻味的讯息。例如，声明中提到有关肃清周永康〝流毒〞的影响，间接印证了此前海外媒体的报导，认为孟宏伟落马同周永康案有关联的猜测。
 <br/>
 <br/>
 实际上，早在10月5日法国电视台《Europe 1》率先爆出孟宏伟在大陆失踪后，海外就有多家媒体指，孟的被失踪甚至被调查〝并不令人意外〞。
 <br/>
 <br/>
 孟宏伟长期在中共公安部任职，周永康2003年3月出任公安部部长后，孟宏伟2004年4月升任副部长、党委委员，成为周永康的主要助手之一。
 <br/>
 <br/>
 周永康在2014年7月29日落马后，他的多名心腹也被查，包括副部长李东生、常务副部长杨焕宁、政治部主任夏崇源；同时周的多名旧部傅政华、黄明等副部长也被调离。
 <br/>
 <br/>
 2017年12月，孟宏伟突然被免去中共国家海洋局副局长、中共海警局局长的兼职；2018年4月，他又被免去已担任14年的公安部党委委员一职。
 <br/>
 <br/>
 同时，孟宏伟近几年接连缺席中共公安部高层的内部会议。中共官场早已传出孟宏伟仕途〝很不妙〞的消息。
 <br/>
 <br/>
 此外，上述提及的所有官员都积极参与了自1999年以来江泽民集团
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_迫害法轮功.html" target="_blank">
  迫害法轮功
 </a>
 的运动。孟宏伟已被海外明慧网列入迫害法轮功学员的涉案责任人之一。
 <br/>
 <br/>
 <h4>
  孟宏伟已非Interpol主席
 </h4>
 <br/>
 孟宏伟9月底从法国返回中国后便失踪。随后，法国警方接到孟宏伟妻子报案，并于10月5日宣布立案调查。
 <br/>
 <br/>
 10月7日深夜近12点，中纪委和国家监察委员会官网发布通报称，中共公安部副部长孟宏伟因为〝涉嫌违法〞，目前正接受监委监察调查。
 <br/>
 <br/>
 当天，国际刑警组织（Interpol）在其官网上说，法国里昂的国际刑警组织总秘书处接到孟宏伟辞去国际刑警组织主席的职务，并立即生效。
 <br/>
 <br/>
 根据国际刑警组织的章程和内部规定，来自韩国的国际刑警组织高级副主席金正阳成为代理主席。
 <br/>
 <br/>
 国际刑警组织说，今年11月18日至21日，该组织将在阿联酋迪拜举行第87届大会，届时将选出剩余两年的新主席。
 <br/>
 <br/>
 同一天，孟洪伟的妻子格蕾丝．孟（Grace Meng）也在法国里昂的一家酒店举行了记者会，有两名负责保护她的法国警察陪同。但她不愿意公开自己的相貌，外界亦不知晓她的中文姓名。
