### 知情人：北京紧盯美国中期选举  选情呈报北京
------------------------

<div class="wysiwyg">
 【新唐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 时间2018年11月05日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 即将登场，对于这场被视为检验美国总统川普（特朗普）政绩的选情。有台媒称，中共正紧盯着川普的这场〝期中考试〞，来自中共驻美各单位的资讯正在源源不断向中共各个驻美总领事馆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_回报.html" target="_blank">
  回报
 </a>
 ，最后向北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_呈报.html" target="_blank">
  呈报
 </a>
 。
 <br/>
 <br/>
 美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中期选举.html" target="_blank">
  中期选举
 </a>
 即将在11月6日登场，不仅要改选众议院所有435名议员和参议院33名议员，还要改选36个州的州长以及其它大小地方选举。
 <br/>
 <br/>
 中期选举的时间点，正好是川普上任约2年，因此中期选举也被视为是川普总统的〝期中考试〞，用来检测人民对总统施政的满意度。同时，中期选举距离2020年总统大选仅有2年光景，因此这次选举也普遍被认为是下届总统大选的风向标。
 <br/>
 <br/>
 如果川普领军的共和党，能在中期选举大有斩获，而且能在国会两院都获得多数党地位，将证明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普新政.html" target="_blank">
  川普新政
 </a>
 受到美国人民的普遍肯定，川普也很有可能在2020年大选赢得连任。
 <br/>
 <br/>
 因此，此次的中期选举，被外界视为是选择美国的未来、选择世界的未来，并将影响中共政权存亡。
 <br/>
 <br/>
 在这些背景下，美国此次的期中选举、特别是参众两院议员改选的选情，受到中共高层特别紧密的关注。而一些关键州的州长改选选情，也被中共高度关注。
 <br/>
 <br/>
 台湾《中央社》4日引述一名与中共外交智库关系密切的人士的话透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 对历次美国中期选举的关切程度，〝从来没有像这次这么高〞。
 <br/>
 <br/>
 美国总统川普上任近两年来，大力推行各项新政改革，包括大幅减税、创造就业、强化军事战力、实施严格移民政策、革新基础建设、恢复对神信仰、提倡传统价值等，让本已衰退的美国国力获得滋养复苏，成效卓着。
 <br/>
 <br/>
 同时川普还整顿华府官僚体制，希望改革政治风气，将权力从华府政客与精英手中还给人民，恢复真正由人民作主的民主政治。他也积极投入中期选举的辅选工作，希望组织出一支改革团队，在国会里与他并肩作战，扩大未来的政治改革能量。
 <br/>
 <br/>
 所以，这次中期选举可说是〝川普新政改革保卫战〞。
 <br/>
 <br/>
 《中央社》分析说，川普施政近2年来，在美国国内评价分歧，却对中共形成了超乎预期的庞大压力，不但正式将中共视为〝战略竞争对手〞，还对中国商品加征了四轮关税，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中贸易战.html" target="_blank">
  美中贸易战
 </a>
 仍在持续升温中。
 <br/>
 <br/>
 再加上美国副总统彭斯10月初发表严厉抨击中共的演说，令北京深感芒刺在背。
 <br/>
 <br/>
 面对川普政府的的强势炮火，中共原本有意拖到中期选举落幕后，再行协商。中共打的算盘是，如果共和党在中期选举失利，将削弱川普在党内与美国国内的支持度与气势；届时加上泛左派媒体与政客的助拳，将有利于中共在贸易战中增添筹码，甚至扳回一城。
 <br/>
 <br/>
 上述中共人士对《中央社》披露，中共驻美国大使馆及驻纽约、芝加哥、旧金山、洛杉矶、休士顿5个中共总领事馆，正按日搜集、汇整选情，并向大使馆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_回报.html" target="_blank">
  回报
 </a>
 ，最后向北京
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_呈报.html" target="_blank">
  呈报
 </a>
 。
 <br/>
 <br/>
 而各个中共总领事馆，透过美国各地由中共直接成立，或立场亲北京的华人，或在当地社团、学术单位、侨社、媒体等，将搜集选情的触角向下延伸。搜集内容包括，媒体公开的报导、民调数据，以及各自的社群网路探听得来的消息。
 <br/>
 <br/>
 截至10月底为止，北京对美国中期选举的评估与美国主流看法一致，即民主党有机会从共和党手中取得众议院多数席次，但参议院仍将由共和党掌握。
 <br/>
 <br/>
 这名人士说，中共在美的上述方式搜集选情，基本不会引起美方特别注意。因为搜集选情掌握驻在国政情发展，本来就是各国领馆的工作，它国的使馆也在做。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿)
