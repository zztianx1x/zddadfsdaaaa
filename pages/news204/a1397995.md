### 兰州货车失控连撞31车已致15死44伤（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】11月3日，甘肃省兰州市发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重大连环车祸.html" target="_blank">
  重大连环车祸
 </a>
 。一辆大货车因刹车失灵，从某高速公路收费站前的坡上冲下，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_连撞31辆车.html" target="_blank">
  连撞31辆车
 </a>
 。目前已知至少造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_15人死亡.html" target="_blank">
  15人死亡
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_44人受伤.html" target="_blank">
  44人受伤
 </a>
 。
 <br/>
 <br/>
 据兰州官方微博发布的消息，11月3日傍晚7点21分左右，G75临洮到兰州方向距离收费站200米处发生一起重大交通事故。一部重型半挂载重牵引车，沿兰海高速公路由南向北行驶，经17公里长下坡路段行驶至距兰州南收费站50米处，重型货车失控冲下坡，撞向排队等候过收费站的诸多车辆，造成重大交通事故。
 <br/>
 <br/>
 从网传现场视频显示，有多辆汽车被撞毁，有的车辆还在冒烟，有伤者被民众抬出车厢急救，还有一些人在车外焦急地等待救援。
 <br/>
 <br/>
 消息指，驾驶大货车的司机李丰，是辽宁人，据他自述，是因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大货车刹车失灵.html" target="_blank">
  大货车刹车失灵
 </a>
 导致的车祸。
 <br/>
 <br/>
 官方通报，截至11月4日凌晨5时，事故已造成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_15人死亡.html" target="_blank">
  15人死亡
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_44人受伤.html" target="_blank">
  44人受伤
 </a>
 （包括肇事司机受轻伤）、31辆车不同程度受损。但有当地网友透露，死亡人数还在不断上升。目前受伤人员已全部送医接受救治，肇事司机也被警方拘留进一步调查。
 <br/>
 <br/>
 据陆媒报导，发生车祸路段是一个连续下坡、多弯道的危险路段，经常发生车祸。
 <br/>
 <br/>
 有网民说，他走过那条高速，将近40公里的大下坡，遇到那种没经验的司机，不靠发动机制动，一路踩刹车到收费站必定失控。
