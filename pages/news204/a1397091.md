### 重庆大巴闪避轿车 冲破护栏坠长江 死伤不明(视频)
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月28日讯】中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万州区.html" target="_blank">
  万州区
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长江二桥.html" target="_blank">
  长江二桥
 </a>
 上28日上午发生一起严重交通事故，一辆大巴士为阬闪避逆向行驶的红色轿车，冲破护栏掉入长江，目前不清楚巴士上乘客数量，但已打捞出2具遇难者尸体。
 <br/>
 <br/>
 综合媒体报导，事发在上午10时左右，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重庆.html" target="_blank">
  重庆
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_万州区.html" target="_blank">
  万州区
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_长江二桥.html" target="_blank">
  长江二桥
 </a>
 上，一名驾驶红色轿车的女司机逆向行驶，一辆当地的22路公车疑为闪避红色轿车而冲出护栏，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_坠入长江.html" target="_blank">
  坠入长江
 </a>
 。
 <br/>
 <br/>
 据悉，事发现场桥下的长江水域较深，车内存有气室的可能性较小，令搜救难度较大，车上人员生还机会极微。事发后，江面可当地水警已派出搜救船在巴士坠江下游进行打捞。
 <br/>
 <br/>
 有民众说，涉事的是一辆环湖巴士，途经江南中学及万州中学，校方正查核是否有学生搭乘该辆巴士。
 <br/>
 <br/>
 目前无法确定公车上的乘客数量，但有网路上有消息称，车上约有26人或31人，全数坠江，估计至少20多人生死不明。稍早已打捞出1名遇难者尸体。随后，下午13：30左右发现第2具死者遗体。
 <br/>
 <br/>
 万州区警方已将肇事邝姓轿车女司机逮捕。
