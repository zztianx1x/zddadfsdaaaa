### 崔永元骂农业部〝最孙子〞 习近平也曾表过态
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月11日讯】央视前名嘴
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 ，在爆料中国娱乐圈逃税的丑闻之后，又将矛头对准了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 。他称〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农业部.html" target="_blank">
  农业部
 </a>
 是最最孙子的地方，现在专吃非转基因〞。尽管中国社会对转基因存在非常大的争论，但
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 曾就转基因表态，强调要确保
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安全.html" target="_blank">
  安全
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自主创新.html" target="_blank">
  自主创新
 </a>
 。
 <br/>
 <br/>
 10月11日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 微博小号〝小崔读书汇〞连续发文，就
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_转基因.html" target="_blank">
  转基因
 </a>
 问题批评中国涉农业机构，他说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_农业部.html" target="_blank">
  农业部
 </a>
 是最最孙子的地方，现在专吃非转基因，却不允许非转基因食品标注。
 <br/>
 <br/>
 他还说，农业部特供基地在新疆，全是有机的，现在严管，不敢了。不承认以前有。
 <br/>
 <br/>
 崔永元讽刺中共原农业部副部长张桃林是个骗子。张桃林曾称，现在对转基因的监管已做到了县一级。
 <br/>
 <br/>
 11月10日，崔永元也曾转网友评论，点名中国农业部称，农业部的特供基地在新疆吐鲁番，全是有机的，还有大鱼、大虾、各种新鲜土特产等，鸡蛋也是特供的。现在国家管得严，不敢了。顺便翻脸不承认以前有过。
 <br/>
 <br/>
 崔永元还转发网友的评论称，〝农业部把转基因鼓吹上了天，请国家把农业部的特供取消，立法让农业部必须吃转基因。〞
 <br/>
 <br/>
 在11月8日的微博中，崔永元炮轰农业农村部关于转基因食品的研究结论造假。他说，农业部说中国转基因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安全.html" target="_blank">
  安全
 </a>
 ，有专门的安全委员会。但转基因中心造假、负责人李奎是转基因安委会成员。
 <br/>
 <br/>
 崔永元上述所指的是，中国农业科学院退学博士生魏景亮，曾实名举报中共国家转基因研究中心事件。魏称，转基因研究中心存在大规模地赶作业式的档案造假、人员的冒名顶替和制作虚假劳动合同、任用实验技能不熟练的硕士、研究生进行检测等问题。
 <br/>
 <br/>
 据悉，崔永元自2013年开始卷入转基因争论，曾〝自费〞100万元人民币，赴美〝调查〞转基因食品的安全问题。
 <br/>
 <br/>
 在崔永元所制作的纪录片中，他称：在美国的超市，转基因食品是有明显标记的，和非转基因的食物是分开排放的。而且大部分的美国消费者对转基因的危害是有认识的。如果能够不买转基因食品的话，他们会尽量避开这类食品，因为对人体的危害性目前还没有深刻的研究。
 <br/>
 <br/>
 崔永元的转基因纪录片，一度在中国引起轩然大波，而且他的结论和力挺转基因的中共知名学者的观点大相径庭。
 <br/>
 <br/>
 尽管中国社会对转基因存在非常大的争论，但在2013年12月，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 曾在〝中央农村工作会议〞上表态，作为新技术、新产业，对转基因有争议、有疑虑属正常。习强调对于转基因一要确保安全，二要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_自主创新.html" target="_blank">
  自主创新
 </a>
 ，大胆研究，慎重推广上，〝占领转基因技术制高点，不能把转基因农产品市场都让外国大公司占领了。〞
 <br/>
 <br/>
 随后，中共农业部称，中共对转基因技术的支持是坚定的，在转基因技术应有一席之地。
 <br/>
 <br/>
 随着中共加强支持转基因作物，包括批准中国化工集团公司花费430亿美元收购瑞士种子巨头先正达，民众的担忧在不断增加。
 <br/>
 <br/>
 中共之所以加强支持转基因作物，分析认为与中国巨大的人口有关，在过去三十年，随着城市化运动吞噬农田，污染侵蚀耕地，生产足够粮食的挑战变得越来越大。
 <br/>
 <br/>
 自1992年以来，中国已成为第一个引入转基因粮食到商业化生产的国家。当年，在中国培育出一种抗病毒的烟草。自从那以后，中共当局开始广泛给转基因食品发安全证书。
 <br/>
 <br/>
 在崔永元站出来反对转基因初期，一名中共少将于2013年写了一篇专栏文章，将转基因食品跟生物武器相比较，说转基因食品的后果，将比鸦片战争给中国造成的伤害更严重。在2014年，广州军区官员要求在他们的部队禁止转基因食品。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：祝馨睿）
