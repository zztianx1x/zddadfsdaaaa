### 崔永元遗书网上疯传 有人出价500万要其手脚
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】引爆中国娱乐圈〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞黑幕的央视前主播
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 ，自曝受税务总局刁难，还遭警察3次传唤。同时他还称，有人出价500万元人民币，要其手脚。面对他自称的死亡恐吓，一份以他名义写的遗书在网上疯传。
 <br/>
 <br/>
 10月12日，一篇题为〈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 ：假如我倒下〉的文章，在中国互联网疯传。
 <br/>
 <br/>
 文章以崔永元口吻给孩子写遗书，〝如果爸有什么不测，请你相信爸不是意外，爸是和一群人战斗倒下，爸倒下的时候是冲着枪口的方向。〞
 <br/>
 <br/>
 文章盘点崔永元这辈子干的三件事，包括：当过中共中央电视台主持人，和转基因食品较上劲，掀开了娱乐界的黑幕。第一件事已经划上句号。
 <br/>
 <br/>
 关于第二件事和第二件事，崔永元称自己一直努力去做。文中还大致讲述了反对转基因和揭露娱乐圈逃税潜规则的缘由。
 <br/>
 <br/>
 这篇疑似崔永元遗书的文章在中国社交平台流传甚广，但是在当晚，这封〝遗书〞被中共官方部分封杀。目前微信中已无法看到，在微博中还是能看到一些。
 <br/>
 <br/>
 对此，崔永元在《今日头条》回应称，自己没有想过写遗书，想想也是，砸了转基因利益集团的饭碗，废了娱乐圈轻松搞钱的路径，得罪了道上的人，人身威胁警察不管。〝我当然要和它们大战一场！〞
 <br/>
 <br/>
 之前的10月10日，一名叫〝天仙哥哥V〞的神秘网友，在〝今日头条〞平台发帖称，为崔永元选好了墓地和遗照，还贴出了图片，该网友引述消息说，有人以500万元要崔永元手脚，并〝提醒〞崔出入要小心。
 <br/>
 <br/>
 崔永元回应说：〝已经做了最坏的打算，那就走着瞧吧。〞目前有关帖文已被删除。
 <br/>
 <br/>
 10月7日，崔永元在微博发贴说，他已经第3次被中共税务总局叫去问话，两名陌生面孔的警察对他补充提供的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 〞证据看都不看。
 <br/>
 <br/>
 他还说，警察拐弯抹角地询问他，从哪里弄到的〝阴阳合同〞。警察还对他和他所参与的所有公司进行了彻底调查，他以前的助理也被彻夜询查。他自称自己的危险来自四面八方。
 <br/>
 <br/>
 崔永元10月4日在微博发文称，在这5年的时间里，有一个名叫司马3忌的人，一直不停地对他进行骚扰和诽谤，而且还对他家人进行人身攻击，他愿意出8亿元钱给这个人，求放过自己。
 <br/>
 <br/>
 在崔永元表达对自身安全的忧虑之后，10月9日，《检察日报》以《举报人遭威胁，法律如何保护》为题做出回应。呼吁公安机关为举报人提供适当的保护。
 <br/>
 <br/>
 10月10日，上海市公安局官方微博也发通报称，上海公安机关已成立调查小组，并多次联系崔永元，相关举报问题一旦查证属实，将依法依规处理。
 <br/>
 <br/>
 有分析认为，引爆中国娱乐圈〝阴阳合同〞丑闻的崔永元，恐怕来头并不简单。另外。有人追溯崔永元爆料的过程，认为其背后似乎一直有支持、有角力。
 <br/>
 <br/>
 崔永元5月底开始爆料〝阴阳合同〞逃税事件，官媒10月3日称，女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 缴纳8.8亿元人民币罚款可免予刑责。有分析说，北京当局选择罚款而非刑罚，意在解决中美贸易战引发的国库空虚问题，范冰冰只是一个开头。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
