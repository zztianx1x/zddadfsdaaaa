### 薄熙来曾大骂周永康是〝常委中混蛋〞 内幕惊人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月14日讯】据报导，前重庆市委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 与薄
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷开来.html" target="_blank">
  谷开来
 </a>
 被抓后，都曾多次大骂中共前政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 是〝常委中的混蛋〞，薄谷开来更是诅咒周是个〝坏蛋〞，在她身上〝占便宜〞，令中共官场的罪恶勾结，荒淫无度曝光天下，震惊舆论。
 <br/>
 <br/>
 据海外中文网爆料称，早在2012年4月中旬，北京就向海外传出了内幕消息，披露中共前政治局常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 事件、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_谷开来.html" target="_blank">
  谷开来
 </a>
 涉嫌杀人案的幕后知情者和后台。
 <br/>
 <br/>
 而薄谷开来被捕后，为避免死刑，曾曝光周永康、薄熙来主谋推翻习近平的政变计划。
 <br/>
 <br/>
 薄谷开来供认，她是薄熙来与周永康之间关于他们策划密谋的信息传递人。她做的一切，周永康都知情，包括她要求英国商人海伍德为周永康和薄熙来所干的事。
 <br/>
 <br/>
 谷开来坚持要与曾保证过她和薄熙来安全的周永康见面。她说，在见到周永康之前，她不会进一步吐露，她还诅咒周是个〝坏蛋〞，在她身上〝占便宜〞，但却从未兑现他的承诺。
 <br/>
 <br/>
 海外网路上也曾有消息人士爆料，〝周永康跟薄谷开来睡过，最害怕薄谷开来被政敌抓到，而把他与薄熙来搞政变的事说出。〞此后，周永康和薄谷开来的花边新闻便频繁出现。
 <br/>
 <br/>
 据港媒报导说，周永康长期接受薄熙来和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王立军.html" target="_blank">
  王立军
 </a>
 提供的美女，周至少玩过包括女主播、模特儿及餐厅服务生约400名女性，为方便淫乐，周永康光在北京就有六处〝行宫〞。
 <br/>
 <br/>
 知情人士称，早在中石油时期，周永康就有〝百鸡王〞的外号。其中一名著名歌星，薄熙来睡过多次后，送给周共享。
 <br/>
 <br/>
 据报，薄熙来和薄谷开来被抓后，都曾多次大骂周永康是〝常委中的混蛋〞。
 <br/>
 <br/>
 2013年8月22日至26日，薄熙来为开脱罪行，在5天的庭审中上演了一出闹剧。在最后的法庭自辩护关头，薄熙来〝咬住〞自己老婆和下属王立军，称他们是〝感情纠结〞、两人〝如胶似漆〞，不能自拔，不惜为自己戴上一顶〝绿帽子〞，令各界哗然。
 <br/>
 <br/>
 不过，薄熙来在庭审时候，并不敢公开其妻薄谷开来，与他的幕后老板周永康之间的淫乱史，因其背后隐藏着巨大秘密。中南海虽然知道这些内幕，但是因为直接牵涉周永康，也不愿公开。
 <br/>
 <br/>
 <h4>
  薄案的核心内幕中共至今还在隐藏
 </h4>
 <br/>
 薄案的核心内幕是政变密谋和活摘器官罪恶，中共至今还在隐藏。薄谷开来是薄熙来周永康政变圈核心人物，同时英国商人海伍德之死，与薄熙来、薄谷开来活摘法轮功学员器官有关。
 <br/>
 <br/>
 根据追查国际调查资料，薄谷开来、薄熙来、王立军，在法轮功遭受残酷迫害期间，为追随
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 、罗干、周永康灭绝性的打压政策，推动和参与用死亡针注射杀害法轮功学员、活体摘取法轮功学员器官，非法贩卖其尸体，从中牟取暴利。
 <br/>
 <br/>
 而英国商人海伍德被薄谷开来谋杀，事实真相是因为他卷入帮助薄家资金海内外运转，并知晓惊人内幕。在海伍德面临国际情报部门及中共中纪委暗中调查时，谷开来和薄熙来唯恐走漏风声，杀人灭口，将其毒死。
 <br/>
 <br/>
 2011年11月14日，海伍德被发现死在重庆南山丽景度假酒店，时任重庆公安局长的王立军组织调查，迅速定论海伍德之死是因〝饮酒过量〞，并将尸体未经检验火化。
 <br/>
 <br/>
 两个多月后，此案出现戏剧性转变。2012年2月6日，王立军出逃美国领馆，揭露海伍德是被其上司重庆市委书记薄熙来的妻子谷开来主导毒杀。
 <br/>
 <br/>
 被抓后在庭审中，王立军供认，因海伍德案与薄熙来翻脸，并被薄痛打一拳，因担心自己也被杀才出逃美领馆。该事件一度震惊国际舆论，并导致三人先后身陷牢狱。
 <br/>
 <br/>
 2012年8月20日，谷开来被以〝故意杀人罪〞判处死刑缓期执行；9月24日，王立军被以〝徇私枉法、叛逃、滥用职权、受贿罪〞判有期徒刑15年；2013年9月22日，薄熙来被以〝受贿罪、贪污罪、滥用职权罪〞，判处无期徒刑。
 <br/>
 <br/>
 该事件犹如多米诺骨牌，引发中共官场大地震一波又一波，令全球目光聚焦中共政局。
 <br/>
 <br/>
