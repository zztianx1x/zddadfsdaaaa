### 瑞媒曝孟宏伟被捕原因：没完成北京交待任务
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月16日讯】近期，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共公安部原副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被捕引发舆论关注。有瑞士媒体称，中共根本就没有明白国际组织的运作方式，因此以为孟宏伟没有完成北京交予的任务而对其抓捕。
 <br/>
 <br/>
 瑞士《新苏黎世报》日前刊发题为《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 没有完成北京交待的任务？》的评论文章说，孟宏伟被抓捕事件对国际刑警组织造成了损害，而最大的受害者，其实是中共自己。
 <br/>
 <br/>
 文章称，很难说，北京究竟是缺乏必要的轻重感，还是精心算计了国际形象受损以及国内权力斗争的得失。北京只是很笼统地对孟宏伟一案给出了涉嫌〝受贿〞，这一常用的理由，因此外界只能猜测。
 <br/>
 <br/>
 虽然国际刑警组织的日常工作由秘书长主持，但是其主席由中共官员担当，依然有助于提升北京的国际形象。文章说，但孟宏伟事件发生后，北京若再提名中国人出任国际组织领导人时，其他国家就会三思。
 <br/>
 <br/>
 文章认为，也许讲究权力尊卑的中共官场，根本就没有明白国际组织的运作方式，因此以为孟宏伟没有完成北京交予的任务而对其抓捕。
 <br/>
 <br/>
 10月7日，中共官方宣布9月底神秘失踪的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 孟宏伟涉嫌违法，正接受中共国家监委监察调查。
 <br/>
 <br/>
 该消息迅速引爆国际舆论，有欧洲媒体曝，孟宏伟利用国际刑警组织主席身份的优势，一直努力为中共做一件事，因此引发该组织的担忧对其特别关注，同时也引来中共方面的威胁。
 <br/>
 <br/>
 据欧洲第一电台报导，孟宏伟与一家中国专门从事网路安全的大型公司美亚柏科有着密切联系，他本人一直争取国际刑警组织与美亚柏科的合作。在美亚柏科的网站上，可以看到孟宏伟与美亚柏科老板的合影，但从10月8日开始，这个页面消失了。
 <br/>
 <br/>
 据美亚柏科官网介绍，该公司成立于1999年，总部位于厦门软体园二期，主要服务于国内各级司法机关以及行政执法部门，目前在中国刑侦器材市场的占有率为45%。
 <br/>
 <br/>
 8月份，美亚柏科曾向俄罗斯销售被称为〝魔方〞的智能手机和移动设备数据提取系统。该系统能够在几分钟内轻松破解脸书、推特、Telegram等各种社交媒体账号内的信息，解码Skype等各种即时通讯软体中的信息存档，也能够获取所有已知智能手机和各种硬碟中的内容。
 <br/>
 <br/>
 5月15日，在北京举办的国际警备警用装备博览会上，美亚柏科曾展示其生产的XDH-CF-5600手机扫描仪，只需几秒钟就可以破解智能手机密码，提取用户数据。
 <br/>
 <br/>
 但国际刑警组织与美亚柏科的合作事件尚未办妥之时，孟宏伟突然落马。
 <br/>
 <br/>
 中国民主运动海外联席会议主席魏京生对美国之音说，国际刑警组织与中共之间的瓜葛已有20多年。最近一个时期，中共开始采取一系列动作，不但为该组织提供大量资金，而且还从中国调很多人到总部，对总部事务进行相当大的干预。
 <br/>
 <br/>
 他说，即便没有孟宏伟，这个组织内仍然有很多中共的工作人员。现在，中共要通过抓捕孟宏伟告诉所有官员，就是无论你们在哪里，都在我们的手心掌控之中。这也使得那些中共官员无论自己有什么想法，都只好替中共卖命。
 <br/>
 <br/>
 日媒《每日新闻》分析称，中共公安部掌握众多机密，其中包括涉及中共高层的相关秘密。孟宏伟拥有中共重要人物违法的证据。报导认为说，公安部的特殊地位，是孟宏伟突然被查的一个重要背景。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
