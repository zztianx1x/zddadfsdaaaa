### 〝八常委〞与胡锦涛齐现身  江泽民被另行处理
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月04日讯】日前，前中共人大副委员长王光英遗体火化仪式在北京八宝山举行，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 等七常委与被称为〝八常委〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 ，以及
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 等退休高层出席致哀，而一向爱出风头的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 也有了最新动态，但被另行处理，只见其名不见其人。
 <br/>
 <br/>
 中共官媒《新华社》报导，11月2日，中共前全国人大副委员长、光大集团前名誉董事长王光英遗体在北京八宝山公墓火化。
 <br/>
 <br/>
 当天上午9时30分许，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 带领中共七常委，被称为〝八常委〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王岐山.html" target="_blank">
  王岐山
 </a>
 ，以及中共前总书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 等现身默哀。
 <br/>
 <br/>
 被中共誉为〝红色资本家〞的王光英，于10月29日晚9点28分在北京病逝，享年100岁。
 <br/>
 <br/>
 报导称，王光英病重期间和逝世后，习近平、李克强、王岐山、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 、胡锦涛等人，前往医院看望或通过各种形式，对王光英表示了哀悼并慰问其亲属。
 <br/>
 <br/>
 报导提及了前中共党魁江泽民的最新动态。分析认为，此次七常委加王岐山、胡锦涛齐现身，不排除在贸易战困境下，胡锦涛现身挺习的意味。而江泽民则被另行处理，只见其名不见其人。一方面可能是被封杀不让其现身；另一方面，也可能是其身体有碍，生死不明。
 <br/>
 <br/>
 10月16日，有上海消息人士透露，江家有异常情况，纷纷〝集合〞，可能江泽民病危或更糟；10月3日，也有美媒披露，江泽民身体据称不断反覆，似乎并不乐观。
 <br/>
 <br/>
 9月30日，中共国务院在北京大会堂举行十一招待会。有美媒爆料，此次招待会上，没有中共元老的镜头，江泽民、李鹏、朱鎔基、胡锦涛、温家宝等人〝集体缺席〞。
 <br/>
 <br/>
 中共每年的十一招待会，早已成为观察其政坛动向的窗口。外界认为，元老集体缺席存在多种原因，其中不排除掩盖某元老病危的消息。当时有中南海知情者对美媒爆料说：由于江泽民和李鹏年事已高，健康频频亮起〝红灯〞，已经遭遇多次病危险情。
 <br/>
 <br/>
 中南海知情者还表示：中共元老们在卸任后一旦突然出现，都会被外界解读为释放出某种政治信号，因为高度敏感，元老们的健康状况，宛如国家最高机密，就更是焦点中的焦点，令人讳莫如深。
 <br/>
 <br/>
 然而在过去的一多月内，胡锦涛、温家宝、胡启立、朱鎔基、李瑞环、吴邦国、吴官正、刘云山等退休常委，接连以不同方式密集〝露面〞，江泽民却一直销声匿迹，引发舆论关注，。
 <br/>
 <br/>
 美媒认为，中共卸任领导人因身份特殊，鲜少出现在公众视野，对于他们的非官方公开活动，官媒普遍秉持不予报导的原则，因此他们露面通常藉助非官方渠道曝光。然而，上述中共退休常委露面，大多被官媒公开报导，被外界认为另有政治意涵。
 <br/>
 <br/>
 由于目前时值
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 白热化，国内经济遭遇严重危机，各种不明朗因素浮现，在北京陷入外忧内患的困境之下，习派、江派退休常委接连现身，或许与这波政坛波动有关。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿)
