### 台媒曝川普将对江泽民动手 江家5000亿美元或被冻结
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 已经蔓延至其他领域，台媒5日爆料称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 即将出手查封中共高官
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_在美资产.html" target="_blank">
  在美资产
 </a>
 ，首批被查封的对象包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 家族。如果消息属实，江泽民家族藏匿在美国的5000亿美金资产，很可能因此遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冻结.html" target="_blank">
  冻结
 </a>
 。
 <br/>
 <br/>
 9月5日，台媒《自由时报》引述海外爆料称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 即将出手，查封上海帮高官在美国的资产，被查封的对象包括中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 。
 <br/>
 <br/>
 消息称，上海帮属江泽民势力，长期控制中国电信、金融等多个领域，美国政府将先对江泽民家族实施依法查封，三周内立法基本完成。
 <br/>
 <br/>
 如果上述爆料属实，江泽民家族藏匿在美国的5000亿美金资产，很可能因此而蒸发。
 <br/>
 <br/>
 中共前党魁江泽民家族，是众所周知的中国巨贪家族，也被坊间称为〝中国第一贪〞。此前就有报导披露，江泽民家族目前在海内外持有现金和资产合计超过5000亿美元。
 <br/>
 <br/>
 据悉，江家控制的海外资产包括，基金、股票、银行、信托，能源股份、科技股份、黄金期货、房地产、海外控股公司、离岸公司等。这些资产均由江泽民的孙子孙江志成代表江家持有。
 <br/>
 <br/>
 海外曝料称，江志成目前持有中、美两国双重国籍。而此次爆料的资料来源于自己聘请的专业团队调查，如果被证实虚假，他愿负一切法律责任。
 <br/>
 <br/>
 《自由时报》引述分析说，美国在9月20日宣布制裁中共军委装备发展部和部长李尚福，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冻结.html" target="_blank">
  冻结
 </a>
 其美国控制范围内的所有资产，如今查江泽民等中共高官的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_在美资产.html" target="_blank">
  在美资产
 </a>
 ，可能是美国逼迫北京重上贸易谈判桌的一项手段。
 <br/>
 <br/>
 美国9月20日宣布，决定对李尚福实施一系列严厉制裁。这一决定立即引起中共官场恐慌。中共除了做出一连串反应，向美国表达抗议之外，中共党媒更是连发评论，称美国此举史无前例。
 <br/>
 <br/>
 分析认为，川普政府针对中共高官的制裁，等于打到了中共的〝七寸〞。
 <br/>
 <br/>
 10月3日，港媒提到，中共不少高官及军方将领，存巨额资金在美国，有部分超级大领导人，在美资产更以万亿元计，家人亲属亦移民美国，并以离婚作为掩饰手段。
 <br/>
 <br/>
 此前也有报导提及，美国是中共外逃贪官的首选，有超过7000名贪官藏匿在美国。中央党校一名教授2010年曾披露，118万官员配偶和子女在国外定居。
 <br/>
 <br/>
 中国民间一项研究发现，中共中央委员当中91%的人都有家人移民海外，甚至加入外籍；中共中纪委成员当中，88%的人都有亲属子女移民海外。
 <br/>
 <br/>
 中共权贵家族转移到美国的财富早已经达到天文数字，美国一旦正式启动对中共权贵阶层个人进行制裁，中共高层必将人人自危，不知道会有多少高官将被美国发起的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_金融战.html" target="_blank">
  金融战
 </a>
 〝一剑封喉〞，从而导致倾家荡产。
 <br/>
 <br/>
 而美国此举对中共高层的冲击、震慑与巨大压力，将远远超过关税、汇率等。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
