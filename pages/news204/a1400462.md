### 范冰冰被重罚8.8亿后 又在好莱坞出事了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 因卷入〝逃税〞案遭中共当局罚款8.8亿元人民币后，有关她将复出的消息不断传出。但有媒体22日称，范冰冰又在好莱坞出事了，不仅她参演的多部影片在中国被禁止上映，她在中国境外和海外的诸多代言也被取消。
 <br/>
 <br/>
 日前，投资过亿的美元好莱坞谍战片《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_355.html" target="_blank">
  355
 </a>
 》制片人称，正考虑用其它演员代替
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 出演女主角。据报导，原本确定范冰冰为女主角的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_好莱坞大片.html" target="_blank">
  好莱坞大片
 </a>
 《355》，极可能因这次的〝逃税〞事件解雇她。据悉剧组已经在酝酿换人，并面试了多位亚洲面孔的女演员。
 <br/>
 <br/>
 好莱坞大片《355》由知名导演西蒙・金伯格（Simon Kinberg）执导，原定由美国金球奖影后杰西卡・查斯坦（Jessica Chastain）、奥斯卡影后玛丽昂・歌迪亚（Marion Cotillard）、戛纳影后佩内洛普・克鲁兹（Penélope Cruz）和圣塞巴斯蒂安影后范冰冰以及奥斯卡最佳女配角露皮塔・尼永奥（Lupita Nyong’o）主演。
 <br/>
 <br/>
 此前消息称，《355》将是范冰冰继《登陆之日》（My Way）、《策马》（Stretch）以及《X战警:逆转未来》（X-Men: Days of Future Past）后，出演的第4部国际电影。
 <br/>
 <br/>
 范冰冰5月受邀出席了电影《355》全球发布会。她在受访时说，非常幸运能和这么多不同国家的优秀女演员合作。她还透露，拍摄该片是全英文对白，前期会进行很多训练，到时会到香港和英国取景。
 <br/>
 <br/>
 然而随着范冰冰卷入〝偷税漏税〞风波，不只是《355》的女主角位置受到影响，范冰冰出演的电影《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大轰炸.html" target="_blank">
  大轰炸
 </a>
 》也被中共官方取消了在院线公映的计划。
 <br/>
 <br/>
 尽管10月26日美国上映了电影《大轰炸》，但原属女主角的范冰冰，只有5个镜头，戏份仅剩9秒。而且她手里至少有6个电影项目面临〝上映困难〞。
 <br/>
 <br/>
 港媒9月份曾称，范冰冰遭中共当局封杀三年，已杀青的多部作品，都将被删除戏份或换角重拍。
 <br/>
 <br/>
 近几个月以来，范冰冰海量的天价代言也不翼而飞。中国境内外多个由她代言的品牌已被取消。
 <br/>
 <br/>
 范冰冰在海外的诸多代言也同时被取消。泰国著名的王权免税店此前宣布，取消了范冰冰担任免税店形象大使的计划；澳大利亚保健品公司SwisseWellness也宣布会暂停与范冰冰的合作。
 <br/>
 <br/>
 外界认为，即使范冰冰再度回到公众视野，演艺生涯恐怕也不再如以前顺利了。
 <br/>
 <br/>
 中共党媒16日再度发文，呼吁加强文艺界法治建设，强调不再让有法律和道德污点的艺人出名获利。有观点认为，这或意味着范冰冰复出之路被堵死。
 <br/>
 <br/>
 不过，近日在上海举办的奢侈品LV的时尚大秀上，LV方面公开力挺范冰冰。品牌方不但将范冰冰曾穿着出席戛纳电影节的礼服展出，还专门放置展示介绍牌，公开表明：范冰冰依然是自家代言人。
 <br/>
 <br/>
 有网友表示，此次品牌方的这波操作，真的是患难中见真情。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明)
