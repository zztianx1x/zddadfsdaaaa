### 中国人大侵害学术自由 康奈尔大学中断与其合作
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月30日讯】美国知名学府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康奈尔大学.html" target="_blank">
  康奈尔大学
 </a>
 近日决定，中断康大与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人民大学.html" target="_blank">
  中国人民大学
 </a>
 的两个学术交流与一个合作研究项目，原因是中国人大惩处了支持劳工维权的在校大学生，康奈尔大学认为那将严重侵害学术自由的原则。这是近年来首例国外大学与中国大学之间中断学术合作的案例。
 <br/>
 <br/>
 据英国《金融时报》的报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_康奈尔大学.html" target="_blank">
  康奈尔大学
 </a>
 工业及劳工关系学院国际项目主任弗莱德曼，在谈及康奈尔大学这次中断与人民大学的学术交流和研究项目的事件时表示，主要原因是人民大学不允许其在校大学生谈论中国的劳工问题，这种做法令双方的合作研究无法进行下去。弗莱德曼还表示，外国大学应当重新评估与中国大学的合作情况。
 <br/>
 <br/>
 据《美国之音》报导，促使康奈尔大学作出上述决定的直接导火索，是人民大学的几位学生两个多月前在网上发声支持工人维权，并参与了声援深圳佳士科技的工人要求组建工会的行动，结果遭到人大校方的惩罚。康奈尔大学认为人大的这种做法严重侵害了学术自由。
 <br/>
 <br/>
 据公开的资讯，中国广东省深圳佳士科技工厂的部分工人，今年7月下旬发起维权行动，指该公司存在超时加班、严苛罚款、欠缴公积金等违法行为，希望通过组建工会来维护自己的权益。但随后有积极组建工会的工人代表遭到不明身份人士的殴打，也有涉事工人被开除，该事件引发海内外舆论界的广泛关注。
 <br/>
 <br/>
 7月27日，一些佳士工人及其支持者前往工厂要求复工却被警方逮捕，令事件进一步发酵。随后不断有工人们和大学生组成声援团到现场举牌声援支持工人维权，呼吁释放被捕工人。
 <br/>
 <br/>
 在之后的近一个月时间内，现场声援团成员不断增加，其中包括部分来自北京大学、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人民大学.html" target="_blank">
  中国人民大学
 </a>
 等高校的学生也参与其中。外界注意到，与以往工人维权不同，这一次工人的维权目标已经从过去的经济诉求转变为政治诉求。
 <br/>
 <br/>
 8月24日，在佳士科技工厂现场举牌声援的人群遭到警方的彻底清场镇压，有数十名工人和声援学生被警方强行带走，其中有4名工人被警方刑事拘留。
 <br/>
 <br/>
 在此期间，人民大学哲学院2016级的本科生杨舒涵，因参与了佳士工人声援团，曾被近十个自称街道办工作人员者拦截，手机被抢夺并遭到人身威胁，之后校方强迫她休学一年。
 <br/>
 <br/>
 人大另一位2017级的本科生张子涵也因声援了佳士科技的工人维权，被老师当着全体同学的面批评，并被要求删除所有有关深圳佳士科技维权工人的信息。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：范铭）
