### 要求习近平〝检讨〞后  多维网紧急修改原文
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】总部设在香港的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海外党媒.html" target="_blank">
  海外党媒
 </a>
 《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_多维.html" target="_blank">
  多维
 </a>
 月刊》，最新一期刊登了一篇要求
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 〝必须进行自我检讨〞的文章。其兄弟媒体多维网转发不久，被迅速撤掉，重新上传的文章经过多次修改后完全改头换面，引发多维高层是否遭约谈的质疑。
 <br/>
 <br/>
 12月3日出版的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_多维.html" target="_blank">
  多维
 </a>
 月刊》刊登了一篇题为〝极左撕裂中国，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 应负责任〞的文章，回顾了近期有关〝民营经济退场论〞，为避免更多这类〝妖风〞接踵而至，习近平作为共产党核心，〝必须进行自我检讨〞。
 <br/>
 <br/>
 文章更指包括重庆、浙江等地高考〝政治审查〞风波、有活动将习近平和毛泽东并列于横幅上等。文章还直接引用外界对习的评语，〝批评者认定这是习近平的本意，他就是想成为另一个毛泽东〞、〝习近平就是要成为毛泽东第二，就是要大权独揽走回头路〞。
 <br/>
 <br/>
 总部设在北京被视为亲北京领导人的多维新闻网转发上述文章后，立即引发多种解读和猜测。
 <br/>
 <br/>
 旅居美国的中国经济学家何清涟发推文说，〝国安派系外宣媒体疑似吹响倒习号角〞，〝文章列举习的几大罪状，将倒习看作党与政权的生死存亡大事。该刊总部在北京，这样做，只有两个可能：一是准备鱼死网破，二是认为己方有胜算。〞
 <br/>
 <br/>
 但多维的文章随后被撤回，重新上载后改头换面，如〝他就是想成为另一个毛泽东〞、〝习近平就是要黄袍加身〞等，大量对习近平的批评质疑句子均被删除。
 <br/>
 <br/>
 而原本要求习近平深刻反思的结论，改为指向习是否另一个毛泽东问题上，〝可能习近平从一开始已不希望成为任何人的‘套娃’，而是想摆脱左加、超越左右〞，并确实获得很好的成绩。
 <br/>
 <br/>
 标题也改为〝防止极左撕裂中国 习近平强势拨乱反正〞。
 <br/>
 <br/>
 同时，多维网还在重要位置发表社论，为习近平〝新时代〞的改革开放背书。
 <br/>
 <br/>
 3日傍晚，多维网对相关文章进行澄清说，上述文章主要是批判近几年，特别是近期官僚的不作为与投机行为，让诸如〝民营经济退场论〞这类极左言论不断发酵。希望习近平充分认识为什么存在极端的言论，以及会给改革带来怎样的风险。
 <br/>
 <br/>
 文章还斥责，部分所谓的海外媒体与〝民主〞人士，对此文进行了恶意曲解，完全背离文章原意和初衷，有些带着强烈逢中必反、逢共必亡强烈意识形态的媒体，居然解读成多维要〝公然逼宫〞，〝有所图谋〞等。
 <br/>
 <br/>
 对于多维网的突然变风转向，何清涟又一次发推文表示：〝看来，多维北京总部一定被有关方面约谈了。〞
 <br/>
 <br/>
 一位接近多维的匿名人士告诉自由亚洲，多维的老总
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_于品海.html" target="_blank">
  于品海
 </a>
 对习近平有几分尊敬，而且经常在朋友面前表示支持习近平，因此不知为何会刊出此文。
 <br/>
 <br/>
 他说，多维新闻网站是在北京编辑部，不知道他们怎么样分工，可能是香港《多维月刊》写的东西比较敏感，然后北京写的不会那么敏感。高层追究的时候，可以有一个解释，就说敏感的东西是香港这边写的，有一个推诿的做法。
 <br/>
 <br/>
 北京一位匿名人士表示，中共高层在严禁搞个人崇拜方面，确实存在不同意见，上半年，城市街道挂有习近平海报，现在都拿下了，因为其他老人对此有不同看法。
 <br/>
 <br/>
 〝多维〞自2009年被商人于品海收购后， 将编辑部分别设在北京和香港两地。香港编辑部人员多为各地新闻组记者，北京方面则是内地新闻从业员，因其立场明显亲共，常常〝小骂大帮忙〞，有中共〝海外大外宣〞之称。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
