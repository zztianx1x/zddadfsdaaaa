### 孟宏伟谋杀陈水扁吕秀莲？法学专家曝内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】国际刑警组织主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 闪电落马后，其往日犯下的各类罪状陆续曝光。知情人士披露，当年做为周永康的助手（中共公安部长助理），孟涉嫌参与谋杀多名政坛高层。其中包括谋杀
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾总统.html" target="_blank">
  台湾总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈水扁.html" target="_blank">
  陈水扁
 </a>
 和副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吕秀莲.html" target="_blank">
  吕秀莲
 </a>
 ，企图影响当时台湾的总统选举。
 <br/>
 <br/>
 中纪委官网日前通报
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 〝收受贿赂、涉嫌违法〞正接受调查后，旅居澳大利亚的法学家袁红冰先生在其推特（Twitter）披露，在孟宏伟官运亨通过程中，袁红冰曾通过《杀佛》和《被囚禁的台湾》两本书，披露孟宏伟参与谋杀
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十世班禅.html" target="_blank">
  十世班禅
 </a>
 大师和参与制造台湾〝三一九枪击案〞等罪过。
 <br/>
 <br/>
 藏传佛教的领袖、第十世班禅额尔德尼．确吉坚赞，1989年1月28猝然离世，终年51岁。当时传言十世班禅是因试图摆脱中共的控制而遭毒杀，但中共官方宣称其〝心脏病〞突发死亡。
 <br/>
 <br/>
 〝三一九枪击案〞则是发生于2004年总统大选的前一日，当时选情呈现紧张胶着，各阵营正紧锣密鼓进行最后冲刺催票，没想到在台南街头却发生了震撼全球的枪击案。
 <br/>
 <br/>
 2004年3月19日下午，时任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台湾总统.html" target="_blank">
  台湾总统
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈水扁.html" target="_blank">
  陈水扁
 </a>
 和副总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吕秀莲.html" target="_blank">
  吕秀莲
 </a>
 在台南搭吉普车扫街拜票，过程中两人在台南市文贤路和金华路附近受到枪击，造成吕秀莲膝盖受伤，陈水扁腹部则有一道长11公分、深3公分的伤口。
 <br/>
 <br/>
 由于枪击案隔天便是投票日，因此绿选民群情激动，枪击案后，绿营立即停止造势活动改用电话拜票，选举晚会也暂停。当时的枪击案冲击之大，甚至改变了11任台湾总统大选的结果。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-15/p9080531a95390103.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-15/p9080531a95390103-ss.jpg"/>
   <br/>
   <font size="-1">
    〝三一九枪击案〞则是发生于2004年总统大选的前一日，造成吕秀莲膝盖受伤，陈水扁腹部受重伤。（网络截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 如今枪击案已经过去14年，爆出孟宏伟当时参与涉杀陈水扁与吕秀莲，似乎也不是空穴来风。
 <br/>
 <br/>
 发动〝三一九枪击案〞之时，孟宏伟担任中共公安部部长助理、交通管理局局长，而其上司是时任中共公安部部长周永康。
 <br/>
 <br/>
 袁红冰在2012年的著作《被囚禁的台湾》中说，为因应陈水扁2000年当选台湾总统后的情势，中共于2002年秋，由中共政治战略专家辛旗，设计出相应的对台战略。
 <br/>
 <br/>
 该战略企图透过对台湾的政治、经济、文化、宗教、社会的全面渗透，把自由的台湾置于中共专制的政治法律制度的管辖之下。
 <br/>
 <br/>
 书中说，〝中共总参二部制造三・一九枪击案的具体过程、透过怂恿发动红衫军运动，污名化陈水扁和民进党政府，用司法迫害和政治诬陷强加在陈水扁身上的一切莫须有的‘罪名’〞。
 <br/>
 <br/>
 据台湾媒体披露，〝三一九枪击案〞当时经美籍刑事专家李昌钰等人调查后，由台湾台南地方法院检察署及刑事警察局结案，认定该事件的嫌犯为已自杀身亡的陈义雄，但外界认为此案的真相仍扑朔迷离。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
