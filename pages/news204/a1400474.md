### 毛泽东秘书李锐病危 曾曝光毛强奸杨开慧堂妹秘闻
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月23日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李锐.html" target="_blank">
  李锐
 </a>
 之女李南央向外界透露消息称，父亲因被探视者传染感冒，病情恶化，刚刚被抢救回来，但情况不乐观。李锐因敢言曾遭受中共迫害，他曾直言
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛泽东.html" target="_blank">
  毛泽东
 </a>
 在文革中搞的那一套就是邪教。还对外界曝光毛强奸杨开慧堂妹秘闻。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李锐.html" target="_blank">
  李锐
 </a>
 今年已101岁高寿，他是著名的中共党史专家，曾任中共中央委员，中共中央组织部常务副部长，水电部副部长，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_毛泽东.html" target="_blank">
  毛泽东
 </a>
 兼职秘书。
 <br/>
 <br/>
 今年3月30日，李锐因身体不适被家人送往北京医院住院治疗，一直留院观察，很多人都关心他的病情。
 <br/>
 <br/>
 11月22日，原新华社记者高瑜在推特援引李南央的消息称，近日，李锐因被一位患感冒的探视者感染，持续发烧，刚刚才从凶险的边缘拉了回来，但病情恶化，现在已经离不开呼吸机的帮助。
 <br/>
 <br/>
 今年4月份曾一度传出李锐病危的消息，中国作家铁流引述消息称，李锐从昏迷中清醒后，交代家人三点身后事安排：一不开追悼会；二不在胸前覆盖党旗；三不进八宝山公墓。
 <br/>
 <br/>
 另有消息人士披露，李南央简讯告知：〝李老向家人交待死后不开追悼会，不覆盖党旗，不进八宝山。〞
 <br/>
 <br/>
 一般来说，中共官员死后葬在八宝山被认为是一种〝荣耀〞。不过，此前也有多名中共元老，包括刘伯承、华国锋、胡耀邦、赵紫阳等人，生前的遗嘱都是不进八宝山。
 <br/>
 <br/>
 有分析认为，这些中共高层官员都曾经历中共党内的残酷斗争，深知中共的邪恶本质。他们不愿进八宝山，是不愿再与中共为伍。李锐提出死后不盖党旗，不进八宝山的要求，应该也是这种心理。
 <br/>
 <br/>
 另外，还有一些中共高官，身在高位时，曾犯下淘天罪恶，害怕将来被人民清算，自己被挫骨扬灰，因此不敢进八宝山，周恩来和毛泽东就被指属于这一类。
 <br/>
 <br/>
 李锐一生经历坎坷，文革期间，曾被关在秦城监狱8年。文革后获平反出任中组部副部长，兼任中共中央委员。他对外界揭露了很多中共党内的丑恶内幕。
 <br/>
 <br/>
 李锐对毛泽东的评价堪称前所未有，他在101岁生日当天接受美媒采访时说，毛讨厌知识份子，说到底，毛还是个农民，心胸极其狭隘。
 <br/>
 <br/>
 他此前也曾直言毛泽东在文革中搞的那一套就是邪教，〝早请示晚汇报，跳忠字舞，摇小红书，真是邪透了〞。
 <br/>
 <br/>
 李锐还曾向记者高瑜透露，毛泽东的妻子杨开慧发现毛强奸了她的堂妹。杨开慧在遗书中称，毛泽东是〝生活流氓，政治流氓〞，这是被官方唯一删掉的8个字。
 <br/>
 <br/>
 此外，李锐还是三峡工程反对派代表人物之一，以及自由派杂志《炎黄春秋》的顾问。他不但在1989年〝六四〞事件中，支持前中共总书记赵紫阳，还非常厌恶靠〝六四〞镇压学生上位的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 。
 <br/>
 <br/>
 他曾揭露江的丑闻，称其在89学运中因投机整肃《世界经济导报》而惹祸，事后又惧怕追责，以致入进京时一度〝见谁都磕头〞，小人情态毕露。
 <br/>
 <br/>
 在中共〝十八大〞上，李锐曾猛烈抨击江泽民，并要求〝结束中共老人干政〞。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
