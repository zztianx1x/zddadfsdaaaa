### 女星赵薇扶贫拉9旬妇拍照 网友求饶：别来了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广西.html" target="_blank">
  广西
 </a>
 传媒21日发布中国女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵薇.html" target="_blank">
  赵薇
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林心如.html" target="_blank">
  林心如
 </a>
 到当地
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扶贫.html" target="_blank">
  扶贫
 </a>
 的消息，图片显示，两人与当地村民一起就餐、散步。但有网友发文抱怨，求演员不要来他们乡下扶贫了，说他外婆90多岁了，大冷天的，还硬拉来，又陪拍照又陪吃饭的。
 <br/>
 <br/>
 11月21日，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_广西.html" target="_blank">
  广西
 </a>
 机关报《广西日报》发布消息，女星
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵薇.html" target="_blank">
  赵薇
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林心如.html" target="_blank">
  林心如
 </a>
 现身广西贫困乡村
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_扶贫.html" target="_blank">
  扶贫
 </a>
 。报导称，两人于19日在〝中国长寿之乡〞等地露面。图文可见，赵薇、林心如与当地村民一同就餐，一同散步。
 <br/>
 <br/>
 但一名网友却在社交媒体抱怨此事。这位网友发贴说，〝求明星不要来我们乡下扶贫了，我外婆都90多岁了，快走不动路，大冷天的，还硬拉来，又陪摄影，又陪吃饭，而且跟他外婆沟通又不会说普通话。〞
 <br/>
 <br/>
 这位网友还说，〝赵阿姨〞（赵薇）又来，他们不用扶贫，不是他们穷，而是乡政府穷，旺旺大礼包还是送给乡政府，他们自己可以种地脱贫。
 <br/>
 <br/>
 这位网友质疑当地乡政府涉嫌贪污。他称，男星古天乐当时承诺建教学楼，但此后未见动静，反而乡长新盖了三层楼，并购置宝马车。
 <br/>
 <br/>
 扶贫一直是中共的头号工程，但因多年以来，中共的所谓扶贫被认为是作秀而令民众反感。
 <br/>
 <br/>
 赵薇和林心如现身广西扶贫的消息一出，网友们马上联想到女星范冰冰，20年前，三人曾共同出演风靡一时的电视剧《还珠格格》，引起很大的反响，一夜之间红遍大江南北，此后她们各自发展，几乎很少同框。
 <br/>
 <br/>
 此次赵薇、林心如同框出镜，让人关注到范冰冰的处境。范冰冰因卷入娱乐圈逃税案，10月被中共国家税务机关重罚超8.8亿人民币，而从镜头前消失已久。
 <br/>
 <br/>
 不过，11月18日晚，《中国电影报道》栏目发布消息显示，赵薇和林心如现身广西扶贫，其实是拍电影。
 <br/>
 <br/>
 报导说，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_小燕子.html" target="_blank">
  小燕子
 </a>
 〞赵薇、〝紫薇〞林心如为脱贫攻坚战星光行动再度合作，赵薇担任纪录电影《星光》总导演，策划拍摄林心如走进广西等地开展扶贫行动的点点滴滴。19日，林心如团队，跟随电影频道参与《星光》行动来到广西等地调研。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：戴明）
