### 川习会谈不拢怎么办？川普提前向北京摊牌
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 进入倒计时，川普（特朗普）总统日前对美媒透露谈判底牌称，唯一可能的协议是，北京必须向美国开放市场，和美国公平竞争。他还指出，如果和习近平谈判失败，将会对另外2670亿美元中国商品加征关税。
 <br/>
 <br/>
 4天后，川普将与习近平在阿根廷举行峰会。此次会晤是美中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的关键转折点，更关系到未来美中两国会否全面开启冷战，因此倍受外界关注。
 <br/>
 <br/>
 11月26日，川普接受《华尔街日报》采访时称，预计将如期执行对华2,000亿美元商品提升关税比例至25%，他〝极不可能〞会接受北京提出推迟这一计划的请求。提升关税的举措将自明年1月起施行。
 <br/>
 <br/>
 川普指出，此次谈判美方唯一能够接受的协议是，北京必须向美国开放市场，和美国公平竞争。他此前曾向外界表示，北京非常想要达成协议，但协议必须是对美国有益的。北京已经占了美国很多年的便宜，这种情况不会再继续了。
 <br/>
 <br/>
 如果谈判失败，川普表示将会对另外价值2,670亿美元的中国商品加征关税，税率将为10%或25%。他还提到，下一轮关税里，将包括从中国进口的苹果手机和笔记本。
 <br/>
 <br/>
 此前，彭博社曾引述三名消息人士说，如果11月底举行的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 没能缓解贸易战，美国将于12月初宣布，对其余的中国进口商品征税。征税时间大约是在2月中国黄历新年期间，这好比是美国送给北京一份〝新年大礼包〞。
 <br/>
 <br/>
 而北京方面为了能让此次川习会达成协议，日前罕见地向美国提交了一份让步清单，涉及142项内容。不过，川普已表示，清单漏掉了4、5个关键内容，他还不能接受。他认为，北京应该将这些内容加上。
 <br/>
 <br/>
 据悉，北京提交的清单依然是老调重谈，对于知识产权，工业补跌和市场开放等美国关切的问题，并未给出承诺。北京只不过是做出一个让步的姿态，没有实际的意义。
 <br/>
 <br/>
 对于此次川习会能否达成协议，外界普遍不乐观。美国商务部长罗斯（Wilbur Ross）日前告诉媒体，川习会最可能的成果是双方达成一项往前推进的〝框架协议〞。不过，双方能否达成框架，达成何种程度的框架都是未知数。
 <br/>
 <br/>
 白宫贸易顾问纳瓦罗近日更直言：〝即使美国与中国（共）在贸易方面达成协议，也无法保证中国（共）能够真正执行。〞中共加入世贸组织时有诸多承诺，但16年过去，这些承诺都没有兑现。
 <br/>
 <br/>
 美国三一学院经济系教授文贯中23日接受俄媒采访时也表示，即将举行的G20峰会是一个拐点，如果中美继续针锋相对，两国关系或将全面恶化。
 <br/>
 <br/>
 他认为，即使美中两国对一系列的经济、贸易分歧达成妥协，休战也只是暂时的。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：文慧）
