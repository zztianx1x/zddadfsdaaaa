### 华为〝公主〞身份成迷 传被捕时查出外国护照
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月07日讯】加拿大政府应美国请求拘押
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司首席财务官孟晚舟事件引爆舆论。随之孟晚舟的身份也被起底，网曝孟晚舟与其父华为创办人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 均拥有香港身份证，孟本人被捕持加拿大护照入境。中共外交部发言人对此不证实，也不否认。
 <br/>
 <br/>
 12月5日，加拿大司法部证实，中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司财务总监、创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任正非.html" target="_blank">
  任正非
 </a>
 的女儿孟晚舟，涉嫌违反美国对伊朗的制裁令，在温哥华被捕。美国正寻求将她引渡回美国受审。
 <br/>
 <br/>
 加拿大司法部发言人说，孟晚舟一案7日举行保释听证，因孟晚舟申请的〝禁止报导令〞有效，当局此刻无法提供更多详情。
 <br/>
 <br/>
 禁令规定，在司法程序中途，或者聆讯期间，禁止报导案情。根据法院资料，禁止报道令可根据《加拿大刑法》（Criminal Code of Canada）、《加拿大引渡法》（Extradition Act）等法律依据发出，而此案禁令，是根据《引渡法》第26节发出。
 <br/>
 <br/>
 禁令还称，孟晚舟以Cathy或Sabrina为别名。1972年出生，现年46岁。
 <br/>
 <br/>
 根据推特网民爆料，孟晚舟是加拿大国籍，持有美国护照。还有网民称，由于中共不承认双重国籍，如果孟晚舟真的入籍美国或者加拿大就等于放弃中国国籍，不再是中国公民，这就让中共驻加拿大大使馆的声明十分尴尬。
 <br/>
 <br/>
 6日的记者会上，中共外交部发言人耿爽，被记者问及〝孟晚舟持有加拿大和中国两国护照〞一事时，他回答，无法在此讨论关于此案的细节。
 <br/>
 <br/>
 中国网友们认为，孟晚舟应该具有加拿大公民资格，加拿大政府才能在她转机时将其逮捕归案。
 <br/>
 <br/>
 而根据香港公司注册处及土地注册处资料，孟晚舟拥有香港身份证，住在大角咀豪宅维港湾1座，由华为技术投资有限公司持有的高层单位。她本人在香港持有3个维港湾高层单位，与丈夫刘晓棕共同持有擎天半岛高层单位。
 <br/>
 <br/>
 孟晚舟还担任三家香港注册的公司董事，包括华为技术投资有限公司、香港华为国际有限公司及华盈管理有限公司。华为技术投资有限公司在香港持有18个维港湾单位。
 <br/>
 <br/>
 孟晚舟是华为创始人任正非的长女，是任正非与前妻所生，父母离婚后随母亲的姓氏。据悉，任正非也持有香港身份证。
 <br/>
 <br/>
 华为官网显示，孟晚舟现任副董事长。她毕业于华中理工大学，硕士学历，于1993年加入华为，历任公司国际会计部总监、华为香港公司首席财务官、账务管理部总裁。
 <br/>
 <br/>
 孟晚舟在2011年首次成为华为董事会成员之一，今年3月底的华为董事会换届中，任正非卸任副董事长，由孟晚舟接任，外界将之视为孟晚舟正式接班的信号。
 <br/>
 <br/>
 据了解，孟晚舟此前极少在公开场合亮相和发表演讲，为人极为低调。她在华为蛰伏了20年，直到2013年，任正非之女的身份才首次曝光。
 <br/>
 <br/>
 孟晚舟曾在一段访问中透露自己已婚，并育有一对儿女，丈夫是圈外人。这也否定了早前指她嫁给一名华为高级职员的传闻。
 <br/>
 <br/>
 虽然孟晚舟的弟弟孟平，叔叔和父亲的现任妻子目前都在华为等子公司上班，没有人像她如此身居高位。因此，孟晚舟被视为任正非的准接班人。
 <br/>
 <br/>
 在福布斯公布的2015福布斯中国商界女性榜中，孟晚舟曾名列第九位，2017年2月6日，福布斯发布〝2017中国最杰出商界女性排行榜〞，孟晚舟排名第八。
 <br/>
 <br/>
 华为公司于1987年在中国深圳成立，自2012年以来一直是全球最大的电信设备制造商。华为也生产消费类电子产品，今年超过苹果成为全球第二大智能手机制造商，仅次于三星电子。
 <br/>
 <br/>
 华为公司的产品从智能手机、到路由器、交换机、基站等等，目前全世界145个国家及500个电讯运营商使用华为产品。
 <br/>
 <br/>
 华为从未上市，管理架构相当神秘，被指长期帮助中共监视中国民众。其创办人任正非曾是中共军队团职干部；刚刚卸任的董事长孙亚芳，加入华为之前，曾长期在中共国安部任职。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
