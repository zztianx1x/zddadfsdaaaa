### 遭中共党媒攻击多年 众院未来议长被指强硬反共
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】美国中期选举大局已定。虽然民主党拿到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 主导权，但外界普遍认为，这对北京来说并非好消息。特别是，有望明年接任众议院议长的民主党领袖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佩洛西.html" target="_blank">
  佩洛西
 </a>
 （Nancy Pelosi）属于对中共的强硬派，已被中共官媒批判多年。
 <br/>
 <br/>
 美东时间11月7日，共和党确定获得参议院的过半席位，而民主党则将成为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_众议院.html" target="_blank">
  众议院
 </a>
 的多数党。消息传出后，中国股市应声下跌。
 <br/>
 <br/>
 近期以来，外界多有分析指，遏制中共已是美国两党共识，中期选举结果对中美贸易战影响不大。
 <br/>
 <br/>
 美国知名商业杂志《福布斯》资深撰稿人Kenneth Rapoza 11月5日发表题目为〝亲爱的中国（共）政府，民主党不会挽救你〞的文章。
 <br/>
 <br/>
 文章指出，贸易战或贸易冲突主要是一个行政决定，这是川普（特朗普）职权内掌管的事务。即便川普在这方面需要国会帮助，他仍然可以指望两党的支持，因为民主党的两院领袖都认同川普对中共的强硬政策。
 <br/>
 <br/>
 其中，参议院的民主党领袖舒默（Chuck Schumer）多次表态，对川普的关税政策表示肯定，并谴责中共盗窃美国知识产权。
 <br/>
 <br/>
 今年3月22日，川普宣布考虑对中国进口商品征收惩罚性关税时，众议院的民主党领袖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佩洛西.html" target="_blank">
  佩洛西
 </a>
 （Nancy Pelosi）当天就发表声明，支持川普政府〝为美国工人和产品而战〞，并呼吁美国政府采取更多行动来应对中共的〝各种糟糕行为〞。
 <br/>
 <br/>
 如无意外，随着明年元月众议院议长改选，已成为众议院多数党领袖的佩洛西将成为新一任议长。
 <br/>
 <br/>
 多年以来，佩洛西一直主张对中共采取强硬措施，被中共官方视为〝敌对人物〞，也被中共党媒列为美国对中共的〝鹰派〞之一。
 <br/>
 <br/>
 早在2008年，中共喉舌媒体就曾盘点佩洛西的所谓〝反华历史〞，指其从1989年六四事件之后就开始谴责中共人权，并曾在访华期间〝擅自〞到天安门拉开黑布条，纪念六四中被中共屠杀的〝烈士〞。
 <br/>
 <br/>
 此后，佩洛西曾公开反对给予北京最惠国待遇，抵制北京奥运等。
 <br/>
 <br/>
 在去年4月的首次川习会前，佩洛西曾致函川普，敦促他向北京提及中国人权问题，以支持中国那些因为宗教和政治信仰遭到迫害的人。
 <br/>
 <br/>
 此外，佩洛西一直公开表示对法轮功的支持。
 <br/>
 <br/>
 2016年10月，世界各地的法轮功学员在美国旧金山市政厅前广场集会，谴责中共活摘法轮功学员的器官。佩洛西发去了表达支持的信件。
 <br/>
 <br/>
 2016年12月，神韵艺术团莅临美国北加州4个城市演出，佩洛西给神韵发去贺信。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
