### 江西数千村民抗议建垃圾焚烧厂遭镇压
------------------------

<div class="wysiwyg">
 【新唐人2018年11月03日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江西九江市.html" target="_blank">
  江西九江市
 </a>
 柴桑区赤湖工业园村民集体抗议建垃圾焚烧发电厂，已持续一周。人数最多时达到数千人，大批特警两次到场暴力镇压，打伤并抓走多名村民。
 <br/>
 <br/>
 赤湖工业园附近村民3号告诉《新唐人》，村民得知垃圾焚烧发电厂动工后，每天都去阻止施工，维权行动已持续一周，其中受害最深的城子镇去的人数最多。10月28号和11月1号两次遭到大批特警镇压。10月29号，上千名村民手持〝坚决抵制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_垃圾焚烧厂.html" target="_blank">
  垃圾焚烧厂
 </a>
 ，保卫赤湖〞的横幅在园区内游行示威。
 <br/>
 <br/>
 赤湖村村民易先生:〝那个厂是污染相当严重的，哪一天都要去的，一个星期是有啊，老百姓多的有几千人，警察来了好多，抓了两次了，打了两次人，打伤的还在医院，第一次抓的八个人放了，第二次抓了五、六个都还没有放。〞
 <br/>
 <br/>
 村民还表示，该项目选址不当，与赤湖候鸟保护区仅一堤之隔，自来水厂就在长江边上，离取水口太近， 因此村民都反对。
 <br/>
 <br/>
 易先生:〝选址湖边上，把那个场地就填埋了办厂，这个地方是在长江边，前面是赤湖，后面是长江，都是水源的地方，长江水取水口1.5公里左右，周边住了有两、三个乡，有城子镇的、港口街镇的。〞
 <br/>
 <br/>
 该项目最初选址在港口街镇刘仓村，遭到村民激烈反对，从今年4月15号起扎营护地至今，与施工队发生多次冲突，当地政府于是重新选址在赤湖工业区内。
 <br/>
 <br/>
 江西九江港口街镇刘仓村村民刘先生:〝选址是靠近长江嘛，靠近饮水资源嘛，就长江边上嘛，现在天鹅就全部来了，这种环境保护的地方，栖息地嘛。(刘仓村抗争)差不多一年了，(冲突)也发生过，抓了好多次了。〞
 <br/>
 <br/>
 该项目是由中国光大国际有限公司投资兴建，占地300亩，总投资达10亿元。村民透露，所征用的土地，原先村民用来养鱼，目前已填土建厂，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_村民抗议.html" target="_blank">
  村民抗议
 </a>
 后停工，但政府没说不建，因此维权行动还在进行中。
 <br/>
 <br/>
 新唐人记者熊斌、舒灿采访报导
</div>
