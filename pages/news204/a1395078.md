### 孟宏伟被抓牵连谁？三只更大老虎浮出
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 被捕的消息公布后，公安部连夜召开会议通报事件，并宣布查处孟的同党。分析认为，接下来，公安系统甚至整个政法系统或将大震荡，目前已有三只更大的老虎浮出水面。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 9月底从法国返回中国后〝失踪〞。10月7日深夜，中纪委官网通报，孟宏伟〝收受贿赂、涉嫌违法〞，正接受调查。
 <br/>
 <br/>
 中共公安部8日凌晨开会通报孟宏伟事件称，〝坚决彻底肃清
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 流毒影响。〞并宣布公安部成立工作组，查处与孟宏伟共同收受贿赂的官员。
 <br/>
 <br/>
 同日，大陆微信公众号〝政知圈〞发文指，孟宏伟长期在中共公安系统工作，是中共前政治局常委、前公安部长及政法委书记周永康的老下属。
 <br/>
 <br/>
 除了周永康之外，孟宏伟的仕途升迁与现任中共政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭声琨.html" target="_blank">
  郭声琨
 </a>
 也有着千丝万缕联系。
 <br/>
 <br/>
 据称，郭声琨与孟宏伟关系密切。两人即是湖南中南工业大学校友，同时又都曾担任公安部正副部长。与公安部形成鲜明对比的是，郭声琨辖下的政法委，对于孟被查事件至今仍未发声，令外界颇为奇怪。
 <br/>
 <br/>
 而在孟宏伟落马后，已有多个公安高官没有露面也引发舆论注意。
 <br/>
 <br/>
 港媒《明报》在评论中称，从中共公安部连夜召开会议来看，公安部党委11名委员，只有新提拔的部长助理兼网络安全保卫局长林锐一人缺席外，10人全部到齐，颇有露面〝报平安〞之意。
 <br/>
 <br/>
 评论分析说，这次孟宏伟案会否牵连更广范围，殊难预料，十一长假，原公安部外调的高官中，除应急部党组书记黄明外，司法部长傅政华、统战部副部长侍俊、国家监察委副主任刘金国暂时都未见露面。
 <br/>
 <br/>
 司法部10日召开会议，由部党组书记袁曙宏主持，傅政华也未出席。而中共公安部虽对孟案表态，但中央政法委迄今未表态，箇中玄机，值得留意。
 <br/>
 <br/>
 评论还称，按官方通报暗示，孟似乎涉及周永康等政法系统贪官旧案，但会否牵连他近年的上司
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟建柱.html" target="_blank">
  孟建柱
 </a>
 、郭声琨仍需观察。
 <br/>
 <br/>
 中共公安部过去20年来，一直被江派人马把持，历任部长分别为江派的要员贾春旺、周永康、孟建柱、郭声琨。
 <br/>
 <br/>
 孟建柱、郭声琨则被指是江派二号人物
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆红.html" target="_blank">
  曾庆红
 </a>
 的嫡系。据悉，郭声琨还是曾庆红的表外甥女婿，郭声琨妻子的祖母是曾庆红的母亲邓六金的亲妹妹。
 <br/>
 <br/>
 习近平上任以来，对江派大员、前政法委书记周永康长期掌控的中共公安、政法系统进行了大清洗。包括周永康及其亲信李东生、张越、杨焕宁、夏崇源等被查。周的多名旧部傅政华、黄明等公安副部长也被调离。
 <br/>
 <br/>
 如今，长期被江派把持的中共公安部高层，基本被习近平阵营接管。自今年7月以来，中共公安系统有20多名公安局长、副局长被查。
 <br/>
 <br/>
 有知情人士对自由亚洲电台透露，孟宏伟被查后，他的多名部下亦受牵连，但具体情况不明。
 <br/>
 <br/>
 有海外媒体分析，作为浸淫中共公安系统多年的高官，孟宏伟的落马或将牵动中共公安系统，甚至整个政法系统的又一轮反腐风暴，目前已经浮出水面的郭声琨和孟建柱，以及两人的大后台曾庆红会否受牵连，还有待观察。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：戴明）
