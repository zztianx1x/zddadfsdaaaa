### 大陆马拉松选手丢金内幕曝光 知情者揭潜规则
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】日前，中国大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马拉松.html" target="_blank">
  马拉松
 </a>
 选手何引丽，因冲刺时刻被递
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国旗.html" target="_blank">
  国旗
 </a>
 而丢失金牌的事件引发所谓〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱国.html" target="_blank">
  爱国
 </a>
 〞的热议。有外媒引述知情者披露，何引丽丢金在某种程度上是中共体育局的潜规则造成。
 <br/>
 <br/>
 11月18日举行的江苏省苏州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马拉松.html" target="_blank">
  马拉松
 </a>
 上，中国选手何引丽与非洲选手争冠时，突然遭遇志愿者围追堵截〝送
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国旗.html" target="_blank">
  国旗
 </a>
 〞，以期冲线时营造〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_爱国.html" target="_blank">
  爱国
 </a>
 〞景象。不料适得其反，何引丽在干扰下以几秒之差屈居亚军，更因旗帜从手中掉落而背负〝不爱国〞的骂名。
 <br/>
 <br/>
 苏州高新区体育局声称，志愿者递国旗导致运动员错失冠军，是志愿者的个人行为，原则上不允许其他人进入赛道，但体育局认为不应该追究某个人的责任，因为有人递红旗〝可能是出于爱国〞。
 <br/>
 <br/>
 不过韩国《中央日报》21日的报导却引述一位大会的相关人员透露，〝中方有一个潜在规则，那就是冠亚季军的中国选手一定要身披国旗到达比赛终点〞。
 <br/>
 <br/>
 一位有9年跑龄的马拉松爱好者方传忠告诉BBC说，一般马拉松都是全程封闭，有警察、保安或武警看守，严禁行人穿入赛道，更不可能出来在离终点200米的地方，允许志愿者自发递国旗的行为。但志愿者不太可能被追究，〝真要认真起来，他们很可能会说出真相，这更可怕。〞
 <br/>
 <br/>
 香港《东网》的评论文章也说，类似行为几乎已经沦为马拉松比赛〝潜规则〞。遇到有外国选手参与的赛事，但凡有望争胜的中国跑手必须在冲刺阶段〝国旗加身〞，或双手高举着冲过终点，以凸显所谓的〝爱国精神〞，否则将招来〝爱国贼〞的道德绑架和口诛笔伐。
 <br/>
 <br/>
 评论写道，以爱国为名行破坏之实的所谓〝爱国主义〞，早已充斥于中国社会的各个层面。如当年钓鱼岛事件引发抵制日货运动，最终演变成打砸抢烧；因萨德风波实施的〝禁韩令〞，导致购买韩国货老翁惨遭踢打；甚至在外国机场大唱国歌，以维权为名行胡搅蛮缠之实。这些行为不但有失人格，亦有失国格，贻笑大方，令人侧目。
 <br/>
 <br/>
 评论还认为，类似这些行为都不是发自内心的爱国，而是将爱国当成政治投机，藉机捞取各种好处。正如同大陆〝贪官最爱国〞的流行说法，没出事之前，个个都是为国为民的清官好官，东窗事发后则个个都是侵吞国家资产、将金帛子女转移到境外的腐败分子。
 <br/>
 <br/>
 评论最后说：〝这些人不管如何将爱国口号挂在嘴边、将国旗披在身上，也难改其祸国殃民本质。〞
 <br/>
 <br/>
 （记者陈远辉报导/责任编辑：戴明）
