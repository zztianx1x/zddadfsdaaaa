### 港媒：习近平粤沪之行两套班子 暗含敏感政治信息
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月09日讯】北京最高领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 近半个月来，先后出行广东和上海。港媒日前分析，两次都事关〝改革开放〞，都是先出席大型活动再开始视察，但随行班子不同，显示出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡春华.html" target="_blank">
  胡春华
 </a>
 的分工和地位的差异。
 <br/>
 <br/>
 香港《明报》11月8日发表的评论文章，总结
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 粤沪之行的异同，解读其中暗含的人事、政策信息。
 <br/>
 <br/>
 文章说，习近平在广东出席珠港澳大桥开通仪式时，国务院副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘鹤.html" target="_blank">
  刘鹤
 </a>
 在场，是因为刘鹤在国务院分管交通运输部。而上海进口博览会是由另一位副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡春华.html" target="_blank">
  胡春华
 </a>
 主管的商务部主办，因此开幕式由胡主持，刘鹤没有参与。
 <br/>
 <br/>
 另外，习近平在大桥开通后视察广东，刘鹤也从头跟到尾；而习在进博会后的视察中，胡春华却不见了踪影。文章分析，这显示两人的分工是胡主外、刘主内。
 <br/>
 <br/>
 文章还说，刘鹤在习的国内视察中从不缺席，但极少陪同他出访，突显他仍为身兼中财办主任的经济智囊角色。而中办主任丁薛祥和发改委主任何立锋则不分内外，一直陪伴习的左右。
 <br/>
 <br/>
 文章还分析了习近平在两地讲话的异同。
 <br/>
 <br/>
 习在广东和上海都强调了〝改革开放〞，但对广东的指示〝提高发展平衡性和协调性〞，似乎暗含了对广东各地区发展不均衡的批评。习要求上海〝更好为全国改革发展大局服务〞和〝深化社会治理创新〞，显示上海已被习指派为〝长三角一体化〞的〝龙头〞，而〝粤港澳大湾区〞至今依然主次未明。
 <br/>
 <br/>
 港澳地区多年来都是江派大佬曾庆红的地盘，广东也曾是江派势力盘踞之地。而上海则是江泽民政治老巢。
 <br/>
 <br/>
 目前，上海主政的是习近平旧部李强。而粤港澳大湾区计划的主持人是江派出身的政治局常委韩正。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
