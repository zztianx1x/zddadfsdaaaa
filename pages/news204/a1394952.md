### 范冰冰事件引发后效应 国际影片准备解雇她
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 逃税事件霸屏舆论场4个月来，有关范的最新动态爆料一直层出不穷，尽管如今范缴纳税款8.8亿元人民币，但范冰冰事件仍未消停。最新消息称，国际动作电影《355》的制片人准备解雇范冰冰。
 <br/>
 <br/>
 10月11日，陆媒引述美国《综艺》杂志消息称，国际动作电影《355》的制片人准备解雇
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范冰冰.html" target="_blank">
  范冰冰
 </a>
 ，制片人说：如果她没有以一种令政府、电影发行商和中国观众满意的方式走出来，他们将选择其他演员替换她。
 <br/>
 <br/>
 电影《355》预计在2019年春季以后才会开机。制片人们已经做好准备。目前，该团队正在观察和等待，寻找新的演员来顶替范冰冰。
 <br/>
 <br/>
 《355》讲述世界各地各组织的顶级女特工们，联合阻止了一个可令世界陷入混乱的全球性组织获得一种武器，355是几位组成的新派系的代号。在5月份戛纳电影节上，华谊兄弟花2000万美元重金拿下该片的中国发行权，范冰冰的去留将决定华谊的投入是否打水漂。
 <br/>
 <br/>
 自央视前主持人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_崔永元.html" target="_blank">
  崔永元
 </a>
 曝光娱乐圈
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_阴阳合同.html" target="_blank">
  阴阳合同
 </a>
 事件后，大陆影视娱乐明星及其公司的偷税漏税问题逐渐浮出水面。
 <br/>
 <br/>
 范冰冰于6月起消失于公众视线，直到10月3日，中共税务机构公布案情，宣布范冰冰涉〝阴阳合同〞逃税案，缴纳约8.8亿元人民币税款，可免于牢狱之灾。至此范案终于落幕。
 <br/>
 <br/>
 然而范冰冰事件引发的巨震仍未消停，舆论场的各种声音仍是此起彼伏。大批中国影视圈人士深受范冰冰事件影响，行业税务地震仍暗潮汹涌，娱乐行业面临着结构性的调整。
 <br/>
 <br/>
 英媒BBC评论称，范冰冰逃税事件引发的关于中共税收〝不公〞的讨论，是中共税制改革不得不面对的问题。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
