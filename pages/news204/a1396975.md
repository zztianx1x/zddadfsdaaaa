### 港珠澳大桥开通白海豚起舞？官媒被轰〝荒谬〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月27日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 近日正式开通，中共官媒为了粉饰太平，在报导大桥附近海域出没的白海豚时，将海豚抬头浮出水面呼吸的举动描述为〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欢腾起舞.html" target="_blank">
  欢腾起舞
 </a>
 〞以庆祝大桥开通，引起公众的反感，被网友批〝太肉麻〞。日前台媒更引述专家称批中共官媒〝荒谬可笑〞。
 <br/>
 <br/>
 耗时近9年才建造完成的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 10月23日正式开通。中共官媒报导称，在举行开通仪式的当天，正在港珠澳大桥东人工岛附近海域值守警戒的广东海事局海巡船，在附近海域发现了5头
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中华白海豚.html" target="_blank">
  中华白海豚
 </a>
 。
 <br/>
 <br/>
 中共官媒描述称，这5头白海豚正〝列队〞向大桥方向游去。报导又称，白海豚们〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_欢腾起舞.html" target="_blank">
  欢腾起舞
 </a>
 〞，或许也是来〝祝贺〞大桥开通的。官媒还发布了5头白海豚在海中游动并时不时浮出海面的视频画面。
 <br/>
 <br/>
 该报导发表后，迅速被其他多个中共官媒以及受中共直接控制的港媒转载。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-27/p9098831a383215183.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-27/p9098831a383215183-ss.jpg"/>
   <br/>
   <font size="-1">
    中共官媒将白海豚抬头浮出水面呼吸的举动描述为〝欢腾起舞〞以庆祝大桥开通，遭专家批〝荒谬可笑〞。（视频截图）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 中共官媒的上述溢美之词引起了中国网民的反感。有网友留言称〝马屁拍得太肉麻，起鸡皮疙瘩啦！〞还有网友质疑：〝确定那些白海豚的视频真的是开通当天拍摄到的吗？考虑到官媒报导假新闻的历史劣迹，这种‘巧遇’白海豚的事，其真实性十分值得怀疑。〞
 <br/>
 <br/>
 日前，台湾《自由时报》引述香港海豚保护协会主席郑家泰批评说：〝荒谬可笑，海豚本来就会抬头呼吸。〞郑家泰表示，其实修建港珠澳大桥已经对白海豚的生存环境造成了污染，严重影响白海豚的生态与繁殖情况。
 <br/>
 <br/>
 据香港渔业和农业部的报告，自从港珠澳大桥项目动工以来，在香港水域生存的中华白海豚的数量就在下降。在2012年这片海域白海豚的数量还有80只，到2017年就只剩下47只，下降幅度达40％。
 <br/>
 <br/>
 据《ChinaFile杂志》报导，香港海豚保护协会谴责建造〝港珠澳大桥〞时，香港政府没有对环境影响评估做好，导致了这样的恶果。该协会的主席郑家泰更表示，在建造桥梁期间，大屿山北部的海豚几乎全部消失。
 <br/>
 <br/>
 香港大学太古海洋科学研究所副教授Leszek Karczmarski则曾经在接受《亚洲科学杂志》的采访时说：〝为了有效保护海豚，应正视栖息地的保育工作，将重点放在海豚生活的核心区域和关键栖息地上，才有可能扭转要灭绝的趋势。〞
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：赵云）
