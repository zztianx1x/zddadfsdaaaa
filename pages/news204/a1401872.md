### 贸战停火协议回避一重大敏感问题 专家警告：别高兴太早
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月04日讯】经过双方首脑的会晤谈判，美中之间达成了为期90天的〝停火〞协议。有美国媒体分析指出，作为美国政府关切的核心问题，〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞 却没有出现在这份〝停火〞协议中，显示双方都有意回避了这一重大敏感问题。在接下来的3个月谈判中，这个问题恐怕会成为阻碍双方真正结束
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的一大障碍。
 <br/>
 <br/>
 美国总统川普（特朗普）与中国国家主席习近平12月1日在阿根廷会晤后，双方同意在接下来的90天内，围绕强制性技术转让、知识产权保护、非关税壁垒、网络入侵及盗窃等议题，进行有关中方结构性改革的谈判。如果90天内无法达成协议，美方仍然会把针对2000亿美元中国输美产品加征的关税从10%上调至25%。
 <br/>
 <br/>
 外界注意到，华盛顿围绕结构改革的问题，向北京提出了五方面的要求，但这次双方似乎都没有提及之前美国政府十分敏感的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划。
 <br/>
 <br/>
 对此，虽然有中共官媒解读为〝两国在寻找围绕高度敏感问题相互妥协的可能性〞，但致力于中国商务与政治经济研究的美国智库《战略与国际研究中心》副主任甘思德（ Scott Kennedy）却指出，〝中国制造2025〞不可能消失，在未来的谈判中〝一定会再次出现〞。
 <br/>
 <br/>
 《美国之音》日前的一篇报导中引述甘思德分析称，〝美国依然高度关注中国的产业政策。‘中国制造2025’只是其中的一项而已。在这样的高峰会中没有必要具体列出某项特殊的产业政策。〞
 <br/>
 <br/>
 华盛顿智库《美国企业研究所》研究美中战略竞争的研究员扎克．库珀（Zack Cooper）也对《美国之音》表示，中国制造2025对中国的产业以及更广泛的发展计划非常关键，很难想像北京当局会完全放弃它。他推测说：〝也许，他们会讨论，如果中国一定要继续这项政策的话， 对它进行一些限制，或者，他们会改个名。〞
 <br/>
 <br/>
 12月3日，新加坡《CFO创新杂志》在一篇分析文章中警告说，不要高兴的太早，因为美中的技术对峙并没有消除。文章写到，〝中国制造2025才是这次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 的根源，双方都没有提及这个问题，美中贸易战的不确定性就依然存在。〞
 <br/>
 <br/>
 〝中国制造2025〞是中共国务院总理李克强2015年所提出的经济发展计划，涉及10大领域，包括新一代信息技术，高档数控机床和机械人，航空航天装备，海洋工程装备及高技术船舶，先进轨道交通装备，节能与新能源汽车，电力装备，农机装备，生物医药及高性能医疗器械和新材料等方面，目标是让中国在2025年从〝制造大国〞晋升为〝制造强国〞。
 <br/>
 <br/>
 甘思德针对北京的这项计划指出：任何国家都有权制定自己的产业政策。但是，中国制造2025的目标和实现手段令人害怕，有可能威胁到全球的技术创新。
 <br/>
 <br/>
 他进一步指出，中国为实现这些目标所采取的手段令人担忧。〝补贴，数字通常是上万亿的；市场准入的限制、地方合同的要求、独特的标准，所有这些政策很明显并没有在中国，甚至超越中国的地方，创造一个平等的竞争环境。这最终会造成扭曲和歧视。〞
 <br/>
 <br/>
 他表示，鉴于中国经济的规模，中国市场的扭曲就意味着全球市场的扭曲。所以，中国国内产业政策影响的不仅仅是中国国内市场，而是超越了中国国境之外。
 <br/>
 <br/>
 事实上，川普政府近几个月来宣布的对中国进口产品加征关税的清单中，很大一部分都跟〝中国制造2025〞有关。
 <br/>
 <br/>
 今年4月，美国贸易代表办公室在一份公告中曾直言，对中国产品加征关税主要是为了回应中共为实现〝中国制造2025〞计划而对美国企业采取的〝胁迫政策〞。川普也曾于今年11月初在一次记者会上说，中国制造2025 〝极具威胁性〞。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：范铭）
