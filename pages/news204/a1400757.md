### 川普联络盟友围剿华为 美特别行动重挫中国股市
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月26日讯】美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府已将美中关系定位为长期战略竞争，虽然川习会在即，依然按部就班地继续反击中共威胁。美媒日前曝光，美国正联合盟友共同抵制有中共军方背景的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 公司，以保障西方安全。消息传出后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 大跌。
 <br/>
 <br/>
 美国《华尔街日报》11月22日援引知情人士消息称，美国政府已向盟国发起一项特别行动，以说服这些国家的无线及互联网提供商抵制
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_华为.html" target="_blank">
  华为
 </a>
 的电信设备。
 <br/>
 <br/>
 报导说，华为因为海外间谍行为，早已被美国列为国家安全威胁。但是在一些拥有美国军事基地的国家，比如德国、意大利和日本，还在广泛使用华为设备。这也成为美国担心的一个主要问题，因为除了特别敏感的通信，美国很多军事设施里大部分还是使用商用网络。
 <br/>
 <br/>
 针对这些盟友，美国官员已将有关使用华为和中兴产品存在安全风险的报告，分发给这些国家的官员和电信高管。美国政府还考虑，将为避开使用中国设备的国家增加相关的财务援助。
 <br/>
 <br/>
 报导引述一些官员表示，这一举措是美中之间技术冷战的一部分。美方担心一些独裁政府的科技巨头崛起，会利用网络不法行为威胁西方安全。
 <br/>
 <br/>
 美方启动抵制华为的行动，正值全球互联网提供商准备购买5G移动技术的新硬件之际。而华为是5G产品主要提供商之一。了解行动的人士表示，美国政府认为，数年后移动网络和互联网将更容易成为网络攻击或间谍活动的目标。
 <br/>
 <br/>
 《华尔街日报》报导发出次日（11月23日），
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国股市.html" target="_blank">
  中国股市
 </a>
 三大指数均低开低走，收市集体重挫。上证综合指数收报2579.48点， 下跌65.95点，跌幅2.49%；深成指数报7636.71点，重挫3.26%；创业板指数则大跌3.34%。
 <br/>
 <br/>
 《美国之音》11月24日报导，美国政府敦促盟国抵制华为设备，以及拟实施严厉技术出口管制的消息，重挫了中国股市。
 <br/>
 <br/>
 华为被指有中共军方背景，背后有中共政府强大财政支持。近年来，华为出口电信设备的后门被广泛揭露，该公司窃取外国技术和情报的间谍行为，越来越引起西方政府的警惕。
 <br/>
 <br/>
 《金融时报》11月22日报导称，北京当局大力推动所谓〝军民融合〞战略，是西方政府噩梦的源头，也是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 政府对中共采取对抗性措施的动机之一。
 <br/>
 <br/>
 近期以来，美国政府推出各种限制措施，以防止美国先进技术经过民用渠道流入中国，也凸显了对中共军方通过〝民营企业〞盗取美国技术的担心。
 <br/>
 <br/>
 《华尔街日报》指，德国政府获得美方有关华为威胁的通报后，已经显示出对华为的新防范情绪。而日本官员早在8月曾表示，正研究对华为进行限制。
 <br/>
 <br/>
 此外，澳大利亚政府8月就已禁止华为和中兴参与该国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_5G网络.html" target="_blank">
  5G网络
 </a>
 。10月，英国政府表示，正在评估该国电信设备市场的组成，被指明显针对华为。
 <br/>
 <br/>
 不过，意大利一家主要电信运营商的董事表示，已了解相关风险，但仍计划继续使用华为设备。意大利政府和北京来往密切，日前表示将加入中共〝一带一路〞。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：云涛）
</div>
