### 杭州劫匪打劫文弱〝书生〞反被散打高手暴打
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月07日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_浙江杭州东站.html" target="_blank">
  浙江杭州东站
 </a>
 日前发生一起离奇抢劫事件。据网传视频显示，一名男劫匪试图打劫一位体型瘦小的文弱〝书生〞，不料该书生是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_散打高手.html" target="_blank">
  散打高手
 </a>
 ，劫匪反被书生打得鼻青脸肿，当场被制服。随后劫匪被拘捕。
 <br/>
 <br/>
 11月4日，在杭州东站的候车室内，25岁的湖南籍男子向某，四处徘徊，寻找〝猎物〞下手打劫。经过一番观察，向某锁定了一名体型瘦小，穿浅色外套，看似文弱〝书生〞的男子肖先生，当时肖先生站在28B检票口尾端，准备进站。
 <br/>
 <br/>
 据网传监控视频画面显示，向某走到肖先生身旁，拍拍他的肩膀，然后从容地从他手中拿走皮包和袋子。
 <br/>
 <br/>
 〝我以为是朋友来送站，跟我开玩笑的〞，肖先生向陆媒说，他一开始有些懵，愣了几秒后才发现自己并不认识对方。他看到向某拿着他的包就要转身离开，连忙上前询问。
 <br/>
 <br/>
 这时，向某又看到了肖先生胸前的皮包，并伸手抢夺。肖先生这才意识到，自己正在被打劫。两人于是扭打到一起。
 <br/>
 <br/>
 穿黑色上衣的向某对肖先生先踹了一脚，肖先生立即回踹他一脚，并打出一套组合拳，向某明显不是对手。
 <br/>
 <br/>
 结果向某抢劫不成，反被这位看似文弱的〝书生〞打得鼻青脸肿。肖先生三两下将劫匪制服。随后，向某被拘捕。
 <br/>
 <br/>
 事后肖先生表示，自己是一名医生，也是业余散打爱好者，已学了三四年拳击。向某没想到，自己〝精挑细选〞的对象，居然是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_散打高手.html" target="_blank">
  散打高手
 </a>
 ，连连喊后悔。目前，向某因涉嫌抢夺罪已被刑事拘留，案件在进一步审理当中。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
