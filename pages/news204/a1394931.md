### 李克强出访不利 尴尬事提前到来
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】中共国务院总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 11日乘专机离开北京，赴亚欧三国进行9天的访问活动。在李克强出访之前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比利时.html" target="_blank">
  比利时
 </a>
 将一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共特工.html" target="_blank">
  中共特工
 </a>
 交给了美国政府，墨西哥前大使称，这件事让李克强尴尬。
 <br/>
 <br/>
 中共政府网报导，国务院总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李克强.html" target="_blank">
  李克强
 </a>
 ，11日乘专机从北京启程前往塔吉克，展开9日的亚欧三国之行。李克强首先会在塔吉克，出席上海合作组织成员国政府首脑(总理)理事会第十七次会议，其后正式访问塔吉克。
 <br/>
 <br/>
 之后李克强会转到荷兰，他亚欧之行的最后一站，会到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_比利时.html" target="_blank">
  比利时
 </a>
 ，在首都布鲁塞尔出席第十二届亚欧首脑会议，同时对比利时进行工作访问。
 <br/>
 <br/>
 报导称，这是李克强在3月续任总理后，首次访问塔吉克斯坦和荷兰，也是中共总理相隔14年后再次访问荷兰。是中共面向亚欧地区的〝一次重大外交行动〞。
 <br/>
 <br/>
 墨西哥前驻华大使瓜哈尔多(Jorge Guajardo），10月10日在推特上转发评论说，比利时将一名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共特工.html" target="_blank">
  中共特工
 </a>
 交给美国政府，李克强即将对比利时进行工作访问。这真是尴尬！
 <br/>
 <br/>
 10月10，美国司法部宣布，在比利时逮捕了一名中共国家安全部情报人员，9日该特工已经被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_引渡.html" target="_blank">
  引渡
 </a>
 到美国，这是美国政府首次将中共情报人员引渡到美国受审。
 <br/>
 <br/>
 〝华盛顿邮报〞报导说，这是首次有中共特工，遭美国引诱至第三国后遭逮捕，并引渡抵达美国进行司法审判。
 <br/>
 <br/>
 美国司法部宣布，以涉嫌经济间谍的罪名，起诉中共特工
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐延军.html" target="_blank">
  徐延军
 </a>
 。司法部官员称，这其实是中共窃取商业秘密计划的一部分。
 <br/>
 <br/>
 美国司法部负责国家安全事务的助理部长戴莫斯（John Demers）表示，这不是一个单一的个案，而是中共整体经济发展政策的一环，却是要美国付出代价，美国〝不能容忍有国家以窃取美国智慧财产的方式，不劳而获。〞
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
</div>
