### 中共内斗新动向？北京惊现大字报为文革翻案
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】日前网传北京城街头出现捧毛批邓的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大字报.html" target="_blank">
  大字报
 </a>
 ，公然为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文革.html" target="_blank">
  文革
 </a>
 招魂，引起外界各种猜测。此事出现有两大背景：一个是当局政策不断左转，一个是中共内斗复杂化。
 <br/>
 <br/>
 17日，海外Youtube网站登出据称在北京拍摄的一段视频，展示了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京街头.html" target="_blank">
  北京街头
 </a>
 悬挂张贴的一批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大字报.html" target="_blank">
  大字报
 </a>
 。
 <br/>
 <br/>
 视频显示，数十张大字报用长绳整齐地悬挂在街边的树干之间，绵延数十米，街上行人纷纷驻足观看。而大字报的内容，则是为毛泽东和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_文革.html" target="_blank">
  文革
 </a>
 翻案，叫嚣〝让美国低头〞，以及对邓小平之后北京政策的批评等等。
 <br/>
 <br/>
 大字报具有典型的〝毛左〞特征，视频中的老年女解说者的语调和口吻，也被指带有〝文革遗风〞。
 <br/>
 <br/>
 视频持续超过4分钟，中间并没有警察和官方人员出现和干预。在〝朝阳群众〞遍地、监控严密的北京街头出现如此大规模的大字报，显得比较异常。
 <br/>
 <br/>
 有海外分析人士指，中共的政治经济都陷入危局，北京可能通过第二次〝文革〞加强社会控制，以图度过危机，然后再继续玩〝斗争加平反〞的〝左右翻转〞游戏。
 <br/>
 <br/>
 《美国之音》近日曾引述分析称，中国经济早在中美贸易战之前就已出现大问题，中共内部其实早就做好了〝最坏的打算〞，那就是在走投无路的情况下，将重返〝计划经济〞的极左路线，以便让中共在经济危局中得以继续执政。
 <br/>
 <br/>
 实际上，薄熙来落马前，就已经在重庆〝唱红打黑〞，抢劫私企财产，并得到高层相当一部分人肯定。北京近年来在政治经济上也不断左转，趋于加强极权管制。
 <br/>
 <br/>
 而另外一方面，也有分析认为，北京出现〝毛左〞的大字报，也和中共内部权力斗争有关。
 <br/>
 <br/>
 中国的毛派势力，当年曾与在重庆〝唱红打黑〞的薄熙来关系密切。今年以来，毛派因为高调参与和炒作深圳佳士的工运事件，屡遭当局打击压制。北京大学和其它高校的左派学生和组织成为重点打击对象。
 <br/>
 <br/>
 此外，因为北京当局坚持〝前后三十年不能相互否定〞，因此前江派常委刘云山掌控的宣传系统，得以使用将习近平和毛泽东并列的手法打击习近平威信。中宣系统这种对习近平〝高级黑〞的〝捧杀〞一直持续至今，习近平要当〝毛泽东第二〞一直是海内外一些中文媒体宣传的主调。
 <br/>
 <br/>
 近日，在中共政法委主持下，高调纪念毛泽东和习近平批示〝枫桥经验〞的会议在浙江召开，也成为一个舆论焦点。政法系统此前也曾纪念毛泽东对〝枫桥经验〞的批示，但将毛和习并列还是第一次。
 <br/>
 <br/>
 值得注意的是，现任政法委书记郭声琨在公安部任职多年，被指是江派大佬曾庆红的亲戚。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：云涛）
