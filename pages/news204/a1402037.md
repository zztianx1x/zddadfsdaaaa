### 习近平安保升级  贴身保镖换人
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】中国国家主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 自上台以来，安保持续升级。习自11月下旬外访，至今仍未回国，中共央视的电视画面透露出一个秘密，目前正在葡萄牙进行国事访问的习近平，换上了新的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贴身保镖.html" target="_blank">
  贴身保镖
 </a>
 。
 <br/>
 <br/>
 12月5日，香港《明报》称，中共央视的电视画面透露出一个秘密：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 换了贴身侍卫。习原来的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贴身保镖.html" target="_blank">
  贴身保镖
 </a>
 是从他做国家副主席时跟过来的，是一个瘦高个的男子。
 <br/>
 <br/>
 报导说，有一年的新年团拜会上，央视镜头拍到时任中共军委委员常万全，当众向该保镖敬礼，吓得当时军衔仅为少校（营长）的他忙鞠躬回礼。
 <br/>
 <br/>
 〝从今年起，这名军衔已至中校（副团长）的侍卫就从习的身边消失了，取而代之的是一个身材健硕的年轻军官，从今次出访画面看，他的军衔是上校（副师长）〞报导称。
 <br/>
 <br/>
 不管是习近平在国内的各种调研活动，还是在他的外访活动中，这个贴身保镖都是出镜率最高的人之一，因为他总是跟随在习近平左右，距离不出三步。
 <br/>
 <br/>
 至于习近平为何更换了保镖，外界并不清楚。而公众对习近平保镖的关注，多半源自对中共内部明争暗斗等秘辛的兴趣。
 <br/>
 <br/>
 自中共十八大习近平上台以来，有关他遭
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_暗杀.html" target="_blank">
  暗杀
 </a>
 的消息频频流出。曾有观察认为，习近平上台后安保严密程度越来越厉害，似乎对应着其防暗杀的心机和中共政局暗涌不断。
 <br/>
 <br/>
 已经停刊的香港《动向》杂2016年曾披露，习近平的保卫人员挑选，有着严格规定，随行的保卫人员人数，一组由原来的8人增加到12人；近身保卫人员人数，一组由原来的6人增加到8人。
 <br/>
 <br/>
 随行和近身保卫人员，除了要求政治上绝对忠诚，亲属政治上可靠，自己朋友清白之外，其本人还需在总参保卫部经6年以上政治审核和专业考核。
 <br/>
 <br/>
 在技击技能方面，相关保卫人员必须能熟练操作8种以上武器，双手能同时开枪击中目标。随行保卫人员通常携带3枝不同型手枪，2把匕首，1包烟雾粉。
 <br/>
 <br/>
 手枪放在西服口袋或右袖内，一枝枪放在裤袋或腰间皮带，一枝枪所穿鞋子内或裤脚内。
 <br/>
 <br/>
 同年的中共中央政治局本年度第25次会议上，习近平表示，他从上任一刻，就在政治局常委会上备了案：我如发生不可预测的情况遇害，天也不会塌下来，祖国前进的步伐不会停止。
 <br/>
 <br/>
 报导称，习近平所指的政治局常委会上备了案，指的是中共十八大一中全会后的当天，新一届中共政治局常委和十七届中共政治局常委联席会。有分析说，这反映出习近平对驾驭和治理中共政治全局，以及面临的困局，是有备而来。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：李泉）
</div>
