### 邓小平的女婿吴建常去世 传聚敛巨额财富
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】已故中共元老邓小平的女婿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴建常.html" target="_blank">
  吴建常
 </a>
 于19日在北京协和医院去世，终年79岁。吴建常妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓林.html" target="_blank">
  邓林
 </a>
 主张丧事从简，不举行告别仪式，吴建常的骨灰将会海葬。
 <br/>
 <br/>
 11月20日，中国冶金报社微信公众号发布消息称，11月19日上午10时40分，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴建常.html" target="_blank">
  吴建常
 </a>
 因病医治无效，在北京协和医院去世。
 <br/>
 <br/>
 消息称，吴建常的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓林.html" target="_blank">
  邓林
 </a>
 是邓小平的长女，邓林主张丧事一切从简，不通知、不发讣告。吴建常遗体于20日下午送北京八宝山火化，骨灰将会海葬。
 <br/>
 <br/>
 公开资料显示，吴建常1939年6月出生，1964年毕业于湖南衡阳矿冶工程学院（现南华大学）有色冶金专业，长期从事有色金属科技情报研究工作。历任北京有色金属研究总院工程师、副处长。中国有色金属工业总公司副总经理、总经理。冶金工业部副部长。国家冶金工业局副局长。中国钢铁工业协会党委书记、副会长、名誉会长。中国有色金属工业协会名誉会长。
 <br/>
 <br/>
 吴建常还是首批前往香港的太子党之一。1992年，他率领有色总公司在香港成立中国有色（香港），成为当时香港媒体追访对象。2012年6月，中国有色矿业集团旗下的中国有色矿业有限公司（港交所：1258）在香港上市。
 <br/>
 <br/>
 同年12月，美国彭博社发表的《中共八元老后代，聚敛巨额财富》长篇报导中，特意提到了邓家女婿吴建常。
 <br/>
 <br/>
 报导披露，吴建常的公司和邓小平的另一位女婿张宏拥有的公司，合伙从美国通用汽车公司（GM）手中购买了稀土磁矿材料主要制造商麦格昆磁。此后，邓小平家族掌控了稀土矿市场，现在这种材料应用在美国灵巧炸弹、风涡轮机和混动汽车中。
 <br/>
 <br/>
 报导指，吴建常控制了中国稀有金属的买卖权，在香港拥有三家上市公司，即东方有色集团、东方鑫源集团、银建国际，吴建常是这三家上市公司的董事会主席。
 <br/>
 <br/>
 此外，他在香港还拥有多家上市公司的股权，如金辉集团、东方鑫源、百利大等，总资产过百亿港元。同时，他也是在奥斯陆上市的金辉船务运输的荣誉董事长和江西铜业的董事会成员。
 <br/>
 <br/>
 (记者罗婷婷报导/责任编辑：李泉)
