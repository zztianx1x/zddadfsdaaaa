### 大陆房价暴跌 任志强：5类人会遭殃！
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】〝如果房价真的暴跌了，有5类人必定会跟着遭殃！〞日前，大陆地产名人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任志强.html" target="_blank">
  任志强
 </a>
 在公开场合，再次发出惊人言论。种种迹象显示，大陆房市已进入多事之秋，多地出现楼盘暴跌现象，地产大佬们断言房市已面临拐点。
 <br/>
 <br/>
 随着
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 开打，不少经济学者预测，中国大陆的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房市泡沫.html" target="_blank">
  房市泡沫
 </a>
 面临被刺破的危机。日前，前华远地产集团董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_任志强.html" target="_blank">
  任志强
 </a>
 公开表示，如果大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_房价暴跌.html" target="_blank">
  房价暴跌
 </a>
 的话，会伤及5类人。
 <br/>
 <br/>
 <h4>
  一．背负房贷的房奴
 </h4>
 <br/>
 近年来，大陆房子卖出天价，买房的民众花费毕生积蓄，贷款买房。一旦房价暴跌，房奴们的血汗钱也就白白蒸发了。与此同时，面对高昂的房贷，是继续还贷还是弃房断供，无论怎么选，似乎都是错误的决定。
 <br/>
 <br/>
 <h4>
  二．背负巨债的开发商
 </h4>
 <br/>
 在大陆，楼盘开发耗费成本相当之高，开发商高价拿地，一般都背负着巨额的贷款，只有房子都按预期卖出去，才能保证资金正常周转。如果房价暴跌，开发商不仅收入骤减，甚至能让他们直接破产。
 <br/>
 <br/>
 <h4>
  三．还在犹豫买房的人
 </h4>
 <br/>
 如果房价真的暴跌，对没买房的人来说，买房反而更加遥遥无期了。对于房价还会下降多少？什么时候才是买房的好时机？现在出手到底靠不靠谱，会不会有风险等问题，会使还在犹豫买不买房的人，更加纠结。
 <br/>
 <br/>
 <h4>
  四．银行可能会降薪或裁员
 </h4>
 <br/>
 房贷业务是大陆银行的主要收益来源。若房价暴跌，一些人还不上贷款将形成大量的银行坏帐。没房的人也不会再申请房贷了。因此，银行前期可能会降薪裁员，后期也就只能宣布倒闭。
 <br/>
 <br/>
 <h4>
  五．普通老百姓
 </h4>
 <br/>
 房地产行业作为大陆的支柱产业，一旦出现巨大震荡，势必会冲击各行各业，如此一来，普通老百姓的生活也将变得十分困难。
 <br/>
 <br/>
 <h4>
  房地产业进入寒冬
 </h4>
 <br/>
 对于大陆房市充满危机感的不只是任志强，最近大陆地产业领头羊之一的万科公司董事会主席郁亮，也发出惊人言论，称万科未来以〝活下去〞为最终目标。暗示房地产业将进入寒冬，悲观情绪正在地产界蔓延。
 <br/>
 <br/>
 最近，环北京的燕郊楼市暴跌，从去年的每平米3万元人民币，跌至现在的1万5千元左右。万科、恒大、碧桂园、泰禾、中粮等地产商，也都开始大肆降价促销，脱手楼盘。
 <br/>
 <br/>
 万科在厦门万科白鹭郡别墅推出107套特价别墅，原价500万人民币，现中间套一口价278万元，边套一口价298万元。
 <br/>
 <br/>
 有大陆业界人士认为，从楼价下跌近半，以及大陆近来的其它资产状况来看，大陆资产泡沫即将破灭。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：李泉）
