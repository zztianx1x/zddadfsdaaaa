### 反共国会+强硬白宫 中共面临双重烦恼
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】美国中期选举落幕，民主党重新夺得了众议院的多数席位，美国政坛格局也随之将发生改变。不过，舆论认为，美国当下的新格局对中共来说并非福音，反而是双重的烦恼，因为重掌众议院控制权的民主党政要大多是比共和党更反共的强硬角色，北京当局接下来将面对的中美关系将比前两年更加复杂艰难。
 <br/>
 <br/>
 美国民主党重新夺回对众议院的控制权后，外界普遍预测，现年78岁的民主党员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_佩洛西.html" target="_blank">
  佩洛西
 </a>
 （Nancy Pelosi）将再次担任众议院议长。
 <br/>
 <br/>
 令北京沮丧的是，佩洛西也是一位对中共态度十分强硬的政治人物，她长年密切关注中国人权问题，并毫不客气地炮轰中共侵犯人权的种种行为。
 <br/>
 <br/>
 1989年的〝六四屠杀〞事件发生后，佩洛西曾积极推动美国政府通过为中国流亡学生提供庇护的〝中国学生保护法案〞，2008年还曾公开呼吁西方国家抵制北京奥运，近期又多次针对中共在新疆建变相拘留营以及西藏人权问题，要求美国政府向中共施压。
 <br/>
 <br/>
 此外，共和党中的对华强硬派候选人，也纷纷在这一次的国会改选中胜出。他们当中最具代表性的人物，就是曾呼吁川普总统封杀华为和中兴的参议员鲁比奥（Macro Rubio）与克鲁兹（Ted Cruz）。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/11-10/p9119201a94481638.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/11-10/p9119201a94481638-ss.jpg"/>
   <br/>
   <font size="-1">
    港媒称，美国2018年中期选举后形成的政治新格局是：反共的国会加强硬的白宫。(Stefan Zaklin/Getty Images)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 针对上述情况，《香港经济日报》发文分析称，美国在中期选举后形成的政治新格局是：一个反共的国会再加上一个强硬的白宫，而这是北京当局〝绝对不欲看到的〞一幕。
 <br/>
 <br/>
 文章推测，今后，在佩洛西的主导下〝肯定不会出现一个对中国（中共）‘温柔’的众院〞，而且可以预料，〝反华（共）议程〞将在众议院〝畅通无阻〞。再考虑到参议院中胜出的共和党议员大多都是鹰派角色，今后对华强硬派比比皆是的美国国会，恐怕将变成中美关系的〝火药桶〞。
 <br/>
 <br/>
 文章进一步分析称，这次的中期选举结果说明，川普对中共的强硬路线得到了美国民意的支持，因此他的鹰派顾问团队不会有大幅调整，其遏制中共的外交理念将延续。
 <br/>
 <br/>
 文章援引美国圣路易斯华盛顿大学东亚系副教授马钊的观点表示，现在的白宫高级顾问团队中，〝反精英〞、〝反建制〞的政界新人正在成为美国的〝权力中枢〞，如鹰派的国家安全顾问波顿（John Bolton）、白宫贸易顾问纳瓦罗（Peter Navarro）等人，他们将影响着美国政府的决策。这些人对中共都持非常强硬的态度，而且〝出招无常〞，这将给中美关系〝增添巨大变数〞。
 <br/>
 <br/>
 文章最后得出的结论是，在这种格局下，北京未来两年要面对的中美关系，〝势比过去两年更复杂。〞
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：明轩）
