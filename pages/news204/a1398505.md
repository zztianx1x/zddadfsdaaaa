### 又一习家军〝黑马〞晋升公安副部长
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】中共国务院日前公布新一波人事任免名单，刚调任中共公安部4个月的〝习家军〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林锐.html" target="_blank">
  林锐
 </a>
 被任命为副部长。一个月前，时任公安部副部长的孟宏伟被查，接替他的同样是被视为〝习家军〞的中共纪检监察组组长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓卫平.html" target="_blank">
  邓卫平
 </a>
 。
 <br/>
 <br/>
 中共人社部11月7日公布国务院任免国家工作人员信息。任命中共公安部党委委员、部长助理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_林锐.html" target="_blank">
  林锐
 </a>
 为公安部副部长。
 <br/>
 <br/>
 这距林锐从福建省厦门市副市长、公安局长，被调任公安部党委委员、部长助理仅有4个多月。
 <br/>
 <br/>
 中共的公安资料显示：林锐，1967年8月生，福建闽侯人。2011年8月起中共任泉州市政府副市长、市公安局局长。2013年10月起任中共厦门市政府市长助理、市公安局局长。
 <br/>
 <br/>
 现年51岁的林锐，被指是习近平的人马。他曾长期在福建公安系统任职，而习近平曾在福建任职18年之久。
 <br/>
 <br/>
 林锐也是公安部常务副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王小洪.html" target="_blank">
  王小洪
 </a>
 的福建公安系统旧部。林锐在福建省公安厅任职期间，王小洪任福建省公安厅副厅长。王小洪2013年8月从中共厦门市副市长、市公安局局长调任中共河南省公安厅厅长后，林锐同年10月调任厦门接替王小洪的职务。
 <br/>
 <br/>
 林锐此次任命为中共公安部副部长的一个月前，时任中共公安部副部长、国际刑警组织主席孟宏伟被宣布调查。孟曾是公安部三号人物，地位仅次于王小洪，接替他的是中纪委国家监委驻公安部的纪检监察组组长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓卫平.html" target="_blank">
  邓卫平
 </a>
 。
 <br/>
 <br/>
 邓卫平生于1955年，江苏人，也属于〝习家军〞。1983年夏至1995年11月，邓卫平历任中共福州鼓楼区委副书记、郊区区委副书记、政法委书记、晋安区委书记。习近平在1990年4月至1996年4月担任中共福州市委书记，是他的老上级。
 <br/>
 <br/>
 邓卫平1998年9月进入中共安全部门，历练十余年后进入中共纪检部门，2011年9月，任中共福建省纪委副书记。2015年进京转任中共公安部纪委书记兼任公安部督察长。2016年1月，改任中纪委驻公安部纪检组组长。一个月前晋升中共公安副部长。
 <br/>
 <br/>
 中共公安部此前一直被江派把持，从1998年3月至2017年10月，公安部部长先后由江派大员贾春旺、周永康、孟建柱、郭声琨担任。
 <br/>
 <br/>
 中共十八大后，习近平当局已多次清洗公安部。周永康2014年落马后，他的多名公安部旧部也落马，除孟宏伟外，还有610办公室主任、公安部副部长李东生，常务副部长杨焕宁，政治部主任夏崇源等人。
 <br/>
 <br/>
 2018年两会后，中共公安部高层再次被调整，周永康的前旧部公安部排名二、三的傅政华、黄明被调离，排名第四的王小洪跃升为排名第二的常务副部长，权力仅次于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_赵克志.html" target="_blank">
  赵克志
 </a>
 。
 <br/>
 <br/>
 截止目前中共公安部已有6名高层被视为〝习家军〞，包括：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部长.html" target="_blank">
  公安部长
 </a>
 赵克志，常务副部长王小洪，公安部纪检组组长邓卫平，副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟庆丰.html" target="_blank">
  孟庆丰
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_许甘露.html" target="_blank">
  许甘露
 </a>
 和林锐。
 <br/>
 <br/>
 新唐人记者查询发现，林锐担任中共福建省厦门市公安局局长期间，因参与迫害法轮功，曾上《追查迫害法轮功国际组织》黑名单。至于其他人是否上过黑名单，目前还没有细查。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：戴明）
