### 中共贪官美国豪宅曝光 揭二奶村神秘面纱（视频）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月08日讯】被网友昵称为〝江苏小哥〞的江苏网友刘某，日前在网路上发布一段视频，曝光中共〝贪官〞在美国的豪宅。美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贪官城.html" target="_blank">
  贪官城
 </a>
 、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二奶村.html" target="_blank">
  二奶村
 </a>
 的神秘面纱被揭。
 <br/>
 <br/>
 刘某网名〝Vlog心声〞，近日因在网路上批评中共政治而爆红。日前，刘某被中共党媒总编辑胡锡进在微博点名批判，更传出他在江苏的老家，遭中共公安上门威胁。
 <br/>
 <br/>
 不过，刘某无惧压力，不但在网上强硬回应胡锡进，称〝这个世界上还有几个要翻墙看YouTube的国家〞？
 <br/>
 <br/>
 11月6日，刘某又发布一段在美国游玩的视频，指视频中的豪宅都是中共〝贪官〞、〝二奶〞、〝小三〞的住所。更有网友在留言处称，影片中〝0分22秒那栋是广西桂林前副市长仇祖和的豪宅〞。
 <br/>
 <br/>
 不少网友佩服江苏小哥的骨气，不过也有不少人担心，江苏小哥这样公开揭露中共丑闻，恐会遭到报复。
 <br/>
 <center>
  <br/>
 </center>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贪官城.html" target="_blank">
   贪官城
  </a>
 </h4>
 <br/>
 事实上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共贪官.html" target="_blank">
  中共贪官
 </a>
 在美国拥有豪宅，曾在媒体上被多次曝光。美国到处藏着携带巨款潜逃出国的中共贪官，他们一到美国就要买豪宅以作安身。在美国各华人聚居的城市，都不难见到中共贪官和富人的二奶。
 <br/>
 <br/>
 大洛杉矶地区的亚凯迪亚市(Arcadia)备受中共贪官青睐，这里有〝贪官城〞之称。亚凯迪亚是一座小城，占地28.9平方公里，人口不到6万。坐落在圣盖博山脚下，整座城市绿树成荫、环境清雅、豪宅成片，豪宅之间拉开距离，豪宅与马路都隔着前院，隐蔽性良好。
 <br/>
 <br/>
 亚凯迪亚居民原本95%以上为白人，但近十多年来，大量大陆贪官入住，如今居民中的亚裔人口已超过白人。
 <br/>
 <br/>
 据外媒报导，落马的前中共中央办公厅主任令计划的弟弟令完成潜逃美国后，就住在亚凯迪亚的一幢豪宅中。美国NBA篮球明星尤德瑞将其在亚凯迪亚的豪宅卖了，买主就是令完成。
 <br/>
 <br/>
 据说令完成带走了大批中共高层权斗和贪腐的秘密文件，亚凯迪亚随时会传出爆炸性新闻，这更使得亚凯迪亚声名远播、万人瞩目。
 <br/>
 <br/>
 亚凯迪亚近几年吸引着来自加州、外州、远至中国大陆的游客到这里一看究竟。特别是传说令完成入住此城，游客更是络绎不绝。据说在豪宅区兜一圈，总能碰上几位外逃的中共贪官，还有人说曾经在此地碰到过令完成。
 <br/>
 <br/>
 <h4>
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_二奶村.html" target="_blank">
   二奶村
  </a>
 </h4>
 <br/>
 美国加州大洛杉矶地区一个名叫罗兰岗(Rowland Heights)的地方，不仅华人无人不晓，在美国主流社会也闻名遐迩。这里是中共贪官或富豪二奶聚集的地方，英文媒体称这里是情妇村。
 <br/>
 <br/>
 有报导称，在洛杉矶国际机场中国航班的出口接机处，每天都可以看到有华人少妇们在焦急等待航班降落。每当一位老男人走出来，便有一位少妇上前拥抱老男人，然后挽着老男人的胳膊，登上奔驰或宝马汽车，直往罗兰岗驶去。
 <br/>
 <br/>
 老男人们都是中共权贵富豪，至少是厅局长以上的官员，或是身家至少十亿八亿的富豪。他们之所以把二奶送出国，一是因为家里的大奶闹得不得安宁，二是把资金转移出国外，需要有个照看资金的人。
 <br/>
 <br/>
 于是这些权贵富豪们就在国外买一幢豪宅，把二奶送出国居住。便有了罗兰岗这样的〝二奶村〞。权贵富豪们不能常来〝二奶村〞，通常每年来一次，住十天半个月，就得回国。剩下的漫长日子，就由住在豪宅里的二奶独自打发了。
 <br/>
 <br/>
 近几年，罗兰岗的〝二奶村〞逐渐衰落，原因是〝二奶村〞名声在外，几乎成了旅游景点，逼使这些权贵富豪不得不另找地方。
 <br/>
 <br/>
 (记者罗婷婷综合报导/责任编辑：文慧)
