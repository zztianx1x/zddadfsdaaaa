### 刘强东现身被问强奸案 京东高管帮忙解围
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月20日讯】中国互联网购物平台京东商城创始人及首席执行长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 中国女留学生一案，被媒体曝光大量不堪细节后，刘强东始终未发声。19日，刘强东首次发声对京东管理团队进行评价，但对强奸事件避而不提。相关问题由京东高管回应。
 <br/>
 <br/>
 陆媒称，11月19日，在京东发布第三季度财报后的电话会议上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 在回答〝京东新的管理团队建成以后，你的角色会发生怎样的转变？〞时称：自己正面临四件事：〝战略、文化、团队和新业务〞。
 <br/>
 <br/>
 被问及在美国涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 中国女留学生事件时，刘强东并未回应。而是由京东高管回应。该高管表示称：〝我们确实没有可以告诉你的新信息，而且我们不能做任何评价，我们要尊重美国司法系统，这是很重要的。就像你看到的，刘强东已经回公司正常工作，并且这件事对公司运营并未产生影响。〞
 <br/>
 <br/>
 虽然京东高管称，刘强东性侵丑闻对京东运营未产生影响。但自刘强东性侵丑闻发生后，京东集团的股价也急转直下，持股机构数量暴跌。2018年第二季度的持股机构达585家，而第三季度仅剩100多家。
 <br/>
 <br/>
 刘强东性侵丑闻事发后的60多天里，刘几乎缺席所有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重磅会议.html" target="_blank">
  重磅会议
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重要场合.html" target="_blank">
  重要场合
 </a>
 。特别是在11月初，习近平主持召开民营企业座谈会，安抚所有民营企业和民营企业家，并称他们都是〝自己人〞。而刘强东已被踢出习近平的〝自己人〞圈子。
 <br/>
 <br/>
 刘强东在美国涉嫌强奸被捕丑闻，发生于8月31日凌晨1点左右，刘当晚被捕入狱，16个小时后获释。警方对刘强东一案的描述是〝一级性犯罪：强奸既遂〞。根据当地法律，刘强东所涉的一级强暴罪被定罪后，刑期至少12年起。
 <br/>
 <br/>
 美国警方9月21日已初步完成调查案件移交检方。美国检方称，这是一个〝棘手的案子〞案件，仍在审查中。
 <br/>
 <br/>
 有华裔刑事案件律师称，刘强东遭起诉的可能性超过90%，涉及重新逮捕等问题。
 <br/>
 <br/>
 11月2日，美媒披露刘强东案的具体性侵细节，报导称刘强东在性侵受害者时，遭到多次反抗，最后仍得逞，刘还告诉21岁的受害女大学生，可以像中共头号女间谍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓文迪.html" target="_blank">
  邓文迪
 </a>
 一样。
 <br/>
 <br/>
 11月13日，《纽约时报》发表万字长文报导刘强东强奸案。报导称，刘强东所参加的明尼苏达大学博士项目，专门为亚洲高管设计，从2017年开始为卡尔森商学院带来超过1,000万美元学费收入，每个学生的学费超过8.5万美元。涉事女生是该项目的志愿者。而刘强东是个白天在校上课，晚上去参加派对的富商。
 <br/>
 <br/>
 今年45岁的刘强东不仅是中国商界亿万富豪，同时也是中共全国政协委员，并兼任中国民间商会副会长。作为中共的红顶商人，在美国涉及强奸犯罪指控，无疑也是中共一大丑闻。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：范铭）
