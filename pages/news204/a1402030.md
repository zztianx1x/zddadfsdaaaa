### 姜春云退党了还是隐瞒身份？副国级移居海外引猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年12月05日讯】中共领导人退休后很少有机会能走出国门，但曾在六四事件中支持中共镇压爱国学生的中共前副总理
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姜春云.html" target="_blank">
  姜春云
 </a>
 ，却被爆与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫩妻.html" target="_blank">
  嫩妻
 </a>
 隐居美国。有分析认为，鉴于姜是中共领导人身份，如果确认其已经移民美国，那么他不是退党了就是隐瞒了身份。
 <br/>
 <br/>
 据港媒2日报导，已退休的中共领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_姜春云.html" target="_blank">
  姜春云
 </a>
 与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_嫩妻.html" target="_blank">
  嫩妻
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_桑潇.html" target="_blank">
  桑潇
 </a>
 已定居美国。桑潇最早在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_山东电视台.html" target="_blank">
  山东电视台
 </a>
 任职，后来到央视任职，因感情问题，渐渐淡出萤屏，后来嫁给了中共高官，没想到是姜春云。
 <br/>
 <br/>
 另有消息称，姜春云与他小20多岁的妻子桑潇是在美国注册结婚，双方均为绿卡身份永居美国。也有知情人士称，姜春云和桑潇早属旧闻，桑潇曾在采访姜春云时两人产生感情，在姜任职山东时就已结婚。姜春云调任中央后，桑潇也悄然消失。
 <br/>
 <br/>
 现年88岁高龄的姜春云是山东人，1989年〝六四事件〞中共在天门屠杀爱国学生期间，姜春云出任中共山东省委书记，他因支持中共屠杀爱国学生运动，于1992年晋升中共政治局委员，1995年任中共国务员副总理，1998年转任中共人大副委员长，2003年退休。
 <br/>
 <br/>
 由于中共领导人退休后鲜有机会走出国门，而在中共前党魁江泽民执政时期担任中共国务院副总理的姜春云移居美国引发舆论猜测。
 <br/>
 <br/>
 有媒体质疑，那么姜春云如何移民美国的？移民美国必须要放弃中共党员的身份，他难道在中国就放弃了吗？据了解，美国还需要提供放弃中共党员身份的证明才会批移民签证。
 <br/>
 <br/>
 根据《移民和国籍法》条款，任何共产党员或附属成员（或任何其它独裁政党成员）都不能入境美国。但也有一些情况可以豁免，包括非自愿加入共产党及其附属组织，或16岁之前加入，或法律要求这么做，或递交申请5年前已退出共产党及其附属组织。
 <br/>
 <br/>
 观察人士认为，鉴于姜春云的中共领导人身份，移民的机会不多，如果确认其已经移民美国，那么他不是退党了，就是隐瞒了中共领导人身份。
 <br/>
 <br/>
 自毛泽东时代，中共领导人退休后就鲜有机会走出国门。美国《外交政策》曾引述霍普金斯大学中国问题专家主任兰普顿（David Lampton）的分析说：在中国，中共前国家领导人几乎不出国。如果没有获得中共现政治局常委的允许，中共前政治局常委不允许出国，这是不成文的规定。
 <br/>
 <br/>
 新加坡国立大学东亚研究所资深研究员薄智跃说，前中共中央政治局委员根本没有个人护照，他们只有官方护照，在中共中央办公厅，因为〝这些人有许多的秘密，他们知道的信息可能对中共的形象不利。〞
 <br/>
 <br/>
 近20年来，由于积极主导并配合中共江泽民集团迫害法轮功等原因，如刘云山、李长春等之流的中共江派党官，都因被国际组织通告追查不敢出国。而罗干、薄熙来、贾庆林、吴官正、江泽民更是遭到全球通缉。
 <br/>
 <br/>
 《周永康与谷开来》一书披露，在几年前，习近平当局准备公开审理薄熙来后，周永康曾向习近平、王岐山申请出国养病，并保证按期回国。
 <br/>
 <br/>
 但王岐山说，此举〝恐怕多有不便〞，因为法轮功学员已在澳大利亚、加拿大、美国等多个国家起诉他，出国就可能面临被抓捕。
 <br/>
 <br/>
 他说，〝你知道西方国家实行司法独立，政府不能管法官判案。你一旦到了那些国家，又没有外交豁免，法院给你送一张传票你接还是不接？法庭去还是不去？这不是很大的麻烦吗？〞
 <br/>
 <br/>
 听到王岐山的一席话后，周永康张口结舌，无言以对，只得打消了出国的念头。但最终周在国内也无法逃脱被审判。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：戴明）
