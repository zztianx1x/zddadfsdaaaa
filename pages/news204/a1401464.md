### 美中讨论协议部分内容曝光 川普：谈不谈成无所谓
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月30日讯】中美双方迄今均无实质让步的迹象，外界对12月1日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 达成协议的期望值越来越低。美媒消息，中方还在为争取美方暂缓升级关税做最后的努力，包括提议恢复购买美国农产品等。但白宫的反应似乎越来越冷淡。
 <br/>
 <br/>
 《华尔街日报》周五（11月30日）引述消息称，美中正探讨能否仿照美方最近和欧盟和日本达成部分协议的模式，暂时形成一个框架性协议，亦即美方把针对中国产品进一步的关税措施暂时推迟到明年春季，而中方同意与美方展开对中共经济政策进行重大调整的谈判。
 <br/>
 <br/>
 为达成这个目的，中方还将同意放宽对采购美国农产品及能源产品的限制。
 <br/>
 <br/>
 报导说，中方和白宫的温和派也在为可能达成的协议做准备。北京可能12月中旬组团赴美谈判，而白宫温和派也在拟定要求北京在哪些方面做出妥协的清单。
 <br/>
 <br/>
 不过，报导没有提到中方承诺会对美方的哪些贸易要求做出实质妥协。而美方此前一再强调，达成协议的前提是中方必须答应美方的贸易要求，但中方一直在回避其中的关键问题，包括强制技术转移、政府补贴、关税和非关税壁垒等等。
 <br/>
 <br/>
 报导也提到，白宫的强硬派，包括贸易代表莱特希泽、国家安全顾问博尔顿以及白宫贸易顾问纳瓦罗等，都强烈反对这种协议，因为担心会因此落入中方的陷阱，陷入几乎不会取得任何成果的无休止谈判中。
 <br/>
 <br/>
 报导称，白宫内部就贸易问题出现分歧时，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 往往会支持强硬派。
 <br/>
 <br/>
 除了白宫强硬派的反对，川普还面临来自民主党的压力。美国众议院的民主党领袖佩洛西一直是中共痛恨的强硬反共人士，而参议院的民主党领袖舒默日前也公开表态将致信川普，希望他不要软化对中共的立场。
 <br/>
 <br/>
 在美国内部制约下，即使白宫有意暂缓对中共的压力，其妥协的空间也十分有限。
 <br/>
 <br/>
 上述《华尔街日报》报导称，在可能达成的协议中，美方只是考虑将进一步的关税措施推迟至明年春季。
 <br/>
 <br/>
 事实上，按照原计划，美国将在明年1月1日把针对2,000亿美元中国商品加征关税的税率从10%提高到25%。川普表示，这个原有计划〝极不可能〞发生改变。而美国还将考虑对剩余2,670亿美元的中国商品加征10%至25%的关税。
 <br/>
 <br/>
 近日川普一直表示对中美达成协议持开放态度，并称中方〝非常想达成协议〞。但是川普日前的表态似乎显示，他本人对达成协议并不抱多大期望。
 <br/>
 <br/>
 川普本周接受《华盛顿邮报》采访时说，他希望中方会向美方作出让步，但美方不一定非要达成协议。〝我认为我们要么达成协议，要么我们就对他们每月收取数十亿美元的关税，而且我对这两种情况的任意一种都无所谓。〞
 <br/>
 <br/>
 29日，川普在推文称，〝由于向中国征收的关税，数十亿美元涌入美国的金库，还有很长的路要走。如果公司不想支付关税，那就在美国建厂吧。让我们的国家比以往更富有！〞
 <br/>
 <br/>
 这显示，川普政府已经确定将关税战长期化的打算，即使中间可能会出现技术性暂缓。
 <br/>
 <br/>
 路透社日前消息，一度被传遭排除出局的白宫超级强硬派纳瓦罗，将确定出席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 。这进一步降低了川习会达成协议的希望，股市应声下跌。
 <br/>
 <br/>
 白宫经济顾问库德洛日前表示，虽然白宫将川习会晚宴视为令贸易战出现转机的机会，但是美方迄今对中方在贸易问题上的回应感到失望。
 <br/>
 <br/>
 库德洛透露，川习会并没有双方顾问当场举行会谈的安排。他说，〝这是一场类似派对的晚宴。〞
 <br/>
 <br/>
 而北京方面，似乎也没有认真准备川习会的迹象。11月中旬，美方表示了对中方让步清单的不满，当时有消息指中共国务院副总理刘鹤将亲自赴美谈判。但最后刘鹤在G20峰会前去了德国，并没有和美方直接接触。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：）
