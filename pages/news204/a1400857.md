### 中共教授首施基因编辑人体实验 全球舆论鼎沸斥疯狂
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】因为担心给人类制造毁灭性灾难，国际医学界一般会在涉及〝改造人体〞的技术面前克制止步。而不受任何道德底线约束的中共体制，往往在这些领域〝后来居上〞。日前世界第一对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 在中国出生，很快引发全球范围的谴责。
 <br/>
 <br/>
 本周一（11月26日），即第二届国际人类基因组编辑峰会开幕的前一天，到香港出席峰会的中国南方科技大学生物系副教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 宣布，一对名为露露和娜娜的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 已于11月在中国诞生。这对双胞胎的一个基因（CCR5）经过修改，使她们一出生就能〝天然抵抗感染艾滋病的可能性〞。
 <br/>
 <br/>
 贺建奎表示，为这项研究，他们团队在实验室里练习编辑老鼠、猴子和人类胚胎已经有好几年。
 <br/>
 <br/>
 贺建奎当天在接受美联社独家专访时宣布了该消息，很快引起全球媒体关注，并引发激烈的有关道德伦理的谴责。
 <br/>
 <br/>
 英国《卫报》当日报导称，基因编辑的科学技术，在大多数国家已被立法禁止，因为这一技术尚处在试验阶段，并且人类DNA作为遗传物质，一旦人为改变，将会遗传给后代，也存在伤害其他基因的风险。
 <br/>
 <br/>
 美国宾夕法尼亚大学任教基因工程专家Kiran Musunuru表示，基因编辑存在脱靶突变性，可能导致在出生和未来生活里发生未知的基因疾病。人类防治艾滋病已有其它有效方法，根本没有必要采用基因编辑方式，让原本健康的婴儿携带高风险。
 <br/>
 <br/>
 报导指，这项操作预期为同时编辑基因的两个副本。但最后这对双胞胎中只有一个成功改变了两个副本；而另一个只是改变了一个副本，仍然可能感染艾滋病毒。对后者来说，没有任何收获，却平添了所有未知的安全风险。
 <br/>
 <br/>
 根据中国临床试验注册中心可查阅的《深圳和美妇儿科医院
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_医学伦理.html" target="_blank">
  医学伦理
 </a>
 委员会审查申请书》（下称〝《伦理审查申请书》〞），这项名为〝CCR5基因编辑〞的科研项目始于2017年3月，项目历时2年，项目负责人为贺建奎。
 <br/>
 <br/>
 在消息曝光的26日下午，中国社交网络微博帐号〝知识分子〞就发表声明，集结122名海内外华人科学家的联合署名，指责上述所谓的〝伦理审查〞形同虚设，直接就基因编辑进行人体实验，只能用疯狂来形容，并表示对此〝坚决反对〞和〝强烈谴责〞。
 <br/>
 <br/>
 贺建奎所在的南方科技大学当天也在官网发布声明，撇清关系，声称贺建奎的研究是〝在校外开展〞，学校和生物系〝对此不知情〞。中共央视也发出疑似〝洗地〞报导称，上述深圳和美妇儿科医院医学伦理委员会这一机构〝未按要求进行备案〞，将展开〝调查〞。
 <br/>
 <br/>
 根据《伦理审查申请书》，开展〝CCR5基因编辑〞项目的目的是：建立基因手术治疗行业的〝质量控制标准〞，占领整个基因编辑相关治疗技术门槛的〝制高点〞，在国际竞争中〝脱颖而出〞。
 <br/>
 <br/>
 该目标和中共一贯争夺〝世界第一〞的作风相合。北京近年来推出的所谓〝中国制造2025〞和〝中国标准2035〞计划，被指企图在全世界夺取尖端科技领域的主导权，并由中共来制定行业标准。
 <br/>
 <br/>
 伴随着中美贸易战，〝中国制造2025〞涉及的各种无数国际规则的操作，包括巨额政府补贴、盗窃知识产权和强制技术转移等，频频被国际聚焦。而中共在医学领域的技术竞争，也以不遵守规则、无视道德伦理著称。
 <br/>
 <br/>
 因为中共治下只有一个形式上的〝医学伦理审查〞，但不制定相关立法，许多在海外被明令禁止的医学手术得以在中国实施。
 <br/>
 <br/>
 去年11月，意大利专家在中国实施了第一例人类头部移植手术。今年1月，中国科学家克隆出首例灵长类动物——两只食蟹猴。这些都曾引发有关医学伦理的巨大争议。
 <br/>
 <br/>
 其中一个典型例子是，自从2000年左右中共开始大规模活摘法轮功学员器官后，中国器官移植产业〝迅速崛起〞。靠无所顾忌的杀人开展的大量人体试验，以及由此积累的丰富临床经验，让中共在多个器官移植领域的技术很快〝领先全球〞。而这也成为党媒宣传中共〝伟光正〞的资本之一。
 <br/>
 <br/>
 （记者穆峰宇综合报导/责任编辑：明轩）
