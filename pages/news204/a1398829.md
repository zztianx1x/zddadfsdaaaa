### 破坏〝龙脉〞惹怒中南海 中纪委副书记坐镇陕西
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月10日讯】中央纪委副书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐令义.html" target="_blank">
  徐令义
 </a>
 ，9日现身陕西省委召开的〝秦岭北麓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建别墅.html" target="_blank">
  违建别墅
 </a>
 整治专题会〞上指导并讲话。被历代帝王尊为华夏文明龙脉的秦岭，因违建别墅多次惊动中共高层，并引发陕西官场持续震荡。
 <br/>
 <br/>
 据陕西传媒11月10日报导，陕西省委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_常委.html" target="_blank">
  常委
 </a>
 班子9日召开〝秦岭北麓
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_违建别墅.html" target="_blank">
  违建别墅
 </a>
 整治专题会〞。陕西省委书记胡和平主持会议。中共专项整治工作组组长、中央纪委副书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐令义.html" target="_blank">
  徐令义
 </a>
 到会指导并讲话。
 <br/>
 <br/>
 中央专项整治工作组副组长、中央纪委八室主任陈章永，以及陕西省政协主席韩勇，陕西省人大常委会副主任刘小燕均有出席。中组部则派员全程指导。
 <br/>
 <br/>
 秦岭被历代帝王尊为华夏文明的龙脉，北麓山坡上有一处别墅项目，以〝国岭〞为名，广告词是〝国之龙脉，岭立天下〞，而这里违规建别墅的问题由来已久，并多次惊动中共高层。
 <br/>
 <br/>
 早前官媒披露，西安秦岭北麓存在大量违法建筑，分布在秦岭北麓5000多平方公里的范围内，对植被和河流等生态环境造成严重破坏。
 <br/>
 <br/>
 官媒调查显示，这些别墅多达300多栋，不少已建成。业主非富即贵，还有党政要员。
 <br/>
 <br/>
 在被官媒披露后，秦岭北麓违法建筑仍屹立不倒，甚至绵延成景、满山开花。一些别墅项目还将售楼广告竖立在进山的必经之路上。
 <br/>
 <br/>
 据报，近年来，习近平先后6次对秦岭北麓西安境内违规建别墅、严重破坏生态环境问题和秦岭生态环境保护作出过重要批示指示，却不见成效。
 <br/>
 <br/>
 在习近平震怒做出彻查要求后，秦岭北麓违规建别墅问题专项整治工作动员部署大会7月30日在西安召开。
 <br/>
 <br/>
 不同以往的是，此次工作队伍阵容空前：中共中纪委副书记徐令义，任中央派驻专项整治工作组组长，中共中纪委八室主任陈章永任副组长。
 <br/>
 <br/>
 官媒强调：针对一个问题，习近平6次批示指示，加之中共中央专门派驻一个专项整治工作组，由中纪委副书记任组长，亲自压阵查办，足见问题严重和中央的震怒。
 <br/>
 <br/>
 徐令义在会上强调，违规建别墅问题反映的是贯彻习近平重要批示指示不坚决、不扎实、不到位的严重问题，既包括政治上的问题，也包括行政业务上的问题。
 <br/>
 <br/>
 中共陕西省委书记胡和平在大会上做动员讲话，表示要提高政治站位，认真领会习近平重要批示精神。
 <br/>
 <br/>
 7月底召开大会之后，8月中旬起，当地政府出动大量人力和大批设备，开始对秦岭违规建别墅群进行大规模拆除。
 <br/>
 <br/>
 截至10月24日，长安区全区共拆除违建724宗、79万平方米，累计拆除违建别墅292栋370套，共计18.2万平方米。
 <br/>
 <br/>
 其中位于西安段单体最大违建别墅，于2008年8月建成砖混建筑1栋1层，另有2处鱼塘，仅1处狗舍面积就达78.03平方米。阔气奢华更令人咋舌。
 <br/>
 <br/>
 近一段时间以来，陕西官场大震荡，高官持续
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_落马.html" target="_blank">
  落马
 </a>
 ，也被指与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_秦岭别墅.html" target="_blank">
  秦岭别墅
 </a>
 案有关。
 <br/>
 <br/>
 台媒中央社评论称，秦岭别墅专项拆除行动凸显了中央和地方之间的博弈和撕裂，且不说还面临着利益集团的隔碍。面对这种政令不出中南海的窘境，中央派出中纪委副书记亲自坐镇，指挥拆违，表明中央对地方的怠惰和软性抵抗极为不满。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：赵云）
