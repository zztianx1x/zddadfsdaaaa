### 揭秘：胡锦涛六字游戏令江泽民中招
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 作为中共总书记的执政10年，权力被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 架空，导致政令难行，习近平上台后大权独揽，以反腐的名义，拿下江泽民的众多亲信大员，令江泽民无比狼狈，而这一切与胡锦涛两次〝捉弄〞江泽民有关。
 <br/>
 <br/>
 据海外媒体报导，习近平能在中共十七大成为政治局常委，立为十八大一把手接班人，与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 同意
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 〝党内民主投票〞有直接关系。
 <br/>
 <br/>
 中共十七大之前，江泽民原计划的〝接班人〞、中共上海市委书记陈良宇，被胡锦涛温家宝合力拿下；曾庆红因为年龄问题，也被胡锦涛抓住不得在十七大连任。在此情况下，薄熙来成为江泽民唯一推动入常的〝自己人〞。
 <br/>
 <br/>
 但薄熙来名声太臭，在推的过程中遇到重重阻力，双方僵持不下的情况下，胡锦涛出了个主意，采用〝党内民主投票〞测试。江泽民稀里糊涂的就同意了。
 <br/>
 <br/>
 可是投票测试的结果出乎江泽民意料，路透社曾援引三个不同消息来源说，习近平在2007年初一个摸底的党内地方选举中，获得最高票数；在上海的一次选举中，获得90%的选票。更有一个消息来源称，习近平在地方党委首脑的支持度是全国第一。
 <br/>
 <br/>
 而薄熙来却在选举中被打败，在倒数之列。这样只是一名普通中央委员的习近平，在十七大直接进常委会，而薄熙来等于被淘汰。
 <br/>
 <br/>
 新当选的十七届政治局常委中，习近平和李克强都是新来者，因为李克强与胡锦涛都是团派出生，江泽民不放心，又耍小聪明强烈建议把习近平排在前位，没想到胡锦涛因为胡耀邦的原因，与习近平家族关系密切，结果同意了习近平排名第六，李克强排名第七。
 <br/>
 <br/>
 这样一来，习近平一步一步接近中共最高领导人的大位，最后终于在胡锦涛〝裸退〞奇招之下，成为江泽民的〝掘墓人〞。
 <br/>
 <br/>
 <h4>
  江泽民出面与胡锦涛达成6字〝共识〞
 </h4>
 <br/>
 中共军中总后勤部基建营房部副部长谷俊山，2012年因涉贪被查，成为中共军委副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_徐才厚.html" target="_blank">
  徐才厚
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郭伯雄.html" target="_blank">
  郭伯雄
 </a>
 案的突破口。
 <br/>
 <br/>
 海外中文媒体披露，谷俊山案发后，郭伯雄和徐才厚曾求助江泽民，江泽民出面与胡锦涛达成6字〝共识〞保护二人。可是，江泽民没想到，胡锦涛又与习近平达成另外6字〝约定〞，巧妙的打破江泽民的包庇屏障。
 <br/>
 <br/>
 谷俊山被胡锦涛拍板调查后，徐、郭二人一度有些惊慌，曾求教于江泽民。当时正逢中共十八大前后，江安抚徐、郭，称已和胡达成共识〝止于谷，不上追〞。
 <br/>
 <br/>
 但江没有料到，胡锦涛有另外一招，既不食言，也不吃瘪，与习近平也达成了共识，习近平与胡锦涛进行分工，〝你查谷、我查上〞。这样一来，查谷俊山是胡拍板决定的，而查徐才厚则是习近平的主意。
 <br/>
 <br/>
 2014年3月31日，谷俊山被提起公诉；同年3月15日，徐才厚被调查，2015年3月患癌死亡；郭伯雄于2015年落马，2016年7月被判终身监禁。至此江泽民军中三〝最爱〞全落马。
 <br/>
 <br/>
 港媒披露，郭伯雄落马后，在监狱服刑期间，不但自曝接受徐才厚转交贿款的详情，还咬出江泽民，把军队变质问题归咎于江泽民对军队的领导。
 <br/>
 <br/>
 徐才厚和郭伯雄两人不但替江把持军权10多年，架空了胡锦涛，两人还曾阻击习近平进军委。在一个有数人在场的军方半公开场合，徐才厚曾对郭伯雄说：〝让他（指习近平）干5年就滚蛋！〞
 <br/>
 <br/>
 而徐才厚统管总后勤部和总政治部期间，各地军队医院发生大量活摘法轮功学员器官的行为，这种骇人听闻的活摘器官黑幕，至今仍被中共掩盖。
 <br/>
 <br/>
