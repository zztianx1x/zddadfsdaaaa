### 贸战效应显 粤苹果供应商裁员8千引爆万人示威
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月17日讯】由于苹果公司的定单减少，全球最大手机屏幕供应商、港资惠州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伯恩光学.html" target="_blank">
  伯恩光学
 </a>
 近日宣布
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大裁员.html" target="_blank">
  大裁员
 </a>
 ，引发近万工人及家属包围工厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_静坐抗议.html" target="_blank">
  静坐抗议
 </a>
 ，地方当局派大批防暴警察驻守工厂。经数日谈判本周四劳资双方初步达成解决方案，工人才疏散离去。舆论认为，这是中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 产生的连锁后效应。
 <br/>
 <br/>
 据香港《苹果日报》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_伯恩光学.html" target="_blank">
  伯恩光学
 </a>
 是苹果和三星最大的玻璃屏幕供应商，该企业60%左右业绩都由苹果和三星贡献。近期，苹果公司大幅削减手机定单，导致供应商陷入无定单可做困境。
 <br/>
 <br/>
 伯恩光学本月9日突然要求各劳务派遣公司（中介）撤走派遣工，分两批先后裁减员工8000人。其中一份被裁名单多达5,000人，而且被裁人员名单完全是由厂方拍板，中介无权决定，而这些被裁员工10月份工资仍未发放。
 <br/>
 <br/>
 被裁工人不满厂方的赔偿方案，加上正值年关将近，不少打工者等钱回家过年，于是被裁工人及家属从9日开始聚集厂部大门外静坐示威，人数最多时达万人。
 <br/>
 <br/>
 地方当局派遣了大批防暴警察到场警戒，不过工人们只是静默包围工厂，并没有做出其他过火举动，因而未爆发警民冲突。
 <br/>
 <br/>
 15日晚，伯恩公司与相关工人达成初步解决方案后，示威工人已疏散离开。
 <br/>
 <br/>
 据报导，除伯恩光学遭苹果公司大幅度削减订单外，苹果代工厂伟创力旗下的子公司伟创力塑胶科技（深圳）公司也同样面临定单锐减的困境。该公司虽然尚未大举裁员，但已决定分批安排员工从本月12日至明年2月1日〝放假〞。
 <br/>
 <br/>
 该公司日前发布通告称，〝受外部环境影响，目前公司定单锐减，公司生产经营发生较大困难，第三季度出现严重亏损。公司做了很多努力，但目前处境仍艰难〞，望员工能与公司一起〝共渡难关〞。
 <br/>
 <br/>
 此外，中国大陆的苹果代工厂富士康在深圳观澜产业园区的生产也下降了10%，有消息指该公司今年的产量高峯结束时间提前个半月，每天有逾千人离职。
 <br/>
 <br/>
 舆论认为，虽然目前中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 加征关税的清单中还没有列入智能电话，但苹果公司所面临的可能遭池鱼之殃的压力并不轻松。
 <br/>
 <br/>
 有分析指，中国是苹果最重要的生产基地，在中共对美展开报复的过程中，很容易就把苹果公司列为攻击的目标。随着贸易战持续升温，苹果和其主要供应商面临的政治风险也日益升高，寻找新的出路在所难免。而早在贸易战一触即发的时候，美国总统川普就曾点名敦促苹果公司将生产线搬回美国。因此，苹果公司大幅削减供应商的订单，与贸易战带来的巨大压力是密切相关的。
 <br/>
 <br/>
 据公开的资讯，随着贸易战的步步升级，多家苹果配件供应商已经在酝酿将工厂搬离中国大陆，以便避开随时可能降临的政治风险，而这些企业主要考虑的搬迁地，大多是台湾或东南亚其他国家。
 <br/>
 <br/>
 例如，上个月就有消息指，负责组装苹果无线耳机AirPods的歌尔声学，正打算将生产线搬至越南。为iPhone提供充电器的正崴精密工业近日也对外宣称，公司正考虑将部份生产线搬回台湾或东南亚。该公司的董事长郭台强曾公开表示，公司正评估泰、越、菲的新厂房。
 <br/>
 <br/>
 针对上述情况，《日经亚洲评论》曾分析称，歌尔声学等公司决定远离贸易战火，折射出苹果公司及其零件供应商正面临的困境。
 <br/>
 <br/>
 （记者唐迪综合报导/责任编辑：秦锋）
