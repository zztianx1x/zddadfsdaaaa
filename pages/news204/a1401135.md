### 中国知名歌手陈羽凡与女友吸毒被抓
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】28日，网曝中国大陆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_男歌手.html" target="_blank">
  男歌手
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈羽凡.html" target="_blank">
  陈羽凡
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吸毒.html" target="_blank">
  吸毒
 </a>
 被抓。对此，陈羽凡所在公司发声明否认。随后，北京警方称，在北京市某小区抓获2名涉毒违法人员，经尿检，陈姓男歌手呈冰毒类和大麻类阳性。陈羽凡方面便删除了辟谣声明。
 <br/>
 <br/>
 11月28日，有网友爆料称歌手
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈羽凡.html" target="_blank">
  陈羽凡
 </a>
 因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吸毒.html" target="_blank">
  吸毒
 </a>
 被抓。对此，陈羽凡所在公司〝巨匠娱乐〞官方微博发布声明否认，称关于陈羽凡的传闻不实。
 <br/>
 <br/>
 当天下午警方通报，石景山公安分局根据群众举报，在本市某小区抓获2名涉毒违法人员陈某（男，43岁，歌手）和何某某（女，25岁，无业），现场起获冰毒7.96克、大麻2.14克。经尿检，陈某呈冰毒类和大麻类阳性，何某某呈大麻类阳性。
 <br/>
 <br/>
 通报称，目前，陈某因吸毒、非法持有毒品，何某某因吸毒均被行政拘留。此案正在进一步工作中。之后，陈羽凡经纪公司删除了关于陈羽凡的辟谣声明。
 <br/>
 <br/>
 平安北京的微博随后转发了平安石景山的微博称，〝毒品，让‘最美’凋零〞。 因陈羽凡所在的组合最初因单曲《最美》走红，通报所指的陈某被指是歌手陈羽凡。
 <br/>
 <br/>
 资料显示，陈羽凡是中国歌唱组合羽・泉成员。他因出轨与前妻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白百何.html" target="_blank">
  白百何
 </a>
 离婚，一度闹得沸沸扬扬。
 <br/>
 <br/>
 陆媒透露，通报中所提及的〝何某〞是陈羽凡同居多年的女友，在一起很久了。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
</div>
