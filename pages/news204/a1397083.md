### 中共海军战舰规模超美  仍被美军吊打
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月28日讯】日前，有美媒引述最新统计数据，显示中共海军作战
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舰艇.html" target="_blank">
  舰艇
 </a>
 总数达到206艘，已超过只有177艘的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 ，但在质量和战略打击能力上，美军依然远胜中共。报导称，对中共海军而言，与美国海军对抗，属于不明智的选择。
 <br/>
 <br/>
 美国网站《工程》（engineering.com）引述最新统计称，在中共海军的206艘作战
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_舰艇.html" target="_blank">
  舰艇
 </a>
 中，包括2艘航母、37艘飞弹驱逐舰、52艘飞弹护卫舰、42艘轻型护卫舰和73艘潜艇。
 <br/>
 <br/>
 而美国海军的177艘舰艇，包括20艘航母／两栖攻击舰、85艘飞弹驱逐舰及72艘潜艇。
 <br/>
 <br/>
 报导说，从舰艇特性看，美国海军最大的优势是舰艇和两栖攻击舰，这两种军舰的数量合计20艘，大部份舰母都是采用核动力设计，且能够搭载第五代战机。
 <br/>
 <br/>
 而反观中共现有的两艘航母中，辽宁舰仍可追溯到苏联在20世纪80年代制造的船体，首艘自主生产的航母仍在海试，加上这两艘航母都是传统动力、滑跳起飞的设计，严重限制作战能力。
 <br/>
 <br/>
 报导强调，虽然中共在在驱逐舰、护卫舰方面，数量上拥有优势。但质量和战略打击能力，美国拥有明显优势，因美国可以将这些军舰部署到日本、澳洲等盟友的港口中，这足以打击中共的高价值目标。
 <br/>
 <br/>
 此外，中共海军装备数量更多的虽然是潜艇，但其中绝大部分是采用柴油动力的传统潜艇，这些潜艇有明显的噪音问题，很容易被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美军.html" target="_blank">
  美军
 </a>
 核潜艇或反潜武器捕获。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-28/p9100161a906875781.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-28/p9100161a906875781-ss.jpg"/>
   <br/>
   <font size="-1">
    图为美国阿利伯克级的导弹驱逐舰米切尔号（USS米歇尔，DDG 57）。(U.S. Navy)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 而美国海军的情况正好相反，潜艇部队完全由核动力潜艇构成，这些攻击武器拥有几乎无限的航程，也很难被对手发现。
 <br/>
 <br/>
 报导最后总结，与美国海军对抗对中共海军而言，属于不明智的选择，在评估谁在冲突中能表现更佳时，还需要考虑更多因素。然而任何冲突对美中两国，甚至全球市场来说都是灾难性的。
 <br/>
 <br/>
 去年4月，就有俄罗斯军事专家称，中美军力对比，中共军队有4大方面远远落后于美国，特别是在武器装备方面，至少落后美国半个世纪。
 <br/>
 <br/>
 一、在战略思想上，美国一直称霸世界，在军事理论和武器装备上，在全球战略要地有其足够快速反应军事力量；而中共一直是以防御与陆军装备为主，只有近距离攻击的作用，这使得中美差距拉大。
 <br/>
 <br/>
 二、大部分的中共部队快30年没打仗了，缺乏实战经验；而且腐败是中共军队战斗力的第一杀手。
 <br/>
 <br/>
 三、在远程投送上，美军的快速反应部队能在24小时内，机动到全球任何一个地点进行作战；而中共军队的机动性就落后美军太多了。
 <br/>
 <br/>
 四、在武器准备上，美国拥有世界一流的战略轰炸机、战略核潜艇和航母战斗群，而中共军队与此相应的武器装备，至少要比美国落后近半个世纪。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
