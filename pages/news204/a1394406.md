### 中国人海外代购：推倒货架疯抢  免税店激烈互殴
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月07日讯】中国流行
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_网上代购.html" target="_blank">
  网上代购
 </a>
 ，许多人看准商机加入〝代购军团〞。近日，美国大型连锁超市好市多，出现了〝代购军团〞为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抢购.html" target="_blank">
  抢购
 </a>
 名牌外套，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_推倒货架.html" target="_blank">
  推倒货架
 </a>
 的事。在韩国乐天免税店，此前还曾出现为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抢名牌.html" target="_blank">
  抢名牌
 </a>
 ，三名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 互殴事件。
 <br/>
 <br/>
 10月6日，有港媒报导，不久前，美国某家好市多大减价，以54.99美元的价格促销汤米席尔菲格（Tommy Hilfiger）修身外套，吸引大批中国代购军团前往扫货。
 <br/>
 <br/>
 网络上疯传当时现场疯抢的一幕，只见有
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 将货架上的外套全推倒在地，一名女子整个人扑上去〝占领〞全部外套，还有代购者一口气买下400件外套，令店内其他顾客极为震惊。
 <br/>
 <br/>
 但这并不是中国代购军团，第一次在国外因
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抢购.html" target="_blank">
  抢购
 </a>
 商品而出丑的场面。据加拿大媒体称，当地一家好市多，也曾出现过两名中国人5分钟搬完整个货架的事件，〝场面一次比一次疯狂、一次比一次壮观〞。
 <br/>
 <br/>
 8月中旬，网上也曾疯传一段3名中国购物者在韩国乐天免税店互殴的视频。
 <br/>
 <br/>
 视频中，两女一男在化妆品柜台前为抢货大打出手。一名白衣女子和另一名黑衣女子扭打在一起，黑衣女子被掀翻在地，而另一名男子，则用脚猛跺倒地的黑衣女子的头。
 <br/>
 <br/>
 中国男子的野蛮行为引发韩国当地围观者的不满，旁边的两名男子强行将打人男子推开。
 <br/>
 <br/>
 据报，代购者疯抢的商品从衣饰到美容妆品、从奶粉到名牌内裤，琳琅满目。
 <br/>
 <br/>
 许多大陆网民也看不下去自己同胞在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_海外.html" target="_blank">
  海外
 </a>
 的行径，希望〝代购们〞别再抢了，〝给中国人留点脸面吧〞。也有人抨击，〝在共产党教育下的中国人都这般的，凡事以暴力解决问题，无法无天。〞
 <br/>
 <br/>
 不过，最近中共人大出台了《电子商务法》，规定从事个人代购和微商需要办理工商登记取得相关行政许可，〝依法纳税〞。
 <br/>
 <br/>
 如违犯该法，不仅代购本人，而且包括平台，都要承担连带责任。情节严重的，将责令停业整顿，及处罚50万到250万的罚款。
 <br/>
 <br/>
 不少人认为，《电子商务法》的实施，将意味着私人代购时代的终结。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：祝馨睿）
</div>
