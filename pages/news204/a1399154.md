### 外媒：中国民企成了政府亲儿子
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】近一段时间以来，中共从中央到地方，从多部委到官媒，密集发声〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纾困.html" target="_blank">
  纾困
 </a>
 〞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民营企业.html" target="_blank">
  民营企业
 </a>
 政策。有媒体称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国民企.html" target="_blank">
  中国民企
 </a>
 受到政府热捧，一时间民企同国企一样成了北京的亲儿子。评论人士认为，只是它惯用的花招。
 <br/>
 <br/>
 《联合早报》11月13日称，近期，中共官方大张旗鼓地支持民营经济发展，将民企同国企一样当作〝亲儿子〞看待，事情缘起于9月份一篇关于〝私营经济离场论〞引发的争议和恐慌。
 <br/>
 <br/>
 尽管中国官媒和高层一再驳斥这一论调，但安抚效果并不显著，官方意识到问题严重性后，频频站出来表态稳定民心。
 <br/>
 <br/>
 如11月9日，李克强在经济形势专家和企业家座谈会上，对参会企业家说，你敞开说话，此处无禁忌；11月4日党媒评论称：让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_民营企业.html" target="_blank">
  民营企业
 </a>
 吃下〝定心丸〞。
 <br/>
 <br/>
 地方一把手甚至亲自走访民营企业，表示支持。上海市委书记李强说，民营企业是上海经济社会发展不可或缺的重要力量。此外，还有湖北、贵州、湖南、浙江、江苏、广州等省市纷纷表态支持民营企业。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 11月1日主持召开民营企业座谈会，接见10名企业家代表。习在座谈会上强调：所有民营企业和民营企业家完全可以吃下定心丸、安心谋发展！民营企业和民营企业家是自己人。是习近平这1月多来，第4次发表言论安抚民营企业。
 <br/>
 <br/>
 中共官方10月份还曾发布《改革开放40年百名杰出民营企业家名单》，官媒称，推荐这个名单的目的是要在所谓的改革开放40年之际，鼓励民企老板〝不忘创业初心〞。
 <br/>
 <br/>
 分析认为，由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 ，导致中国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济疲软.html" target="_blank">
  经济疲软
 </a>
 ，经济增长动力不足，从中央到地方，从多部委到官媒，密集发声新一轮〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_纾困.html" target="_blank">
  纾困
 </a>
 〞民营企业政策，其实只是中共惯用的花招。
 <br/>
 <br/>
 财经人士批评，在过去5年，民企实在是被中共收割得够呛，税务稽查风暴，社保检查浪潮，环保一刀切，股票质押爆仓，银行收缩贷款，等等等等诸如此类。还要叠加供给侧改革带来的原材料价格暴涨因素，民企压根连生存空间都没有。
 <br/>
 <br/>
 如今因为贸易战，外企加速转移产能，已经指望不上，也无法指望官方搞基建和国企垄断撑起中国经济，民企又有利用价值了，所以中共对民企也一直是〝两面人〞。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：赵云）
</div>
