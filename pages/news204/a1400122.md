### 陆就业市场大地震？贸战冲击鸿海集团传裁员34万
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月21日讯】近日，有中国媒体爆料称，iPhone最大代工厂
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富士康.html" target="_blank">
  富士康
 </a>
 所属的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海集团.html" target="_blank">
  鸿海集团
 </a>
 ，为了降低人力成本，将在一年内
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
  裁员
 </a>
 34万，同时对高层管理人员依贡献程度调整薪资。截至目前，鸿海集团并未对此传闻做出回应，富士康的负责人则对媒体表态称〝不评论〞。
 <br/>
 <br/>
 据青岛软媒旗下的〝IT之家〞20日报导，由于新iPhone销量不佳，苹果大幅削减订单，导致相应的代工厂、配件厂生意大减，而苹果的最大组装厂商
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_鸿海集团.html" target="_blank">
  鸿海集团
 </a>
 （Foxconn Technology Group）为降低人力成本，也将于一年内裁减34万名员工。同时，也有台媒称，鸿海集团董事长郭台铭曾表示，要在5年内把该公司的作业员裁掉80%。
 <br/>
 <br/>
 同一天，鸿海集团在中国大陆的经营实体
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_富士康.html" target="_blank">
  富士康
 </a>
 的负责人针对上述传闻回应称：不知道消息从何而来，但富士康不评论市场及媒体上的流言。
 <br/>
 <br/>
 另据台湾《经济日报》11月18日的一篇报导，因明年全球经济情势不确定因素太多，鸿海集团为降低人力成本和固定费用或将
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_裁员.html" target="_blank">
  裁员
 </a>
 ，对于总经理、副总裁等级高年薪的高层管理人员，也将依据贡献程度调整薪资。
 <br/>
 <br/>
 该报导并透露，鸿海内部主管已提出多套检讨高管贡献价值程度的措施，总经理、副总裁等层级的高年薪管理人员皆在逐一检讨名单内。
 <br/>
 <br/>
 有台媒称，鸿海集团员工人数截至今年6月底共有69万人，如鸿海真的实施上述裁员计划，将可节省逾500亿人民币的开支。
 <br/>
 <br/>
 截至目前，鸿海集团并未就上述有关降低成本计划的传闻做出回应。
 <br/>
 <br/>
 据公开的资讯，由台湾商人郭台铭创立的鸿海集团，总部位于新北市土城区，旗下拥有841家公司子公司，在多个国家及地区设有厂房。中国人所熟知的〝富士康国际控股有限公司〞就是鸿海集团旗下的子公司。
 <br/>
 <br/>
 在苹果公司的18家代工厂中，富士康占了7家，而后者近一半的营收都要依赖苹果公司的订单。近年来，iPhone销量下滑，富士康的收益亦随之逐渐减少。
 <br/>
 <br/>
 今年11月13日，鸿海集团公布的第三季度财报显示，截至9月的三个季度，该公司净利润为248.8亿新台币（约合8.0552亿美元），不及分析师预期的282.6亿新台币。
 <br/>
 <br/>
 此外，据TechWeb报道，受新iPhone销售疲软消息影响，苹果亚洲供应商和代工厂股价普遍下跌。其中鸿海集团跌逾3%，其竞争对手和硕联合集团下跌超过5%。
 <br/>
 <br/>
 有舆论感叹称，在贸易战的阴影下，中国企业的寒冬恐怕真来了！
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：秦锋）
