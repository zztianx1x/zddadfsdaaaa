### 北京政协官员汪洋被查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】近日，北京市朝阳区政协原副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汪洋.html" target="_blank">
  汪洋
 </a>
 ，涉嫌严重违纪违法，接受纪律
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审查.html" target="_blank">
  审查
 </a>
 和监察
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调查.html" target="_blank">
  调查
 </a>
 。由于这位落马官员与中共政协主席汪洋同名，而且同属中共政协系统官员，从而引起外界好奇。
 <br/>
 <br/>
 10月30日，北京市纪委监委发消息称，朝阳区已退休两年的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政协副主席.html" target="_blank">
  政协副主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_汪洋.html" target="_blank">
  汪洋
 </a>
 ，涉嫌严重违纪违法，目前正接受纪律
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_审查.html" target="_blank">
  审查
 </a>
 和监察
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_调查.html" target="_blank">
  调查
 </a>
 。
 <br/>
 <br/>
 公开资料显示，被查的汪洋现年58岁，湖北罗田人，拥有经济学硕士学位。自1997年起就在朝阳区担任公职，先后历任区委办公室副主任、街道工委书记、社会建设办公室主任、副区长、区政协副主席等职务，2016年12月已经正式退休。
 <br/>
 <br/>
 由于这位落马官员汪洋与中共领导人汪洋同名引发关注。
 <br/>
 <br/>
 公开资料显示，外界所熟知的中共领导人汪洋，1955年3月生，安徽宿州人，1972年6月参加工作，中央党校大学学历，工学硕士学位。2007年中共十七大晋升副国级，当选为中央政治局委员，兼任广东省委书记，十八大后任国务院副总理。中共十九大上当选政治局常委。
 <br/>
 <br/>
 汪洋〝入常〞之路颇为曲折，2012年换届年，汪洋入常呼声甚高，但最终被刘云山以更低的得票数取代。
 <br/>
 <br/>
 对此内幕，海外中文网曾揭露，因江派大佬江泽民、曾庆红坚持要刘上，最后胡、习只好让汪洋下。据称，汪洋遭江、曾否决的原因，是他主政广东时期〝公然〞得罪了江、曾两人。
 <br/>
 <br/>
 原中国军事学院出版社社长辛子陵在接受媒体采访时披露，十八大组建领导班子时，江泽民还占着相对的优势，七名常委他推荐了三个，刘云山的票数少于汪洋，但江坚持要刘上，胡锦涛、习近平只好让步，让汪洋下。
 <br/>
 <br/>
 现年62岁的汪洋是胡锦涛、习近平阵营的重要人物。中共十八大后，汪洋受习近平重用，在2017年10日十九大上当选中共政治局常委，2018年3月两会上，汪洋当选中共全国政协主席。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉)
