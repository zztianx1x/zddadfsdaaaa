### 习明泽哈佛读书内幕 传回国做了习近平特别幕僚
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】近日，有台媒称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 这几年成了风云人物，习的女儿
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习明泽.html" target="_blank">
  习明泽
 </a>
 也成为全世界媒体追逐的焦点，因为她曾在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杭州.html" target="_blank">
  杭州
 </a>
 念
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_翻译.html" target="_blank">
  翻译
 </a>
 ，所以出现在习身边的翻译人员，常被猜测是习明泽。有美媒披露，习明泽曾在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈佛大学.html" target="_blank">
  哈佛大学
 </a>
 读书，回国后做了习的特别幕僚。
 <br/>
 <br/>
 现年26岁的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习明泽.html" target="_blank">
  习明泽
 </a>
 是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 的独生女儿，小名木子，大名习明泽，据说名字是习近平的父亲习仲勋给起的，〝希望她将来清清白白的做人，做个对社会有用的人。〞
 <br/>
 <br/>
 《商业期刊》日语版10月初报导称，习近平的独生女儿习明泽在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_哈佛大学.html" target="_blank">
  哈佛大学
 </a>
 念书，中美关系友好的时候，美国政府还会加以照顾，不过现在随着中美贸易战的升级，川普很可能采取相反的方式。
 <br/>
 <br/>
 早在2015年明报称，美国哈佛大学名誉教授、著名东亚问题专家傅高义就曾透露，习近平的女儿习明泽，2014年已从哈佛大学毕业回国。报导中还贴出一张疑似习明泽的毕业照。
 <br/>
 <br/>
 傅高义透露，习明泽在哈佛就读期间被当做普通学生，并未受到特别对待。
 <br/>
 <center>
  <br/>
  <a href="http://imgs.ntdtv.com/pic/2018/10-12/p9077201a135592002.jpg" target="_blank">
   <img border="0" src="http://imgs.ntdtv.com/pic/2018/10-12/p9077201a135592002-ss.jpg"/>
   <br/>
   <font size="-1">
    明报称，习近平的女儿习明泽2014年已从哈佛大学毕业回国。报导中还贴出一张疑似习明泽的毕业照。(网络截图)
   </font>
  </a>
  <br/>
 </center>
 <br/>
 美国〝纽约客〞杂志也曾引述消息称，习明泽利用化名在哈佛大学主修心理学和英文，了解她真正身份的人不超过10个，仅限于亲密的友人和几位教职人员。
 <br/>
 <br/>
 媒体还爆料，习明泽回国后，作为特别幕僚辅佐其父亲。她运用自己在哈佛学到的知识，提出谘询意见，成为习的形象顾问，在政治上挺习护习的〝学习小组〞即为其策划。为培养其政治素质，习近平多次在国事访问中将其携带。
 <br/>
 <br/>
 <h4>
  汶川大地震时曾做义工
 </h4>
 <br/>
 据媒体报导，2008年汶川大地震时，还不满16岁的习明泽，曾以志愿者的身份，前往四川绵竹的汉旺东汽小学，当了7天参与抢救伤者、心理辅导的慈善义工。
 <br/>
 <br/>
 当时，中共十七大召开才半年，习近平成为胡锦涛的接班人、国家副主席。至于女儿去了四川哪里，彭丽媛也不知道。
 <br/>
 <br/>
 习明泽向母亲描述自己做志愿者的情况和对四川的好印象，她说，她跟孩子们一起玩，还与当地一些灾区群众、同龄人交上了朋友。她觉得四川人非常好，坚强又善良。她以后想到四川生活。
 <br/>
 <br/>
 此外，习近平2002至2007年任职浙江省委书记时，习明泽就读于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_杭州.html" target="_blank">
  杭州
 </a>
 外国语学校。2007年，习近平在十七大上调中央后，当时15岁的女儿也曾随父亲回到北京念书。
 <br/>
 <br/>
 有自称是习明泽在杭外同班同学的网友称，习明泽在杭外读书时，大家也都知道她爸是习近平。不过，杭外的学生一大半非官即富，一个省委书记虽然是浙江省最大的官儿，但也没有多少人觉得是多么了不得的事。
 <br/>
 <br/>
 这位网友还说，习明泽是那种非常内敛宽厚的人，在班里也不太活跃，有点〝泯然众人〞的意思。毕业后，听说她去了哈佛，大家也觉得很正常，杭外毕业直接去国外名校的人非常多。
 <br/>
 <br/>
 据称，习明泽自16岁之后就从未在媒体露面，迄今为止，外界只能看到她幼年和少女时代的照片，近年来的样貌，除了与她最亲近的家人和亲友外，几乎无人知晓。2014年哈佛大学毕业回国后，外界也无从得知她的去向。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：李泉）
