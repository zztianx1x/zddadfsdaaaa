### 中共官场出大笑话  两官讲稿〝撞车〞抄袭同一网文
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月13日讯】近日江西省抚州市
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_官场.html" target="_blank">
  官场
 </a>
 闹出
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大笑话.html" target="_blank">
  大笑话
 </a>
 ，在一个
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_会议.html" target="_blank">
  会议
 </a>
 上，广昌县两名官员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发言.html" target="_blank">
  发言
 </a>
 时，听上去竟然几乎一模一样，在场的巡察人员感觉蹊跷，叫停了发言，结果发现两人均
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抄袭.html" target="_blank">
  抄袭
 </a>
 网上的同一内容。
 <br/>
 <br/>
 陆媒10月11日报导，广昌县委第一巡察组，近日在进驻县党史办和县房管局巡察动员会上，发现县党史办党支部书记钟某，和县房管局党组书记刘某的表态
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_发言.html" target="_blank">
  发言
 </a>
 ，听上去几乎一模一样，当场叫停了他们的发言。
 <br/>
 <br/>
 巡察人员经比对发现，钟某与刘某提交的纸质发言材料，除了单位名称有所区别之外，其他地方完全相似。经查，两人的表态发言都是从网络上照抄而来，没料到，这两份几乎一模一样的发言材料，竟然在同一个会场〝撞车〞了。
 <br/>
 <br/>
 之后，钟某与刘某被约谈和责令检讨。
 <br/>
 <br/>
 事实上，类似事件在中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_官场.html" target="_blank">
  官场
 </a>
 早已经屡见不鲜。最近，在江西省资溪县召开的一次约谈会上，会上要求被约谈单位结合自身实际作表态发言，结果嵩市镇、县城管局、县交警大队3个党组织负责人的发言稿撞车，经查，都是从同一网文中
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抄袭.html" target="_blank">
  抄袭
 </a>
 拼凑的。
 <br/>
 <br/>
 〝以为表态发言材料只是在会上念一下罢了。〞〝平时工作中的材料，往往都是自己提思路，由办公室整理和撰写。〞面对纪委工作人员的询问，这3个单位党组织负责人道出了出现〝同款〞发言稿的原因。
 <br/>
 <br/>
 网民嘲讽，〝天朝从上至下都一样，开会讲话假大空一大堆，讲稿撞车太普遍了〞； 〝想起了文革，下面迎合频繁的大批判文章和发言，哪个不是抄的？过去没网络，要交大批判稿子，就得上街趴在墙上抄。现在的官员，应该禁止上百度〞。
 <br/>
 <br/>
 曾经任法院检察官的安徽异见人士沈良庆对自由亚洲电台说，类似的情况在各地都有发生，不少官员都会透过其他人代笔撰写
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_工作报告.html" target="_blank">
  工作报告
 </a>
 ，然后就在公开场合上搬字过纸读出来，只是有些很容易蒙混过去，即使被发现了也不会作出很严厉的处罚。
 <br/>
 <br/>
 他说：中共官员的工作报告，一般都有专人代写，就随便在网上抄一下，内容当然都大同小异。只要混过去就行了，如果这次没有被抓包，事情也会过去。
 <br/>
 <br/>
 曾参选北京市人大代表的朱秀玲认为，官员抄袭网络文章的行为，与中共的选举制度有关。官员在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_会议.html" target="_blank">
  会议
 </a>
 上发表报告，在某个程度上是在反映当地的实际情况，然而这些官员缺乏民意基础，平常都不会积极了解民意，所以只能在网上东拼西凑。
 <br/>
 <br/>
 朱秀玲说：事实证明，不是按照选举法或相关法律法规选出来的人，不会替老百姓真正去说话的。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
