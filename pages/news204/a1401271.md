### 习近平〝四菜一汤〞与餐费单惊动西媒
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】近日，在中共举办的改革开放40周年大型展览上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在河北阜平考察时的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四菜一汤.html" target="_blank">
  四菜一汤
 </a>
 〞餐单，以及考察河南兰考时的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_餐费.html" target="_blank">
  餐费
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_收据.html" target="_blank">
  收据
 </a>
 等被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_展出.html" target="_blank">
  展出
 </a>
 。有西媒称，中共官方力图〝塑造〞领导人厉行节约举措，然而网民并不领情。
 <br/>
 <br/>
 在中共国家博物馆25日举办的庆祝改革开放40周年大型展览上，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 于2012年考察河北省阜平县时的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_晚餐.html" target="_blank">
  晚餐
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_菜单.html" target="_blank">
  菜单
 </a>
 、2014年考察河南省兰考县时的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_餐费.html" target="_blank">
  餐费
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_收据.html" target="_blank">
  收据
 </a>
 、以及2015年在原梁家河的午餐费用收据等票据在〝历史巨变〞展区
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_展出.html" target="_blank">
  展出
 </a>
 。
 <br/>
 <br/>
 展出的2012年底习近平在阜平县所住宾馆的晚餐餐单上，是标准的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_四菜一汤.html" target="_blank">
  四菜一汤
 </a>
 〞，有四个热菜为：红烧土鸡块、阜平烩菜（猪肉、豆角、粉条、海带、豆腐、土豆）、五花肉炒蒜薹、拍蒜茼蒿，还有一个冬瓜丸子汤；主食是猪肉馅水饺、西葫芦粉条馅水饺、米饭、花卷，还有杂粮粥。
 <br/>
 <br/>
 与这份晚餐菜单一同展出的，还有习近平考察河南省兰考县时的餐费收据，交款人是习近平，时间是2014年3月18日，交了人民币160元的伙食费，票据上还特别写明现金付款。
 <br/>
 <br/>
 同批展出的还有习近平2015年看望陕西省梁家河村村民时，在原梁家河大队党支部书记梁玉明家的午餐费用收据。
 <br/>
 <br/>
 收据上的内容为：今收到习近平等3人伙食费合计玖拾元正（整），落款人是梁玉明，落款时间是2015年2月13日；另一收据为：今收到习近平随行人员伙食费。从收据看，每人的伙食费是30元。

 对此，有西媒称，在中国，中共官员和商界高管对美食的热衷和庞大需求，已经导致鲨鱼数量减少，酒类制造商的数量则因此增多。而此次大型展览上，展出习近平〝四菜一汤〞，非常明显的是在传达领导人厉行节约的用意。
 <br/>
 <br/>
 然而网民似乎不领会其意，有人称作秀。也有人调侃有点像童话〝皇帝的新装〞。
 <br/>
 <br/>
 有网民表示，勤俭节约本来就是当官应该做的事，如今却大肆展出，越宣传越让人觉得此地无银三百两，更反衬出高官们以前大吃大喝，当下出来作秀。
 <br/>
 <br/>
 也有网民表示，如果不是在中共治下，而是在西方民主政体之下，这样的做法十分可笑。就像童话皇帝的新装，他们不是想让人们相信他们穿着华丽的新衣，而是要让人们相信，他们什么都没穿，他们没有巨额财富。
 <br/>
 <br/>
 网民认为，中国老百姓对中共领导人形成的糜烂奢侈看法，很难改变。如果真想改变老百姓对领导人的看法，必须有实质性的改变，单靠这样作秀是不行的。
 <br/>
