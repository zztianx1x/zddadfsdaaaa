### 贸易战再升级 俄媒：川普决定动用加重型武器
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】美方连串动作显示已启动
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美科技战.html" target="_blank">
  中美科技战
 </a>
 。美高官日前更警告英国，与中企合建核电站存在洩密和安全风险。俄媒称，美国决定动用
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重型武器.html" target="_blank">
  重型武器
 </a>
 ，谴责
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共盗窃技术.html" target="_blank">
  中共盗窃技术
 </a>
 。而深圳官方报告指，美国或在半年至一年内对中共实施高科技全面封锁。
 <br/>
 <br/>
 10月25日，美国国务院国际安全与防核武扩散局(ISN)助理国务卿福特（Christopher Ford）在媒体吹风会上向英国《金融时报》表示，英国欣克利角C(Hinkley Point C)核电站项目的合作方——
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国广核集团.html" target="_blank">
  中国广核集团
 </a>
 (CGN)是中共把民用核技术军事化的先锋之一。
 <br/>
 <br/>
 福特透露，美国已经与英国共享了公开的和美国情报部门搜集的各种证据，以证明中国广核集团窃取民用核技术，并转交给中共军方。他说，中国核工业已经和中共的〝军民融合〞捆绑在一起了。
 <br/>
 <br/>
 福特警告称，中广核窃取来的技术可能帮助军方发展核动力潜艇和航空母舰，包括在南海建造浮动核反应堆。海外项目为中国公司提供了获取核心科技情报的途径，并对西方各国的安全造成威胁。
 <br/>
 <br/>
 福特表示，如果中广核正在帮助中共军方，〝研制有可能瞄准包括伦敦在内的西方国家首都的导弹〞，那么与中广核合作是否一个好主意，就是其他国家值得思考的问题了。
 <br/>
 <br/>
 俄罗斯卫星通讯社报导称，中美贸易战逐渐升级，美国决定动用更重型的武器，即谴责中共窃取技术，并指福特直接谈到了这一点。
 <br/>
 <br/>
 作为西方国家的领袖，美国针对中共的动作能够起到很好的示范作用，再加上美国政府直接向各国宣示中共威胁，并敦促各国采取行动，中共盗取他国技术将越来越遭到抵制。
 <br/>
 <br/>
 俄罗斯卫星通讯社引述分析称，美国对中共盗取他国核技术的谴责，未来还可能转化成政治决议，甚至是针对与中方核企业合作的第三国公司的〝二次制裁〞。
 <br/>
 <br/>
 日前，美国
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）政府高调起诉中共科技间谍，对中国科技公司福建晋华再次启动禁售令，并限制美国企业向中方提供民用核材料和设备，显示针对中共的科技战已经正式开打。
 <br/>
 <br/>
 10月流出的一份深圳官方对美国的考察报告指，美国可能在半年至一年内，在高科技层面对中共实施全面封锁，并建议中共官方在全面封锁前，加紧以〝超常规手段〞引进科研设备和高科技人才。
 <br/>
 <br/>
 川普政府一直将〝中国制造2025〞视为主要威胁。而美媒日前曝光，中共还同时启动了一个〝中国标准2035〞计划。这意味着中共在取得高科技领导权后，还将从新制定规则。
 <br/>
 <br/>
 在中共治下，从政治到法制，完全以中共利益为转移，毫无公正性可言。在高科技主导的未来世界，如果由中共主导规则，意味着西方发达国家的安全无法保障，主权也将逐渐丧失。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
</div>
