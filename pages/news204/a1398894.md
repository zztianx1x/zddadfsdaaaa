### 港媒：北京调整策略 悄然祭出另一套经济计划
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月11日讯】美国中期选举结束后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）在白宫举行的新闻会上披露，中方已经放弃〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞。有港媒称，中共从〝中国制造2025〞计划上撤下来，但又悄然祭出一项名为〝中国制造2035〞的政府新草案。
 <br/>
 <br/>
 11月7日中午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在白宫举行新闻会上谈到美中贸易时说，中国经济已经大大下滑。原本中国经济可能在两年内取代美国作为经济大国，现在已经无法接近美国。
 <br/>
 <br/>
 川普还提到〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国制造2025.html" target="_blank">
  中国制造2025
 </a>
 〞计划，他说，〝中国制造2025〞非常具有威胁性。〝我也已经告诉他们（中方）。我告诉他们‘中国制造2025’很有威胁性。〞川普还表示，中方已经放弃〝中国制造2025〞。
 <br/>
 <br/>
 川普表示，美国在往上走，中国经济在下滑。〝我不希望他们下滑。我们（川普和习近平）将举行一次很好的会议，我们将会看看我们能做些什么。〞〝我们将会尝试与中方达成协议，因为我希望和习主席以及和中国都有良好的关系。〞
 <br/>
 <br/>
 彭博新闻分析称，〝中国制造2025〞是阻碍解决美中贸易紧张局势的绊脚石之一，该计划改变了与美国的经济关系。
 <br/>
 <br/>
 旅美经济学家程晓农表示：〝‘中国制造2025’是要从西方获得技术，真正的目标是用〝中国制造2025〞去占领西方，特别是美国的市场，换句话讲，击垮美国的制造业，包括一枝独秀高科技产业。〞
 <br/>
 <br/>
 对于中共是否会放弃
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_2025计划.html" target="_blank">
  2025计划
 </a>
 ，时政评论员夏小强认为，这只是中共方面向川普承诺的放弃2025计划，并非中共已经放弃。
 <br/>
 <br/>
 他说，对于中共来讲，中共实行2025计划，并非富国强民，只是通过盗窃手段获得技术来超越取代美国，维持中共统治所需要的经济发展而已。
 <br/>
 <br/>
 放弃2025计划，等于放弃了中共掌控中国经济发展的工具，从某种意义上，等于削弱中共政权的稳定和统治。夏小强说，如果从这个角度来讲，中共的底线就是维持统治，影响到政权和统治的事情，中共绝不会干。中共不可能从根本上放弃2025计划。
 <br/>
 <br/>
 不过，香港《亚洲时报》报导，中共官员认为从〝中国制造2025〞计划上撤下来，北京可以安抚川普政府，让川普能在公共关系上取得成绩，同时让自己的工业计划安然无恙。
 <br/>
 <br/>
 报导说，中共当局正在悄悄探索多种方式来达成这样的协议，美国中期选举前，北京发布了一份名为〝中国制造2035〞的政府新草案，供政府智库讨论。一些政府顾问说2035年（而不是2025年）才是实现中共高科技雄心的目标。
 <br/>
 <br/>
 报导认为，这些分歧主要是口头上的，对实际政策影响不大。
 <br/>
 <br/>
 （记者李韵报导/责任编辑：祝馨睿）
