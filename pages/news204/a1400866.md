### 川习会前涌暗流？周永康儿媳突然发声指责官派律师
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月27日讯】中美贸易战令中共陷入执政危机，并由此引发中共内部激烈斗争。川习会前夕敏感时刻，已倒台的中共前江派常委周永康的家族成员，在沉默5年后突然公开发声，变相喊冤并影射北京当局，或显示中共政局不稳，暗流涌动。
 <br/>
 <br/>
 北京时间上周六（11月24日），《新京报》刊出对周永康案官方指派律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_郝春莉.html" target="_blank">
  郝春莉
 </a>
 的访谈。郝春莉表示，自己办理大多数涉高级官员的案子，都得到了当事人家属的委托。她特别提到，为周永康案辩护达到了〝比较理想的效果〞，且周永康〝认罪态度很好〞。
 <br/>
 <br/>
 不过，周永康的长媳、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周滨.html" target="_blank">
  周滨
 </a>
 之妻
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄婉.html" target="_blank">
  黄婉
 </a>
 随后在微信转发该文章，并质问郝春莉什么时候接触过周永康家属，是哪位家属委托她，以及她为周案辩护取得了什么样的〝好效果〞。
 <br/>
 <br/>
 黄婉在微信中公开指责郝春莉〝利用周永康案自我宣传〞，并称当年自己全家都遭控制，根本无权委托律师。
 <br/>
 <br/>
 据自由亚洲电台26日报导，周滨的辩护律师陈以轩向该电台记者证实，微信确系黄婉本人所发，但他没有透露黄婉的联系方式。
 <br/>
 <br/>
 报导称，这是2013年周永康被实际控制后，其家族成员首次发声。周滨至今还在监狱外，黄婉本人也曾一度被关押。
 <br/>
 <br/>
 报导还引述多个〝观察人士〞评论称，周永康倒台是因为权斗，被抓后没有人权，郝春莉是北京的工具云云。
 <br/>
 <br/>
 海外时政评论员唐靖远认为，黄婉公开指责律师，弦外之音就是周家都被控制，律师也好法官也好，都是官方指定，因此周永康的审判不公平，周入狱有冤。质疑官派律师，其实就是质疑当初审判的合法性，这背后的确有微妙的意味。而指责官派律师〝利用周案自我宣传〞，也有含沙射影之嫌。
 <br/>
 <br/>
 事实上，周永康及其同党薄熙来倒台后相当长一段时间，江派掌握的一众海外中文媒体也曾不断炒作，称薄、周被抓是因为〝政治迫害〞，一直遭受〝不公正对待〞等，以此掩盖两人企图推翻北京当局的阴谋，以及主导活摘器官等惊天罪恶，并试图为两人翻案。
 <br/>
 <br/>
 不过，在这个过程中，周永康家族成员从未公开发声。近期，随着中美贸易战引发中共危局，周家人开始逐渐走向台前。
 <br/>
 <br/>
 9月底，中共前公安部副部长兼国际刑警组织主席孟宏伟被抓时，被指为〝周永康余党〞。当时曾有〝周永康家属〞通过香港《南华早报》为此〝辟谣〞。黄婉这次出来指控官派律师，是周永康家族首次亲自发声。
 <br/>
 <br/>
 周家人这次露面的时机也很敏感。一周之后，中美首脑将在阿根廷会晤，商讨关系中共生死存亡的贸易战。据编辑部设在北京的外宣媒体消息，有关此次川习会的中方立场，北京内部经过激烈的争论。
 <br/>
 <br/>
 贸易战爆发以来，中共体制内对北京现当局内外政策的批评，显示出内部反对势力已经蓄势待发。在今年北戴河会议前夕，曾传出一波北京政变流言。虽然该流言被指可信度不高，但也足以证明中南海内部已暗流涌动。
 <br/>
 <br/>
 北京当局不顾国际影响，突然拿下在任的国际刑警主席的孟宏伟，外界普遍猜测北京或有紧急事件发生，很可能关系到中南海高层的权力斗争。
 <br/>
 <br/>
 而孟宏伟案牵出周永康家属罕见出面〝撇清关系〞，观察人士认为不乏此地无银之嫌，这可能显示孟宏伟落马与周氏家族，甚至背后的江派势力有扯不断的联系。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
