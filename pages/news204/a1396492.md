### 与郑晓松关联密切 港中联办主任缺席大桥通车仪式
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】北京领导人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 23日主持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠港澳大桥.html" target="_blank">
  珠港澳大桥
 </a>
 通车仪式，外传澳门中联办主任郑晓松及其前任、现香港中联办主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王志民.html" target="_blank">
  王志民
 </a>
 本被安排一同出席。但郑晓松离奇堕亡，官方迅速淡化处理，被指案涉机密。而王志民也未出席仪式，或有意淡化郑晓松事件。
 <br/>
 <br/>
 多家港媒报导，10月23日，港珠澳大桥开通仪式在珠海口岸旅检大楼举行，由
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 主持，中共广东省委书记李希、香港特首林郑月娥、澳门特首崔世安以及副总理韩正依次致词。
 <br/>
 <br/>
 出席仪式的还包括中办主任丁薛祥、副总理刘鹤、香港前特首梁振英等。
 <br/>
 <br/>
 报导中没有出现香港中联办主任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王志民.html" target="_blank">
  王志民
 </a>
 的名字。不过，王同日现身中共央视接受专访，大赞
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_珠港澳大桥.html" target="_blank">
  珠港澳大桥
 </a>
 〝 打通了大湾区的‘任督二脉’〞。
 <br/>
 <br/>
 而据此前消息，王志民和澳门中联办主任郑晓松，作为名义上中共中央和港澳间的联络人、实质上的港澳〝一把手〞，本已被安排一同出席大桥通车仪式。
 <br/>
 <br/>
 不过，郑晓松在通车前突然离奇堕楼死亡。王志民随后也缺席通车仪式，被怀疑有意避免出现香港中联办主任单独现身的尴尬局面，让人联想起郑晓松事件。而王志民现身央视，亦有将原本准备好的致辞变相发表的嫌疑。
 <br/>
 <br/>
 去年9月，王志民从澳门中联办主任调任香港，郑晓松接替了他在澳门的职务。
 <br/>
 <br/>
 《美国之音》报导指，大桥工程涉及粤港澳，郑晓松理应出席仪式，他的突然死亡，势必给活动蒙上阴影。
 <br/>
 <br/>
 港媒分析，中共官方启动应急程序，20日紧急宣布郑晓松因〝忧郁症〞于19日晚〝跳楼自杀〞，旨在〝从速善后〞，迅速淡化影响。
 <br/>
 <br/>
 另有港媒报导，郑晓松坠楼地点为澳门的中莲大厦，是中联办职员宿舍，距中联办大楼仅500米，保安非常严密。
 <br/>
 <br/>
 对于郑晓松在和习近平见面前突然死亡，外界普遍认为内有黑幕。有人认为是郑怕见习近平，也有猜测指有人怕他见到习近平。
 <br/>
 <br/>
 《美国之音》引述香港浸会大学高级讲师吕秉松的分析指，北京迅速给郑晓松的猝死定调，有稳住澳门中联办的意思，也有意避免有关材料外洩，因为背后肯定涉及〝更大的机密〞。
 <br/>
 <br/>
 有海外媒体指，港澳历来是江派曾庆红的势力地盘，郑晓松在习近平抵达前突然死亡，是曾庆红给了习近平〝当头一棒〞。
 <br/>
 <br/>
 据港媒报导，习近平22日开始的这次〝南巡〞，行程非常隐秘低调。在大桥通车仪式开始时，习近平迟到达半个小时，只用一句话宣布大桥通车即匆匆下台。而台下给他事先安排的座位，也罕见地没有放置名牌。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：）
