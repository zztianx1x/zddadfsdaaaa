### 国际刑警主席落马 北京通报一特殊表述引关注
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月08日讯】10月7日，北京当局发出通报称，已失踪多日的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 已落马。北京当局的通报称孟正在〝接受国家监委监察调查〞。据报，孟宏伟涉嫌北戴河政变案，因案件重大而紧急，北京才不顾国际影响而紧急抓人。
 <br/>
 <br/>
 10月7日晚，中央纪委国监委网站通报，中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部副部长.html" target="_blank">
  公安部副部长
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 涉嫌违法，正接受国监委监察调查。
 <br/>
 <br/>
 大陆公众号〝政知圈〞8日报导称，自2018年3月中共国监委成立以来，部级〝老虎〞落马通报的表述，基本都是〝接受中央纪委国家监委纪律审查和监察调查〞。
 <br/>
 <br/>
 而孟宏伟是第一个在通报中只写有〝接受国家监委监察调查〞的，而且通报中提及孟宏伟涉嫌〝违法〞，而不是〝违纪违法〞。此外，孟宏伟的落马通报比其他〝大老虎〞的通报内容短一些。
 <br/>
 <br/>
 报导说，中共国监委成立时，监察法也同期颁布。监察委的监督范围是所有行使公权力的公职人员。监察法调查的主要内容涵盖了公职人员的所有腐败行为。
 <br/>
 <br/>
 从目前通报来看，孟宏伟的落马原因，与以往落马〝老虎〞相比，确有其特殊之处。但相关详情尚未公布。
 <br/>
 <br/>
 孟宏伟是黑龙江哈尔滨人，历任中共公安部部长助理、交通管理局局长，公安部副部长、党委委员，国际刑警组织中国国家中心局局长（兼），中共国家海洋局副局长、党组副书记兼中共海警局局长等职。
 <br/>
 <br/>
 2016年底，孟宏伟当选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 ，任期将于2020年届满。
 <br/>
 <br/>
 除了上述职位以外，孟宏伟还是中共国家反恐办主任。2011年8月4日，中共国家反恐协调小组在乌鲁木齐召开反恐会议，孟宏伟曾经作为公安部副部长、反恐办主任出席并发言。
 <br/>
 <br/>
 孟宏伟是中共前政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 的亲信。周永康2003年3月出任公安部部长后，孟宏伟2004年4月升任副部长、党委委员，成为周永康的主要助手之一。周永康2013年落马后，他的多名公安部旧部也被查，包括副部长李东生、常务副部长杨焕宁、政治部主任夏崇源等人。
 <br/>
 <br/>
 对于此次孟宏伟落马，香港《苹果日报》称，孟宏伟可能涉嫌参与2015年的北戴河政变未遂案。文章说，因为案件重大而紧急，北京当局才不顾国际影响而紧急对兼任国际刑警组织主席的孟宏伟问话。
 <br/>
 <br/>
 据悉，2015年的北戴河会议前夕，中共前党魁江泽民和江派二号人物曾庆红，联合周永康的前秘书、时任河北省委书记周本顺等人，炮制了一份《河北政情通报》，试在北戴河会议期间，向习近平、王岐山发难。
 <br/>
 <br/>
 但在北戴河会议前夕，周本顺正在参加由中共国务院召开的〝京津冀协同发展工作推动会议〞期间，被中纪委突然带走调查。
 <br/>
 <br/>
 去年10月中共十九大上，河北省委书记赵克志接任中共公安部长一职，多次要求公安部继续清洗周永康的〝余毒〞。孟宏伟于2017年底被免除中共国家海洋局副局长、中共海警局局长的兼任职务，2018年4月，又被免去已担任14年的公安部党委委员一职。
 <br/>
 <br/>
 10月4日，孟宏伟的妻子向法国里昂警方报案说，孟宏伟9月29日返回中国后音讯全无。
 <br/>
 <br/>
 国际刑警组织7日发表声明称：当日位于法国里昂的国际刑警组织总秘书处，收到国际刑警组织主席孟宏伟先生的辞呈，立即生效。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：祝馨睿）
