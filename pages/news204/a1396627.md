### 习近平深圳视察 数万警进驻一小区
------------------------

  习近平
 </a>
 24日
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视察深圳.html" target="_blank">
  视察深圳
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_龙华区.html" target="_blank">
  龙华区
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_龙悦居.html" target="_blank">
  龙悦居
 </a>
 。有港媒称，龙悦居23日晚已开始
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戒严.html" target="_blank">
  戒严
 </a>
 ，居民出入均被严格限制，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商户.html" target="_blank">
  商户
 </a>
 亦要提早
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关门.html" target="_blank">
  关门
 </a>
 ，政府从外地调派数万
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警力.html" target="_blank">
  警力
 </a>
 ，挨家挨户敲门登记，且入屋〝陪聊天〞，防止有人闹事，以致于整个屋苑仿如〝死城〞。
 <br/>
 <br/>
 深圳
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_龙悦居.html" target="_blank">
  龙悦居
 </a>
 居民杨女士向《苹果日报》爆料，10月19日就收到通知，会有〝国家领导人〞于24日到访，连日来当局多次动员，通知居民24日上午不要返回住所，如果留在住所的居民就不要外出。
 <br/>
 <br/>
 当地政府还从外地调派数万名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_警力.html" target="_blank">
  警力
 </a>
 ，挨家挨户敲门登记，杨女士说：如果待在家里，就有警员一对一的陪聊天，主要是怕居民到中心会场闹事，当局还向每家每户，派发毛毯收买人心。
 <br/>
 <br/>
 报导说，由于龙悦居23日晚已开始
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_戒严.html" target="_blank">
  戒严
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_商户.html" target="_blank">
  商户
 </a>
 也要提早
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_关门.html" target="_blank">
  关门
 </a>
 ，以致整个屋苑如同〝死城〞，24日早已当局容许临街的一些商铺及食肆恢复营业，并找了一批疑似警察、公务员或经〝筛选〞信得过的市民〝扮演〞客人充撑场面。
 <br/>
 <br/>
 中午10时许，龙悦居附近一处空地有大批民众被拦在塑胶〝铁马〞。在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_龙华区.html" target="_blank">
  龙华区
 </a>
 民治街道社区老人日间活动中心，也有大批男女疑是被当局安排的人扮演龙悦居〝居民〞，近距离迎接
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 。
 <br/>
 <br/>
 当地警方还以保安理由，要求附近的大厦必须关窗开窗簾，物业管理处人员在习近平到访前逐一排查，有些住户离开时忘记关窗，被急召回家关窗，联系不到的住户，请来开锁师傅破门而入强行关窗。
 <br/>
 <br/>
 公开资料显示，龙悦居是深圳市的一个保障性住房项目，位于龙华新区玉龙路与白龙路交汇处，与金龙路共同围绕一至四期建筑物。
 <br/>
 <br/>
 住户为深圳市2007年度和2009年度保障性住房申请配售遗漏人员，以及该市参战退役人员、1983年底前集体转业到深圳的原基建工程兵。整个龙悦居分4期开发，被称为中国首家建筑面积超过20万平方米的工业化住宅小区。
 <br/>
 <br/>
 报导说，习近平选择到类似香港公屋的龙悦居探访，估计是为了体察民情，贯彻十九大报告中提及的〝坚持房子是用来住的、不是用来炒的〞定位，提出要加快建立多主体供给、多渠道保障、租购并举的住房制度。
 <br/>
 <br/>
 （记者李韵综合报导/责任编辑：李泉）
