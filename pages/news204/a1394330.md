### 3名中国人被中非打死  未见中共〝战狼〞式反应
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月06日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中非.html" target="_blank">
  中非
 </a>
 消息人士证实，至少3名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 ，10月4日在位于中非共和国西南部一处
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_矿区.html" target="_blank">
  矿区
 </a>
 ，被当地人乱棍打死，另外3名中国人重伤住院，目前生死不明。但值得关注的是，截止6日下午，中共官方仍未对该事件做出回应。这与此前在瑞典和英国发生的两起事件中，中共的〝战狼〞式反映形成鲜明对比。
 <br/>
 <br/>
 根据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中非.html" target="_blank">
  中非
 </a>
 共和国新闻报导，一个大约由10名
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国人.html" target="_blank">
  中国人
 </a>
 组成的团队，招募中非青少年领袖、国民议会第二副议长的弟弟伊格纳斯．迪姆贝勒．纳科（Ignace Dimbele-Nakoe）做向导，带他们前往一处金矿。
 <br/>
 <br/>
 不料想，他们所搭乘的独木舟在途中倾覆，导致这名年轻向导失踪，他的尸体后来被发现。
 <br/>
 <br/>
 该消息激怒了当地的居民，当这批中国人回到村里，向警察做陈述时，愤怒的村民用棍棒殴打这些中国采矿队人员，3人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_被打死.html" target="_blank">
  被打死
 </a>
 ，其他几个人都受了伤，当时现场只有少数警察。
 <br/>
 <br/>
 报导说，除了3名中国男子死亡外，医院方面称，另有3名中国男子受重伤，目前情况危急，在一家医院接受治疗。目前，死者的遗体已经被飞机运往首都班基。
 <br/>
 <br/>
 然而截止10月6日下午，中共外交部官网上，仍未见针对这3名中国人在中非被打死的回应。
 <br/>
 <br/>
 美联社报导说，中共驻班基大使馆没有回复记者所提出的评论请求。
 <br/>
 <br/>
 中共官方近来对外营造强力保护中国公民在海外权益形象，先前有大陆游客前往瑞典旅游时，因在旅馆外哭闹，遭当地警方驱离，随后，此事被中共上升为外交事件，中共驻瑞典大使馆与中共外交部多次为此发话。
 <br/>
 <br/>
 9月底，在英国保守党筹办的一个讲座上，央视女记者孔琳琳大闹会场，并且掌掴工作人员，遭英警以袭击罪拘捕。事件发生后，中共驻英国使馆发言人反咬一口，说央视记者受人身侵犯，并且呼吁英国保守党不要再〝干涉内政〞，还要求活动的主办者应向孔琳琳道歉。
 <br/>
 <br/>
 外界分析，中共的上述〝战狼〞式反映，目的是想再次激化民族情绪，让民众一致对外，转嫁中美贸易战带来的空前危机。但在当今网络信息发达的时代，大多数海内外民众了解真相后，不但未被中共蛊惑，反觉得上述两起事件太给中国人丢脸。
 <br/>
 <br/>
 但面对3名中国人在中非被打死，以及3人被打重伤，中共究竟将做何反应，值得关注。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉）
</div>
