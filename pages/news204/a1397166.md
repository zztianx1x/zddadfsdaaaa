### 央视新闻罕见破例 南海冲突局势难料
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】中日首脑会晤当天，央视新闻联播罕见未将相关新闻放在头条要闻播放，而是把
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视察.html" target="_blank">
  视察
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南部战区.html" target="_blank">
  南部战区
 </a>
 的消息放在头条。韩媒分析，这在一定程度上预示了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海.html" target="_blank">
  南海
 </a>
 情况不妙。除贸易战之外，如今中美在南海的冲突也面临升级。
 <br/>
 <br/>
 日本首相
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_安倍.html" target="_blank">
  安倍
 </a>
 晋三10月25日至27日对北京访问，被称为〝破冰之旅〞。26日是安倍
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_访华.html" target="_blank">
  访华
 </a>
 行程中最重要的一天。
 <br/>
 <br/>
 他先是于上午与李克强举行重要磋商，达成一系列协议，包括敲定约52项合作等。下午，安倍又与刚刚结束〝南巡〞返京的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 ，举行了80分钟的会谈，当晚习在钓鱼台国宾馆特设晚宴，款待安倍伉俪。
 <br/>
 <br/>
 10月29日，韩媒《东亚日报》报导说，习近平与安倍26日举行首脑会晤及晚宴的消息，并未成为当天央视新闻联播的头条要闻进行播放，当天的头条要闻播出的是，25日习
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_视察.html" target="_blank">
  视察
 </a>
 管辖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南海.html" target="_blank">
  南海
 </a>
 区域的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_南部战区.html" target="_blank">
  南部战区
 </a>
 的相关消息。
 <br/>
 <br/>
 而这完全打破了新闻惯例，一般而言，央视应首先播放习近平出席首脑会晤，再播放习接见南海官兵的新闻。而这一新闻惯例显然被央视颠倒。
 <br/>
 <br/>
 习近平在南部战区，要求各级指挥员带头加强军事斗争实践锻炼，〝大兴战争和作战问题研究之风〞，〝增强打赢现代战争实际本领〞。
 <br/>
 <br/>
 习近平还要求南部战区，大抓〝实战化〞军事训练，加强指挥训练和联合训练，加强检验性、对抗性训练，提高练兵备战质量和水平，准备应对各种复杂情况的最佳作战方案等。
 <br/>
 <br/>
 韩媒分析认为，习近平的上述言论直指中美间在南海的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_军事冲突.html" target="_blank">
  军事冲突
 </a>
 可能性。
 <br/>
 <br/>
 在南海方面，中美的矛盾也在不断升级，9月底刚刚出现了中美军舰差点发生冲突的事件。中美军事紧张局势不断升级之时，美军11月还将在南海、台湾海峡附近海域举行大规模实弹军演。
 <br/>
 <br/>
 人民日报27日也将习近平视察南部战区的新闻，放在要闻头条新闻的位置进行展示，充分展示了中共充分认识到南海区域中美冲突的可能性。
 <br/>
 <br/>
 美国霍奇斯将军24日在波兰的华沙安全论坛上甚至警告，美中很可能爆发太平洋战争，〝你们将看到我们继续在欧洲投资、训练、轮防以及驻防，以便做好准备，10年或15年之内我们可能要在太平洋打一场仗。〞
 <br/>
 <br/>
 美国总统川普近日也提出退出由美俄签署的《中导条约》一事，更是加大了有关太平洋可能发生战争的猜测。
 <br/>
 <br/>
 时评家文昭分析说，美国退出《中导条约》是推动中共加入军备竞赛的一步棋，它意味着川普要放手重整美国军备，在战略上，川普仍然是沿袭美国前总统里根的星球大战计划。当年苏联解体在很大程度上就是被里根这种军备竞赛拖垮的。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
