### 被指江曾家族白手套   徐翔狱中情绪差身体不好
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月09日讯】素有〝中国私募一哥〞之称的徐翔去年1月底因操纵证券市场及内幕交易罪被判囚5年零6个月，被罚没的款项合计超过200亿元（人民币，下同）。日前，徐妻应莹接受陆媒的专访透露了徐翔在狱中的一些情况，称徐〝情绪比较差〞，身体状况也不好。其妻并称，徐家人目前正正为被查封的资产奔波，希望当局能尽快甄别并解冻被查封
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_冻结资产.html" target="_blank">
  冻结资产
 </a>
 中的合法资产。
 <br/>
 <br/>
 中国大陆新媒体野马财经日前报导称，该媒体通过采访徐翔的妻子应莹，了解到徐翔案的最新进展情况。
 <br/>
 <br/>
 据应莹称，目前徐翔正在青岛监狱服刑，她每个月都要去青岛探望丈夫，而徐翔在狱中有空也在阅读一些书籍，但他〝因为面临各方面的压力和比较多的想法，自身情绪比较差，身体不是特别好。〞
 <br/>
 <br/>
 应莹称：由于泽熙公司的银行帐户从2015年11月1日后已全部被冻结，公司的公章、电脑和财务凭证等被查封带走，导致公司已无法正常经营，而公司员工的工资以及公司所欠的房租等一系列应付款项亦无法正常支付。
 <br/>
 <br/>
 她说：〝徐翔出事后，徐翔父母和我们家庭名下所有的银行帐户和资产，包括股权和房产等都被查封和冻结，这其中也包括我父母的唯一住房。在那之后，老人们情绪波动比较大，身体也都不太好，我父亲还长期住院。〞
 <br/>
 <br/>
 当被问及徐翔的家里老人情绪波动的原因时，应莹称：〝我们家庭的资产在判决前都受到查封、扣押、冻结。有一些属于合法资产，这些合法资产是我们家庭及我和徐翔夫妻共同合法财产，老人主要对这些查封有看法。〞
 <br/>
 <br/>
 她进一步解释说，〝徐翔案〞判决书中已认定徐翔的全部违法所得均已追缴，其它财产就都是合法财产。根据法律规定，这些合法财产应当返还给家属，并对夫妻财产进行分割。
 <br/>
 <br/>
 应莹还透露说，有一些徐翔的朋友因为受到徐翔案的牵连，资产也被冻结，至今未能解封。这些人曾多次到青岛市中级法院要求对相关资产进行甄别，但到目前还没有得到解决。现在大家都指望着青岛市中级法院能尽快完成资产甄别工作，把涉案的无关资产尽快解封。
 <br/>
 <br/>
 她表示，按照有关部门当初的统计，徐翔及其家人遭到查封的资产数额是足以覆盖所有罚款的，但是那些被查封的资产大多是金融资产，这几年股市低迷，导致这些资产都已贬值。
 <br/>
 <br/>
 应莹最后再次呼吁青岛市中级法院〝尽快甄别和公平对待尚处查封、冻结状态的合法资产〞，并表示自己希望徐翔能早日归来。
 <br/>
 <br/>
 徐翔是浙江宁波人，现年41岁，是中国金融交易市场知名的投资人。他因炒股〝快准狠〞而出名，被中国股民戏称为〝私募一哥〞。
 <br/>
 <br/>
 早年间，徐翔作为一名散户在宁波炒股而获得了巨额收益。2009年，徐翔成立上海泽熙投资管理有限公司并自任总经理，开始进军私募界。泽熙旗下的产品因多次取得私募冠军而受到中国资本市场的关注。
 <br/>
 <br/>
 2015年11月1日，徐翔因涉嫌非法手段获取股市内幕信息，从事内幕交易、操纵股票交易价格等违法犯罪，被公安机关刑事拘留；2016年4月29日，因涉嫌操纵证券市场及内幕交易犯罪，被公安机关依法批准逮捕。2017年1月23日，徐翔以操纵证券市场一审被判有期徒刑5年零6个月，被没收的违法所得逾90亿，并处罚金110亿元。
 <br/>
 <br/>
 当初徐翔被抓捕的消息在网络上传出后，有关徐翔的内幕消息开始井喷。据信，徐因擅长股市资本投资而成为了中共权贵家族的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_白手套.html" target="_blank">
  白手套
 </a>
 〞，利用大批受其控制的子公司或相关公司为中共权贵从股市大规模套利，其中就牵涉到内幕交易和人为操控股市的黑幕。徐翔被指主要是替江泽民、曾庆红、李长春和刘云山等家族服务，更有消息指徐翔和其后也已被拘捕的富豪肖建华是2015年内地股灾的幕后黑手，卷入了中共反习联盟发起的〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经济政变.html" target="_blank">
  经济政变
 </a>
 〞。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：秦锋）
