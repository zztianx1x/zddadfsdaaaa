### 兰州高速重大撞车事故 15人死亡31车受损
------------------------

<div class="wysiwyg">
 【新唐人2018年11月04日讯】中国甘肃省西北部兰海高速周六（11月3日）晚发生
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重大撞车事故.html" target="_blank">
  重大撞车事故
 </a>
 ，兰州南收费站一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重型卡车.html" target="_blank">
  重型卡车
 </a>
 失控，撞向等待缴费的一排车辆。截至目前，事故造成15人死亡、包括
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肇事司机.html" target="_blank">
  肇事司机
 </a>
 在内的45人受伤、31辆车受损。
 <br/>
 <br/>
 事故发生在周六晚7点半左右，一辆拉运塔吊的辽宁籍半挂车沿兰海高速公路由南向北行驶，经17公里长下坡路段，行驶至距兰州南收费站50米处，与排队等候缴费通行的车辆发生碰撞，造成至少15人丧生，在45人伤者中重伤11人。31部车辆受损。
 <br/>
 <br/>
 据
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_肇事司机.html" target="_blank">
  肇事司机
 </a>
 说，事故发生时因连续下坡，刹车失灵导致车辆失控，目前司机已经被拘押。
 <br/>
 <br/>
 而上周，中国重庆刚刚发生惨烈车祸，一辆公交车行驶于长江大桥时，突然冲向对面车道并坠入水深70米的江中。车上15人，目前已经发现13人遗体，还有两人仍失踪，警方上周五发布监控视频，车辆失控原因是一位女乘客与司机对骂并升级到肢体冲突。
 <br/>
 <br/>
 新唐人记者严枫、天羽综合报导
</div>
