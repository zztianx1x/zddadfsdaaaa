### 两律师依法为法轮功做无罪辩护 却被中共处罚
------------------------

<div class="wysiwyg">
 【新唐人2018年11月13日讯】首先来关注中国的消息。湖南两名律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡林政.html" target="_blank">
  胡林政
 </a>
 和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾武.html" target="_blank">
  曾武
 </a>
 ，在法庭上依法为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 学员做
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无罪辩护.html" target="_blank">
  无罪辩护
 </a>
 ，却遭到长沙市司法局停止执业六个月的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_行政处罚.html" target="_blank">
  行政处罚
 </a>
 。
 <br/>
 <br/>
 11月5号，长沙市司法局网站刊登了对
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_湖南律师.html" target="_blank">
  湖南律师
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡林政.html" target="_blank">
  胡林政
 </a>
 律师的处罚决定书。处罚书中说，2017年12月1号，云南省红河州开远市法院的庭审中，胡林政作为张某的辩护人发表：〝没有证据证明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_法轮功.html" target="_blank">
  法轮功
 </a>
 是x教，张某信仰法轮功是宪法36条赋予的宗教自由，限制法轮功信仰是违法行为。〞等辩护意见，司法局决定处罚胡林政停止执业六个月。
 <br/>
 <br/>
 湖南律师胡林政：〝主要是因为在法庭上，他指控我说做性质辩护，就是说法轮功不是邪教。我的辩护意见，我是说没有证据证明说法轮功是邪教，然后就被处罚了。〞
 <br/>
 <br/>
 与胡林政一起为法轮功学员张红英做
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_无罪辩护.html" target="_blank">
  无罪辩护
 </a>
 的湖南律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾武.html" target="_blank">
  曾武
 </a>
 ，也遭到司法局停止执业六个月的处罚。
 <br/>
 <br/>
 胡林政：〝我跟曾武律师，我们俩一起开庭的，他也被处罚了，在云南红河州开远市开庭，为张红英辩护的时候，因为我们两个发表了这种言论，曾武律师的辩护意见是，没有法律规定法轮功是邪教，结果我们两个都被处罚了。〞
 <br/>
 <br/>
 胡林政和曾武对于司法局的指控，放弃申辩和听证的权利。怕被找麻烦，胡林政强调他不接受采访。
 <br/>
 <br/>
 胡林政：〝现在这种情况下，你通过法院去起诉啊，或者申请覆议啊，我们认为不能改变这个结果，所以就不做这种无谓的工作了。〞
 <br/>
 <br/>
 法轮功是由李洪志大师在1992年传出的佛家上乘修炼大法，短短几年内传遍了中国大江南北，使上亿人身心受益。1999年，江泽民集团动用国家资源对修炼〝真、善、忍〞的法轮功学员展灭绝人性的残酷迫害，至今已19年，期间有几十位依法为法轮功学员辩护的大陆律师遭到打压。
 <br/>
 <br/>
 新唐人记者熊斌、钟元采访报导
 <br/>
</div>
