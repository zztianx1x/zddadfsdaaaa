### 黄兴国余毒？天津天房集团董事长邸达落马
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月24日讯】中共中纪委官网10月23日发布消息称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天津.html" target="_blank">
  天津
 </a>
 市房地产集团有限公司（下称
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天房集团.html" target="_blank">
  天房集团
 </a>
 ）前董事长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邸达.html" target="_blank">
  邸达
 </a>
 ，涉嫌严重违纪违法，正在接受调查。
 <br/>
 <br/>
 根据公开资料，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邸达.html" target="_blank">
  邸达
 </a>
 自1976年起，一直在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天津.html" target="_blank">
  天津
 </a>
 工作长达42年。1998年7月，邸达进入天津市房地产信托集团有限公司，担任总经理助理，后历任副总经理、党委副书记、总经理。自2014年6月至2018年8月期间，邸达任
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_天房集团.html" target="_blank">
  天房集团
 </a>
 党委书记、董事长。
 <br/>
 <br/>
 此前，天津市官网曾于8月31日通告，时任天房集团董事长邸达、总经理熊光宇，被同时免去现有职务。新董事长由前天津市国资委副主任王振宇担任。
 <br/>
 <br/>
 作为老牌国有房地产企业和天津楼市〝一哥〞的天房集团，自去年进行混改以来一直没有定论，加上经营的亏损以及巨额偿债的压力，使天房陷入了尴尬的僵局。
 <br/>
 <br/>
 此外，天房集团开发的一个18栋高端住宅的樾梅江项目，其主体建筑已全部完成，却因混凝土强度不符合设计要求将全部拆除重建。
 <br/>
 <br/>
 今年5月，港媒〝凤凰网〞报导称，天房集团约有1,800亿元（人民币，下同）债务，恐有违约风险。
 <br/>
 <br/>
 8月底，天房集团发布了2018年半年度报告。经营数据显示，2018年上半年，天房发展实现合同销售面积6.68万平方米，同比减少51.66%；实现合同销售金额16.13亿元，同比减少39.72%。
 <br/>
 <br/>
 据悉，天房集团仅控股子公司就有40多家，涉及产业包括土地整理、房地产开发、基础设施投资、建筑施工、新型建材等多个领域。
 <br/>
 <br/>
 有分析人士认为，天房集团近年内存在严重的债务问题，是邸达被调查的重要原因之一。
 <br/>
 <br/>
 也有分析人士指，邸达是天津市委前代理书记、天津市长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_黄兴国.html" target="_blank">
  黄兴国
 </a>
 嫡系，其落马很可能与清洗〝黄兴国余毒〞有关。
 <br/>
 <br/>
 公开资料显示，黄兴国2003年11月到天津任职，2016年9月被调查，在天津宦海沉浮近13年，历任天津市常务副市长、市长，市委代理书记。这13年中，天津市一直在大规模投入城市建设。从履历上看，天房集团前董事长邸达与黄兴国存在交集。
 <br/>
 <br/>
 2017年7月21日，大陆财新网发表长篇特稿，披露了黄兴国放任亲属插足天津、浙江等地的相关产业，尤其他的弟弟黄兴余、黄兴荣傍着黄兴国的权势，在房地产、开发建设、滨江道改造等领域里，充当掮客谋利。
 <br/>
 <br/>
 2016年曾有海外中文媒体引述消息人士的话透露，被外界称为是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_张高丽.html" target="_blank">
  张高丽
 </a>
 在天津的〝城建女总管〞的马白玉，交代了黄兴国利用职权违纪插手天津一些重大工程的内幕。黄兴国曾经暗示马白玉将多个天津市的重大工程项目，交由其亲属〝浙江帮〞参与承包，黄兴国的亲属或从中获取巨大利益。
 <br/>
 <br/>
 黄兴国被指是江派人马安插到天津市的一枚棋子。黄兴国主政宁波期间，宁波高速公路的各个出口，都竖起了江泽民的巨幅画像。有报导称，黄兴国通过巴结江泽民而迅速发迹。
 <br/>
 <br/>
 黄兴国与前天津市委书记张高丽关系密切，两人在天津官场内外相互抬举吹捧。而天津则一直受控于江派要员张高丽。张主政天津时大搞地产投资，此后天津房地产业投资进入疯狂增长时期，成为目前城建领域腐败严重的根源。
 <br/>
 <br/>
 （记者萧静综合报导/责任编辑：明轩）
