### 刘强东性侵最新细节 受害女生：他说我能像邓文迪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】中国亿万富翁、京东公司创始人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 ，在美国明尼苏达州涉嫌
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_强奸.html" target="_blank">
  强奸
 </a>
 再曝新细节，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_受害女生.html" target="_blank">
  受害女生
 </a>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_供词.html" target="_blank">
  供词
 </a>
 中揭露，当时刘强东侵害她时，她不顾一切的反对。刘还说一句〝你可以像
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓文迪.html" target="_blank">
  邓文迪
 </a>
 一样〞。
 <br/>
 <br/>
 据明尼苏达州当地媒体11月2日报导，8月30号晚，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘强东.html" target="_blank">
  刘强东
 </a>
 让助理订了一家日式餐厅，宴请同学和同行人。21岁的受害女大学生是10名志愿者之一，但女大学生发现只有她一个志愿者被邀请，所以她带着一名同为志愿者的男性友人前往。
 <br/>
 <br/>
 晚宴期间，女大学生坐在刘强东身边，被不停敬酒，女大学生被迫饮酒有点喝醉了。与她同来的男性友人在吃饭期间，两次被刘强东的助理叫走。
 <br/>
 <br/>
 晚上11点左右，刘强东把女大学生送回住所，也跟着下了车，并告诉司机和助理〝一会儿就回来〞。进入房间后，刘强东不顾女大学生的反对，有了进一步的接触，刘强东此时还说了一句话〝你可以像
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓文迪.html" target="_blank">
  邓文迪
 </a>
 一样〞。
 <br/>
 <br/>
 几次挣扎后，最后事情还是发生了。深夜2点零5分，女大学生发消息给朋友说〝我被他睡了〞〝我不是自愿的〞。她还给陪同她前往晚宴的男生发消息，该男报了警。
 <br/>
 <br/>
 刘强东8月31日晚间到达学校被逮捕。事后，女大学生也很害怕，不想启动调查。9月1日刘强东获释，女大学生在与朋友的讨论后，她最终决定控告刘强东。
 <br/>
 <br/>
 而刘强东提到的邓文迪，是传媒大亨
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_默多克.html" target="_blank">
  默多克
 </a>
 的第三任妻子，2013年已经离婚，被曝光是中共头牌间谍。
 <br/>
 <br/>
 2013年6月，泛华网援引北京内幕人士消息称，目前在全球范围内搞得沸沸扬扬、正在闹离婚的女强人邓文迪，真实身份竟然是中共总政治部联络部宣传局的头牌间谍！
 <br/>
 <br/>
 报导称，来自总参谋部的消息源说，总政目前正在大地震，以应对突如其来的离婚消息。
 <br/>
 <br/>
 消息称，邓文迪在大学一年级时，就被总政联络部在广州的分局吸收，试图培养成为外派香港的间谍。因邓文迪凭着过人手腕，在极短时间内取得美国绿卡，而获中共重视，决定予以培养。
 <br/>
 <br/>
 北京内幕人士说，当时正值中共收回香港主权，而控制整个香港媒体的任务，就是由总政负责。总政不仅直接操办凤凰卫视，还通过控制股份来操控香港报纸。对于一些西方传媒控制的香港媒体，将中共人员打入进去成为最有效的方法。
 <br/>
 <br/>
 北京内幕人士称，〝完全出乎总政的意料，那个女强人工作能力非常强大，不仅成为该卫视管理层中唯一的中国女性，而且开始与传媒王国的大老板谈上了恋爱，并最终成为其夫人。〞
 <br/>
 <br/>
 邓文迪两人离婚后，默多克也多次告诉那些肯听得进去的人说，其前妻邓文迪是中共间谍，从结婚到离婚都是。
 <br/>
 <br/>
 澳大利亚亿万富翁帕尔默（Clive Palmer）2013年在接受9号电视网访问时，也公开指称：邓文迪是中共间谍。他说：邓文迪已经秘密监视默多克多年了，还把钱送回中共情报当局。她是在中共南部接受训练的。这就是为什么默多克要跟她分手。
 <br/>
 <br/>
 而刘强东性侵女大学生时告诉对方可以像邓文迪，是否想让受害女大学生学邓文迪做间谍？引起外界猜测。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
