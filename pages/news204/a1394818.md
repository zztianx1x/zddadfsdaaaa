### 广州律师孙世华遭警方围殴脱衣羞辱
------------------------

<div class="wysiwyg">
 【新唐人2018年10月10日讯】广州
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_维权律师.html" target="_blank">
  维权律师
 </a>
 隋牧青的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙世华.html" target="_blank">
  孙世华
 </a>
 律师，近日在办理一宗涉及访民的刑事案时，遭警方构陷袭警并围殴，还被强制脱光衣服以证明身上没有武器。孙世华形容，这是她近二十年律师生涯中最黑暗、恐惧、屈辱的一天。
 <br/>
 <br/>
 10月7号，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孙世华.html" target="_blank">
  孙世华
 </a>
 在维权网公布自己的经历，事发在9月20号，她和委托人到广州华林街派出所，办理委托人丈夫的保释手续，但经办警察拒绝讨论案件，只说把申请书寄给他，于是孙查问对方的名字和警号，这时，这名警察突然把证件甩向她，她本能举手遮挡，却被指抢证和袭警，遭到一群警察围殴，随后被抓进办案区。
 <br/>
 <br/>
 隋牧青律师：〝这个案件重点在于构陷，即使按照他公布的情况，也就是说，拉扯那个工作牌，但事实上，连拉扯都没有。我要看一下工作证，看一下你的名字，这样才能寄给你取保申请，你这一甩来，我本能的用手一挡，你说我是拉扯工作证，这个是赤裸裸的构陷。〞
 <br/>
 <br/>
 到了办案区，警察以怀疑藏有凶器为由，喝令孙世华脱光衣服。孙世华形容，整个脱衣、裸身过程持续仅约20分钟，但对她却有如一生的漫长！之后她被审讯长达6小时，直到深夜12点才离开。事后孙世华得知，现场一位拍摄警察施暴画面的女访民也被强制脱衣搜身；而她的委托人也被强制在孩子面前脱光衣服。
 <br/>
 <br/>
 隋牧青：〝脱了3个女人衣服，还抓了一个男的梁某，他是没有脱衣服，脱衣服是因为怀疑你藏有凶器，一个壮年男性、一个无业男性，没有这个危险。说明什么呢？说明他们有脱女人衣服的爱好，而且这种脱衣服肯定是没有法律根据和合理的理由的，那就是一种羞辱。〞
 <br/>
 <br/>
 第二天，孙世华报警投诉，要求调查警方滥权，但督查警官称，若要取得案发现场的录音录像证据，必须向当事的华林派出所申请。10月10号，广州警方发布通报称，不存在孙世华等3人被警察殴打和羞辱的情况。
 <br/>
 <br/>
 隋牧青：〝最重要的问题在于什么，他自己公布结论，把自己做法官。这个，我们绝对不能接受。〞
 <br/>
 <br/>
 新唐人记者致电华林派出所和广州荔湾区公安分局询问案情，对方均以不方便、不清楚为由拒绝回答。
 <br/>
 <br/>
 孙世华在自述经历的文章中写道：我知道社会黑暗，但从未想到会黑暗如斯！我不知自己是否还有勇气继续执业。我根本没有资格同情弱者，作为女律师，我是弱者中的弱者。
 <br/>
 <br/>
 新唐人记者陈汉、陈洁、柏妮采访报导
</div>
