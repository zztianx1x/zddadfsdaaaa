### 王全璋妻李文足荣获国际〝人权奖 〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】中国〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_709案.html" target="_blank">
  709案
 </a>
 〞被抓捕律师
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 的妻子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 ，日前荣获2018年瑞典〝爱德尔斯塔姆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权奖.html" target="_blank">
  人权奖
 </a>
 〞（The Edelstam Prize），表彰其对于捍卫人权事业的杰出贡献和勇气。因李文足遭中共限制出境，奖项由他人代为领取。
 <br/>
 <br/>
 11月27日下午5点左右，2018年瑞典〝爱德尔斯塔姆
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_人权奖.html" target="_blank">
  人权奖
 </a>
 〞在瑞典斯德哥尔摩举行颁奖仪式，有200多人参加。中国流亡人权活动人士陈光诚的妻子袁伟静代替
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李文足.html" target="_blank">
  李文足
 </a>
 领奖。
 <br/>
 <br/>
 颁奖仪式结束后，自由亚洲电台记者采访了袁伟静。
 <br/>
 <br/>
 袁伟静介绍说，她是受李文足委托代为领奖的。她表示，颁奖典礼办的很好，安排的细节方面也很好。
 <br/>
 <br/>
 袁伟静说，典礼组织者注意到了李文足在大陆抗争时用的小红桶，所以他们也准备了很多小红桶，桶上写着〝爱〞这样的字。他们以这种方式来支持文足。还有一些孩子拿着红桶唱歌。
 <br/>
 <br/>
 另外，（基金会的）标志物也做成红绳，上面写着这个奖项的名字，每一个出席人的手上都系着一根这样的红绳。他们还在每一个出席典礼的人身上贴一个中文的〝爱〞字。
 <br/>
 <br/>
 袁伟静说，李文足没有在现场进行电话连线，但是事前发过来一个视频，说到丈夫
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王全璋.html" target="_blank">
  王全璋
 </a>
 被抓后，一开始她是如何担心、害怕，后来是如何抗争的过程。
 <br/>
 <br/>
 李文足的丈夫〝709〞维权律师王全璋，被中共当局抓捕关押已经超过3年，但当局一直不让李文足和律师会见王全璋。不仅如此，当局还对李文足采取长期监视和维稳措施。
 <br/>
 <br/>
 迄今为止，王全璋和李文足的5岁儿子泉泉，已经三年多没有见到爸爸，两年前还曾因为王全璋案被北京一家幼儿园拒绝接收。
 <br/>
 <br/>
 报导称，袁伟静的丈夫陈光诚以前在国内被迫害的时候，也有过和李文足类似的经历，被国保贴身监视、跟踪。
 <br/>
 <br/>
 袁伟静表示，李文足获奖，对于其他为了丈夫和家人的自由而抗争的人来说，都是一个很大的鼓励和支持，让他们更有信心地支持他们的丈夫和家人，为他们的自由而呼吁。
 <br/>
 <br/>
 〝这是有很大的帮助的。如果妻子和家人能支持维权律师和维权人士的话，对维权人士来说，也是一个很大的鼓励，这是一个良性的循环。〞袁伟静表示。
 <br/>
 <br/>
 据此前报导，王全璋被关押后拒绝认罪，坚持要求程海律师与李文足担任辩护人，但不获中共当局同意。王全璋是〝709〞案件中唯一至今未判决的当事人。这期间，李文足一直坚持不懈地向中共要求探视王全璋，维护自己和丈夫的合法权益。
 <br/>
 <br/>
 最近，自称是受王全璋委托的律师刘卫国在网上发布消息说，他和王全璋会见了两次，王全璋被控〝颠覆国家政权罪〞，目前被关押在天津第一看守所。
 <br/>
 <br/>
 李文足近日表示，得知王全璋的最新消息，对于他的处境更感忧心。她从刘卫国律师那儿得知，王全璋在关押期间被强迫服用抗高血压药物，会见过程中王全璋不敢大声讲话，而是用口型表达。
 <br/>
 <br/>
 日前，现年51岁的大陆维权律师余文生，也荣获德法两国政府联合颁发的〝人权奖〞。不过，余文生目前身陷囹圄，被中共以〝煽动颠覆国家政权罪〞羁押10个多月。余文生律师曾代理多起法轮功学员辩护案、〝709事件〞被抓维权律师案。外界相信，获奖对他现在的处境会有正面影响。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：戴明）
