### 北京贸战无计可施 专家猜测〝最后一张牌〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】中美贸易冲突经过几个月的攻防较量，美国政府节节进逼，而中共手中几乎无牌可打。日前有美国媒体引述专家分析称，中共可能将影响美国利率的能力作为最后一张牌，而退出购买或抛售美债的可能性最大。但港媒却对此不以为然。
 <br/>
 <br/>
 《纽约时报》10日发表的一篇署名评论文章称，越来越多的金融学家、经济学家和地缘政治分析家分析：中共可能把其影响利率的能力当作向美国总统川普打出的最后一张牌来使用，而具体的作法可能是公开退出购买美国国债，或在公开市场倾销中共所持有的美国国债。
 <br/>
 <br/>
 文章表示，人们通常会觉得这种做法在经济方面是非常不合理的，因为中共政府
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_抛售美国国债.html" target="_blank">
  抛售美国国债
 </a>
 虽然会导致利率上升而损害美国，但它同时会严重损害中共政府自己持有的美国国债。不过，在这个问题上，人们的常规思维也许〝有误〞。
 <br/>
 <br/>
 文章认为，因为中共政府没有更多美国进口商品可以征收关税了，它已经在把目标瞄准美国企业的交易，也就是说，中共手中已经没有什么可供选择的牌可打了，因此〝越来越多的金融学家、经济学家和地缘政治分析家正悄悄提出这样一种前景：中共政府可能将其影响利率的能力作为向川普打出的最后一张牌。
 <br/>
 <br/>
 文章分析称，中共政府在最后阶段并不一定要确保中国在今年或明年的财政健康，为了至少不会失去它已拥有的任何所谓优势，〝它可能愿意在今天承受点煎熬〞。
 <br/>
 <br/>
 中共政府目前是美国外债的最大持有者，持有总量已超过一兆美元。外界注意到，在中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 开打以来，中共政府正陆续减持美债。
 <br/>
 <br/>
 不过，也有评论人士对中共政府抛售美债以反制美方的可能性表示不以为然，他们认为，中国自身的经济太过脆弱，不能冒着风险作出任何可能引发政局不稳的事。
 <br/>
 <br/>
 香港《南华早报》此前曾引述北京智库天则经济研究所研究员张林（Zhang Lin，音译）的分析说，中共政府不太可能以抛售美债的方式来因应贸易战。
 <br/>
 <br/>
 那篇文章分析称，美国发行的国债总计约20兆美元，其中有6兆美元为外国投资者所持有，而中共政府持有近1.2兆美元的美债，虽然约占外国投资者所持比例的20%，但只占全部美国国债的6%。也就是说，美国政府部门持有债券的比例是中国的5倍，北京若决定抛售，美国也有足够的因应办法，所以即使北京真的低价抛售美国国债，也不会对美国造成多大的影响。
 <br/>
 <br/>
 反而对中方来说，中共政府可能已经将以美元为主的外汇存底中一大部分投资于美国国债，因此确保美元的价值对北京来说十分重要。北京如果真的低价抛售美国国债，还会导致中共央行的资产大幅缩水，弱化其财务槓杆和对世界舞台的影响力。因此中共政府如果抛售美国国债，肯定会伤到自己，却可能不会对美国造成太大影响。
 <br/>
 <br/>
 （记者竺颖综合报导/责任编辑：李泉）
