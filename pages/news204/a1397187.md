### 携带法轮功资料 八九学运领袖周勇军被捕
------------------------

<div class="wysiwyg">
 【新唐人2018年10月29日讯】日前，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_八九学运.html" target="_blank">
  八九学运
 </a>
 领袖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周勇军.html" target="_blank">
  周勇军
 </a>
 被广西警方逮捕，现羁押在东兴市看守所。据报导，当局以周勇军背包里查到法轮功真相资料为由，把他抓捕后正式逮捕。法律界人士相信，广西当局是陷害。
 <br/>
 <br/>
 据〝维权网〞10月25号报导：
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_八九学运.html" target="_blank">
  八九学运
 </a>
 领袖
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周勇军.html" target="_blank">
  周勇军
 </a>
 9月26号，被广西东兴警方正式逮捕。
 <br/>
 <br/>
 报导说，8月19号，周勇军与维权人士农定才准备从广西东兴过境前往越南游玩时，被边境检查站扣押，农定才稍后获释，周勇军则因被查到背包里有法轮功资料，遭刑事拘留。
 <br/>
 <br/>
 农定才事后对媒体表示，当时周勇军的包里有一张讲述法轮功真相的光盘，但是抓捕他的罪名是不成立的，他没有炼法轮功。再说，法轮功也不是见不得人的，是正常的信仰。
 <br/>
 <br/>
 维权人士李燕军说，警方抓人应该是有预谋的，周勇军在广西逗留期间，有很多人在监视他。
 <br/>
 <br/>
 大陆维权人士李燕军：〝我们那一天见面的时候，从里面（宾馆）出来，然后他忘记拿东西，又回宾馆拿，都看见很多人在楼道，在他住的宾馆房间门口，要是不忘记东西，不回去的话都不知道这个事。〞
 <br/>
 <br/>
 另一名维权人士谭爱军告诉新唐人，周勇军被抓，可能与广西维权律师覃永沛新开的律师俱乐部有关。
 <br/>
 <br/>
 大陆维权人士谭爱军：〝他来我们广西南宁的时候，他已经跟覃永沛律师谈好在这边工作了，跟覃永沛一起合作做生意的。〞
 <br/>
 <br/>
 周勇军原籍四川，被拘留后，广西当局派人到他成都的寓所搜查。律师覃永沛说，当局以他电脑有法轮功材料为藉口对他治罪，是陷害。因为按照中国法律，持有法轮功资料并不违法。
 <br/>
 <br/>
 周勇军是1989年北京学生运动领袖，1993年流亡美国，5年后回国被判处劳教3年。之后再遭拘捕判刑9年，2015年减刑获释。
 <br/>
 <br/>
 新唐人记者熊斌、元真采访报导
 <br/>
