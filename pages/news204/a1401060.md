### 英媒揭邓朴方讲话被封杀内幕  一句话激怒北京
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月28日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓朴方.html" target="_blank">
  邓朴方
 </a>
 早前在中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_残联.html" target="_blank">
  残联
 </a>
 换届大会上，大谈其父
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 功绩，被中共官方封杀，在海内外引起舆论关注。近日，有英媒曝光邓朴方的讲话之所以被封杀，是因为一句话激怒北京当局。
 <br/>
 <br/>
 11月27日，英国金融时报报导说，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓小平.html" target="_blank">
  邓小平
 </a>
 之子
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_邓朴方.html" target="_blank">
  邓朴方
 </a>
 ，9月在中共〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_残联.html" target="_blank">
  残联
 </a>
 第七次全国代表大会〞闭幕式上的内部讲话中，呼吁回归改革时代的重点是解决国内问题，同时保持稳定的外部关系。
 <br/>
 <br/>
 邓朴方在讲话中还暗讽当下中美贸易战、国内经济增长放缓。考虑到中国仍相对贫穷，而且国际上不稳定不确定等因素增多，他说：〝要害是把中国自己的事情办好。〞
 <br/>
 <br/>
 报导认为，邓朴方这些话一定是让北京当局不悦了，因为中共媒体没有报导此次讲话。
 <br/>
 <br/>
 实际上，邓朴方在讲话中还回顾了由其父邓小平推动的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_改革开放.html" target="_blank">
  改革开放
 </a>
 40年的〝历史进程〞。邓朴方还说，一定要有这种实事求是的态度，保持清醒的头脑，知道自己的份量，既不妄自尊大，也不妄自菲薄，坚持立足国情，从实际出发谋划工作。
 <br/>
 <br/>
 邓朴方的上述讲话是在中共残联换届大会上，选出他继续担任残联名誉主席时他发表的演讲。最初中共残联官网全文刊登了邓朴方的讲话。但从10月初，残联官网就很难上去。被认为是高层下了封杀令。
 <br/>
 <br/>
 美媒称，当年发动改革开放工程的邓小平，如今〝被消失〞，做儿子的邓朴方跳出来发声，网站都遭遇整体〝休克〞，印证了邓小平被彻底边缘化。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 10月下旬考察广东期间，强调改革开放，但避谈邓小平。而《人民日报》10月底刊发改革开放40周年社评，也只字没提邓。再加上日前习近平和
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王沪宁.html" target="_blank">
  王沪宁
 </a>
 有关改革开放的公开讲话中，同样没提邓小平。被外界解读为中共内部的〝路线之争〞再次加剧。
 <br/>
 <br/>
 （记者李芸报导/责任编辑：李泉)
