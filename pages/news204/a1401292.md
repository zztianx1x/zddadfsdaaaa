### 失踪马航内幕惊人 家属找到5片新残骸
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】载有239人的马航班机MH370
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 已经4年多。
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马航MH370.html" target="_blank">
  马航MH370
 </a>
 的搜寻任务早已结束，至今无解。英媒28日称，有失踪者家属称，找到客机残骸，将移交给马来西亚政府。之前媒体曝出的马航失踪内幕相当惊人。
 <br/>
 <br/>
 马航班机MH370遇难者家属Calvin Shim11月28日在英媒发简短声明称，找到5片
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 客机残骸，11月30日与马来西亚交通部长会面，并移交新发现的客机残骸。
 <br/>
 <br/>
 Calvin的妻子是MH370客机上一名机组人员，Calvin说，残骸是他8月份在马达加斯加附近发现。此前，马达加斯加附近也找到部份客机残骸。目前全球各地共收集了27片客机残骸，只有3片被冲上印度洋海岸的机翼碎片，确认是MH370客机残骸。
 <br/>
 <br/>
 2014年3月，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马航MH370.html" target="_blank">
  马航MH370
 </a>
 班机从吉隆坡飞往北京，在起飞后不到1个小时，就在马来西亚与越南海域交界处失联，机上大部分是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中国旅客.html" target="_blank">
  中国旅客
 </a>
 ，客机失踪10个月后，马来西亚政府正式宣布航班失事，推定机上239人全数罹难。
 <br/>
 <br/>
 4年多来，有多达25个国家参与搜救，但始终没有找到生还人员。
 <br/>
 <br/>
 去年9月6日，身在美国的中国富商郭文贵在视频中爆料称，马航MH370失联事件，是一起〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_政治暗杀.html" target="_blank">
  政治暗杀
 </a>
 事件〞，机上人员有中共公安人员和替江泽民之子江绵恒换肾的关系人，这些人的身份为中共官方人员，死后却被当成平民身份。
 <br/>
 <br/>
 郭文贵说，帮江绵恒和中共政法委书记孟建柱家人换肾脏的人，就是南京军区总医院副院长、中国工程院院士、国际著名肾脏病专家黎磊石。
 <br/>
 <br/>
 不过，黎磊石却在2010年4月16日从自家跳楼身亡。郭文贵提到的李保春。陆媒曾报导：2007年5月4日下午，44岁的著名肾脏病学专家李保春跳楼自杀。
 <br/>
 <br/>
 郭文贵说，参与江绵恒和孟建柱家人换肾手术的医生的家属和知情人，有跑到马来西亚的，中共警方办案人员前去劝回，回程班机失踪，中共政法委和孟建柱制造杀人灭口案，是为了避免江绵恒等人的活摘器官换肾真相洩露。这就是马航事件的真相。
 <br/>
 <br/>
 马航刚失踪时，中共驻马来西亚大使黄惠康在新闻发布会上曾表示，马航失踪涉及刑事案件，〝大家要有耐心，很多事情超出了想像和控制力。〞
 <br/>
 <br/>
 3月中旬，澳洲工程师彼得．麦克马洪（Peter McMahon）曾对英媒说，当局〝并不想找到飞机，因为飞机早已被子弹打得千疮百孔了。如果这一事实被证实，又要开启另一项调查。不过，澳洲官员否认此说法。
 <br/>
 <br/>
 （记者李芸综合报导/责任编辑：李泉）
 <br/>
