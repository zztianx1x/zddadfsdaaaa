### 揭秘：胡锦涛紧握习近平的手 流着泪交待4个字
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 执政10年，权力被
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 架空，中共十八大上胡
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_祼退.html" target="_blank">
  祼退
 </a>
 支持
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_掌权.html" target="_blank">
  掌权
 </a>
 。据港媒披露，胡在一次中共组织
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生活会.html" target="_blank">
  生活会
 </a>
 上发言时，仍悲愤难抑，导致血压速升中风，胡被送入
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疗养院.html" target="_blank">
  疗养院
 </a>
 时，他紧握习的双手眼泪直下，对习说出4个字。
 <br/>
 <br/>
 香港《争鸣》曾报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_胡锦涛.html" target="_blank">
  胡锦涛
 </a>
 2015年8月20日在中共组织
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_生活会.html" target="_blank">
  生活会
 </a>
 上发言时，悲愤难抑，致血压速升中风，胡锦涛被送医救治，诊断出帕金森症。胡锦涛对前去探望的中央政治局常委提出，他支持、接受公开病情及不保密等。
 <br/>
 <br/>
 胡锦涛出院后，继续接受中西医物理治疗，以恢复手足功能。胡说：〝总该趁有利条件和环境，把工作大事回顾、记录下来，对历史有个交代。〞
 <br/>
 <br/>
 胡锦涛在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_疗养院.html" target="_blank">
  疗养院
 </a>
 期间，还写了封致中央委员会暨退离休原中央委员的信，亲自交给前来探望的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 、栗战书。据称，胡锦涛悲愤心情仍难以平静。他紧握习近平双手，眼泪直下，并对习近平表示要〝勇往直前〞等。
 <br/>
 <br/>
 胡锦涛致中央委员会暨退离休原中央委员的信中，有12点建议和呼吁，主要内容包括：
 <br/>
 <br/>
 要带头申报、公开、公布个人、配偶、子女的工作、经济收入、财产来源；请求、建议撤销退休后总书记、国家主席办公室编制及规格；要推行、执行对中共高层政治问责和严重渎职的〝法律追究〞，要结束有规不行、有法不依的局面等。
 <br/>
 <br/>
 这12条建议中，多项是针对中共前党魁
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 。如建议撤销中共前总书记、国家主席办公室编制及规格；对中共党政高层官员的政治问责和严重渎职的〝法律追究〞等。
 <br/>
 <br/>
 <h4>
  胡锦涛关键时刻〝裸退〞
 </h4>
 <br/>
 胡锦涛执政10年，权力被江泽民架空。在中共16大上，江泽民发动军事政变，逼迫胡锦涛表态同意，并得以延任两年军委主席。两年后，江泽民被迫卸任，但继续通过自己安插的亲信掣肘胡锦涛，让胡、温的政令出不了中南海。
 <br/>
 <br/>
 江泽民退而不休，干预〝朝政〞已是尽人皆知。直到2012年中共18大时，胡锦涛以自身全退，欲终止江泽民干政。
 <br/>
 <br/>
 当时，不少舆论将此举比喻为〝胡锦涛学董存瑞炸碉堡〞，与一直想置他于死地的江泽民〝同归于尽〞，同时也给习近平全面接管权力开路。
 <br/>
 <br/>
 港媒披露，在2014年9月28日的中共党内生活座谈会上，胡锦涛侧重表达的两个方面是，批评江泽民乱政和提议对江泽民全面评价。
 <br/>
 <br/>
 报导还说，胡锦涛在会上曝光江泽民在2002年12月初、中共十六大后，突然在中共政治局常委会议上提出多项内则修订，其中包括中共政治局常委会内则和政治局内则。
 <br/>
 <br/>
 胡锦涛还表示，江泽民采取的不正当手段使中共常委会和政治局内部山头林立，重要事宜、政策出台后即夭折停顿。
 <br/>
 <br/>
 港媒2015年初也曾披露，胡锦涛在一次中共党内生活会上，痛揭自己在两届中共总书记任内，若干重大政策和问题均因争议和分歧被搁置，其中一个重要原因是退而不休的江泽民乱发表意见和建议，造成政策被搁置或被否决。
 <br/>
 <br/>
 同年北戴河会议前后，陆媒屡屡抨击〝老人干政〞现象。中共喉舌《人民日报》刊发的文章〈辩证看待〝人走茶凉〞〉引发热烈讨论，外界普遍认为此文是暗指江泽民。港媒说，这是习近平当局要斩断这些干政乱政的幕后黑手。
 <br/>
 <br/>
 据美媒披露，习近平接班前后，从背伤到腹痛至少十次遇险，多与江氏集团暗杀有关。港媒也披露，胡锦涛在位时遭遇江氏集团三次暗杀。提起江泽民，胡总是心情悲愤难以平静。
 <br/>
 <br/>
