### 国际刑警组织主席孟宏伟回中国后失踪  法国介入调查
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月05日讯】法国警方周五对外界透露，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 、中国公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 ，今年9月底前往中国后失去踪迹，其妻子日前在法国报案，法国警方已对此展开调查。
 <br/>
 <br/>
 据《美国之音》报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 与家人目前都居住在里昂。今年9月29日，孟从法国里昂乘飞机前往中国后即与家人失去联系。他的妻子日前告知法国警方，孟宏伟已
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 ，家人无法与他联系，不知他现在身在何处。法国警方目前对此事件展开了调查。
 <br/>
 <br/>
 就孟宏伟失踪事件，国际刑警组织仅对外表示，该组织已关注到有关报导，但这是法国警方和中国有关当局的事。该组织并对外声明，负责国际刑警组织的日常运作的是组织的秘书长Juergen Stock，所以孟宏伟的失踪不会影响到该组织的日常运作。
 <br/>
 <br/>
 除上述声明外，国际刑警组织没有进一步对孟宏伟的失踪作出评论。
 <br/>
 <br/>
 据公开的资讯，孟宏伟现年64岁，他于2004年4月升任公安部副部长，并从同年的8月起兼任国际刑警组织中国国家中心局局长。2016年11月孟宏伟当选
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织主席.html" target="_blank">
  国际刑警组织主席
 </a>
 时，曾引发国际人权组织的广泛批评和谴责。这些组织担忧孟宏伟作为广泛侵害人权的中共政府的高官，会令国际刑警组织成为在全球通缉和抓捕中国异见人士和政治难民的工具。
 <br/>
 <br/>
 孟宏伟在中国大陆失踪的消息传出后，外界注意到，孟宏伟2017年12月被北京当局免去其2012年3月开始兼任的中共国家海洋局副局长、海警局局长的职务；今年4月又被免去公安部党委委员的头衔，令外界猜测他的仕途可能遭遇了〝滑铁卢〞。
 <br/>
 <br/>
 鉴于孟宏伟是在前中共政治局常委、政法委书记周永康在2003年担任公安部部长后获得提拔的高官，外界猜测他可能是周永康的党羽。周永康本人已经于2015年6月11日天津市中级法院以受贿罪、滥用职权罪、故意泄露国家秘密罪，判处无期徒刑剥夺政治权利终身并没收全部财产，目前周正在北京昌平的秦城监狱里服刑。有舆论怀疑，孟宏伟这次在返回中国大陆后蹊跷失踪，或许是作为周永康的〝余毒〞 被北京当局〝拿下〞了。
 <br/>
 <br/>
 （记者何雅婷综合报导/责任编辑：明轩）
