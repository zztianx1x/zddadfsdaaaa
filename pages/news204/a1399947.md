### 〝请用新华社通稿〞 中共APEC输出新闻管制舆论大哗
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月19日讯】北京在亚太经济合作会议（APEC）峰会上遭遇围堵，中共官员强闯主办国外交部被撵，成舆论焦点。日媒日前披露了中方又一惊人举动，指
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 在APEC期间召集太平洋岛国开会，竟不许
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外国记者.html" target="_blank">
  外国记者
 </a>
 入场，还让他们引用新华社报导。
 <br/>
 <br/>
 《美国之音》19日报导，国际记者联合会发表书面声明，对中共官员提出严厉谴责，指其在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_APEC峰会.html" target="_blank">
  APEC峰会
 </a>
 期间，不准有采访资格的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_外国记者.html" target="_blank">
  外国记者
 </a>
 采访。
 <br/>
 <br/>
 声明说，11月12日至18日在巴布亚新几内亚举行的APEC会议期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 与太平洋国家领导人举办了一个论坛，有库克群岛、斐济、密克罗尼西亚、纽埃、萨摩亚、汤加、瓦努阿图和巴布亚新几内亚等国参加。中共官员以场地和安保为由，只允许中国媒体采访，而阻止已获得APEC采访许可的外国记者入场采访，并建议他们在报导时采用新华社的通稿。
 <br/>
 <br/>
 被阻止入场的，还包括APEC会议东道国巴布亚新几内亚的记者。
 <br/>
 <br/>
 国际记者联合会声明指，中共官员阻止外国记者采访作为APEC会议组成部分的论坛，侵犯了获取信息权和新闻自由。
 <br/>
 <br/>
 此外，新西兰国家广播电台报导，习近平在巴布亚新几内亚莫尔兹比港口一条公路开通仪式上演讲时，也只允许中共中央电视台录音。
 <br/>
 <br/>
 在此次峰会前后，美国已明确将联合日本和澳大利亚等盟友，加大对太平洋岛国的援助，以削减北京在这些国家的影响力。而北京也宣布继续在太平洋〝大撒币〞。
 <br/>
 <br/>
 外界猜测，在中美竞争的背景下，中方召集太平洋岛国开会，可能有一些〝不适合〞外界报导的〝私房话〞要说。
 <br/>
 <br/>
 近期以来，随着当局高调推进所谓〝人类命运共同体〞，北京有将国内轻车熟路的舆论审查向海外〝推广〞的趋势。
 <br/>
 <br/>
 据中央社报导，10月10日，中联部部长宋涛在中日执政党交流机制会议上强调，为了〝中日关系朝友好方向发展〞，日本政府要加强对日本媒体的〝管制〞，应成为媒体的〝领导者〞，〝引导〞有关中日关系的〝舆论形成〞。
 <br/>
 <br/>
 除了对外国媒体进行〝舆论管制〞，中方在APEC峰会上还有一个惊人的举动，那就是强行闯入主办国的外长办公室，直到保安人员介入后才不得不离开。
 <br/>
 <br/>
 多家外媒报导，美国在APEC峰会上谴责中共不公平的贸易行为，得到多数与会国家的支持。为了阻止会议联合公报中写入谴责北京的措辞，中共官员才如此霸道地〝求见〞巴布亚新几内亚外长，但遭到该国外长拒绝。
 <br/>
 <br/>
 事后，中共外交部官员指相关报导是〝散播谣言〞，但巴布亚新几内亚官方出面证实了此事。
 <br/>
 <br/>
 最后，因为中方阻挠，此次峰会第一次破例没有发表联合公报，而是授权主办国事后代为发表一个闭幕声明。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
