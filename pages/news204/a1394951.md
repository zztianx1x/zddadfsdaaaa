### 上证、深证暴跌创4年新低 千股跌停
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月11日讯】受
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 大跌、外资继续流出等因素影响，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上证综指.html" target="_blank">
  上证综指
 </a>
 跌5.22%，失守2600点，创2014年11月以来新低，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深证成指.html" target="_blank">
  深证成指
 </a>
 跌幅更达6.07%，两市逾千股跌停。
 <br/>
 <br/>
 据中央社报导，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上海证券交易所综合股价指数.html" target="_blank">
  上海证券交易所综合股价指数
 </a>
 （
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上证综指.html" target="_blank">
  上证综指
 </a>
 ）今天开盘报2643.07点，收盘报2583.46点，下跌142.38点，跌幅5.22%。成交量人民币1700.58亿元。
 <br/>
 <br/>
 深圳证券交易所成份股价指数（
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_深证成指.html" target="_blank">
  深证成指
 </a>
 ）开盘报7753.03点，收盘报7524.09点，下跌486.60点，跌幅6.07%。成交量1885.63亿元。
 <br/>
 <br/>
 鸿海旗下工业富联下跌9.99%，股价仅人民币12.34元，远低于发行价的13.77元。
 <br/>
 <br/>
 财新网引述香颂资本执行董事沈萌表示，A股大跌是受到
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美股.html" target="_blank">
  美股
 </a>
 及亚太股市下跌的影响。美股道琼工业指数前一夜大跌831.83点，跌幅3.15%，带动各国股市今天走跌。
 <br/>
 <br/>
 沪深两市今天开盘跌幅都逾3%，上证综指今天收盘前一度跌幅扩大到6.03%，深证成指收盘前也一度跌幅达6.59%，最后再小幅拉升。
 <br/>
 <br/>
 (责任编辑：夏明义)
</div>
