### 港媒：周永康7次跪求免死 供出〝重量级人物〞
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】中共前政治局常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 因〝受贿、滥用职权、故意泄露国家秘密〞三罪，被判处无期徒刑。多名专家指，周永康被免死轻判，说明背后有政治博弈，〝周有一些筹码〞。港媒此前曾爆料，周永康有7次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跪求免死.html" target="_blank">
  跪求免死
 </a>
 ，并供出〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_重量级人物.html" target="_blank">
  重量级人物
 </a>
 〞。
 <br/>
 <br/>
 2015年6月11日，北京当局通报，中共前政治局常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_周永康.html" target="_blank">
  周永康
 </a>
 已被判处无期徒刑。周永康被证实泄露了5份绝密级文件、1份机密级文件，周当庭表示认罪悔罪。
 <br/>
 <br/>
 对此，香港大陆问题专家梁国梁认为，法院判处周永康刑罚有多方考虑，一是犯罪的轻重，二是认罪态度，三是是否立功。从周永康被免死轻判来看，不排除其在审讯过程中，揭露了其它北京方面不知晓的问题。
 <br/>
 <br/>
 时政评论员丁律开也表示，周永康之所以会被轻判，应不仅仅在于其〝认罪悔罪〞，很可能是因为其有〝重大立功表现〞，即〝揭发〞了更大的〝老虎〞。
 <br/>
 <br/>
 时评人士陈破空此前曾断言，周永康若被免死，等于留下活口，为随时拿下其背后靠山
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 及其家族，留下重要人证。
 <br/>
 <br/>
 据港媒此前披露，从2015年1月19日至3月上旬，周永康被先后提审27次，记录提审时间112小时，周永康递交了22份交代材料。周被列出的条条证据，量刑都是从无期至最高死刑。
 <br/>
 <br/>
 报导称，周永康可能有预感死罪难逃，在被提审及最高检察院常务副检察长胡泽君见周时，周有7次
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_跪求免死.html" target="_blank">
  跪求免死
 </a>
 。
 <br/>
 <br/>
 报导还披露了周永康接受审查的一些内幕细节。当被问及与薄熙来的〝非组织政治活动〞时，周永康并没有直接回答问题，而是怒声斥责薄熙来是懦夫。周还声称，他不会指望江泽民插手干预，似乎在表达对江泽民的不满情绪。
 <br/>
 <br/>
 另有港媒披露，指周永康供出了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_曾庆红.html" target="_blank">
  曾庆红
 </a>
 。时事评论员夏小强此前曾发表评论文章称，周永康掌控中共政法委十年之久，掌握巨大情报网路资源和高层秘密，掌握了大量的江泽民集团的核心内幕，〝周永康的招供将会使得江泽民、曾庆红的罪证被当局掌握。〞
 <br/>
 <br/>
 夏小强分析，周永康已经是一只〝死虎〞，习近平当局审判周永康，其实用意并不在周永康。通过审判周永康，来为打击身后的曾庆红和江泽民作铺垫准备，才是真正目的。
 <br/>
 <br/>
 （责任编辑：唐颖）
