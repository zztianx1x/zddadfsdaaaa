### 中南海政局诡异 又一上海帮大佬携妻露面
------------------------

<div class="wysiwyg">
 【新唐人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 时间2018年10月08日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 令北京遭国际围困之际，中共政坛更显诡异。最新消息称，江派前政治局常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴邦国.html" target="_blank">
  吴邦国
 </a>
 在安徽合肥露面。而在中南海权斗中失势的前中共国家副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李源潮.html" target="_blank">
  李源潮
 </a>
 ，在江苏省盐城市现身。外界猜测这其中或有政治意涵。
 <br/>
 <br/>
 10月7日，中共海外党媒称，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_吴邦国.html" target="_blank">
  吴邦国
 </a>
 于〝十一〞期间，在祖籍合肥市肥东县露面，并且发布了一张吴邦国和妻子章瑞珍两人的合影。
 <br/>
 <br/>
 舆论分析，中共卸任领导人因身份特殊，鲜少出现在公众视野，对于他们的非官方公开活动，官媒普遍秉持不予报导的原则，因此他们露面通常藉助非官方渠道曝光。
 <br/>
 <br/>
 然而，这种有意安排的行踪，往往被外界认为另有政治意涵。
 <br/>
 <br/>
 同日，中国作家高伐林在推特透露，〝十一〞期间，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李源潮.html" target="_blank">
  李源潮
 </a>
 于10月4日携妻子高建进在江苏省盐城市现身，到他50年前工作过的地方参观。
 <br/>
 <br/>
 而在5天前的9月30日晚，李源潮和前中央军委副主席
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_范长龙.html" target="_blank">
  范长龙
 </a>
 ，同时现身中共国务院举行的十一〝招待会〞，两人都曾一度传出受到调查的消息。
 <br/>
 <br/>
 与此同时，在中共十九大卸任之后，一直销声匿迹的前中共政治局常委
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_刘云山.html" target="_blank">
  刘云山
 </a>
 ，也与9月27日突然露面，出现在河南嵩山少林寺。
 <br/>
 <br/>
 而上述几人均有浓厚江派色彩。此次在中共建政69年纪念日前后接连亮相，背后是否另有政治意涵，引人猜测。
 <br/>
 <br/>
 刘云山是江派对抗习近平的前台人物，中共十九大前，刘还在联手江派常委张德江持续针对习的政变计划，十九大退休后极少露面。
 <br/>
 <br/>
 吴邦国则是〝上海帮〞的主要成员，在中共十六大上，吴邦国从中共政治局委员一跃成为排名第二的中共政治局常委，并在次年接任中共全国人大委员长，主控所谓的立法机构。
 <br/>
 <br/>
 据说这是江泽民下台之前特意安排这种人事布局，意在架空胡锦涛，并维持江泽民集团在中共最高权力层的影响力。
 <br/>
 <br/>
 而李源潮的背景横跨团派、江派及太子党等多个派系，被指涉令计划案，今年3月两会上已裸退；同样在两会上卸任军委副主席的范长龙，先前爆出因涉徐才厚案，被军纪委立案调查。
 <br/>
 <br/>
 中共十九大前，习近平以反腐之名，拿下了包括徐才厚、令计划、周永康等众多江派要员，基本收拢了党政军大权。十九大一度被外界视为，是习近平经过5年打〝虎〞铺排、彻底拿下江泽民等政敌的好机会。
 <br/>
 <br/>
 但外界认为，在中共十九大之后，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_北京.html" target="_blank">
  北京
 </a>
 当局的打虎势头明显放缓。自此之后，中南海政坛鬼影幢幢，8月的北戴河会议前，甚至一度传出，中共老常委们要求开会解决中央主要领导问题，北戴河会议后，北京当局内忧未解，外患又至，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中美贸易战.html" target="_blank">
  中美贸易战
 </a>
 令北京召来国际社会的围堵。
 <br/>
 <br/>
 在外忧内患困境之下，习近平留下的政敌是否会藉机添乱？而江派人马的突然现身，是否与这波政坛波动有关？都是外界聚焦所在。
 <br/>
 <br/>
 （记者文馨综合报导/责任编辑：李泉）
