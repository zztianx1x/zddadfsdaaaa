### 前经纪人宋喆获刑6年 王宝强与律师合影乐开花
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月18日讯】10月18日，中国男演员
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王宝强.html" target="_blank">
  王宝强
 </a>
 的前
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经纪人.html" target="_blank">
  经纪人
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宋喆.html" target="_blank">
  宋喆
 </a>
 ，涉嫌职务侵占罪一案宣判，宋喆获刑6年，当天王宝强与其律师张起淮晒握手合影，两人喜笑颜开，背景板上还有〝乐开花〞三个字。
 <br/>
 <br/>
 根据法院通报，被告人
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_宋喆.html" target="_blank">
  宋喆
 </a>
 于2014年至2016年在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_王宝强.html" target="_blank">
  王宝强
 </a>
 （上海）影视文化工作室任职期间，单独或伙同被告人修雨乐，侵占王宝强工作室演出、广告代言等业务款共计人民币232.5万元（人民币，下同）。修雨乐参与侵占167.5万元。
 <br/>
 <br/>
 宋喆被指利用职务上的便利侵占单位财物；修雨乐则为宋喆提供帮助，构成宋喆的共犯。
 <br/>
 <br/>
 10月18日上午，宋喆获刑6年宣判后，王宝强与其律师张起淮晒握手合影，两人喜笑颜开，背景板上还有〝乐开花〞三个字。
 <br/>
 <center>
  <br/>

    王宝强与其律师张起淮晒握手合影，两人喜笑颜开，背景板上还有〝乐开花〞三个字。（微博图片）
   </font>
  </a>
  <br/>
 </center>
 <br/>
 宋喆是王宝强
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_离婚.html" target="_blank">
  离婚
 </a>
 案的涉事人，被指与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_马蓉.html" target="_blank">
  马蓉
 </a>
 有染。2016年8月14日凌晨，王宝强通过个人微博发表声明，宣布和结婚7年的马蓉离婚，原因是马蓉婚内出轨自己的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_经纪人.html" target="_blank">
  经纪人
 </a>
 宋喆。
 <br/>
 <br/>
 次日，王宝强到北京朝阳法院起诉马蓉，要求离婚。他称，马蓉有隐藏、转移夫妻共同财产，破坏家庭、伤害家人的恶劣行为；宋喆也存在隐藏、转移王宝强夫妻共同财产，破坏家庭、伤害家人的恶劣行为，致使双方感情彻底破裂。
 <br/>
 <br/>
 2017年8月16日，马蓉起诉王宝强侵犯名誉权。同年10月18日，王宝强离婚案在朝阳法院开庭审理。在起诉离婚后，王宝强查询自己的银行帐户，帐面余额只剩下十几万，其他的财产部分被转移。
 <br/>
 <br/>
 同年9月12日，宋喆因〝涉嫌职务侵占罪〞被警方刑拘。马蓉本人也被警方限制出境，其筹备已久的移民计划被迫终止。9月26日，宋喆等人因涉嫌职务侵占罪被批捕。
 <br/>
 <br/>
 据陆媒报导，宋喆被抓后，马蓉曾亲自联系过王宝强，求其放宋喆一马。
 <br/>
 <br/>
 2018年2月11日，法院一审判决王宝强马蓉离婚。宣判后，马蓉提出上诉。
 <br/>
 <br/>
 6月22日，北京市三中院再次开庭审理并当庭宣判，王宝强马蓉离婚案、名誉纠纷案两案，马蓉的上诉理由不能成立，驳回上诉，维持原判。
 <br/>
 <br/>
 （记者文馨报导/责任编辑：李泉）
