### 中共经济非常糟？川普：他们想达协议想的不得了
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月05日讯】美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 日前表示，中美贸易冲突刚刚打完〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前哨战.html" target="_blank">
  前哨战
 </a>
 〞，就让中共经济陷入困境，北京现在〝非常想〞和美方停战。不过，他也同时警告称，如果美中不能达成公平协议，美方将继续加码关税战。
 <br/>
 <br/>
 《美国之音》报导，周五（11月2日）下午，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 赶赴西弗吉尼亚州参加竞选活动前，在白宫对媒体谈及前一天的中美首脑通话。
 <br/>
 <br/>
 他说〝他们（北京）非常希望达成协议。他们想得不得了。我认为我们将和中国达成协议，我认为对每个人来说这都将是一项非常公平的协议，但对美国来说这将是一项好协议。〞
 <br/>
 <br/>
 川普表示，〝如果我们能够有史以来首次让中国（中共）敞开大门，公平行事，这早就该由其他总统做却没有做到的事，我愿意去做。〞
 <br/>
 <br/>
 川普还确认，中美双方同意在G20峰会期间会面，并共进晚餐。他说，事情已有进展，相信美中最终能达成良好协议。
 <br/>
 <br/>
 川普形容，对2500亿美元的中国产品加征关税只是〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_前哨战.html" target="_blank">
  前哨战
 </a>
 〞，而这已经让中共的经济〝非常糟〞。中方也明白，如果无法达成协议，将会面临美方在关税上继续加码。
 <br/>
 <br/>
 川普在周四一早和北京通完电话后，也曾在推特表态称〝进展良好〞。随后，这通电话和川普的表态，引来各种混乱的〝解读〞和炒作。
 <br/>
 <br/>
 中共官方开始放风〝中美各让一步〞，〝贸易战有望叫停〞。而美国彭博社周五则引述〝四名消息人士〞称，川普〝急于和北京和解〞，〝已下令白宫草拟协议〞。
 <br/>
 <br/>
 不过，美国CNBC记者随后引述白宫消息说，中美要达成协议〝还有很长的路要走〞。《华盛顿时报》也刊出周四专访川普的内容指，川普本人预计11月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 不会达成协议。
 <br/>
 <br/>
 白宫经济顾问库德洛也对CNBC表示，美方并没有起草协议，〝我们还没到达成交易的临界点〞，并且美中两国在正式贸易谈判中的立场仍然相距甚远。
 <br/>
 <br/>
 事实上，在中美首脑通话的前一天，中共政治局会议首次承认了经济下行压力加大。外界认为，这说明中共内部已对经济危局有了共识，北京在贸易战中妥协的倾向也在加大。
 <br/>
 <br/>
 不过，美方要求中共实现真正的贸易公平，必将危及中共的经济体制和政治安全。一些分析人士认为，北京将顽抗到底，暂时的示弱不过是一种拖延战术。
 <br/>
 <br/>
 而美国经济强势，遏制中共也已是美国朝野共识。对于一再誓言实现中美贸易公平的川普政府而言，对中共让步的可能性也越来越小。因此多有分析称，中美对抗或将持续升级。
 <br/>
 <br/>
 目前美国已经在经济、军事、外交等领域接连推出对中共的反击措施。伴随着美国副总统全方位谴责中共的重磅演讲，中美关系正在进入新冷战时代。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
