### 一次投票令江泽民心慌 牵出习近平接班背后内幕
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月13日讯】中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十七大.html" target="_blank">
  十七大
 </a>
 前的一次非公开的信任投票，江派选为十八大
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_接班人.html" target="_blank">
  接班人
 </a>
 的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 ，得票情况非常差，这一结果让
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
  江泽民
 </a>
 十分心慌，在不得已的情况下，江派暂时接受了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 作为接班人选，并和曾庆红、薄熙来策划了十八大后夺位的政变计划。
 <br/>
 <br/>
 <h4>
  一次令
  <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_江泽民.html" target="_blank">
   江泽民
  </a>
  心慌的投票
 </h4>
 <br/>
 据日媒透露，2007年6月25日下午3时，也就是中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_十七大.html" target="_blank">
  十七大
 </a>
 前夕，400多名中共高级干部在北京举行了一次非公开的投票，目的是考察将来有可能进入中共最高决策层、年龄在63岁以下的干部的能力。
 <br/>
 <br/>
 据悉，这种形式的所谓〝党内民主〞尚属首次，胡锦涛亲自主持了会议。
 <br/>
 <br/>
 投票结果至今未对外公布。一名中共党内人士（其亲属是原省部级高官）透露，当时的投票引发了一场风波。江派看好的时任商务部部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_薄熙来.html" target="_blank">
  薄熙来
 </a>
 的得票情况非常差，特别是军队内干部对其评价甚低，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 得票则位于前列。
 <br/>
 <br/>
 据称，这一结果让江泽民十分心慌，因为江泽民在执政时得到了原国务院副总理薄一波的支持，江成了薄一波之子薄熙来的后盾。
 <br/>
 <br/>
 当时，作为江泽民权力基础的〝上海帮〞的核心人物之一
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_陈良宇.html" target="_blank">
  陈良宇
 </a>
 因贪腐下台，习近平接任上海市委书记。陈良宇的旧部下与江泽民之间尚保持一定联系，而习近平主政上海后并未冷遇这些陈的旧部下。
 <br/>
 <br/>
 这让江泽民在不得已的情况下暂时接受了习近平作为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_接班人.html" target="_blank">
  接班人
 </a>
 选。
 <br/>
 <br/>
 这位党内人士认为：投票结果表明〝薄熙来在党内树敌较多，因此江泽民把目光投向了得票数居于前列的习近平。〞
 <br/>
 <br/>
 一名熟悉党内情况的原学者对记者说：〝江和曾是反共青团派的各种势力的利益代言人，这些势力包括党内元老、军队、‘红二代’等。〞而习近平正是这些势力的代表性人物。
 <br/>
 <br/>
 <h4>
  习近平接班背后藏惊人内幕
 </h4>
 <br/>
 据报导，由于中共高层各种因素制约，江泽民被迫选定习近平，作为十八大中共最高层接班人。但习近平不是江泽民和曾庆红的铁杆，其上台只是江曾的权宜之计。
 <br/>
 <br/>
 江曾的算盘是先在2007年阻止胡锦涛的接班人上台，在2007年～2012年期间内，让江、曾真正的接班人薄熙来锻炼成熟、取得威望和权势，在2012年的中共十八大上至少得到常委和中央政法委书记的位置。
 <br/>
 <br/>
 江曾预计在十八大后再经过二年左右的时间，利用薄熙来在全国通过〝唱红打黑〞取得的对于全国的挟持和操控，把〝重庆模式〞推向全国，再利用薄熙来掌控的全国政法委等强力政权、武警部队、以及全国众多被薄熙来掌握的军队、江泽民在军中的力量等，采取邓小平废黜华国锋等方式，罢免甚至逮捕习近平等人，到时候全国又是江曾的天下。
 <br/>
 <br/>
 本来江曾的计划得到了顺利的贯彻，薄、周已经完成了一半的进程，被原重庆市公安局长王立军出逃美领馆事件给曝光摧毁、全盘崩溃。
 <br/>
 <br/>
 重庆事件爆发后，薄熙来周永康等人政变计划，以及活体摘除法轮功学员器官的罪恶逐一曝光，牵涉出诸多中共党政军要员，引爆了中共高层的公然分裂与激烈权斗，直到现在，由此事件所引发的中南海震荡仍在持续中。
 <br/>
 <br/>
