### 港珠澳大桥惊爆大混乱 三口岸千人排长龙（组图）
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月29日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 通车后首个周末惊现大混乱场面。因大桥配套尚未完善，穿梭巴士班次严重不足，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_上千人滞留.html" target="_blank">
  上千人滞留
 </a>
 ，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三地口岸.html" target="_blank">
  三地口岸
 </a>
 的候车旅客大排长龙。有旅客苦等两小时才上车，为体验大桥赴港一日游的大陆游客称，这是〝花钱买罪受〞。还有港民直言〝以后就是有人送车票，我都不会再用港珠澳大桥〞。
 <br/>
 <br/>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_港珠澳大桥.html" target="_blank">
  港珠澳大桥
 </a>
 24日正式开通，香港、澳门、珠海三地有不少旅客乘坐巴士来体验这座跨海大桥。然而，不少人体验之后却表示非常糟糕。
 <br/>
 <br/>
 据香港《苹果日报》报导，10月28日，港珠澳大桥迎来首个周末，也是首个人流高峰日。但因来往三地的穿梭巴士——〝金巴〞班次严重不足，加上繁忙时段双层金巴太少，主要是单层金巴运行，载客量也严重不足，导致
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_三地口岸.html" target="_blank">
  三地口岸
 </a>
 出现混乱场面，人潮难以疏散，各口岸分别有逾千人大排长龙候车。
 <br/>
 <br/>
 报导称，当日早上在珠海口岸，大陆的林小姐在金巴站等了一个多小时才上车。
 <br/>
 <br/>
 在香港口岸，一位一日内往返港澳两地的谭先生称，早上出境还算顺畅，但金巴站等车的人就非常多，他也等了约一小时才等到车。

 <br/>
 另外，在澳门及珠海口岸，因为售票机不收现金，只收支付宝和澳门通等，不少乘客要排长龙到售票处购票，再排队搭金巴，乘客们纷纷抱怨〝太不方便〞。
 <br/>
 <br/>
 到当日傍晚7点左右，三地口岸人潮滞留情况更为严重。在香港口岸，有逾千人排长队等候金巴返回澳门或珠海，不少人面容疲惫。有为体验大桥赴港一日游的大陆游客称，傍晚6点已办好出境手续，但在金巴站苦等近两小时才等到车。
 <br/>
 <br/>
 来自顺德的女游客陈女士则直言，今天的香港一日游，简直就是〝花钱买罪受〞。
 <br/>
 <br/>
 在澳门口岸，也有逾千人包括记者排队等金巴，结果记者排队45分钟才上到金巴，其间有司警带着数名拖着行李的男女插队登车，引起乘客不满。
 <br/>
 <br/>
 上述的香港谭先生由澳门返港时，在澳门口岸等了50分钟才上到金巴，若以他从澳门酒店出发的时间计算，前后用了两个半小时才返回香港。
 <br/>
 <br/>
 另一位香港市民则直言，〝以后就是有人送车票，我都不会再用港珠澳大桥〞。
 <br/>
 <br/>
 对于港珠澳大桥惊爆人潮滞留的混乱的场面，香港民主党立法会议员林卓廷表示，有逾千人排队滞留是不能接受的，希望当局改善大桥配套设施。
 <br/>
 <br/>
 他相信，出现这种混乱情形，应该与大桥仓卒通车有关，〝巴士营办商都讲啦，他们找工人很困难，政府没有给足时间准备〞。
 <br/>
 <br/>
 事实上，在大桥开通前，已有不少香港议员认为，大桥配套尚未完善，仓促开通恐引发诸多问题。
 <br/>
 <br/>
 另有业界人士透露，港珠澳大桥三地口岸出现混乱场面，和珠海金巴司机集体〝吃饭〞脱班，疑似作无声抗议有关。
 <br/>
 <br/>
 据业界消息，港珠澳三地的司机薪酬各异，珠海金巴司机薪酬比香港和澳门的司机要低很多，司机们对此表示不满。但金巴运营商对此未给出回应。
 <br/>
 <br/>
 （记者罗婷婷报导/责任编辑：祝馨睿）
