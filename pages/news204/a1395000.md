### 美国会发布300页人权报告 痛斥中共犯下反人类罪
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月12日讯】
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国会.html" target="_blank">
  美国国会
 </a>
 周三（10月10日）出台的一份报告指，中国人权状况今年极度恶化。其中特别提到，中共在新疆关押迫害上百万的少数族裔民众，可能已构成
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反人类罪.html" target="_blank">
  反人类罪
 </a>
 。
 <br/>
 <br/>
 该内容出自是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国国会.html" target="_blank">
  美国国会
 </a>
 及行政当局中国委员会（Congressional-Executive Commission on China，英文简称CECC）发表的2018年年度报告。
 <br/>
 <br/>
 CECC是美国国会于2000年10月成立的独立委员会，其法律职能是监察中国法治发展和人权，每年向美国总统和国会提交一份年度报告。
 <br/>
 <br/>
 该委员会共同主席之一、来自新泽西州共和党籍众议员史密斯（Rep. Chris Smith, R-NJ）对《美国之音》表示，今年是委员会发布年度报告17年来，中国人权状况最糟的一年。
 <br/>
 <br/>
 报告指出，中国共产党近年来使用国家机器进行打压、监控和思想灌输，以极力维护对权力的垄断。
 <br/>
 <br/>
 报告特别谈到，中共政府在新疆将上百万的维吾尔和其他少数族裔关押进〝再教育营〞，是自二战以来对少数民族最大规模的关押行动，可能构成了危害人类罪。
 <br/>
 <br/>
 史密斯在报告发布记者会上说，中共对少数民族，特别是对新疆维吾尔族采取了自文革以来最严厉的镇压手段。
 <br/>
 <br/>
 报告还提到中共蚕食香港自由。其中引述不同团体指出，香港参选人被指面对政治审查及筛选，高铁〝一地两检〞被指严重削弱公众对一国两制的信心，反映〝一国两制〞承诺的香港自治和人权自由继续受到侵蚀。
 <br/>
 <br/>
 报告还提到中共镇压侵扰西藏人，焚烧圣经拆毁教堂，迫害基督徒等。
 <br/>
 <br/>
 史密斯表示，中共大力打压香港民主派、中国维权人士和劳工权利倡议者的行为，尽管是在人权标准低的中国，仍可说是胆大妄为。
 <br/>
 <br/>
 报告还指出，中共政府加强了使用现代科技渗入到民众生活，包括扩大收集生物识别数据，增加监控网络，以及大力发展社会信用体系。
 <br/>
 <br/>
 报告也显示，中共对社会和政治管理都与智能科技和人工智能的进步有密切关联，其中包括建造一套〝无所不在的监控系统〞。
 <br/>
 <br/>
 报告建议，美国须订定既有的相关法律，以集中国会资源应对中共大规模拘留新疆少数民族。报告又促请国会，尽快通过去年提出的数条相关法律草案，包括《香港人权和民主法》。
 <br/>
 <br/>
 该委员会另一位共同主席，来自佛罗里达州的共和党籍参议员鲁比奥（Sen. Marco Rubio, R-FL）表示，中国人权状况今年再次恶化，对美中关系产生了不利影响，并导致中国民众无法行使他们的基本自由权利。
 <br/>
 <br/>
 法新社报导说，美国国会民主党和共和党议员一致谴责，中共肆意大规模关押新疆维吾尔人的行为构成了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_反人类罪.html" target="_blank">
  反人类罪
 </a>
 。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
</div>
