### 党媒痛批意大利无视危机又〝撒钱〞 网络炸锅
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月15日讯】中共不顾国内经济危局，继续维持〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大撒币.html" target="_blank">
  大撒币
 </a>
 〞金钱外交，大陆民众日益不满。在此背景下，党媒喉舌日前痛批
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 政府〝无视危机又‘撒钱’〞，引起网友热议。
 <br/>
 <br/>
 10月8日，《参考消息》发表报导&lt;
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_意大利.html" target="_blank">
  意大利
 </a>
 政府无视危机又〝撒钱〞 每月向贫困者发放数百欧元&gt;。
 <br/>
 <br/>
 报导说，深陷国家财政危机的意大利近日有意率先在欧洲推行〝基本收入计划〞，即每月向失业和贫困者发放可观补贴（超过700欧元）。不过，报导称，这并不是一项普惠计划，约束条件非常严格。
 <br/>
 <br/>
 率先提出类似计划的是瑞士，但因为民众反对而作罢。其次是芬兰，去年刚刚挑选2,000名失业者开始试点。而其他欧洲国家目前还因为担心不良后果，拒绝踏上这条路。
 <br/>
 <br/>
 报导指，意大利经济状况明显不如瑞士和芬兰，并且已陷入巨额的国家债务，失业率也已爆表，但该国执政联盟两党之一的五星运动党却执意要推行该计划，而且走在芬兰的前面。
 <br/>
 <br/>
 该报导对意大利〝撒钱〞的批评态度，引起网友的吐槽。
 <br/>
 <br/>
 有网友称，〝人家给自己国家的穷百姓发钱也惹到中共了？〞并指《参考消息》此文分明就是对北京〝
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_大撒币.html" target="_blank">
  大撒币
 </a>
 〞的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_高级黑.html" target="_blank">
  高级黑
 </a>
 。
 <br/>
 <br/>
 网络活跃人士〝秀才江湖〞讥讽道：〝一个败家子嘲笑一个辛勤养家的人：你太不负责任了！你怎么可以把钱给家里人花！我都是请别人喝酒吃肉，不管家里人死活的。〞
 <br/>
 <br/>
 中共政权数十年来不顾国内人民生活困苦，坚持〝大撒币〞收买其他穷国，甚至包括很多独裁暴政的国家。这种做法如今在海内外遭遇越来越多的批评和谴责。
 <br/>
 <br/>
 在海外，中共的〝撒币〞援助被指为殖民、操控小国的经济陷阱，在这些国家越来越被抵制。而在国内，陷入经济困境的中国社会也越来越不以为然。
 <br/>
 <br/>
 近期，山东大学退休教授孙文广，因为批评海外〝大撒币〞不如给资助山区儿童上学、改善贫困民众生活，一度遭当地警方深夜绑架。
 <br/>
 <br/>
 （记者栗捷报导/责任编辑：明轩）
