### 川普与习近平进行〝非常好〞通话 长谈贸易和朝鲜
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月02日讯】中共官媒报导，中美元首刚刚通过电话，就双边贸易和朝鲜问题交换意见。美国总统
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 （特朗普）在推特表示，双方在通话中重点讨论了贸易问题，并称〝进展良好〞。
 <br/>
 <br/>
 美东时间11月1日早7点（北京时间1日晚7点）左右，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川普.html" target="_blank">
  川普
 </a>
 在推特发文称，刚刚跟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_习近平.html" target="_blank">
  习近平
 </a>
 进行了一个长时间并〝非常好〞的通话，双方讨论了很多话题，而贸易问题是重中之重。
 <br/>
 <br/>
 川普称，这些讨论向着阿根廷会晤的方向〝进展良好〞。此外，有关朝鲜问题也有一个良好的讨论。
 <br/>
 <br/>

 <br/>
 <div style="clear:both;display:block;">
 </div>
 <br/>
 据此前美媒消息，川普和习近平在11月底的阿根廷G20峰会开始前，可能将有一个短暂会晤。这次通话或是月底会晤的一次〝预热〞。
 <br/>
 <br/>
 北京时间11月1日晚11点左右，中共央视也报导了这次通话。
 <br/>
 <br/>
 在央视通稿中，川普和习近平都表示对阿根廷会晤表示期待。有关贸易问题，川普称愿意〝继续扩大对华出口〞，双方经济团队需要加强沟通。
 <br/>
 <br/>
 而通稿中用了两大段报导习近平有关贸易问题的表态。习近平表示，〝中方已就中美关系多次阐明原则立场〞，并再次强调北京不愿看到中美经贸关系出现〝分歧〞。
 <br/>
 <br/>
 习近平称，中美经济团队要加强接触磋商，以推动达成一个〝双方都能接受的方案〞。
 <br/>
 <br/>
 值得注意的是，在川习通话前一天（北京时间10月31日），中共政治局刚刚召开会议，首次承认了中国经济下行压力加大，有关经济局势的说法也从〝稳中向好〞变成了〝稳中有变〞。
 <br/>
 <br/>
 中共官方报导中称习近平〝应邀〞与川普通话。这与中方团队此前〝应邀〞赴美谈判的说法相同。但川普曾经多次透露，是北京希望进行谈判。
 <br/>
 <br/>
 有关11月底的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_川习会.html" target="_blank">
  川习会
 </a>
 ，川普日前表示，他认为中美可能达成一个公平的协议，而且必须公平。川普还说，如果谈不成，他还可以对2,670亿美元的中国商品加征关税。
 <br/>
 <br/>
 彭博社援引消息称，如果川习会不能缓解贸易战局势，美国准备最晚12月初宣布对所有剩余中国商品加税。
 <br/>
 <br/>
 美国白宫国家经济委员会主任库德洛10月31日对CNBC表示，川习会后对中国商品加税并非定局。
 <br/>
 <br/>
 他表示，川普在近期接受访问时曾说过，〝如果（会晤）一切进展顺利，或许会取消一些关税也说不定。〞但库德洛没有说明他指的是哪次访问。
 <br/>
 <br/>
 库德洛被指属于白宫温和派人士。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
