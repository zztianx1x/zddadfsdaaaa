### 习近平蓄势待发？港媒：四中会期失常引三大猜测
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年10月31日讯】中国共产党中央委员会第4次全体会议（简称〝四中全会〞）的会期，通常会安排在10月至11月之间，并提前两三个月就宣布会期。但今年应该举行的中共十九届四中全会的会期却迟迟未公布，再加上中共总书记习近平近期频频在公开场合〝失语〞、〝失态〞，引发外界的各种揣测。
 <br/>
 <br/>
 近期，身为中共党总书记的习近平接连几次在发表公开讲话时的表现都〝不在状态〞，引起外界的关注。
 <br/>
 <br/>
 其中最明显的是，习近平在港珠澳大桥通车仪式上，仅简单宣布大桥开通却不做任何其他政策意义之类的舆论宣导与引申。其后又有消息指习在与日本首相安倍晋三会谈时一度神情有异、出现失态的状况。外界还发现，习近平与安倍会谈结束后，竟在记者拍摄时间需要发言时照念讲稿，完全没有了过往侃侃而谈的自信与淡定。
 <br/>
 <br/>
 加之原本早就应该公布的中共四中全会的会期也迟迟未能定下，中美
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贸易战.html" target="_blank">
  贸易战
 </a>
 又日益加剧，中共似已陷入无力反击、内外交困的境地中，中共国内的政治、经济各领域也乱象横生，因此引发了外界对中共政治局势的种种揣测。
 <br/>
 <br/>
 日前，香港苹果日报发表了一篇署名评论文章，针对上述异常现象，就习近平在中共高层的权斗中是已处于弱势还是正蓄势待发的问题，提出了三个猜测：
 <br/>
 <br/>
 其一，习近平权力失控的问题，可能并未在今年的北戴河会议期间得到解决。
 <br/>
 <br/>
 文章认为，中共前总理朱鎔基、全国人大前委员长吴邦国、中共前政治局常委刘云山等近日接连现身，显示出中共老人政治的压力仍然存在，而习近平近期在公开场合出现的上述〝失常〞表现，也在一定程度上印证了习正面临很大政治压力的猜测。
 <br/>
 <br/>
 其二，习近平近日到广东〝南巡〞视察期间，虽有表态中国将会继续〝改革开放〞，但对邓小平的所谓〝历史贡献〞却并未提及。这个现象被认为是印证了外界有关邓小平长子邓朴方曾在中国残联讲话暗批习近平倒退回毛泽东时代的传闻并非空穴来风。
 <br/>
 <br/>
 文章推测，习近平对中国改革开放问题〝失语〞的现象背后隐藏的问题，要么是习可能已〝陷入了空前复杂的权力挑战格局〞，要么是习的执政活力已〝趋近穷尽〞，意味着中国政治的长期停滞阶段正在到来。
 <br/>
 <br/>
 其三，习近平及其亲信幕僚对川普和美国对华政策严重误判，以致贸易战中步步被动。因此，习近平当局现在只好采取〝以拖待变〞的策略，指望着川普以及共和党在中期选举中失利而获得贸易战的和解机会，所以四中全会要等美国的中期选举后才举行。
 <br/>
 <br/>
 文章也分析称，此前北戴河会议前中共政坛一度山雨欲来，有关习近平遭遇党内反对，其本人与嫡系人马失势的传闻甚嚣尘上，但会后习近平满血归来，核心地位依旧，更兼对美政策更为强硬，令人疑惑此次习近平是否再次蓄势？
 <br/>
 <br/>
 实际上，对于中共四中全会究竟何时举行的问题，路透社一周前曾引述北京消息人士透露称，中共一些地方要员已被要求要把11月份的时间留出来，预计四中全会将在11月上旬召开。
 <br/>
 <br/>
 该报导并分析称，北京方面之所以迟迟不对外宣布四中全会的会期，主要原因是在中美贸易谈判无果的情况下，北京希望在〝外部大的不确定因素稍有明确结果〞后，再确定后续的〝改革方案〞，而这个所谓〝外部大的不确定因素〞恐怕与
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 有直接关系，也即是说，四中全会可能会在11月6日以后召开。
 <br/>
 <br/>
 《自由亚洲电台》也曾在相关报导中分析称，北京当局正期盼着美国的民主党能在美国中期选举中重新掌控美国国会参、众两院，要等美国的这场选举结束后，才会针对贸易战最新形势来确定后续应对方案，所以估计中共的四中全会可能会安排在11月10日前后举行。
 <br/>
 <br/>
 不过，这篇报导同时也引述原香港城市大学政治学教授郑宇硕的观点表示，无论美国中期选举结果如何，对中共施压是美国民主共和两党的共识，所以美国的中期选举不应是四中全会延期的根本原因。
 <br/>
 <br/>
 郑宇硕认为，四中全会延期的真正原因是中共领导层意见不统一，北京当局在究竟要如何回应美国政府的要求等问题上仍〝举棋不定〞。
 <br/>
 <br/>
 值得一提的是，中共国务院副总理刘鹤曾接受媒体专访，当时他谈到中国经济仍然处在〝三期叠加〞的状态中，承认中国经济〝在内外因素的共同作用下，历史上积累的一些风险和矛盾正在水落石出〞，急需拿出新的应对方案。
 <br/>
 <br/>
 外界舆论认为，刘鹤的上述这番谈话，可视为对中共四中全会所要讨论的主要议题做的〝预热〞。
 <br/>
 <br/>
 （记者黎明综合报导/责任编辑：明轩）
