### 知情人：北京仍赌美选举 四中全会或延至明年
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月03日讯】中共十九届四中全会迟迟不开，引发关注。美媒引述知情人士消息称，北京还在等待
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 结果，然后才会制定贸易政策，四中全会或将推迟到明年。
 <br/>
 <br/>
 正在举行中的上海进口博览会要11月10日才结束，而
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美国中期选举.html" target="_blank">
  美国中期选举
 </a>
 也将在那段时间公布结果。因此有猜测称，四中全会很可能在10日前后开幕。
 <br/>
 <br/>
 不过，自由亚洲电台11月2日引述熟悉中共高层动向的人士透露，四中全会可能进一步延迟到明年。
 <br/>
 <br/>
 他解释说，北京正在等待中期选举结果，然后再制定应对方案。而方案制定需要数个月时间，因此今年举行四中全会的可能性越来越小。
 <br/>
 <br/>
 大陆学者张女士也认为，四中全会迟迟不开，与制定下一步对美贸易政策有关。据她分析，北京可能押宝川普（特朗普）在中期选举失败，然后再开四中全会，〝起码脸面有光〞。现在中国经济还在下滑，7月时中共政治局要求的〝6个稳〞，一个也没有稳成。
 <br/>
 <br/>
 张女士表示，今年加开了确认修宪的二中全会，又举行了三中全会，如果再开四中全会，一年三次全会就会显得比较反常。因此，四中全会开还是不开，能不能在今年开还是个问题。
 <br/>
 <br/>
 此外，也有分析认为，所谓〝等待中期选举结果〞的说法，可能只是个托词，真正原因是北京已经没有缓解经济危局的有效策略，因此四中全会没法开。按照中共惯例，此次全会的议题应该是经济改革。
 <br/>
 <br/>
 还有观点认为，四中全会推迟的原因是中共内斗激烈，政局不稳，党内还无法〝统一思想〞。
 <br/>
 <br/>
 （记者穆峰宇报导/责任编辑：明轩）
