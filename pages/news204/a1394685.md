### 中共深圳政法委书记李华楠被查
------------------------

<div class="wysiwyg">
 【新唐人2018年10月09日讯】中共官媒周二通报，深圳市委副书记、政法委书记
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李华楠.html" target="_blank">
  李华楠
 </a>
 ，因涉嫌〝严重违纪违法〞，正在接受调查。
 <br/>
 <br/>
 通报没有说明
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_李华楠.html" target="_blank">
  李华楠
 </a>
 违法违纪的具体原因。
 <br/>
 <br/>
 据公开资料，1997年8月起，李华楠先后任深圳市法制局副局长、局长、党组书记、市法制办主任、党组书记、市人大副主任等职；2010年6月起，任深圳中级法院院长、党组书记、市委常委、秘书长；2015年5月起，任深圳市委副书记、政法委书记等职。
 <br/>
 <br/>
 另据《明慧网》报导，李华楠积极追随江泽民集团迫害法轮功，对当地迫害法轮功学员负有责任，被多次通报。
 <br/>
 <br/>
 海外〝追查迫害法轮功国际组织〞，也将其列入迫害法轮功学员责任人，对其进行追查。
</div>
