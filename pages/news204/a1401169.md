### 贺建奎露面回应基因编辑婴儿 自相矛盾回避尖锐提问
------------------------

<div class="wysiwyg">
 【新唐人北京时间2018年11月29日讯】因为编辑婴儿基因饱受全球舆论谴责的南方科技大学副教授
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 ，28日在香港如期出席了第二届人类基因组编辑国际峰会，对其基因编辑项目进行现场说明。但其说法多处自相矛盾，对一名教授提出的尖锐问题避而不答。
 <br/>
 <br/>
 在当天的峰会上，主持人给了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_贺建奎.html" target="_blank">
  贺建奎
 </a>
 一个专场讨论
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_基因编辑婴儿.html" target="_blank">
  基因编辑婴儿
 </a>
 项目的机会。贺建奎在演讲中，对其项目情况进行了简单的说明。
 <br/>
 <br/>
 贺建奎称，刚出生的基因编辑双胞胎中，其中一个在胚囊阶段就已出现脱靶效应（即基因编辑失败，出现异常），但在〝父母了解到相应的风险后〞，贺的团队仍然将两个胚囊都植入了母体。
 <br/>
 <br/>
 在讨论会的讨论环节和问答环节，贺建奎回应了部分外界质疑，并回答了现场专家和记者的一些问题。但其多处说法和公开信息不符。
 <br/>
 <br/>
 贺建奎声称，不应该在没有完成同行评议的时候提前公布这个项目，〝这个新闻是不小心提前洩露出去的，所以一切都是我的预料之外〞。
 <br/>
 <br/>
 但中国基因编辑婴儿事件之所以迅速发酵，始于贺建奎在26日，即峰会开幕前一天接受美联社独家专访，高调公布婴儿诞生的消息。因其太过敏感和轰动，迅速引来全球舆论谴责。
 <br/>
 <br/>
 美联社在全球拥有巨大影响力，贺建奎选择在峰会前一天对美联社发布消息，显然是为了制造轰动效应。
 <br/>
 <br/>
 另外，有关项目经费，贺建奎声称，参加实验的志愿者的医疗费用，是其本人负担，其它研究经费则由南方科技大学提供。但他声称，为其提供经费的南方科技大学对其实验〝完全不知情〞。
 <br/>
 <br/>
 贺建奎还强调，他名下的所有企业〝都没有为项目提供资金〞。
 <br/>
 <br/>
 但据贺建奎研究室官网的一份基因项目知情同意书，项目组为参与实验的每对夫妇提供28万元人民币临床费用。而贺建奎在演讲中透露，共有8对夫妇参与项目，其中一对中途退出。按7对夫妇参与来计算，医疗费用总计196万人民币。如此巨额费用全由贺建奎个人承担的说法，令人生疑。
 <br/>
 <br/>
 根据中共官方的中国临床试验注册中心网站的注册信息，贺建奎团队相关项目的研究实施单位为南方科技大学，而项目的经费或物资来自深圳官方的市科技创新委员会资助的〝科技创新自由探索项目〞。
 <br/>
 <br/>
 事后，南方科技大学和深圳市科技创新委员会都高调否认和贺建奎项目有关。
 <br/>
 <br/>
 在讨论会现场，有专家质问贺建奎，如果是自己的孩子，他会不会这么做。贺建奎回答说，〝如果我的小孩有先天缺陷，我会首先做这样的尝试。〞
 <br/>
 <br/>
 但据贺建奎的介绍，开展该基因编辑项目的目的，是让婴儿能先天免疫艾滋病，接受其实验的夫妇都是丈夫感染艾滋病，而妻子没有感染。而稍有医学常识的人都知道，虽然母亲在怀孕和分娩期间将艾滋病毒传染给婴儿的机率比较大，但艾滋病毒不能感染精子和卵子，因此艾滋病根本不会遗传，父亲患艾滋病并不会影响胎儿。
 <br/>
 <br/>
 虽然贺建奎声称其招募的志愿者〝知识背景很好〞，但外界怀疑其团队是通过欺瞒的手段，骗取不知情的人来为其充当小白鼠。
 <br/>
 <br/>
 在现场，还有人追问基因编辑婴儿涉及的伦理问题。贺建奎的回答是，〝我从美国回来，不熟悉中国在这方面的规定。〞
 <br/>
 <br/>
 而事实是，在美国，基因编辑婴儿不但违反医学伦理，美国还有专门立法禁止。据外媒报导，贺建奎团队还招募了一名美国科学家。此人之所以来中国参与该项目，就是因为美国法律不允许进行相关实验。
 <br/>
 <br/>
 此外，现场还有专家质问有关基因编辑风险的关键专业问题，贺建奎则答非所问。
 <br/>
 <br/>
 这名专家指问，贺建奎团队选择编辑基因序列中的
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_CCR5.html" target="_blank">
  CCR5
 </a>
 部分，这是否一个可靠的靶点，该团队是否足够了解CCR5的功能，编辑CCR5是否会引起其他复杂的情况，比如会不会导致人体更容易感染西尼罗病毒或者流感病毒等。
 <br/>
 <br/>
 他指出，CCR5在免疫系统里，一定还有其它和HIV感染无关的功能，编辑CCR5可能导致未知的健康问题。他再次质问，贺建奎团队是否真的了解CCR5和它在免疫系统里的功能。
 <br/>
 <br/>
 而贺建奎回答称，其团队研究CCR5，是因为它是一个〝简单且受人了解〞的基因。
 <br/>
 <br/>
 不过，从现场专家的提问来看，对世界上其它国家的顶级医学家来说，CCR5并非贺建奎所称的那样〝简单且受人了解〞。
 <br/>
 <br/>
 贺建奎26日曝出中国基因编辑婴儿的爆炸性新闻，给医学界带来巨大危机，引发科学界的迅速反应。包括122名海内外华人科学家在内，都对其〝疯狂〞行为进行了严厉的谴责。
 <br/>
 <br/>
 这让中共官方陷入非常被动的境地。虽然该项目涉及的多家官方和非官方机构都迅速出面〝撇清关系〞，但很多人怀疑，贺建奎如此高调的背后，少不了中共官方的大力支持。
 <br/>
 <br/>
 根据网络曝光的《深圳和美妇儿科医院医学伦理委员会审查申请书》，贺团队开展此项目的目标是：建立基因手术治疗行业的〝质量控制标准〞，占领整个基因编辑相关治疗技术门槛的〝制高点〞，在国际竞争中〝脱颖而出〞。
 <br/>
 <br/>
 而北京近年来推出的所谓〝中国制造2025〞和〝中国标准2035〞计划，其目标正是不择手段地夺取全球尖端科技领域的主导权，并由中共来制定行业标准。
 <br/>
 <br/>
 （记者栗捷综合报导/责任编辑：明轩）
