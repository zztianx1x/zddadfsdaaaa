### 文武：孟宏伟找到了吗
------------------------

<div class="wysiwyg">
 9月29日，
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 了，孟宏伟是中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部.html" target="_blank">
  公安部
 </a>
 排名第三的副部长，也是中共现在在任的任职年数最长的副部长，孟宏伟从2004年起任中共公安部副部长至今已有14年，算得上是个资深的老公安了，孟宏伟不光是中共的公安部的副部长而且还是国际刑警组织的主席。国际刑警组织设在法国里昂，是世界各国警察机关在国际上合作的一个机构，这一届的国际刑警组织的主席就是孟宏伟。
 <br/>
 <br/>
 由于
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 是国际刑警组织的主席，所以孟宏伟的家也就在法国里昂。可是9月29日，孟宏伟说是回到中国后，孟宏伟的家人，孟宏伟的妻子就联系不上孟宏伟了，孟宏伟的妻子到处打听也打听不到孟宏伟的下落，音讯全无，这一下子，孟宏伟的妻子就着急了，前两天，她就跑到法国里昂的警察处报案，说孟宏伟
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_失踪.html" target="_blank">
  失踪
 </a>
 了，人不见了。
 <br/>
 <br/>
 法国警方立即开始寻人，结果得出的结论是，孟宏伟确实于9月29日到了中国北京，然后就不知下落了，就是说孟宏伟在北京失踪的，这就与法国没有关系了。
 <br/>
 <br/>
 一个中共
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_公安部.html" target="_blank">
  公安部
 </a>
 的副部长怎么会在北京突然失踪呢？一个大活人怎么一下子就音讯全无了呢？这可是个千古奇案了。
 <br/>
 <br/>
 大家都知道，中共的公安在中国是最有实权的部门，基本上是想抓谁就抓谁，只有公安的让别人突然失踪，怎么还有人让公安部的副部长突然失踪呢？
 <br/>
 <br/>
 这对于一个正常的社会来说是无法理解的，不说让一个公安部的副部长突然失踪，就是让一个普通的平头百姓突然失踪，在西方法制社会里都是一件不可思议的事情，就算一个人犯了再大的罪，在抓捕时，那也是要法律手续的，抓到哪里去了，什么时候抓的，也都会告知家人。
 <br/>
 <br/>
 可是中共却不同，抓个人象抓个蚂蚁一样，说抓就抓了，什么时候抓捕的，为什么抓捕，关在哪里，全不说，先抓捕了再说，根本不管什么法律程序不程序，根本不管什么家人不家人，也根本不去管什么社会上的反应，更不会去管什么国际上的反应，这就是中共，中共就这样，任何法律、任何规矩在中共这里全都是零。
 <br/>
 <br/>
 现在一些跟着中共跑的人，特别是一些警察，盲目的听从中共的命令，所谓的上级命令，叫做什么就做什么，叫抓捕维权人士就抓，叫抓法轮功的学员就去抓。而这些人全都是无辜的，全都是无罪的，他们只是在用和平、合法的方法，维护自己应该有的权力，就因为中共不喜欢这样的人，就叫警察去迫害这些人，可是一些良心丧失了的 警察，就听从中共的，就冲上去前迫害这些善良的中国人。
 <br/>
 <br/>
 可是帮助中共迫害了中国百姓后，这些参加迫害的人，中共能保你们平安吗？中共能保你们永远升官发财吗？
 <br/>
 <br/>
 周永康都当上中共的政法委书记了，算得上是最大的中共政法的官了吧，可是他迫害完了中国百姓后，他自己的结果呢？不是也进了秦城监狱了吗？孟宏伟都当上中共公安部的副部长了，当上了国际刑警组织的主席了，可是结果呢?却是失踪了，却是音讯全无了。
 <br/>
 <br/>
 何况一般的警察，何况一般的官员，那中共要整死你们的时候，那不就是随便整死吗？跟着中共走，哪里还有活路？只有死路一条。
 <br/>
 <br/>
 所以，孟宏伟失踪了，不用找了，结果无外就是，要么从此人间蒸发了，地球上从此没有这个人了；要么是被双规了；要么是传来死讯，说是自己上吊之类的死了。
 <br/>
 <br/>
 反正对于孟宏伟及孟宏伟的家庭来说就是大祸已经降临，因为跟着中共跑必然会大祸降临，这也是孟宏伟跟着中共做恶后的必然下场。
 <br/>
 <br/>
 文武
 <br/>
 2018年10月7日星期日

 <br/>
