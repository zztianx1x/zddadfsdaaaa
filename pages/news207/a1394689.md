### 陈破空：美国副总统演说，不可忽视的另一层含义
------------------------

<div class="wysiwyg">
 10月4日，美国副总统迈克尔．彭斯在哈德逊智库发表长篇演说，阐述美国政府的最新中国政策。演说简要回顾了
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 历史，责备
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 对美国恩将仇报。演说详细列举了中共在贸易、军事、高科技、国际战略、文化等方方面面对美国的渗透、进攻、颠覆和破坏。表达了反击中共、捍卫美国利益和普世价值的决心。
 <br/>
 <br/>
 这篇演说，是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_美中关系.html" target="_blank">
  美中关系
 </a>
 重大转折的官方阐述，标志着，历经46年之后，美中关系根本转向。新冷战宣言，讨共檄文。怎么说都不为过。其重大意义和内涵，相信世界各国都能领略，并从中找到各自的机会与角色定位。
 <br/>
 <br/>
 然而，不可忽视的，还有另一层含义：只要
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_中共.html" target="_blank">
  中共
 </a>
 领导人改过自新，停止倒行逆施，中美关系仍有转圜的可能。这一回，由副总统而不是总统出面宣示美中关系的转折，本身就预留了最后的空间。
 <br/>
 <br/>
 在
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_彭斯演说.html" target="_blank">
  彭斯演说
 </a>
 的结尾部分，有如下段落：
 <br/>
 <br/>
 〝我们应该记住，‘竞争并不总是意味着敌意’，它也不必如此。特朗普总统已经明确表示，我们希望与北京建立建设性关系，让我们的繁荣与安全成长于一道，而不是分离。尽管北京一直在更远地偏离这一愿景，但中国统治者仍然可以改变路线，回归几十年前两国关系开始时的改革开放精神。美国人民不是要得到更多，中国人民也不应该得到更少。〞
 <br/>
 <br/>
 这是对中共领导层改弦易辙的最后呼吁。根据彭斯的演说内容，具体而言，要挽救中美关系，中共当局需要做到如下三件事：其一，遵守入世承诺，接受公平贸易观念，停止损害美国利益，与美国达成平等、公平和互惠的贸易协定。其二，停止压迫中国人民，让中国人民获得更大自由；尤其停止宗教迫害，停止对西藏人、维吾尔人和基督徒的迫害。其三，停止军事挑衅，尤其要停止在南海和台海随时可能挑起战争的冒险举动。
 <br/>
 <br/>
 对中共当局而言，哪些事做得到？哪些事做不到？客观而论，第一件事和第三件事，中共都并不难做到。
 <br/>
 <br/>
 第一件事，让步，与美国达成新的贸易协定，就像其他国家所做的那样，中共也可以做到，从而长远改变美中贸易逆差巨大的格局。事实上，就在彭斯发表演说的头一天，10月3日，中共驻美国大使崔天凯微妙表态：中国愿意让步，准备与美国达成贸易协定。尽管，他假装抱怨说：美国的立场始终在变化，不知道美国的重点到底是什么，不知道美国到底想要什么。
 <br/>
 <br/>
 第三件事，无论是南海争端还是
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_台海危机.html" target="_blank">
  台海危机
 </a>
 ，都是中共挑起，依照〝解铃还须系铃人〞的古老法则，只要中共停止推进其南海扩张、停止武力威胁台湾，南海和台海都会安宁下来。看上去，中共军机绕台、军舰绕台的疯狂举动近期似乎有所收敛。
 <br/>
 <br/>
 对中共而言，最难做到的是第二件事。指望中共让中国人民获得更大自由，那是不切实际的幻想。停止宗教迫害，原本不是那么难，但鉴于中共作为一个巨大的利益集团，其地方领导人与维稳系统的大小官员都从宗教迫害和镇压少数民族的巨大专案中获益（比如分吃、坐享庞大的维稳经费），要他们停下来，阻力重重。
 <br/>
 <br/>
 在彭斯演说的最后一段，明确提到中国国家主席（习近平）。彭斯这样表述：
 <br/>
 <br/>
 〝相信特朗普总统的领导力和远见，以及他与中国国家主席建立起的关系。相信美国人民和中国人民之间的持久友谊。相信上苍能看到未来——在上帝的恩典下，美国和中国将一起到达那样的未来。〞
 <br/>
 <br/>
 这是美国政府给习近平个人发出的最后奉劝，或曰，最后通牒，意思是：球在你脚下，进退由你。既然你手握大权，就请做出明智决定。如果习近平继续执迷不悟、冥顽不化，那么，下一回出面宣示美中关系政策的，极可能，将不再是美国副总统，而是总统本人。
 <br/>
 <br/>
 英国首相温斯顿．邱吉尔于1946年3月5日在美国富尔顿城威斯敏斯特学院发表〝铁幕演说〞，正式拉开美苏冷战或东西方冷战的序幕。如果特朗普总统本人出面，发表类似彭斯的演说，就将正式拉开美中新冷战或东西方新冷战的序幕。
 <br/>
 <br/>
