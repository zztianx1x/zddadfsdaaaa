### 本世纪最有害的十本书
------------------------

<div class="wysiwyg">
 美国 Human Events 杂志，邀请学者评比19到20世纪，世界十本最有害的书，一起来看看得分最高的是...！
 <br/>
 <br/>
 美国 Human Events杂志邀请著名学者组成评委会评比19到20世纪世界十本最有害的书。
 <br/>
 <br/>
 评比揭晓后得分最高的是《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_共产党宣言.html" target="_blank">
  共产党宣言
 </a>
 》、
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_希特勒.html" target="_blank">
  希特勒
 </a>
 的《
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_我的奋斗.html" target="_blank">
  我的奋斗
 </a>
 》为第二害、《毛主席语录》紧挨着为第三害。
 <br/>
 <br/>
 自1848年《共产党宣言》宣布共产主义幽灵出世，已经100多年过去了，〝这个幽灵，在上个世纪的一百年中，像传染病般在全世界泛滥展开，屠杀了数以千万人的性命，剥夺了亿万人的个人财产，甚至他们原本自由的精神和灵魂。〞(《九评》)。
 <br/>
 <br/>
 正是这个邪恶〝宣言〞掀起的共产主义运动给人类带来了极大祸害。
 <br/>
 <br/>
 共产党为何能在人间为祸百年？《九评》一针见血指出共产党是一个反天理灭人性的邪恶生灵，它的使命就是要用暴力颠覆人类社会。但实践已证明，共产主义在全球范围内彻底失败了，连共产党都不相信共产主义理论。
 <br/>
 <br/>
 现今中共内部的贪腐已深入政权骨髓，民众普遍恐慌、不安不满和无奈。
 <br/>
 <br/>
 《资本论》是马克思继《共产党宣言》之后，为无产阶级提供的又一个理论武器，它将资本主义列为人类社会的罪恶阶段。号召全球无产者革命预言资本主义必然灭亡，但讽刺的是历史见证20世纪末东欧和苏联解体。
 <br/>
 <br/>
 《毛主席语录》与《我的奋斗》两害相当。希特勒自传《我的奋斗》直接导致第二次世界大战和对犹太人的屠杀，给世界带来了巨大灾难。毛泽东1966年发表的《毛主席语录》强制中国人人手一册，掀起文化大革命，给中国带来空前的浩劫。
 <br/>
 <br/>
 《九评共产党》，全面揭露中共的黑暗历史和邪恶本质，是认清共产党的必读之书。
 <br/>
 <br/>
 多位专家智库赞誉《九评》给中国人带来精神觉醒。《九评》引发的退党大潮将给中国带来巨变。
 <br/>
 <br/>
 第四名以后的有害之书：
 <br/>
 第四害：金赛性学报告（金赛）
 <br/>
 第五害：民主主义与教育（杜威）
 <br/>
 第六害：资本论（马克思）
 <br/>
 第七害：女性的奥秘（贝蒂．弗里丹）
 <br/>
 第八害：实证主义哲学（孔德）
 <br/>
 第九害：善恶之外（尼采）
 <br/>
 第十害：就业、利息与货币的一般理论（凯恩斯）
