### 崔士方：孟宏伟最后放出的那把〝刀〞
------------------------

<div class="wysiwyg">
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_国际刑警组织.html" target="_blank">
  国际刑警组织
 </a>
 主席、中共公安部副部长
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在北京被纪监委拘押，这个身兼两要职的〝大人物〞在最后时刻发给妻子的那个刀的图片，留给了外界无尽的遐想。
 <br/>
 <br/>
 有分析认为，刀图可能发自他人，因为
 <a href="http://www.ntdtv.com/xtr/gb/articlelistbytag_孟宏伟.html" target="_blank">
  孟宏伟
 </a>
 在纪委人员管控之下，不可能发出信息。
 <br/>
 <br/>
 其实不见得是这样。
 <br/>
 <br/>
 孟宏伟在WhatsApp给妻子发出〝等我电〞的信息是傍晚的6：26分，较大可能是他下飞机前觉察到了异常，但还没太确定；4分钟后的6：30分，孟发现确有个牢笼在等自己的时候，再通话报信已来不及了。
 <br/>
 <br/>
 之所以发刀图，既是因为刀的表情在WhatsApp里只需要两三下手指动作就可调出，也是因为孟可能觉得，实在不能在WhatsApp里给纪委留下内幕文字作为〝罪证〞。
 <br/>
 <br/>
 WhatsApp在中国是被封杀的，除非孟因其特殊身分在翻墙上有豁免，否则很大可能，孟的〝飞刀〞是在下飞机前发出。孟宏伟夫妇用WhatsApp密聊，而非中国最流行的微信联系，这只能说，身为公安部高官的他，对微信的监控本性是鸡吃放光虫——心知肚明。
 <br/>
 <br/>
 孟妻Grace Meng选择公开发声，这是自保的选择。但很多人也注意到了，仅给外界留下背面照的她，背影显得很年轻。而孟宏伟已经快65岁了。
 <br/>
 <br/>
 在中国大陆，女性没有随夫姓的习俗，如果是孟宏伟的结发妻子，这把年纪了很难想像还会取个Grace的外国名字，加上Grace Meng声称要保护〝年幼〞的孩子，以及她用的iPhone手机的鲜红色时尚外套，这种种迹象都表明，她确实比孟宏伟年轻很多。
 <br/>
 <br/>
 这一点，容易让人想起周永康曾在央视任职的年轻二婚妻子贾晓晔。而周永康当年任公安部长时，正是孟宏伟的顶头上司。
 <br/>
 <br/>
 值得玩味的另一个例子，是前政协副主席苏荣的原配、号称〝于姐〞的于丽芳。苏荣的家族式腐败，与这位〝于姐〞从中穿针引线有莫大关系。苏荣的所作所为，于丽芳基本上都是知情人，甚至就是那只白手套，所以当纪委抓住了〝于姐〞，苏荣基本上就完蛋了。
 <br/>
 <br/>
 但是，〝时尚生〞贾晓晔与〝老江湖〞于丽芳却大有不同。尽管周永康被判无期徒刑的同时，贾晓晔也以犯受贿罪、利用影响力受贿罪被判囚9年，但实际上，对周永康参与的各种密谋，与周亲而不密的贾晓晔很可能只是〝局外人〞。所以我们才看到，当局打周老虎寻找突破口时，拳头都落在了周的秘书帮和石油帮身上。
 <br/>
 <br/>
 总而言之，孟宏伟在最后关头虽然得以放出了那把〝刀〞，但刀的余劲只怕有限。年轻的孟妻虽然没有〝于姐〞的油滑和瞻前顾后，敢于走到聚光灯前喊出〝克服恐惧，走向对历史真相、正义和责任的追求〞，但如果她虽是孟宏伟生活上的伴侣，却是孟工作上的〝外人〞，我们就很难指望她可以把这把刀变成力度大得多的长矛，在中共政权上扎出一个大窟窿。
 <br/>
 <br/>
 ──转自《大纪元》 本文只代表作者的观点和陈述。
 <br/>
 <br/>
